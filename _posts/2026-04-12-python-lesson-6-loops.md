# 6강: 파이썬 기초: 반복문 (for, while)

안녕하세요! 👋 여러분, 이번 강의에서는 여러분에게 정말 특별한 능력인 **반복문**을 배워볼 거예요. 이거 없으면 코딩은 완전히 다른 맛이죠. 진짜 신기하죠? 🔥

## 반복문이란?

반복문이라는 것은 말 그대로 반복적으로 코드를 실행하는 것을 의미합니다. 예를 들어, "안녕하세요"라는 메시지를 10번 출력하고 싶다면, 매 번 직접 쓸 수도 있겠지만, 이건 귀찮고 실수할 수 있습니다. 그래서 반복문을 사용하면 아주 쉽게 해결할 수 있어요!

## for 문

`for` 문은 우리가 미리 정해진 횟수만큼 코드를 반복적으로 실행합니다. 예를 들어, 1부터 5까지의 숫자를 출력하는 코드를 보겠습니다.

```python
# 1부터 5까지의 숫자를 출력하는 예제
for i in range(1, 6):
    print(i)
```

코드 한 줄 한 줄 뜯어보면:

- `for i in range(1, 6):` : 이 부분이 반복문의 시작입니다. `range(1, 6)`은 1부터 5까지의 숫자를 생성하는 함수죠. 왜 6까지 안가요? 그건 Python에서 범위를 정할 때 끝 값을 포함하지 않기 때문입니다.
- `print(i)` : 이 부분이 반복적으로 실행되는 코드입니다. 여기서는 `i`라는 변수에 각 숫자가 들어가면서 출력됩니다.

결과는:

```
1
2
3
4
5
```

처럼 나옵니다. 멋지죠? 💪

## while 문

`while` 문은 조건이 참인 동안 계속해서 코드를 반복적으로 실행합니다. 예를 들어, 사용자가 "종료"라고 입력할 때까지 메시지를 출력하는 코드를 보겠습니다.

```python
# 사용자가 "종료"라고 입력할 때까지 메시지를 출력하는 예제
user_input = ""
while user_input != "종료":
    user_input = input("메시지를 입력하세요 (종료하면 프로그램이 종료됩니다): ")
    print(f"당신이 입력한 메시지는: {user_input}")
```

코드 한 줄 한 줄 뜯어보면:

- `while user_input != "종료":` : 이 부분이 반복문의 시작입니다. `user_input`이라는 변수가 "종료"와 같지 않을 동안 계속해서 코드를 실행합니다.
- `user_input = input("메시지를 입력하세요 (종료하면 프로그램이 종료됩니다): ")` : 이 부분에서 사용자로부터 메시지를 입력받습니다.
- `print(f"당신이 입력한 메시지는: {user_input}")` : 이 부분은 반복적으로 실행되는 코드입니다. 여기서는 사용자가 입력한 메시지가 출력됩니다.

## 실무주의보

반복문은 정말 중요한 개념이에요! 꼭 잘 기억해두세요. 예를 들어, 데이터를 처리할 때, 파일의 각 줄을 읽어서 처리할 때, 또는 여러 개의 아이템을 순회하면서 작업을 수행할 때 등등, 반복문은 아주 많이 사용됩니다.

## 초보자 폭풍 질문!

그럼 여러분, `for` 문과 `while` 문의 차이점은 무엇이라고 생각하시나요? 🤔

답변 예시:

- `for` 문은 미리 정해진 횟수만큼 반복합니다. 예를 들어, 리스트의 각 요소를 순회할 때 사용합니다.
- `while` 문은 조건이 참인 동안 계속해서 반복합니다. 예를 들어, 사용자가 특정 값을 입력할 때까지 기다릴 때 사용합니다.

정확한지 모르겠다면 언제든지 저에게 물어봐 주세요! 😊

## 실용적인 활용 예제

반복문을 실제로 어떻게 사용하는지 보여드릴게요. 예를 들어, 1부터 10까지의 숫자 중에서 짝수만 출력하는 코드를 작성해보겠습니다.

```python
# 1부터 10까지의 숫자 중에서 짝수만 출력하는 예제
for i in range(1, 11):
    if i % 2 == 0:
        print(i)
```

코드 한 줄 한 줄 뜯어보면:

- `for i in range(1, 11):` : 이 부분이 반복문의 시작입니다. 1부터 10까지의 숫자를 생성합니다.
- `if i % 2 == 0:` : 이 부분은 조건문입니다. `i`가 짝수인지 확인합니다.
- `print(i)` : 이 부분은 짝수인 경우만 실행되는 코드입니다.

결과는:

```
2
4
6
8
10
```

처럼 나옵니다. 멋지죠? 💪
