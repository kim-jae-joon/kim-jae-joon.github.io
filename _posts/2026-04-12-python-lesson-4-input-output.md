# 파이썬 기초: 입력과 출력

안녕하세요! 👋 파이썬 초보자 여러분! 제 이름은 Qwen, 오늘은 파이썬의 기본을 다루는 4강입니다. 이 강의에서 우리는 **입력(Input)**과 **출력(Output)**에 대해 배우게 될 거예요. 컴퓨터와 대화하면서 정보를 주고 받는 방법을 알아보면, 프로그래밍에 있어서 가장 기초적이고 필수적인 부분 중 하나입니다! 🚀

## 1. 입력(Input)

먼저, **입력**부터 시작해볼까요? **입력**은 우리가 프로그램에게 어떤 데이터를 주고받는다는 개념이에요. 파이썬에서는 `input()`이라는 함수를 사용해서 사용자의 입력을 받을 수 있어요.

### 예제 1: 간단한 입력

```python
# 사용자로부터 이름을 입력받습니다.
name = input("당신의 이름은 무엇인가요? ")
print(f"안녕하세요, {name}님!")
```

**코드 해석:**
- `input()` 함수를 호출하면, 콘솔에 "당신의 이름은 무엇인가요? "라는 메시지가 표시됩니다.
- 사용자가 키보드에서 입력한 값을 `name` 변수에 저장합니다.
- 그 다음으로, `print()` 함수를 사용해서 "안녕하세요, {name}님!"이라는 메시지를 출력합니다. `{name}` 부분은 사용자로부터 받은 이름을 대입하는 것입니다.

💡 **초보자 폭풍 질문!** 왜 `input()` 함수에 괄호 안에 문자열이 들어가야 할까요? 이건 바로 `input()` 함수에게 프롬프트(메시지)를 전달하라는 뜻이에요. 사용자가 입력하기 전에 어떤 내용을 입력해야 하는지를 알려주는 역할이죠.

## 2. 출력(Output)

다음은 **출력**입니다. **출력**은 우리가 프로그램에서 계산한 결과나 메시지를 화면이나 파일 등으로 보여주는 것을 의미합니다. 파이썬에서는 `print()` 함수를 사용해서 쉽게 출력할 수 있어요.

### 예제 2: 간단한 출력

```python
# 숫자 더하기
num1 = int(input("첫 번째 숫자를 입력하세요: "))
num2 = int(input("두 번째 숫자를 입력하세요: "))

# 두 숫자를 더합니다.
result = num1 + num2

# 결과 출력
print(f"{num1}와 {num2}의 합은 {result}입니다.")
```

**코드 해석:**
- `input()` 함수를 사용해서 사용자로부터 숫자를 입력받습니다. 여기서는 `int(input(...))`로 문자열을 정수형으로 변환해야 합니다.
- 두 숫자를 더하고 그 결과를 `result` 변수에 저장합니다.
- 마지막으로, `print()` 함수를 사용해서 계산 결과를 화면에 출력합니다.

🚨 **실무주의보!** 입력받은 데이터의 타입이 중요하다는 것을 잊지 마세요. 예를 들어, 사용자가 숫자를 입력했지만 문자열로 저장되어 있다면, 이후에 계산할 때 오류가 발생할 수 있습니다. 그래서 `int()`나 `float()` 등의 함수를 사용해서 타입을 변환해주는 것이 중요합니다.

## 3. 다양한 출력 포맷

`print()` 함수는 다양한 방법으로 출력 형식을 지정할 수 있어요. 가장 많이 사용하는 방식은 **포맷팅**이에요.

### 예제 3: 다양한 출력 포맷

```python
# 이름과 나이 입력받기
name = input("당신의 이름은 무엇인가요? ")
age = int(input("당신의 나이는 몇 살까요? "))

# 기본 포맷팅 (f-string)
print(f"안녕하세요, {name}님. 당신의 나이는 {age}살입니다.")

# %-포맷팅
print("안녕하세요, %s님. 당신의 나이는 %d살입니다." % (name, age))

# str.format() 메서드
print("안녕하세요, {}님. 당신의 나이는 {}살입니다.".format(name, age))
```

**코드 해석:**
- **f-string**: 가장 간단하고 직관적인 방식으로, 문자열 앞에 `f`를 붙이고 중괄호 `{}` 안에 변수 이름을 넣어주면 됩니다.
- **%-포맷팅**: `%s`는 문자열을, `%d`는 정수를 나타냅니다. 이 방식은 C 언어 스타일로 쓰여서 좀 더 구식적입니다.
- **str.format() 메서드**: `format()` 함수를 사용해서 문자열에 값을 대입하는 방법으로, 중괄호 안에 위치나 키워드를 넣을 수 있습니다.

💡 **초보자 폭풍 질문!** 왜 세 가지 방식이 있을까요? 각각의 장단점이 있으니, 상황에 맞게 사용하면 좋습니다. 예를 들어, f-string은 파이썬 3.6 버전부터 지원되며, 문자열 내에서 변수를 쉽게 넣을 수 있어서 많이 사용합니다.

## 4. 여러 줄 출력하기

때로는 한 줄보다 더 많은 정보를 보여줘야 할 때가 있어요. 그럴 때는 `print()` 함수에 여러 개의 인자를 전달하거나, `\n` 문자를 사용해서 줄을 바꿀 수 있습니다.

### 예제 4: 여러 줄 출력

```python
# 여러 줄 출력하기 (개행문자 사용)
print("안녕하세요!\n파이썬 세계로 오신 것을 환영합니다.\n오늘은 입력과 출력에 대해 배우는 날입니다.")

# 여러 개의 인자를 전달하여 출력하기
print("안녕하세요!", "파이썬 세계로 오신 것을 환영합니다.", "오늘은 입력과 출력에 대해 배우는 날입니다.")
```

**코드 해석:**
- `\n` 문자를 사용해서 줄을 바꿉니다. 이 문자는 "newline"의 약자이며, 새 줄을 시작한다는 뜻입니다.
- `print()` 함수에는 여러 개의 인자를 전달할 수 있습니다. 이렇게 하면 각각의 인자는 공백으로 구분되어 출력됩니다.

🚨 **실무주의보!** 가독성을 위해, 긴 문자열은 여러 줄에 나눠서 작성하는 것이 좋습니다. 이때는 역슬래시 `\`를 사용해서 줄을 계속할 수 있습니다.

## 5. 파일로 출력하기

대화형 콘솔뿐만 아니라, 프로그램의 결과를 파일에 저장하는 것도 중요합니다. 파이썬에서는 `open()` 함수와 함께 `write()` 또는 `writelines()` 메서드를 사용해서 파일에 데이터를 쓸 수 있습니다.

### 예제 5: 파일로 출력하기

```python
# 이름과 나이 입력받기
name = input("당신의 이름은 무엇인가요? ")
age = int(input("당신의 나이는 몇 살까요? "))

# 데이터를 문자열로 변환
output_data = f"이름: {name}, 나이: {age}"

# 파일에 쓰기
with open("output.txt", "w") as file:
    file.write(output_data)

print("데이터가 output.txt 파일에 저장되었습니다.")
```

**코드 해석:**
- `open()` 함수를 사용해서 파일을 열거나 생성합니다. `"w"` 모드는 쓰기 모드입니다.
- `with` 문은 파일 작업이 끝난 후 자동으로 파일을 닫아줍니다.
- `write()` 메서드로 문자열을 파일에 저장합니다.

💡 **초보자 폭풍 질문!** 왜 `with` 문을 사용해야 할까요? 이는 파일 작업 중에 예외가 발생해도 파일이 정상적으로 닫히도록 보장해주는 문법입니다. 또한, 코드를 더 깔끔하게 만들 수 있어요.
