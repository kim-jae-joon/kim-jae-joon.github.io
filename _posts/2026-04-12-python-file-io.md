**9강: 파일 입출력, 코딩을 시작하는 초보자들에게 필수적인 강의!**

안녕하세요, 이번에는 '파일 입출력'이라는 주제로 오늘부터는 초보자들도 따라가기 쉬운 스타일로 가르치겠습니다. 💪

**1. 왜 파일 입출력이 중요한가요?**

파일 입출력은 컴퓨터를 사용하다 보면 반드시 마주치는 주제입니다. 데이터를 저장하거나, 다른 프로그램으로부터 자료를 읽어올 때 사용하는 기술입니다. 예를 들어, 이미지 파일을 열어보면 그 안에 있는 정보가 보이는데요, 그것도 바로 파일 입출력의 기반이 되는 기술입니다.

**2. 파일 입출력의 개념**

파일 입출력은 컴퓨터 시스템에서 데이터를 파일로 저장하고, 다른 프로그램으로부터 파일을 읽어올 수 있도록 하는 기능입니다. '입력'은 파일에 있는 자료를 읽는 것을 말하며, '출력'은 새로운 자료를 파일에 써주는 것입니다.

**3. 파일 입출력의 종류**

파일 입출력에는 '읽기'(Read)와 '쓰기'(Write)가 있습니다. 읽기는 파일의 내용을 프로그램으로 가져오고, 쓰기는 프로그램에서 생성한 데이터를 파일로 내보내는 것입니다.

### 🚨 실무주의보

파일 입출력을 제대로 사용하지 않으면 프로그램이 동작하지 않는 경우가 발생할 수 있으므로 주의하셔야 합니다!

**4. 파이썬에서 파일 입출력하기**

파이썬에서는 `open()` 함수를 사용하여 파일 입출력을 할 수 있습니다.

```python
# 파일 읽기 예제
f = open('example.txt', 'r')
data = f.read()
print(data)
f.close()

# 파일 쓰기 예제
f = open('example.txt', 'w')
f.write('Hello, World!')
f.close()
```

*   `open()` 함수의 첫 번째 인자는 파일 이름이고, 두 번째 인자는 파일을 읽기('r')나 쓰기('w') 모드로 열 수 있습니다. 👉

### 💡 초보자 폭풍 질문!

-   파이썬에서 'example.txt'라는 파일이 없으면 어떻게 될까요? 🤔
    *   FileNotFoundError가 발생합니다! 😅

### 🚨 실무주의보

파일 입출력을 할 때는 항상 예외 처리를 하셔야 합니다. 🔒

```python
try:
    f = open('example.txt', 'r')
except FileNotFoundError:
    print("파일이 존재하지 않습니다.")
```

*   예제 코드에서는 `try-except` 문법으로 파일 입출력을 시도한 후 예외가 발생할 경우 처리 방법을 지정합니다.

### 📝 중요 내용 요약

*   파이썬에서 파일 입출력은 `open()` 함수를 사용하여 가능합니다.
*   읽기('r') 모드는 파일을 열어 데이터를 읽고, 쓰기('w') 모드는 새로운 데이터를 파일에 써줍니다.
