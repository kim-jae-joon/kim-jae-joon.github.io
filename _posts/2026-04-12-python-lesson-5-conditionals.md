# 5강: 파이썬 기초: 조건문 (if, elif, else)

안녕하세요 여러분! 👋 오늘은 파이썬을 배우는 여러분께 "조건문"에 대해 알려드릴게요. 이 조건문 덕분에 우리의 코드가 세상을 좀 더 현명하게 이해하고 대응할 수 있게 됩니다. 🔥

## 1. 조건문이란?

조건문은 우리가 어떤 조건에 따라 다른 동작을 하도록 하는 특별한 구조입니다. 예를 들어, 오늘 날씨에 따라 우산을 가져갈지 말지 결정하는 것처럼요. 🌧️☀️

```python
# 기본적인 if 문
if weather == "비":
    print("우산을 가져가세요!")
```

여기서 `weather`라는 변수가 "비"일 때만 "우산을 가져가세요!"라고 출력합니다. 이는 우리가 조건에 따라 코드를 선택적으로 실행할 수 있게 해줍니다.

## 2. if 문의 구조

```python
if condition:
    # condition이 True일 때 실행되는 코드
```

- `condition`은 참(True)인지 거짓(False)인지 평가됩니다.
- 만약 `condition`이 True라면, 들여쓰기된 블록 안의 코드가 실행됩니다.

## 3. 예제: 날씨에 따라 다른 메시지 출력하기

```python
weather = "맑음"

if weather == "비":
    print("우산을 가져가세요!")
elif weather == "눈":
    print("패딩과 우산을 가져가세요!")
else:
    print("오늘은 맑습니다. 즐기세요!")
```

- `elif`는 "else if"의 약자로, 여러 조건 중 하나를 만족할 때 실행됩니다.
- `else`는 모든 조건이 거짓일 때 실행됩니다.

### 💡 초보자 폭풍 질문!
Q: 왜 들여쓰기가 중요한가요?
A: 파이썬은 들여쓰기를 통해 코드 블록을 정의합니다. 따라서 조건문 내부 코드를 올바르게 인식하려면 반드시 들여쓰기해야 합니다. 예를 들어, `if` 문 다음에 오는 코드가 여러 줄일 경우, 각 줄은 같은 들여쓰기로 시작해야 합니다.

## 4. 실무 활용: 조건문을 사용한 간단한 점수 시스템

```python
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"

print(f"당신의 등급은 {grade}입니다.")
```

- 여러 조건을 체크할 때는 `elif`를 순서대로 사용합니다.
- 마지막으로 모든 조건이 거짓일 때 실행되는 `else`가 있습니다.

### 🚨 실무주의보!
조건문을 작성할 때는 항상 가능한 한 명확하고 간결하게 코드를 작성하는 것이 좋습니다. 여러 개의 `elif`를 사용하면 가독성이 떨어질 수 있으니, 필요 이상으로는 조건을 줄이는 게 좋습니다.

## 5. 논리 연산자와 함께 사용하기

조건문은 논리 연산자(`and`, `or`, `not`)와 함께 사용할 수도 있습니다.

```python
age = 25
has_license = True

if age >= 18 and has_license:
    print("운전이 가능합니다.")
else:
    print("운전을 할 수 없습니다.")
```

- `and`는 두 조건 모두가 참일 때 결과가 참입니다.
- `or`은 두 조건 중 하나라도 참이면 결과가 참입니다.
- `not`은 조건을 반대로 합니다.

### 💡 초보자 폭풍 질문!
Q: 논리 연산자는 어떻게 사용하면 좋을까요?
A: 논리 연산자를 사용할 때는 코드의 가독성을 위해 괄호를 적절히 사용하는 것이 좋습니다. 예를 들어 `(age >= 18) and (has_license)`와 같이 각 조건을 괄호로 감싸면 명확해집니다.

## 6. 삼항 연산자: 짧은 if-else 문

종종 간단한 조건문을 한 줄로 표현하고 싶을 때가 있습니다. 이럴 때 사용하는 것이 "삼항 연산자"입니다.

```python
x = 10
y = 20

max_value = x if x > y else y
print(f"최대값은 {max_value}입니다.")
```

- `x if x > y else y`는 `x`가 `y`보다 크면 `x`, 그렇지 않으면 `y`를 선택합니다.

### 🚨 실무주의보!
삼항 연산자는 코드를 간결하게 만들 수 있지만, 너무 복잡한 조건을 한 줄로 표현하면 가독성이 떨어질 수 있습니다. 따라서 짧고 직관적인 경우에만 사용하는 것이 좋습니다.

## 7. 조건문의 실무적 활용 예제

조건문은 실제로 다양한 상황에서 유용하게 사용됩니다. 예를 들어, 입력값 검증이나 파일 존재 여부 확인 등에 활용할 수 있습니다.

```python
# 입력값 검증
user_input = input("숫자를 입력하세요: ")

if user_input.isdigit():
    number = int(user_input)
    print(f"입력된 숫자는 {number}입니다.")
else:
    print("유효한 숫자가 아닙니다.")

# 파일 존재 여부 확인
import os

file_path = "example.txt"

if os.path.exists(file_path):
    print("파일이 존재합니다.")
else:
    print("파일이 없습니다.")
```

### 💡 초보자 폭풍 질문!
Q: `input()` 함수와 `isdigit()` 메소드는 어떻게 동작하는가요?
A: `input()` 함수는 사용자로부터 입력을 받습니다. 여기서 입력받은 값은 문자열로 저장됩니다. `isdigit()` 메소드는 문자열이 숫자로 구성되어 있는지 확인합니다. 만약 그렇다면 True, 그렇지 않으면 False를 반환합니다.

## 8. 조건문의 주의사항

조건문을 작성할 때 몇 가지 주의사항도 있습니다.

- **조건이 항상 참(True) 또는 거짓(False)**해야 합니다.
- 여러 개의 `elif`가 있다면 첫 번째로 참인 조건만 실행됩니다.
- 들여쓰기는 매우 중요합니다. 파이썬은 들여쓰기를 통해 코드 블록을 구분합니다.

### 🚨 실무주의보!
조건문 내부에서 변수를 수정할 때는 조심해야 합니다. 예를 들어, `if` 문 안에서 변수 값을 변경하면 그 값이 바뀌어서 예기치 않은 결과가 발생할 수 있습니다. 따라서 필요한 경우 복사본을 만들어 사용하는 것이 좋습니다.
