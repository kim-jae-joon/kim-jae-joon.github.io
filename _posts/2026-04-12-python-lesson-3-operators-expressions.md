# 3강: 파이썬 기초: 연산자 및 표현식

안녕하세요! 이번에는 파이썬에서 가장 기본적인 개념 중 하나인 **연산자와 표현식**에 대해 알아보겠습니다. 코드를 작성하는 데 필수적인 도구들이니 꼭 기억해두세요!

## 1. 기본 연산자
우선, 우리가 자주 사용하는 기본적인 연산자를 살펴보겠습니다.

### 1.1 산술 연산자
산술 연산자는 숫자들에 대해 계산을 수행하는 연산자입니다.

| 연산자 | 설명 |
| --- | --- |
| `+` | 덧셈 |
| `-` | 뺄셈 |
| `*` | 곱셈 |
| `/` | 나눗셈 (결과가 항상 실수) |
| `%` | 나머지 계산 (나머지를 반환) |
| `//` | 정수 나눗셈 (소수점 이하를 버리고 정수 부분만 반환) |
| `**` | 거듭 제곱 |

```python
# 예제 코드: 기본 산술 연산자 사용하기
a = 10
b = 3

print(a + b)  # 덧셈: 13
print(a - b)  # 뺄셈: 7
print(a * b)  # 곱셈: 30
print(a / b)  # 나눗셈: 3.333...
print(a % b)  # 나머지 계산: 1
print(a // b) # 정수 나눗셈: 3
print(a ** b) # 거듭 제곱: 1000
```

### 1.2 비교 연산자
비교 연산자는 두 값을 서로 비교할 때 사용합니다.

| 연산자 | 설명 |
| --- | --- |
| `==` | 같음 |
| `!=` | 다름 |
| `>` | 크다 |
| `<` | 작다 |
| `>=` | 크거나 같다 |
| `<=` | 작거나 같다 |

```python
# 예제 코드: 비교 연산자 사용하기
x = 5
y = 3

print(x == y) # False
print(x != y) # True
print(x > y)  # True
print(x < y)  # False
print(x >= y) # True
print(x <= y) # False
```

### 1.3 논리 연산자
논리 연산자는 두 개 이상의 조건을 하나로 결합할 때 사용합니다.

| 연산자 | 설명 |
| --- | --- |
| `and` | 모두 참일 경우에만 True |
| `or` | 하나라도 참이면 True |
| `not` | 반전시킵니다 |

```python
# 예제 코드: 논리 연산자 사용하기
p = True
q = False

print(p and q) # False
print(p or q)  # True
print(not p)   # False
```

## 2. 표현식과 문의 차이점
연산자를 통해 이루어지는 계산을 **표현식**이라고 합니다.

```python
# 예제 코드: 표현식 사용하기
result = 10 + 5 * (3 - 2)
print(result) # 결과: 15
```

반면, 실제로 어떤 동작을 수행하는 것은 **문**입니다. 일반적으로 문은 줄 끝에 세미콜론(`;`)으로 표시하지만 파이썬에서는 생략 가능합니다.

```python
# 예제 코드: 문 사용하기
x = 10
y = x + 5 # 이 줄 자체가 하나의 문입니다.
print(y)   # 이 줄도 하나의 문입니다.
```

## 3. 할당 연산자와 복합 연산자
할당 연산자는 변수에 값을 저장하는 역할을 합니다.

```python
# 예제 코드: 기본 할당 연산자 사용하기
z = 20
print(z) # 결과: 20
```

복합 연산자는 한 줄에서 할당과 연산을 동시에 수행할 수 있게 해줍니다.

| 복합 연산자 | 설명 |
| --- | --- |
| `+=` | 더하기 후 할당 |
| `-=` | 빼기 후 할당 |
| `*=` | 곱하기 후 할당 |
| `/=` | 나누기 후 할당 |
| `%=` | 나머지 계산 후 할당 |
| `//=` | 정수 나눗셈 후 할당 |
| `**=` | 거듭 제곱 후 할당 |

```python
# 예제 코드: 복합 연산자 사용하기
a = 10
a += 5 # a = a + 5와 동일합니다.
print(a) # 결과: 15

b = 20
b *= 3 # b = b * 3과 동일합니다.
print(b) # 결과: 60
```

## 4. 연산자의 우선 순위
연산자가 여러 개 있을 때 어떤 연산을 먼저 수행해야 하는지 규칙이 있습니다.

| 순위 | 연산자 |
| --- | --- |
| 1 | `**` (거듭 제곱) |
| 2 | `*`, `/`, `%`, `//` (부등한 우선순위) |
| 3 | `+`, `-` (부등한 우선순위) |

괄호(`()`)를 사용하여 연산의 순서를 강제할 수 있습니다.

```python
# 예제 코드: 연산자 우선 순위와 괄호 사용하기
result = 10 + 3 * 2 ** 2 - (8 / 4)
print(result) # 결과: 17.0
```
