# 파이썬 기초: 변수와 자료형 💡🔥

안녕하세요! 👋 여러분, 이번 강의에서는 파이썬에서 가장 기본적인 개념 중 하나인 **변수와 자료형**에 대해 깊이 있게 알아보겠습니다. 이 두 가지는 모든 프로그래밍 언어에서 필수적으로 사용하는 요소입니다! 😎

## 변수란 무엇인가요?

변수는 컴퓨터 프로그램에서 데이터를 저장하고 관리하는 역할을 합니다. 간단히 말해, 변수는 "값"의 집합체라고 생각하면 됩니다. 예를 들어, 여러분이 이름이나 나이 같은 정보를 저장하려면 이들 정보를 변수에 담아두고 필요할 때마다 쉽게 접근할 수 있어요.

### 🚨 주의! 💡
변수는 프로그래밍에서 매우 중요한 역할을 합니다. 잘못된 변수 사용은 코드가 제대로 작동하지 않을 수 있습니다! 따라서 변수를 정확하게 선언하고 관리하는 것이 중요합니다.

## 자료형이란 무엇인가요?

자료형은 데이터의 종류를 나타냅니다. 파이썬에서는 다양한 자료형을 제공하고 있는데, 그 중 가장 기본적인 몇 가지를 살펴보겠습니다.

### 1️⃣ 문자열(String)
문자열은 텍스트 데이터를 저장할 때 사용하는 자료형입니다. 큰 따옴표(`"`)나 작은 따옴표(`'`)로 감싸진 텍스트가 바로 문자열입니다.

```python
# 이 코드는 문자열 변수를 선언하고 출력합니다.
name = "홍길동"
print(name)  # 출력: 홍길동

age = '25'
print(age)  # 출력: 25
```

### 💡 초보자 폭풍 질문!
문자열을 작은 따옴표와 큰 따옴표로 감싸도 똑같아요? 왜 그렇게 하는 거죠?
> 사실은, 파이썬에서는 문자열을 작은 따옴표나 큰 따옴표로 감싸는 것은 아무 차이가 없습니다. 하지만 한 문장에 두 가지 따옴표를 함께 사용할 때 편리합니다. 예를 들어, `"He said, 'Hello!'"`와 같이 쓰면 더 명확해집니다.

### 2️⃣ 정수(Integer)
정수는 우리가 일상에서 사용하는 숫자와 비슷한 형태의 데이터입니다. 소수점이 없는 숫자를 의미합니다.

```python
# 이 코드는 정수 변수를 선언하고 출력합니다.
count = 10
print(count)  # 출력: 10

temperature = -5
print(temperature)  # 출력: -5
```

### 💡 초보자 폭풍 질문!
음수도 정수에 포함되나요?
> 맞습니다! 파이썬에서 음수도 정수로 간주합니다. 예를 들어, `-1`, `-100` 등은 모두 정수입니다.

### 3️⃣ 실수(Float)
실수는 소수점이 있는 숫자입니다. 우리가 일상 생활에서 사용하는 부동소수점 숫자와 비슷한 형태의 데이터입니다.

```python
# 이 코드는 실수 변수를 선언하고 출력합니다.
height = 175.5
print(height)  # 출력: 175.5

price = 3.99
print(price)  # 출력: 3.99
```

### 💡 초보자 폭풍 질문!
소수점이 있는 숫자만 실수인가요?
> 맞습니다! 파이썬에서 소수점이 없는 숫자는 정수(integer)로, 소수점이 있는 숫자는 실수(float)로 구분합니다.

## 자료형 변환(Type Conversion)

때로는不同类型的数据를 서로 변환해야 할 필요가 있습니다. 이런 경우, **자료형 변환**을 사용할 수 있습니다.

### 문자열 → 정수 (String to Integer)
문자열이 숫자를 나타내고 있다면, 이를 정수로 변환할 수 있습니다. 예를 들어, `"123"`과 같은 문자열을 `123`이라는 정수로 변경할 수 있습니다.

```python
# 이 코드는 문자열을 정수로 변환합니다.
num_str = "123"
num_int = int(num_str)
print(num_int)  # 출력: 123

print(type(num_int))  # 출력: <class 'int'>
```

### 💡 초보자 폭풍 질문!
문자열이 숫자가 아닌 문자를 포함하고 있을 때 어떻게 변환해야 하나요?
> 예를 들어, `"abc"`와 같은 문자열을 정수로 변환하려고 하면 오류가 발생합니다. 이런 경우에는 `int()` 함수를 사용할 수 없으니 주의하세요.

### 정수 → 실수 (Integer to Float)
정수를 실수로 변환하는 것도 가능합니다. 예를 들어, `10`이라는 정수를 `10.0`이라는 실수로 변경할 수 있습니다.

```python
# 이 코드는 정수를 실수로 변환합니다.
num_int = 10
num_float = float(num_int)
print(num_float)  # 출력: 10.0

print(type(num_float))  # 출력: <class 'float'>
```

### 💡 초보자 폭풍 질문!
정수를 실수로 변환할 때 소수점이 붙는 건 왜 하나요?
> 파이썬에서 정수를 실수로 변환하면 항상 소수점을 포함한 형태로 나타납니다. 이는 데이터의 유연성과 정확성을 유지하기 위함입니다.

## 자료형 확인(Type Checking)

변수의 자료형을 확인하고 싶다면 `type()` 함수를 사용할 수 있습니다. 이 함수는 변수가 어떤 자료형인지를 알려줍니다.

```python
# 이 코드는 변수의 자료형을 확인합니다.
name = "홍길동"
age = 25
height = 175.5

print(type(name))    # 출력: <class 'str'>
print(type(age))     # 출력: <class 'int'>
print(type(height))  # 출력: <class 'float'>
```

### 💡 초보자 폭풍 질문!
`type()` 함수를 사용하면 어떤 자료형이 나올 수 있나요?
> 파이썬에서 기본적으로 제공하는 주요 자료형은 다음과 같습니다:
> - `int`: 정수
> - `float`: 실수
> - `str`: 문자열
> - `list`: 리스트 (다음 강의에서 다룰 예정!)
> - `dict`: 딕셔너리 (다음 강의에서 다룰 예정!)
> - `bool`: 불리언 값 (참/거짓)
> 등등 더 많은 자료형이 있지만, 지금은 이 기본적인 것들을 잘 이해해두세요.

## 실무주의보 🚨
변수와 자료형은 프로그래밍의 기초를 이루는 요소입니다. 변수를 통해 데이터를 관리하고, 자료형을 이용하여 데이터를 적절히 다루면 코드가 더 효율적이고 가독성이 좋아집니다. 따라서 항상 변수를 명확하게 선언하고, 각각의 자료형이 어떤 역할을 하는지 잘 알고 사용하는 것이 중요합니다.

## 정리 🔥
- 변수: 데이터를 저장하고 관리하는 역할
- 자료형: 데이터의 종류를 나타내며, 기본적으로 문자열(str), 정수(int), 실수(float)가 있습니다.
