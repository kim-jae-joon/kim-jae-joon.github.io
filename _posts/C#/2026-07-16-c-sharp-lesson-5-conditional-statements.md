##  5강: 조건문: if, else, switch - 프로그래밍의 '도망길'을 찾아서!

**"프로그래밍은 계산하는 건 아니야, 문제를 해결하는 거지!"** 😎 

저희 코딩 여정이 시작된 이후, 우리는 변수처럼 데이터를 저장하고 연산, 루프와 같은 놀라운 도구들을 사용하며 코드 세계의 기본적인 문법을 마스터했죠. 그런데 프로그래밍은 단순히 "자동 계산기"가 아닌 **문제 해결 전략**을 담고 있는 매력적인 언어! 

💥 이번 강좌에서는 프로그래밍의 가장 근본적인 '도망길'인 **조건문**에 대해 알아볼 거예요. 코드를 실행하는 순서를 바꿀 수 있게 하는, 마법 같은 기술이라고 해도 과언은 아니죠! 🤔

### 🗺️ 조건문: 우리 코드의 지혜로운 판단관

어떤 상황에서만 특정 작업을 하고 싶은 경우가 종종 생기죠. 예를 들어, 게임 캐릭터의 체력이 적으면 공격을 중지해야 하는데, 바로 이때 조건문이 활약할 수 있어요! 🚀 

* **if 문**: " 만약 ~라면" - 가장 기본적인 조건문! 특정 조건이 참이면 코드를 실행하고, 거짓이면 생략하는 기능을 가지고 있습니다.
    ```csharp
    int 체력 = 20;

    if (체력 < 30) // 만약 체력이 30보다 작다면?
    {
        Console.WriteLine("힘들어 보이는구나, 휴식 시간!"); // 이 문장은 출력됩니다!
    }
    ```

* **else 문**: " 그렇지 않으면" - if문과 함께 사용하여 조건이 참이 아니면 실행할 코드를 정의합니다. 

    ```csharp
    int 체력 = 70;

    if (체력 < 30) // 만약 체력이 30보다 작다면?
    {
        Console.WriteLine("힘들어 보이는구나, 휴식 시간!"); 
    }
    else
    {
        Console.WriteLine("아직 충분한 에너지가 있네!"); // 이 문장이 출력됩니다!
    }
    ```

* **else if 문**: "혹시 그렇지 않으면" - 여러 조건을 순차적으로 검사하여, 가장 적합한 코드 블록만 실행하는 기능을 제공합니다. 🤯

    ```csharp
    int 점수 = 85;

    if (점수 >= 90) // 만약 점수가 90 이상이라면?
    {
        Console.WriteLine("우와! 최고의 점수!");  
    }
    else if (점수 >= 80) // 아니면 80 이상이라면?
    {
        Console.WriteLine("잘했어요!");
    }
    else if (점수 >= 70) // 아니면 70 이상이라면?
    {
        Console.WriteLine("참 잘하고 있어요!");
    }
    else // 나머지 경우
    {
        Console.WriteLine("더 노력해봐도 될 것 같아요!");
    }
    ```

### 🤔 **초보자 폭풍 질문!** 🚨


* "if, else if 문이 여러 개 있으면 어떤 순서로 실행될까요?" -> 딱 좋아! 이 부분은 코드의 순서대로 하나씩 확인한다는 점을 명심하세요.  
* "조건식(파라미터)에 들어가야 할 데이터 유형은 무엇일까요?" -> 조건문은 수치 비교, 문자열 연산 등 다양한 형태로 사용됩니다! 숫자, 문자열, 불린 값이 모두 가능해요!


### 🚀 switch 문: 최적의 선택

여러 가지 케이스 중 하나를 선택하는 작업을 빠르고 효율적으로 처리하고 싶을 때 **switch 문**이 활약하죠. 🤔  
* `int`, `string` 타입과 같이 제한적인 값 범위를 가진 변수에 사용됩니다.

    ```csharp
    int 요일 = 3; // 세 번째 요일 (예: 수요일)

    switch (요일)
    {
        case 1: // 월요일
            Console.WriteLine("새로운 일주일의 시작!");
            break;
        case 2: // 화요일
            Console.WriteLine("힘내세요, 화요일은 언제나 어렵죠.");
            break;
        case 3: // 수요일
            Console.WriteLine("이번 주 중간입니다! 잘 지냅시다.");
            break;
        default: // 기타 요일
            Console.WriteLine("즐거운 주말을 맞아보세요!");
    }
```

* 각 케이스 값과 코드 블록은 `case` 키워드로 연결됩니다. 🎯
* `break` 문은 switch문에서 특정 경우 실행 후 나머지 case를 건너뛰도록 지시합니다.
