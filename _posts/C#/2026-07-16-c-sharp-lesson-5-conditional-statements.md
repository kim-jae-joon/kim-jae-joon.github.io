## C# 5강: 조건문 (if, else if, else) - 당신의 코드를 스마트하게 만들다 🚀

안녕하세요! 오늘부터 C# 개발에 도전하는 여러분을 위한 특별한 여정에 함께 갈게요. 🔥 저는 대한민국 최고의 C# 일타 강사이자 15년 차 시니어 개발자, 죠아~ 😎  오랜 경험을 통해 얻은 지식과 유머를 담아 코딩 초보도 천재가 되도록 가르쳐드릴 거예요!

**💡 오늘의 주제: 조건문 (if, else if, else)** - 코드를 생각보다 더 똑똑하게 만들어줄 마법 🚀

조건문은 프로그램이 특정 상황에 따라 다른 실행 과정을 선택하는 강력한 도구예요.  인공지능이 아닌 우리의 머리처럼 "만약 ~라면, 그렇지 않으면" 식으로 판단할 수 있도록 만들어주는 친구죠. 🤩

### 1. 처음부터 알아볼까? 🤔

자칫봐서는 복잡해 보이는 조건문은 사실 매우 직관적이에요!  그러니까, 우리 머리 속의 "만약 ~라면" 논리를 코드로 나타내는 것일 뿐이라는 점을 기억하세요. 💪

예를 들어, 어려운 과제가 있고 해결하려면 두 가지 방법이 있습니다: A) 책을 읽으면 이해하고 B) 친구에게 질문하면 도움을 받을 수 있어요.  어떤 방법을 선택할지는 지금 당신의 상황에 따라 달라지죠? 💡

코드에서도 마찬가지로, 특정 조건이 만족될 때는 A를 실행하고, 만족하지 않으면 B를 실행하는 코드를 만들 수 있어요.  이것이 바로 **조건문**의 역할입니다!

### 2. if 문: 가장 기본적인 선택 🚀

`if` 문은 "만약 ~라면"을 표현하는 가장 기본적인 조건문입니다. 🤔

```csharp
int score = 85;

if (score >= 90) { 
    Console.WriteLine("축하합니다! 장학금 받으셨어요!"); // 만약 점수가 90 이상이면 이 문장을 출력
}
```

* **`int score = 85;`**:  점수를 저장하는 변수 `score`를 선언하고 85로 초기화했습니다. 🎲
* **`if (score >= 90)`**: 점수가 90 이상인지 확인하는 조건입니다. 조건이 참이면 다음 코드 블록을 실행합니다! ✅
* **`Console.WriteLine("축하합니다! 장학금 받으셨어요!");`**:  조건이 참일 경우, "축하합니다! 장학금 받으셨어요!"를 화면에 출력하는 코드입니다. 🎊

💡 초보자 폭풍 질문!
* `score >= 90` 부분은 무엇을 의미할까요? 🤔

### 3. else if 문: 여러 가지 선택지 추가 🎉

`else if` 문은 여러 가지 조건에 대한 대처 방법을 추가하는 데 사용됩니다. 📚

```csharp
int score = 75;

if (score >= 90) {
    Console.WriteLine("축하합니다! 장학금 받으셨어요!"); // 점수가 90 이상이면
} else if (score >= 80) {
    Console.WriteLine("잘했어요! 학생권 신청 가능!"); // 점수가 80 이상이면
} else if (score >= 70) {
    Console.WriteLine("노력하고 있으세요? 계속 그래도 좋아요!"); // 점수가 70 이상이면
} else {  
    Console.WriteLine("좀 더 노력해 보세요!"); // 다른 조건을 만족하지 않으면
}
```

* 위 코드는 여러 가지 점수 기준에 따라 다르게 출력하는 예시입니다. ✨


### 4. else 문: 마지막 선택 지정 💪

`else` 문은 모든 조건이 거짓일 때 실행되는 블록을 정의합니다. 🤔

```csharp
int age = 15;

if (age >= 19) {
    Console.WriteLine("운전면허 시험 준비하세요!");
} else {
    Console.WriteLine("조금 더 기다려야 해요! 😊");
}
```

* `else` 문은 "아니라면"이라는 의미를 담고 있어요. ✨


### 🚨 실무주의보 🚨

실제 프로그램에서는 조건문을 여러 개 겹쳐서 사용하는 경우가 많습니다. 💪 다양한 상황에 맞춰 코드를 제어할 수 있도록 마음껏 활용해 보세요!  ✨

* **조건 논리 연산자**: `&&` (AND), `||` (OR), `!` (NOT)를 사용하여 조건을 더욱 복잡하고 정교하게 만들 수 있습니다. 
* **단계적인 실행**: 코드의 실행 순서를 명확하게 표현하는 것은 프로그램의 논리적 이해도를 높입니다.
