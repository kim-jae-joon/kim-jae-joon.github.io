##  🔥 역할의 시작: 함수, C# 프로그래밍의 신비로운 마법사! 🚀

안녕하세요, 여러분! 😎 저는 대한민국 최고의 C# 일타 강사이자 15년 차 시니어 개발자입니다. 🔥 오늘은 C# 프로그래밍 세계로 새로운 문을 열어줄 특별한 주제, 바로 **'함수'**에 대해 알아보겠습니다!  💡 함수는 마치 프로그래머를 위한 신비로운 마법사, 코드의 핵심 요소라고 할 수 있습니다. 함수만 있다면 복잡한 작업을 작은 조각으로 나누어 효율적으로 처리할 수 있죠. 진짜 신기하죠? 😉

###  ❓ '함수'란 무엇일까요?

함수는 C# 프로그램에서 특정 작업을 수행하는 블록입니다. 🧙‍♂️ 마치 요리 레시피와 같아요! 당신은 재료를 준비하고, 함수(레시피)에 따라 순서대로 조작하면 원하는 음식(프로그램 결과)을 만들어낼 수 있습니다. 예를 들어, "파이 만들기"라는 함수가 있다고 가정해 봅시다. 이 함수는 '강아지'와 '햄버거' 두 가지 재료를 넣고 특정 순서로 조합하여 완성된 파이를 출력하는 마법같은 기능을 합니다. 😎

###  ✏️ 함수 정의: C# 프로그래밍에서 코드 작성의 기본!

함수는 **'정의'** 과 **'호출'** 두 단계로 작동합니다. 🗝️ '정의'란 함수에 이름을 지어주고, 어떤 작업을 수행할지 명확히 하는 것입니다. 마치 레시피 이름을 정하고 재료와 조리법을 적는 것과 같습니다.

```csharp
public int 더하기(int a, int b) 
{
  return a + b; // 두 숫자를 더한 값을 반환합니다.
}
```

이 코드 블록은 **'더하기'**라는 함수를 정의하고 있습니다.

* `public`: 다른 부분에서 사용할 수 있음을 나타냅니다.
* `int`:  함수가 整数(정수)값을 반환한다는 것을 의미합니다.
* `더하기`: 함수의 이름입니다. 짓는 법은 간단하지만, 명확하고 이해하기 쉽도록 하는 것이 중요합니다! 😉
* `(int a, int b)`:  함수가 받는 두 개의 정수 입력값을 나타냅니다. 'a'와 'b'는 변수 이름이며, 함수가 실행될 때 사용됩니다.

```csharp
return a + b; // 두 숫자를 더한 값을 반환합니다.
}
```

* `return`: 함수가 계산된 결과값을 반환하는 키워드입니다.


###  🎬 '함수 호출': 프로그램에서 마법사의 활동! ✨

'정의' 한 함수를 사용하기 위해서는 **'호출'**이 필요합니다. 💻 함수 호출은 마치 레시피를 따라 요리를 하는 것과 같습니다. 함수 이름 뒤에 괄호 안에 입력값을 넣고 실행하면, 함수가 정의한 작업을 수행하고 결과값을 반환합니다.

```csharp
int 결과 = 더하기(5, 3); // 더하기 함수를 호출하여 8을 저장합니다.
Console.WriteLine(결과); // 8을 출력합니다.
```


* `더하기(5, 3)`:  '더하기' 함수를 호출하고, '5'와 '3' 두 정수값을 입력합니다.
* `int 결과 = ...`: 함수가 반환한 값을 '결과'라는 변수에 저장합니다.
