##  4강: 조건문 - 프로그래밍의 '가짜 스피릿' 발전시켜 보자!

**"무슨 일을 할지는 상황에 따라 달라져야지!" 💪** 이렇게 생각하는 당신, 맞죠? 🧐 개발도 마찬가지랍니다. 특정 상황이 발생하면 어떤 코드를 실행해야 하는지 정하기 위해 **조건문**을 사용하는 거예요! 😎

오늘부터 'if', 'else if', 'else' 조건문의 세계로  🚀  뛰어들어, 프로그램 진행을 이루는 스릴있는 게임처럼 재밌게 코딩해보자! 🎉

### 🤔 조건문이란 무엇인가요?

우선 기본 질문부터 생각해봐야겠네요. 🧠 '조건문'은 **특정 조건을 만족하면 특정 코드를 실행하고, 만족하지 않으면 다른 코드를 실행하는 구문**입니다. 마치 친구와 게임을 할 때, "내가 이기면 영화 볼게!" 라고 약속했는지 처럼! 🍿  

만약 우리가 '조건문'을 사용하면 프로그램도 더욱 **똑똑하고 유연하게** 작동할 수 있어요! ✨

### 🚀  if 문 - 상황에 맞춰 행동하는 마법의 주문!

일단 가장 기본적인 조건문인 **'if 문'**부터 익히도록 하겠습니다. 🤔

```csharp
if (조건) {
    // 만약 조건이 참이면 실행할 코드 블록
} 
```

*  `if (조건)` 부분에서, `조건`은 **참(true) 또는 거짓(false)**을 나타내는 값으로 평가됩니다.
* 예를 들어, '변수 X의 값이 10보다 크다'라는 조건을 사용하면:

```csharp
if (X > 10) {
    Console.WriteLine("X는 10보다 크지!"); // 만약 X가 10보다 큰 경우에만 출력
} 
```

* 위 코드를 실행하면, 변수 X의 값이 10보다 크면 "X는 10보다 크지!" 라고 출력합니다.  반대로 작거나 같으면 아무것도 출력하지 않아요! 😎

### 💡 초보자 폭풍 질문!
> 그럼 조건으로 '같은'이나 '다름'처럼 사용할 수 있을까요? 🤔

**답변:** 물론이지요! 🤝  `==`(같다)와 `!=`(다르다) 연산자를 사용하여 같은, 다른 조건을 만들 수 있습니다. 예를 들어, "a와 b가 같으면 출력" 라는 조건은 `if (a == b)` 로 구현할 수 있어요!


### 💡 else if 문 - 여러 상황에 맞춰 동작하는 주문사!

'if' 문 하나만으로는 여러 가지 조건을 처리하기에는 한계가 있습니다. 😕 이때 등장하는 것이 바로 **'else if' 문'**입니다! 💪

```csharp
if (조건1) {
    // 조건1이 참이면 실행할 코드 블록
} else if (조건2) {
    // 조건1이 거짓이고, 조건2가 참이면 실행할 코드 블록
} else if (조건3) {
    // 조건1과 조건2 모두 거짓이고, 조건3이 참이면 실행할 코드 블록
} else {
    // 모든 조건이 거짓일 때 실행할 코드 블록
} 
```

* 위 코드에서는 **다양한 조건을 순차적으로 확인**합니다. 첫 번째 조건이 참이면 그에 해당하는 코드만 실행하고, 만약 첫 번째 조건이 거짓이라면 두 번째 조건을 확인하며 이런 식으로 계속 진행됩니다!

### 🚨 실무주의보
> 'else if' 문들은 **한 줄씩 작성**하지 마세요! 코드를 보기 쉬운 형태로 작성하는 것이 중요합니다. 🤓


### 💪 else 문 - 최후의 수비대, 모든 조건을 거부할 때!

마지막으로 소개할 것은 **'else' 문'**입니다! 👍

```csharp
if (조건1) {
    // 조건1이 참이면 실행할 코드 블록
} else {
    // 모든 조건이 거짓일 때 실행할 코드 블록
} 
```

* 'else' 문은 **모든 이전 조건들이 거짓일 때만 실행**되는 코드 블록입니다. 생각하면서 'if문에서 마지막 주역!' ✨


### 💪  다양한 조건문을 활용한 실제 예시!

예를 들어, 우리가 사용자의 나이에 따라 다른 메시지를 출력하는 프로그램을 만들고 싶다고 하면, 다음과 같이 조건문을 사용할 수 있습니다. 🧑‍💻

```csharp
Console.WriteLine("나이를 입력해주세요:");
int age = int.Parse(Console.ReadLine()); // 사용자의 나이 입력 받기

if (age < 18) {
    Console.WriteLine("아직 미성년자입니다."); // 만약 나이가 18보다 작으면 출력
} else if (age >= 18 && age < 65) {
    Console.WriteLine("성인입니다."); // 만약 나이가 18 이상이고 65 미만이면 출력
} else {
    Console.WriteLine("노년층입니다."); // 만약 나이가 65 이상이면 출력
} 
```
