## 🔥 2강: C# 기초: 코드 작성 및 실행 🚀 - 처음이라도 걱정 마세요!

안녕하세요, 여러분! 😎 저는 대한민국 최고의 C# 일타 강사이자 15년 차 시니어 개발자가 되돌아온  **저**. 오늘부터 함께 C# 언어를 배우게 될 거예요. 으깬 팥처럼 쉬운 설명과 재밌는 비유로, 처음 코드를 접하는 당신도 천재적인 프로그래머가 된다는 것을 보여드릴 테니까 용기를 내세요! 💪

### 🤔 먼저 알아보자: C#이란 무엇일까요?

C# (C sharp)은 마치 레시피처럼, 컴퓨터에게 어떤 일을 해야 하는지 명확하게 알려주는 언어예요. Microsoft에서 만들었고, Windows 프로그램부터 웹 애플리케이션까지 다양한 곳에서 사용되는 유명한 프로그래밍 언어랍니다.

> *C#이라는 이름은 '샵' 기호(#)가 들어있는 것을 의미해요. 마치 주방에 있는 쇠를 이용해서 재료들을 자르듯, 코드를 작성할 때도 이 샵 기호를 사용하여 명령을 정확하게 전달하는 역할을 해요!*

### 💻 코드 작성: C# 개발의 시작!

우리가 C#으로 하는 첫 작업은 바로 '코드'를 쓰는 거예요. 마치 아침 식사 시 "밥, 김치, 계란"이라고 주문하는 것처럼, 우리가 컴퓨터에게 원하는 행동을 명확하게 말해주고 그 명령어가 '코드' 라는 이름으로 표현됩니다.

**💡 초보자 폭풍 질문!**: "그럼 이 코드를 어디에 쓰고 실행해야 할까요?" 🤔  우선, Visual Studio Code 같은 편집기에서 C# 코드를 작성하고, 명령을 넣으면 C# 코딩이 가능합니다.

### ⚙️ 코드 실행: 우리가 만든 코드의 작동!

'코드 실행'은 마치 레시피대로 요리를 하는 것과 같아요.  우리가 준비한 재료(데이터)와 조리법(코드)을 사용하여 최종 결과물(출력)를 얻는 과정이지요.

### 🧩 첫 코드 예제: '안녕하세요!' 출력하기

다음은 C#로 "안녕하세요!"라고 화면에 출력하는 간단한 코드예제입니다. 각 줄의 의미를 하나하나 자세히 알아보겠습니다!

```C#
using System; // 외부 라이브러리 가져오기 - 'System' namespace 사용 가능하게 해주는 부분

public class HelloWorld { // 클래스 선언 - 프로그램의 뼈대
    public static void Main(string[] args) { // 메인 함수 선언 - 프로그램 시작 지점
        Console.WriteLine("안녕하세요!"); // 화면에 출력하는 명령어 실행!
    }
}
```

1. `using System;`:  'System' 라이브러리를 사용할 수 있게 하는 부분입니다. 이 라이브러리에는 컴퓨터에 메시지를 표시하는 기능이 포함되어 있어요. 마치 요리 레시피에서 '칼'이나 '그릇'처럼 필요한 도구를 가져오는 것과 같습니다.

2. `public class HelloWorld`:  프로그램의 뼈대라고 할 수 있는 클래스를 선언합니다. "HelloWorld"라는 이름을 가지고 있으며, 이 안에 우리가 원하는 코드들을 담아서 실행할 수 있습니다. 마치 요리 레시피의 제목처럼 프로그램 전체를 나타내는 역할을 합니다.

3. `public static void Main(string[] args)`:  이 부분은 'Main' 함수라고 하며, 프로그램 실행 시 처음으로 실행되는 코드 블록입니다. C#에서 "프로그램 시작 지점"에 해당하는 곳이죠! 마치 요리 레시피의 주요 순서와 같습니다.

4. `Console.WriteLine("안녕하세요!");`:  화면에 "안녕하세요!"라고 출력하는 명령어입니다.  `Console`은 '콘솔 창'을 나타내고, `WriteLine()` 함수는 해당 콘솔 창에 메시지를 입력하고 화면에 표시합니다. 마치 레시피의 특정 단계에서 재료를 조절하거나 섞는 과정과 같습니다.

### 🚀  결론: C# 세계로 발걸음 내딛기!
