### 💻 변수: 생각을 저장하는 상자📦

변수는 **프로그래머가 데이터를 저장하고 관리하기 위해 사용하는 이름 지정된 공간**입니다. 예를 들어, 우리가 "사과"라는 과일의 개수를 세고 싶다고 가정해봅시다. 이때 변수에 "사과개수" 라는 이름을 붙여서  그 값을 저장하면 좋겠죠? 🍎

```csharp
int 사과개수 = 5; // '사과개수'라는 변수에 숫자 5를 저장!
Console.WriteLine("저희가 가지고 있는 사과 개수는 " + 사과개수 + "개입니다.");
```

이 코드에서 `int 사과개수 = 5;` 부분을 살펴보세요. 여기서  `int` 는 변수의 **자료형**을 의미하는데, 이번에는 정수 (숫자)를 저장할 수 있는 자료형입니다. 그리고 `사과개수` 라는 이름을 붙인 변수에 값 5를 할당했습니다.  

🔥 `Console.WriteLine("저희가 가지고 있는 사과 개수는 " + 사과개수 + "개입니다.");` 부분은 화면에 "저희가 가지고 있는 사과 개수는 5개입니다."라는 메시지를 출력하는 코드입니다! 변수의 값을 활용하여 다양한 정보를 표현할 수 있죠?

**💡 초보자 폭풍 질문!**
>  `int` 말고 다른 자료형들이 있나요?


### 📚 자료형: 데이터의 종류를 정의하는 규칙📑

자료형은 변수에 저장할 수 있는 데이터 유형을 정의하는 것입니다. C#에는 크게 두 가지 주요 자료형이 있습니다!


* **정수형 (Integer):** 숫자만 저장할 수 있는 자료형입니다.
    * `int`:  4바이트를 사용하여 양의/음의 정수를 저장합니다. (예: -10, 0, 500)

* **실수형 (Floating Point Number):** 소수점을 포함한 숫자를 저장할 수 있는 자료형입니다.
    * `float`: 4바이트를 사용하여 부동소수점을 저장합니다. (예: 3.14, -2.5)
    * `double`: 8바이트를 사용하여 더 정확한 부동소수점을 저장합니다. (예: 3.1415926535)

**💡 초보자 폭풍 질문!**
> 자료형이 왜 중요할까요?


### **다양한 자료형들을 활용해 봐요!**

```csharp
int 나이는 20; // 사람의 나이를 저장하는 변수
double 키는 1.75; // 사람의 키를 저장하는 변수
string 이름 = "김민지"; // 사람의 이름을 저장하는 변수
bool 성격 = true; //  "행복한 성격"인 사람을 나타내는 변수 (진/거짓 값)

Console.WriteLine("이름: " + 이름);
Console.WriteLine("나이: " + 나이는);
Console.WriteLine("키: " + 키는 + "m");
Console.WriteLine("성격: " + 성격); 
```


### 🚨 실무주의보!

변수의 이름은 **명확하고 이해하기 쉬운 명칭**을 사용하는 것이 좋습니다! `사과개수` 보다는 `appleCount` 같은 이름이 더 유용합니다. 😉
