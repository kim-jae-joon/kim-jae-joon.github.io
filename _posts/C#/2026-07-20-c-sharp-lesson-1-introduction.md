###  C# 기초 문법: 우리 첫 걸음 👣

지금부터 C#의 기본적인 문장 구조에 대해 알아보겠습니다. C#은 영어처럼 구문을 따르기 때문에 처음 접하는 사람들에게도 다소 친숙하게 느껴질 거예요!


**💡  코드 예제:** 1

```C#
Console.WriteLine("안녕하세요, 세계!"); // 이 문장은 화면에 "안녕하세요, 세계!"를 출력합니다.
```



> **설명**: `Console.WriteLine()` 함수는 화면에 정보를 출력하는 데 사용됩니다.  `"안녕하세요, 세계!"` 부분이 출력될 텍스트라는 의미입니다.

**💡  코드 예제:** 2

```C#
string name = "홍길동"; // 변수 선언 및 초기화
Console.WriteLine("당신의 이름은 {0} 입니다.", name); //  매개변수 대체 출력
```



> **설명**: `string` 이라는 자료형을 사용하여 "name"이라는 변수를 만들고, `"홍길동"`을 할당합니다.

### C# 데이터 타입: 정보를 정리하는 요소

C#에서는 다양한 데이터 타입을 사용하여 정보를 저장합니다. 🗂️ 데이터 타입은 변수가 어떤 종류의 값을 저장할 수 있는지 결정하는 역할을 합니다.  인터페이스, 기본형 자료형 등 다양한 유형들이 존재하며 이 중 가장 중요한 기본 자료형들은 다음과 같습니다:


* **int:** 정수 (예: 10, -5)
* **float:** 부동 소수점 (예: 3.14, -2.718)
* **double:** 더 정확한 부동 소수점 (예: 1.23456789)
* **bool:** 참/거짓 값 (`true` / `false`)
* **string:** 문자열 ("Hello World!")
