##  🔥 1강: C# 기초 - C#이란 무엇인가? 🔥 (진짜 신기하죠?)

안녕하세요! 저는 C#에 빠져들어 5년차 주니어 개발자가 된, 당신의 프로그래밍 여정을 함께 나서는 스승이자 친구입니다. 😎  오늘부터 시작하는 이 강좌가 당신에게 새로운 세상의 문을 열어줄 거예요!

**🤔 뭐야 C#?! 🤔**

일단 말씀드리자면, C#은 마치 요리를 하는데 사용하는 레시피와 비슷한 존재죠. ✨  "요리사가 이 레시피대로 제품을 만들고 사람들이 먹는다!" 그런 식으로 이해하면 돼요. 하지만 우리의 "레시피"는 컴퓨터를 만든 개발자들이 알아서 실행할 수 있도록 작성하는 코드입니다.  

**💡 초보자 폭풍 질문!** 🤔
_코드는 무엇을 하는 게 있나요?_

* **답변:** 코드는 컴퓨터에게 "무슨 일을 해야 하는지" 명확하게 알려주는 지시서와 같습니다. 마치 요리사의 레시피처럼, 컴퓨터도 명령어를 따라 쉬운 작업부터 복잡한 작업까지 수행할 수 있게 된답니다!

**💻 C#이란 무엇일까요?** 🤔
C#은 **Microsoft에서 개발한 객체 지향 프로그래밍 언어입니다.**  그냥 "프로그래밍 언어"라고 생각하면 됩니다. 이 언어는 웹사이트, 게임, 애플리케이션 등 다양한 프로그램을 만드는데 사용됩니다! 🚀

**🤩 C#이 왜 인기 있나요? 🤔**
* **쉬운 문법**:  C#은 배우기 쉽고 이해하기 쉬운 문법을 가지고 있어 초보자들에게 추천되는 언어입니다. (저도 처음 배울 때 그랬죠! 😄)
* **퍼포먼스**: C#은 빠르고 효율적인 코드를 실행할 수 있어 게임, 애플리케이션 등 성능이 중요한 프로그램 개발에 적합합니다. 😎
* **다양한 도구와 라이브러리**: Microsoft는 C# 개발을 위한 다양한 도구와 라이브러리를 제공하며 커뮤니티도 활발하여 문제 해결에도 큰 도움이 됩니다!  🙌

**🚀 C#로 무엇을 만들 수 있나요? 🤔**
* **웹사이트**: 온라인 스토어, 블로그, 포트폴리오 웹사이트 등 다양한 종류의 웹사이트를 만들 수 있습니다. 🌐
* **모바일 애플리케이션**:  iOS, Android 등 모바일 플랫폼에서 사용되는 어플을 개발할 수 있습니다! 📱
* **게임**: C#은 Unity 게임 엔진과 함께 활용되어 다양한 종류의 게임을 만들 수 있는 강력한 도구입니다. 🎮
* **데스크탑 애플리케이션**: Microsoft Office와 같은 데스크탑 애플리케이션도 C#로 개발할 수 있습니다.  💻

**🌟 이제부터 C#를 배우기 시작하겠습니다! 🌟**


##   ##  C# 기본 문법 🧑‍💻

### 변수란? 🤔
변수는 **프로그램에서 데이터를 저장하는 공간**입니다. 마치 상자처럼, 우리가 정보를 담고 관리할 수 있는 곳이죠. 📦

```csharp
int age = 25; // 숫자 데이터를 저장하는 변수 
string name = "홍길동"; // 문자열 데이터를 저장하는 변수
bool isStudent = true; // 참/거짓 값을 저장하는 변수
```

💡 **변수 이름은 알파벳, 숫자, _  (언더바)로 시작하고, 중간에는 알파벳, 숫자, _만 사용할 수 있습니다.**


### 데이터 타입 🗄️

다양한 종류의 데이터를 저장하기 위해 C#는 다양한 **데이터 타입**을 제공합니다.

* `int`: 정수 (예: 10, -5)
* `double`: 실수점 (예: 3.14, -2.718)
* `string`: 문자열 ("Hello World!", "김철수")
* `bool`: 참/거짓 값 (true, false)


### 출력문 (`Console.WriteLine()`) 🎉

출력문은 우리가 **프로그램에서 결과를 화면에 보여주기 위한 기능**입니다.  

```csharp
Console.WriteLine("Hello World!"); // "Hello World!" 문자열을 화면에 출력합니다.
Console.WriteLine(age); // 변수 `age`의 값을 화면에 출력합니다.
```



### ⚠️ **주의 사항!**

* C#에서 코드를 실행하려면,  **C# 컴파일러**를 사용하여 코드를 실행 가능한 프로그램으로 변환해야 합니다.
