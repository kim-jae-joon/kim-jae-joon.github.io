##  C# 기초: 변수와 자료형 (2강) - 코드는 물건과 같아! ✨

안녕하세요, 5년 차 C# 개발자지만 당신들보다 열정적인 저기저희입니다! 🔥 이번 강좌에서는 C#의 정석, **'변수와 자료형'** 에 대해 자세히 알아보고, 이걸 배우면 코드를 쓰는 것도 마치 장난감을 조립하는 것처럼 재미있어질 거예요! 😎 

###  1. 변수: 정보를 담는 상자 생각해봐 🤔

변수라는 것은 프로그램에서 **정보를 저장하고 사용**할 수 있는 공간이라고 생각하면 돼요. 
마치, 실생활에서 당신의 가방이나 지갑처럼 짐을 담고 다니죠? 😄  똑같이 변수는 우리가 코드에 필요한 정보들을 담아 관리하는 역할을 합니다!

예를 들어, "나의 이름은 홍길동" 라고 할 때, '홍길동'이라는 단어를 저장하는 공간이 변수일 거예요. 

**💡 초보자 폭풍 질문!** 🤔
> 변수는 어떤 종류가 있을까? 🧐 답변은 다음 강의에서 알려드릴게요! 😉


###  2. 자료형: 변수 안에 들어갈 정보의 종류 ✨

다음은 변수에 저장할 수 있는 **정보의 종류**를 나타내는 개념인 **자료형**입니다. 
마치, 가방이 있으면 그 가방에 책을 담거나, 사과를 담고 다닐 수 있잖아요?  똑같이 자료형도 변수에 어떤 종류의 정보를 저장할 수 있는지 정해줍니다!

**예시**:

- `int`: 온전히 숫자만 저장하는 공간. 마치 계산기처럼 1, 2, 3,... 이런 깔끔한 숫자들을 담을 수 있어요!
- `string`: 글자열을 저장하는 공간. 예를 들어 "안녕하세요!", "C# 배우겠습니다!" 와 같은 문장들을 담아낼 수 있습니다!

###  3. 변수 선언 & 초기화: 코드에서 '상자' 만들기 🛠️


`int 나이 = 25;`
* `int`:  숫자 저장 공간을 정해줘요. (변수의 자료형)
* `나이`: 변수 이름! (별로 기억하기 쉬운 이름은 좋지 않아요? 😄)
* `=`: 값을 할당하는 연산자라고 합니다. 생각하면 이해가 용이해요! (정보를 상자에 담는다는 의미!)
* `25`: 변수 '나이' 에 담을 값!

**💡 초보자 폭풍 질문!** 🤔
> 코드 예시에서 각 부분은 무슨 역할을 하나요? 🧐  


###  4. 자료형 종류: 다양한 정보 저장하기 📚

C#에는 변수에 저장할 수 있는 다양한 자료형들이 있습니다. 몇 가지 예를 들어보겠습니다!
* `bool`: 참(true) 또는 거짓(false) 값을 저장하는데, 프로그램의 논리적 판단에 사용됩니다. 예시: "나는 오늘 아침에 공부했다." 와 같이 진실/거짓 여부를 나타낼 때 유용해요!
* `double`: 소수점을 가진 숫자를 저장하는 자료형입니다. 실제 세계에서 나타나는 값들을 표현할 때 사용됩니다. 예시: "내 키는 175cm" 와 같이 부분적으로 정확도가 필요한 데이터 처리에 적합합니다!
* `char`: 한 글자를 저장하는 자료형입니다. 문자열을 구성하는 데 사용되며, 코드 안에서 특정 문자를 조작할 때 유용하게 사용됩니다. 예시: "A", "B", "C" 와 같은 알파벳 문자를 저장할 수 있어요!

**🚨 실무주의보:** 자료형을 잘못 선택하면 프로그램이 엉망이 될 수도 있으니 주의해야 해요! 😩


###  5. 변수 사용: 정보를 활용하기 🚀


변수에 정보를 저장한 후에는 이 정보들을 활용하여 다양한 작업을 수행할 수 있습니다. 예를 들어, `나이`라는 변수가 저장된 값을 이용해서 다음과 같은 코드를 작성할 수 있습니다.

```C#
int 나이 = 25;

Console.WriteLine("당신의 나이는 " + 나이 + "입니다.");
// 출력 결과: 당신의 나이는 25입니다. 
```
