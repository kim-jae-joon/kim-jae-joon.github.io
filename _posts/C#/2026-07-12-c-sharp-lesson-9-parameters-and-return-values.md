## 💖 9강: 파라미터와 리턴 값 - C# 프로그래밍의 신비로운 마법! 💖

안녕하세요, 프로그래머 친구들! 👋  지금까지 C#의 기본 문장들을 배우고, 변수를 사용하는 법을 배웠죠? 멋진 시작이었어요! 🎉 지금부터는 더 진한 세계로 들어갈 거예요. 바로 **파라미터와 리턴 값**에 대한 비밀을 밝혀낼 시간입니다!

### ✨ 파라미터: 함수의 마법봉! ✨

상상해 보세요, C# 프로그램은 정말 재능있는 요리사 같습니다. 우리는 레시피를 만들고 요리사에게 주면, 요리사가 재료들을 멋지게 조합해서 완벽한 음식을 만듭니다.  요리사에게 필요한 재료들이 바로 **파라미터**입니다! 😋

파라미터는 함수에 들어갈 때 사용되는 변수들인데요, 함수를 '활성화'시키는 마법봉과 같아요! 🪄 함수가 진짜 활동하기 위해서는 파라미터처럼 특정 정보가 필요한 경우가 많습니다.

**예를 들어,** "거리 계산" 함수를 만들어 보겠습니다. 🤔 이 함수는 두 점 사이의 거리를 계산하는데, 마법봉으로 변수 'x1', 'y1' 그리고 'x2', 'y2'를 받아야 합니다! 

```csharp
public static double 거리계산(double x1, double y1, double x2, double y2)
{
    double result = Math.Sqrt(Math.Pow(x2 - x1, 2) + Math.Pow(y2 - y1, 2)); // 피타고라스 정리 활용!

    return result;
}
```

* **`public static double 거리계산(double x1, double y1, double x2, double y2)`**:  이 부분은 함수의 이름(거리계산), 접근 권한 (public, 모든 프로그램에서 사용 가능), 반환 값 (double, 계산된 거리는 숫자)과 파라미터를 정의합니다.

* **`double result = Math.Sqrt(Math.Pow(x2 - x1, 2) + Math.Pow(y2 - y1, 2));`**:  파라미터로 받은 값들을 이용하여 피타고라스 정리를 활용해 거리를 계산합니다!

* **`return result;`**: 계산된 결과를 다른 부분으로 돌려줍니다.

### ✨ 리턴 값: 함수의 선물! ✨

함수가 파라미터로 받은 정보를 사용하여 작업을 완료하고, 그 결과를 다시 우리에게 보여주는 것을 **리턴 값**이라고 합니다. 마치 요리사가 맛있는 음식을 만들고, 당신께 건네주듯! 🎁

```csharp
double 거리는 = 거리계산(1, 2, 5, 8); // 함수를 실행하고 결과를 변수에 저장
Console.WriteLine("두 점 사이의 거리는: " + 거리는);
```

**💡 초보자 폭풍 질문!**

* 파라미터는 함수에게 무엇을 주입하는 것인가요? 🤔
* 리턴 값은 함수가 우리에게 무엇을 돌려주는 것일까요? 🤔
