##  2강: 자료형 - 프로그래밍의 바탕을 다지자! 🎉

**"프로젝트를 시작하려고 해도 무슨 코드를 쓰면 좋을지 모르겠어요!"**  😭 혹시 이런 고민이 있다면, 당근! 제가 알려드릴게요. 프로그래밍은 마치 건축처럼요!

🧱 벽돌부터 시작하면서 하나씩 쌓아 올라간다는 생각 해 보세요. 그렇다면 프로그래밍에서 가장 기본적인 '벽돌'이 바로 **자료형**입니다.  🔥

### 🎉 💃 자료형의 세계로 떠나볼까요? 🕺 🎊

자료형은 데이터를 저장하고 처리하는 방식을 정의해주는 친구랍니다. 마치 수, 문자, 그림 등 다양한 것을 담을 수 있는 상자라고 생각하면 좋아요!  📦  각 상자가 특정 종류의 물건만 담을 수 있도록 설정되어있죠.

**예를 들어:**

* `숫자 5`는 **정수형**에 들어갑니다. 💪
* `123.45` 와 같은 `소수점 값`은 **실수형**에 들어갑니다.  🧮
* 이름처럼 `텍스트`는 **문자열** 에 들어갑니다. ✍️


#### 🤔 자료형의 종류를 살펴보고, 각각이 어떤 역할을 하는지 알아보자!

###  💯 정수형 - 온전히 수를 담으라! 💯

정수형은 0 또는 양의 정수만 저장하는 자료형입니다. 우리가 일상생활에서 쓰는 숫자들, 그저 간단한 숫자들을 표현할 때 사용됩니다. 마치 계산기를 생각하면 좋겠죠?  🧮

**💡 예시 코드**:
```csharp
int age = 25; // 나이를 나타내기 위한 정수형 변수 선언 및 값 설정
```

**✨ 설명:**

* `int` 키워드는 정수형을 나타냅니다. ✨
* `age` 는 우리가 만든 변수 이름입니다. 변수는 데이터를 저장하는 틀이라고 생각하면 됩니다. 😉
*  `25` 은 정수 값이며, 이를  `age` 변수에 저장합니다.

**🚨 실무 주의보:** 🤔 "정수형은 음수도 표현할 수 있을까요?" 🤔

사실, 음수도 담고 싶다면 `short` 와 같은 다른 자료형들을 활용하면 됩니다.  


### ✨ 실수형 - 소수점까지 정확히! ✨

실수형은 정수와 함께 소수점까지 표현할 수 있는 자료형입니다.  우리가 일상생활에서 사용하는 가격, 길이, 시간 등을 나타낼 때 주로 쓰입니다. 마치 컴퓨터가 "2.5개"처럼 작은 값도 정확하게 담고 계산해줄 수 있게 해주는 친구랍니다!

**💡 예시 코드:**
```csharp
double price = 19.99; // 상품 가격을 나타내기 위한 실수형 변수 선언 및 값 설정
```

**✨ 설명:**
* `double` 키워드는 실수형을 나타냅니다. 💪


###  ✍️ 문자열 - 글자를 담아 이야기를 만들어 보자! ✍️

문자열은 여러 글자가 모여서 문장, 제목 등을 이루는 자료형입니다. 마치 책이나 웹페이지에 쓰이는 글씨와 같습니다. 😄

**💡 예시 코드:**
```csharp
string name = "김철수"; // 이름을 나타내기 위한 문자열 변수 선언 및 값 설정
```

 **✨ 설명:**
* `string` 키워드는 문자열을 나타냅니다. ✨ 


###  🌈 자료형의 세계, 끝이 없다! 🌈

오늘은 정수형, 실수형, 문자열 이 세 가지 자료형에 대해 알아봤죠? 🤔 프로그래밍에는 마치 '우주'처럼 방대한 자료형들이 존재합니다. 🚀 각 자료형마다 특별한 기능을 가지고 있기 때문에, 다양한 자료를 효율적으로 처리하고 코드의 완벽함을 향상시킬 수 있어요!
