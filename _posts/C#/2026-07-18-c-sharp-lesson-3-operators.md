##  🔥 C# 세번째 강의: 연산자 - 산술연산자와 비교연산자!  🔥

안녕하세요 여러분! 👋  5년차 주니어 개발자인 저, **코딩왕**이라고 합니다. 😎 이번 주는 C# 세계에서 가장 신비로운 존재 중 하나인 "**연산자**"에 대해 깊이 있게 알아볼 거예요! 마치 수수께끼 같은 문장들이 코드로 변신하는 경험을 함께 하시고,  초보 개발자도 금방 코딩 실력 UP! 시키는 강의를 준비했어요. 기대 많으세요? 🤩

### 🎯 산술 연산자: 숫자들의 놀라운 활약!

우선 가장 익숙한 **산술 연산자**부터 시작해볼까요? 🤔 이 친구들은 그냥 덧셈, 뺄셈, 곱셈, 나눗셈 등의 기본적인 연산을 하죠. 마치 우리 일상에서 하는 계산처럼!

-  `+`: 더하기 - 두 수를 합쳐주는 거예요. 예를 들어 `5 + 3`은 `8`이 되니까요. 🚀
- `-`: 빼기 - 앞의 수에서 뒤의 수를 뺀 결과를 보여줍니다. 예를 들어 `10 - 4`는 `6`이 된답니다! ➖
-  `*`: 곱하기 - 두 수를 곱해주는 연산자예요. `7 * 2`는 쉽게 생각하면 '7을 두 번 더하는 것'과 같아서 `14`가 나오죠. 🧮
- `/`: 나눗셈 - 첫 번째 숫자를 두 번째 숫자로 나누어 결과 값을 보여줍니다. 예를 들어 `12 / 3`는 `4`입니다.  ➗

**💡 초보자 폭풍 질문!** 🤔 혹시 연산자들은 항상 정수만 계산할까요? 아닌가봐요?! 😉 곧 알아갈 거예요!


### 📊 코드 예제: 산술 연산자의 활약!

```C#
int a = 5; // 변수 'a'에 숫자 5를 저장해주세요.
int b = 3; // 변수 'b'에 숫자 3을 저장해주세요.

int sum = a + b;  // 'sum' 변수는 'a'와 'b'의 합 (8)을 저장합니다.
Console.WriteLine(sum); // 결과값 8을 화면에 출력해줍니다!
```


### 🚀 비교 연산자: "같다"나 "아니야", 그 정도로만 알고 있던 당신에게

이제부터는 숫자가 아닌 **비교**를 하는 연산자들을 알아볼 거예요. 🧐 이 친구들은 두 값을 비교하여 '같은지', '다른지' 등의 조건을 확인하는 데 사용되죠.

- `==`: 같다는 의미 - 두 변수의 값이 동일할 때 참(true)을 반환합니다. 예를 들어 `5 == 5`는 `true`이고 `3 == 7`는 `false` 입니다. 🤔
- `!=`: 다르다는 의미 - 두 변수의 값이 서로 다를 때 참(true)을 반환합니다. 예를 들어 `10 != 5`는 `true`이고 `8 == 8`는 `false`입니다. 🤷‍♀️
- `>`: 큰 거야! - 첫 번째 변수가 두 번째 변수보다 크면 참(true)을 반환합니다. 예를 들어 `7 > 3`는 `true`이고 `2 < 5`는 `true` 입니다.
- `<`: 작은 거야! - 첫 번째 변수가 두 번째 변수보다 작으면 참(true)을 반환합니다. 예를 들어 `3 < 7`는 `true`이고 `9 > 6`는 `true`입니다. 😏
- `>=`: 크거나 같다는 의미 - 첫 번째 변수가 두 번째 변수보다 크거나 같으면 참(true)을 반환합니다. 예를 들어 `5 >= 5`는 `true`이고 `10 >= 8`는 `true` 입니다. 💪
- `<=`: 작거나 같다는 의미 - 첫 번째 변수가 두 번째 변수보다 작거나 같으면 참(true)을 반환합니다. 예를 들어 `2 <= 5`는 `true`이고 `4 <= 4`는 `true`입니다. 👍

**🚨 실무 주의보!** 👀 비교 연산자는 'bool' 자료형의 값을 반환하니까, 이 결과값을 활용해서 프로그램의 흐름을 조절할 수 있습니다.


### 📊 코드 예제: 비교 연산자를 활용해 보세요

```C#
int age = 20;

if (age >= 18) // 만약 나이가 18세 이상이면...
{
    Console.WriteLine("성인입니다!");  // 성인 메시지 출력
}
else
{
    Console.WriteLine("미성년자입니다."); // 미성년자 메시지 출력
}
```
