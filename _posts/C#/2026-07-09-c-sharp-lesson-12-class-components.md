##  ✨12강: 구성 요소 (필드, 프로퍼티, 메서드) - 객체 프로그래밍의 핵심!✨

### 🔑 객체 프로그래밍: 세상을 블록으로 이해하기

> '사람처럼 프로그램을 만들고 싶다!' 🤔 라고 생각했던 개발자들이 오랫동안 고민해왔죠. 그런 안타까움에 마침내 떠오른 아이디어가 바로 **'객체 프로그래밍'**입니다! 🎉  무슨 일이 일어나고 있는지 이해하기 위해 우리는 세상을 주변 환경, 사람들, 사물 등으로 나누어 생각하는 것 같죠? 객체 프로그래밍도 마찬가지인데요. 🧩 프로그램을 구성 요소들을 모아서 만들고, 각 요소들이 서로 작용하며 복잡한 문제를 해결하는 방식이에요! 🤩

###  🧱 필드: 객체의 '내부 저장 공간'

필드는 객체가 가지고 있는 **속성**이나 **데이터**를 나타냅니다. 💻 마치 사람의 이름, 나이, 성격처럼 프로그램에서도 정보를 저장하는 곳을 만들어야죠! 😉  예를 들어, '강아지 객체'를 만들면 필드로는 '이름', '품종', '나이' 등이 있겠죠? 🐶

```csharp
public class Dog
{
    // 강아지의 이름을 저장하는 필드입니다.
    public string Name;  
}
```


* `public class Dog`: 강아지를 나타내는 객체를 만들기 위한 틀! 🔨 
* `public string Name;`: 강아지가 가지고 있는 이름 정보를 저장할 공간을 선언합니다. 이때 `string`은 문자열을 나타내는 자료형이라는 뜻이에요!

💡 초보자 폭풍 질문!: 객체 프로그래밍에서 필드와 변수의 차이점이 무엇일까요? 🤔


###  🪄 프로퍼티: '외부로 접근하기 위한 창'

프로퍼티는 필드를 통해 저장된 데이터를 **조금 더 안전하고 조정 가능하게** 제어하는 메커니즘입니다. 생각보다 단순한 변수인 프로퍼티가, 마치 집의 엄마처럼 데이터를 보호해주고 필요하면 나눠주는 역할을 하지요!🦸‍♀️

```csharp
public class Dog
{
    private string _name; // 필드명은 '_'로 시작하여 접근 제한!🤫

    public string Name // 마치 현관문처럼, 외부에서 이름을 얻을 수 있게 해줍니다.
    {
        get { return _name; }  // 이름을 읽어보려고 할 때, '_name'를 반환합니다.
        set { _name = value; } // 새로운 이름을 설정할 때, '_name'에 저장합니다.
    } 

}
```


* `private string _name;`: 강아지의 이름을 내부적으로 저장하는 필드로 접근 제한! 🔐
*  `public string Name { ... }`: 'Name'이라는 문자열 변수를 통해 강아지 이름을 읽고 설정할 수 있게 합니다. 마치 현관문처럼, "안녕하세요"라고 말하겠다고 할 때 이름을 확인하는 것과 같습니다!

🚨 실무 주의보!: 프로퍼티는 필드 값에 대한 **조건 처리** 또는 **데이터 검증** 등 추가적인 로직을 구현할 수 있어요. 데이터를 더 안전하고 효율적으로 관리하는 데 유용합니다!


### 💪 메서드: 객체의 '행동 패턴'

메서드는 객체가 할 수 있는 **기능이나 동작**을 정의합니다. 마치 강아지가 짖거나, 배고프면 음식을 먹는 행동처럼, 코드에서도 특정 작업을 수행하는 기능들을 모듈로 만들어 사용할 수 있죠! 🦴

```csharp
public class Dog
{
    // ... 필드와 프로퍼티 정의 ...

    // 강아지가 짖는 동작을 나타내는 메서드입니다.
    public void Bark() 
    {
        Console.WriteLine("멍!"); // "멍!" 출력!🐶
    }

    // 강아지를 산책시키는 동작을 나타내는 메서드입니다.
    public void Walk()
    {
        Console.WriteLine("강아지가 산책을 좋아해요!");
    }

}
```



* `public void Bark()`: 'Bark' 라고 이름이 붙은 메서드로, 강아지가 "멍!" 이라고 소리를 내는 동작을 수행합니다. 'void'는 메서드가 값을 반환하지 않는 것을 나타냅니다.

💡 초보자 폭풍 질문!: 메서드의 역할이 무엇이고, 코드를 더 가독성 있게 만드는데 어떤 도움이 될까요? 🤔
