### ✨  ASP.NET Core란 무엇일까요?

> ASP.NET Core는 Microsoft에서 제공하는 **개방형 웹 애플리케이션 프레임워크**입니다! 
마치 우리가 요리를 할 때 레시피처럼, 웹 애플리케이션을 개발할 때 필요한 모든 것을 알려주고 도와주는 친구라고 생각하면 돼요.

### 🌈  ASP.NET Core의 장점은 무엇일까요?

* **빠름과 효율성:** 🚀 경량화된 아키텍처로 빠르고 효율적인 웹 애플리케이션을 구축할 수 있어요!
* **개방성**: 😈 유연하고 확장 가능한 프레임워크로 다양한 기술과 도구와의 연동이 용이합니다.  자유롭게 선택할 수 있다는 건 환상적이죠? 🤩
* **다양한 플랫폼 지원**: Windows, Linux, macOS 등 여러 플랫폼에서 작동하여 개발 편의성을 높여줍니다! 💻

### 🚀  ASP.NET Core로 웹 애플리케이션 구축하기: 실제 예시!

지금부터 ASP.NET Core를 사용해서 간단한 "Hello World" 웹 애플리케이션을 만들어보겠습니다!

**1. 개발 환경 설정:**
   * Visual Studio 2022 또는 Rider IDE를 설치합니다. 🚀 💻  Visual Studio는 저에게 있어서 최고의 선택이지만, 편안하게 사용할 수 있는 IDE를 선택하시면 좋겠어요. 😉

**2. 새로운 프로젝트 생성:**
   * Visual Studio에서 "프로젝트" -> "새 프로젝트"를 선택합니다.  👉
   * "ASP.NET Core Web Application (.NET 6)"을 선택하고 "설명 추가" 버튼을 클릭하여 "Hello World"라는 이름으로 프로젝트를 생성합니다. 👍

**3. 코드 수정:**
   * `Pages/Index.cshtml` 파일을 열고 다음과 같은 코드로 변경합니다:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8" />
    <title>Hello World!</title>
</head>
<body>
    <h1>안녕하세요! 이제 ASP.NET Core로 웹 애플리케이션을 구축하는 것을 배웁니다!</h1> </body>
</html>

```
 * `Pages/Index.cshtml` 파일은 웹 페이지의 콘텐츠를 담당하는 파일입니다. 🎨

**4. 프로젝트 실행:**
   * F5 키를 누르거나 "프로젝트" -> "실행"을 선택하여 프로젝트를 실행합니다.  👉 🚀

**5. 결과 확인:**
   * 브라우저에서 http://localhost:7000/ 주소로 접속하면 "안녕하세요! 이제 ASP.NET Core로 웹 애플리케이션을 구축하는 것을 배웁니다!" 라는 문구가 표시됩니다. 🎉

### 💡  개념 정리 시간!


* **Razor**: C# 코드와 HTML을 함께 사용할 수 있게 해주는 기술입니다. 😎
* **Views**: 웹 페이지의 콘텐츠를 담당하는 파일들이 모여 있는 폴더입니다.
* **Controllers**: 요청을 처리하고 데이터를 전달하는 역할을 합니다.
