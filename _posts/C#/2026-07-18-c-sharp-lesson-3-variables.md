##  C# 기초: 변수와 자료형 - 코드의 건축 블록을 알아보자!

안녕하세요! 저는 5년 차 C# 개발자 '코딩챔피언'입니다. (😎 참, 나만 부르지 마세요!) 오늘부터 함께 C#를 배우기 시작할 거예요. 이 강좌가 당신의 코딩 여정의 첫걸음이 되길 바라며, 흔들리지 않고 멋진 코드 세계로 들어갈 수 있도록 최선을 다하겠습니다!

**어떻게 시작하는 걸까요? 🥁 드럼 소리가 들리네요! 바로 변수와 자료형부터 시작하죠! 🎉 **

### 변수: 데이터를 담는 상자

변수라고 하면 어떠신가요? 마치 코드 속에 책상이나 서랍처럼 데이터를 담을 수 있는 '상자' 같은 존재예요. 📝  하지만, 이 상자는 형태나 용량이 다양해요! 🤯  일부 데이터들은 작은 상자에서도 충분히 들어가고, 어떤 데이터는 더 큰 상자를 요구할 수 있어요.

**💡 초보자 폭풍 질문!** 🤔
- 변수 이름을 정하는 데 유의해야 할 점이 있나요?
    - 물론 있습니다! 변수 이름은 명확하고 이해하기 쉬워야 합니다. 'x', 'y'처럼 무슨 의미인지 알 수 없는 이름보다는 '계산된합계', '사용자이름' 처럼 구체적인 의미를 담아주는 것이 좋습니다. 😊

**변수 선언**: 변수라는 상자를 만들 때, 먼저 자료형을 지정해야 합니다! 자료형은 데이터의 종류와 용량을 정해주죠. 예를 들어, '숫자'를 저장할 상자라면 `int` 자료형을 사용하고, '텍스트'를 저장할 상자라면 `string` 자료형을 사용합니다.

```csharp
// 숫자 데이터를 저장하는 변수 선언 (int 자료형)
int 나이;

// 텍스트 데이터를 저장하는 변수 선언 (string 자료형)
string 이름;
```

**변수 할당**: 변수에 값을 담는 것을 '할당'이라고 합니다.  "=", 이 연산자가 사용됩니다. 예시 코드에서 보세요!

```csharp
// 나이 변수에 25를 할당
나이 = 25;

// 이름 변수에 "강태훈"을 할당
이름 = "강태훈";
```

### 자료형: 변수의 종류

C#은 다양한 자료형을 제공하며, 각 자료형은 데이터의 종류와 용량을 결정합니다. 

**1. 정수 자료형 (int, long):** 숫자를 저장하는데 사용됩니다.

- `int`: 4바이트 크기로 보통 실제 사람들의 나이나 점수 등 간단한 숫자를 표현할 때 사용합니다.
- `long`: 8바이트 크기로 매우 큰 수를 저장해야 하는 경우에 사용합니다 (예: 거대 기업의 매출액).

**2. 실수 자료형 (float, double):** 소수점을 포함하는 숫자를 저장하는데 사용됩니다.
- `float`: 4바이트 크기로 일반적인 실수 값을 표현할 때 사용합니다.
- `double`: 8바이트 크기로 더 정밀한 실수 값을 표현할 때 사용합니다 (예: 과학적 계산).

**3. 문자열 자료형 (string):** 글자를 저장하는데 사용됩니다. 예를 들어, 사람의 이름이나 제목 등을 저장할 때 사용합니다.

```csharp
// 숫자 데이터
int 나이 = 25;
long 거리 = 1234567890;

// 실수 데이터
float 높이 = 1.75f;
double 가격 = 12345.6789;

// 문자열 데이터
string 이름 = "강태훈";
```

**💡 초보자 폭풍 질문!** 🤔

- 자료형은 어떻게 선택해야 할까요?

    - 자료형을 선택할 때는 저장될 데이터의 종류와 크기를 고려해야 합니다. 예를 들어, 나이처럼 작은 정수 값을 저장한다면 `int` 자료형이 적절하며, 대서양과 같은 큰 수를 저장하려면 `long` 자료형이 필요합니다!

### 변수 활용: 코드를 짜는 실력 향상

변수와 자료형은 C# 코드의 기본 구성 요소입니다. 이것들을 이해하면 코드를 더 명확하고 효율적으로 작성할 수 있습니다. 💪

### 기본형 자료 변환:

C#에서는 다양한 자료형 간의 변환이 가능합니다. 예를 들어, `int` 자료형의 값을 `string` 자료형으로 변환하거나, 반대로 `string` 자료형의 값을 `int` 자료형으로 변환할 수 있습니다. 이러한 변환은 데이터의 형태를 변경하는 데 유용하게 사용됩니다.

```csharp
// int를 string으로 변환하기 
int 점수 = 85;
string 문자열점수 = 점수.ToString(); // '85' 형식으로 변환
Console.WriteLine(문자열점수);  // 출력: 85


// string을 int로 변환하기 (주의! 잘못된 값은 오류 발생)
string 입력값 = "123";
int 변환된값 = Convert.ToInt32(입력값); // int 자료형으로 변환

```



**💡 초보자 폭풍 질문!** 🤔

- 기본형 자료 변환이 어떤 상황에서 유용할까요? 

    - 데이터 표현 형식을 변경해야 할 때, 또는 다른 함수나 라이브러리를 사용하기 전에 필요한 형태로 변환해야 할 때 유용합니다. 예를 들어,  '숫자'를 '문자열' 형태로 바꾸어 파일 이름이나 출력 결과에 활용할 수 있습니다.
