###  알고리즘이란 무엇일까요? 🤔

인공지능, 자율주행 자동차, 데이터 분석 등 현대 사회는 알고리즘 없이는 살아갈 수 없습니다. 🤯 하지만 그저 '알고리즘' 이라는 단어만 들으면 어떤 것인지 모르겠죠?  진짜 신기하죠?! 😳

**알고리즘은 문제 해결을 위한 단계별 지침서입니다.** 요리 레시피처럼 문제의 입력을 받아 특정한 과정을 거쳐 원하는 출력 결과를 얻는 방법이라고 생각해 보세요. 🤔  예를 들어, 햄버거 만들기 알고리즘은 다음과 같겠죠:

1. 빵 두 조각을 준비한다.
2. 패티를 구워서 익힌다.
3. 상추, 토마토 등 재료들을 올린다.
4. 두 빵 사이에 패티와 재료들을 끼운다.


### C#로 알고리즘 구현하기 💪

자, 그럼 이제 C#을 이용하여 실제 알고리즘을 구현해 보도록 하죠! 가장 기본적인 알고리즘 중 하나인 **"합계를 구하는 알고리즘"**을 예시로 들어볼까요? 🧮

```csharp
int sum = 0; // 초기값 설정: 합계는 처음부터 0으로 시작합니다.
for (int i = 1; i <= 10; i++) // 1부터 10까지 반복문을 돌립니다.
{
  sum += i; // 각 번호를 sum에 더해줍니다.
}
Console.WriteLine("1부터 10까지의 합계: " + sum); // 결과값 출력! 
```

이 코드를 살펴보면 어떤 과정들이 이루어지는지 알 수 있습니다. 👍

-  **`int sum = 0;`**: 변수 `sum`를 선언하고, 처음에는 0으로 초기화합니다. 이 변수가 합계를 저장하는 역할을 할 거예요.
- **`for (int i = 1; i <= 10; i++)`**:  숫자 `i`를 사용하여 1부터 10까지 반복문이 돌아갑니다. 🤔 이제 알고리즘의 주요 부분인 작업 단계가 시작됩니다!
- **`sum += i;`**: 각 반복에서는 현재 값인 `i`를 `sum`에 더합니다. 이 과정은 순서대로 숫자들을 더해나가는 합계 구하기 알고리즘의 정의를 보여줍니다.

- **`Console.WriteLine("1부터 10까지의 합계: " + sum);`**: 마지막으로, `sum`에 저장된 최종 합계 결과값을 화면에 출력합니다. 🚀🚀🚀
