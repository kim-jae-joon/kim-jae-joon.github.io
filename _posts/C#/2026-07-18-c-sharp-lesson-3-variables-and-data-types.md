##  🔥 C# 배우기, 시작은 변수부터! 🔥: int, double, string 

안녕하세요! 여러분의 C# 여정을 함께 걸어갈 최고의 C# 강사, yours truly가 자리 잡았습니다 😎! 오늘은 C# 프로그래밍의 기초를 다지는 **변수와 자료형**에 대해 알아볼 거예요. 바로 시작할까요? 🚀

### 변수란 무엇일까요? 🤔  

변수는 생각만큼 여러가지 정보를 담을 수 있는 상자라고 생각해보세요! 📦 이 상자에는 숫자, 문자열, 진실 여부 등 다양한 종류의 값을 저장할 수 있습니다. 마치 게임에서 캐릭터의 체력이나 경험치처럼, 프로그램에서는 변수를 통해 정보를 관리하고 계산하는 작업을 합니다.

예를 들어, "나는 25살이다" 라는 문장을 C#로 나타내면 다음과 같습니다.

```csharp
int 나이 = 25;
```

여기서 `나이`라는 변수에 '25'라는 숫자 값을 저장했습니다.  변수 이름은 의미가 잘 드러나도록 "나이" 처럼 명확하게 정해 주는 것이 좋습니다! 👍


### 자료형: 정보의 종류를 나타내는 태그 ✨

마치 우리 주소록처럼, 변수에 어떤 종류의 정보를 저장할 수 있는지 알려주는  '자료형'이 존재합니다. C#에서는 'int', 'double', 'string' 등 다양한 자료형을 사용하여 정보를 정확하게 나타낼 수 있습니다.

**💡 초보자 폭풍 질문!** 
어떤 종류의 변수가 있을까요? 🤔  예시 들어볼까요?


#### 1. int:  전쟁터에서 불어오는 강렬한 바람처럼, 정수를 담아내리겠습니다! 💪

`int` 자료형은 **정수(integer)**를 나타냅니다. 즉, 소수점 없는 숫자를 저장하는 변수입니다. 예를 들어, '10', '-5', '0' 와 같은 값을 저장할 수 있습니다.


```csharp
int 학생수 = 30; // 학생들의 힘! 💥
int 연도 = 2023; // 현실의 추월적인 통찰력! 🚀
```

*  `학생수`라는 변수에 '30'이라는 정수 값을 저장. (예: 1년 기준 학급 수)
*  `연도` 라는 변수에 현재 연도인 '2023'를 저장.

#### 2. double: 소수점까지 자유로운 해적처럼, 부동소수점을 담아내리겠습니다! 🌊

 `double` 자료형은 **부동소수점(floating-point)**을 나타냅니다.  즉, 소수점이 포함된 숫자를 저장하는 변수입니다. 예를 들어, '3.14', '-2.718', '0.0' 와 같은 값을 저장할 수 있습니다.

```csharp
double 키 = 1.75; // 자유로운 기회! 🎉
double 가격 = 99.99; // 섬세한 계산! 😎
```

*  `키`라는 변수에 '1.75'라는 부동소수점 값을 저장. (예: 인간의 키)
*  `가격` 라는 변수에 상품 가격인 '99.99'를 저장.


#### 3. string: 이야기꾼처럼, 문자열을 담아내리겠습니다! 💬

 `string` 자료형은 **문자열(text)**을 나타냅니다.  즉, 글자로 이루어진 값을 저장하는 변수입니다. 예를 들어, 'Hello World!', 'Python', 'C# is Awesome!' 와 같은 값을 저장할 수 있습니다.


```csharp
string 이름 = "김철수"; // 숨겨진 이야기! 👀
string 문구 = "오늘은 좋은 날입니다."; //  일상의 기쁨! 🥰
```

*  `이름`라는 변수에 '김철수'라는 문자열을 저장. (예: 사람의 이름)
* `문구` 라는 변수에 '오늘은 좋은 날입니다.'라는 문자열을 저장. 



###  🚨 실무주의보! : 자료형 선택이 중요합니다. 👀

잘못된 자료형을 사용하면 프로그램이 이상하게 작동하거나 오류가 발생할 수 있습니다. 예를 들어, '10'라는 값을 `string`으로 저장하려고 하면 프로그램에 문제가 생길 수 있다는 것을 기억하세요!  🤔
