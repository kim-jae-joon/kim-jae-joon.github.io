### 🚀 for 문: 정해진 횟수를 위한 명령전사

'for' 문은 가장 **기본적이고 흔히 사용되는 반복문**이에요. 이름에서 알 수 있듯이, 특정 횟수만큼 반복되도록 설정하는 데 최고죠! ✨

예를 들어서 "나는 1부터 5까지 좋아한다!" 라고 표현하고 싶다면 어떻게 코드를 작성할까요? 🧐

```C#
for (int i = 1; i <= 5; i++) // 'i'는 시작값이 1이고, 5보다 작거나 같을 때까지 반복하게 된다.
{
    Console.WriteLine("나는 " + i + " 좋아한다!"); // 'i' 값이 변할때마다 출력문 실행된다.
}
```

**코드 해설:**

- `for (int i = 1; i <= 5; i++)`: 이 부분은 반복문의 시작 부분입니다.
    - `int i = 1;` : 변수 'i'를 정의하고 초기값을 1로 설정합니다. 이는 반복문이 처음부터 어디서 시작할지 나타내죠!
    - `i <= 5;`:  'i' 가 5보다 작거나 같으면 반복문이 계속 실행됩니다. 우리가 원하는 만큼 몇 번 반복할지를 정의하는 부분이에요.
    - `i++`: 'i' 값을 1씩 증가시키는 작업입니다. 매번 반복이 끝난 후 'i' 값이 바뀌어 다음 실행에서 사용됩니다. 마치 계산기 버튼을 누르면서 카운터를 올리는 것과 같아요!

- `Console.WriteLine("나는 " + i + " 좋아한다!");`: 이 부분은 반복문 안에 있는 코드입니다. 'i' 값이 변하는 대로 몇 번씩 출력될지 결정합니다.


### ✨ while 문: 조건을 만족할 때까지 지속적인 열정

`while` 문은 특정 조건이 **만족될 때까지 반복**되도록 하는 문입니다. 마치 "수박 먹고 싶으면 계속 더 먹는다!" 라는 욕심이 생기면서 수박을 먹는 상황과 같죠! 🤣


```C#
int count = 0;

while (count < 5) // 'count' 값이 5보다 작을 때까지 반복
{
    Console.WriteLine("수박 먹고 있어요!");
    count++; // 'count' 값을 1씩 증가시키기
}
```

**코드 해설:**

- `int count = 0;`: 변수 'count'를 정의하고 초기값을 0으로 설정합니다.
- `while (count < 5)`: 'count' 값이 5보다 작으면 반복문 코드가 실행됩니다. 조건에 따라 반복되는 행동의 기준이죠!

### 🤔 do-while 문: 한번은 반드시 실행해야 하는 마지막 선택

`do-while` 문은 **먼저 반복문 안의 코드를 한 번 실행하고**, 그 후 조건을 만족하면 다시 반복하는 방식입니다. 🤯 처음부터 "내가 할 수 있어!" 라고 자신감이 가득해서, 그 다음에 확인해보는 상황과 같죠! 💪

```C#
int number = 1;

do // 먼저 코드를 실행하고 조건을 확인
{
    Console.WriteLine("안녕하세요! 현재 숫자는 " + number + " 입니다.");
    number++; // 다음 반복에 사용할 숫자를 증가시킵니다.
} while (number < 5); // 'number' 값이 5보다 작거나 같을 때까지 반복됩니다.

```

**코드 해설:**

- `int number = 1;`: 변수 'number'를 정의하고 초기값을 1로 설정합니다.
- `do { ... } while (number < 5);`:  반복문 구조입니다.

### 🎉  실무 적용: 더욱 풍부한 코드 시대를 열어봐!

반복문을 사용하면 코드를 간결하게 작성하고, 복잡한 작업을 효율적으로 처리할 수 있어요. 예를 들어, 여러 숫자들을 더하는 프로그램을 생각해보세요. 단순히 하나씩 더하기만 해도 되겠지만, 반복문을 사용하면 훨씬 코드가 간결하고 보기 좋습니다!

```C#
int sum = 0;

for (int i = 1; i <= 10; i++) // 1부터 10까지의 숫자를 더합니다.
{
    sum += i; // 'sum' 변수에 현재 값을 더하는 작업을 수행합니다.
}

Console.WriteLine("1부터 10까지 합: " + sum); // 결과 출력합니다.
```
