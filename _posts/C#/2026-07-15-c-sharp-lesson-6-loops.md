## 🔥  C# 6강: 반복문 - for, while, do-while (초보자도 통하는 완벽 가이드!) 🚀

**안녕하세요!** C# 탐험대 여러분 👋. 저는 대한민국 최고의 C# 일타 강사이자 15년 차 시니어 개발자 김덕수입니다 😎!  오늘은 우리 코드 작가들이 가장 자주 활용하는 마법같은 도구, **반복문**에 대해 알아볼 거예요!

**🤔 "반복문"이란 무엇일까요?**

상상해보세요. 저는 매일 30번 버스를 타고 집으로 돌아오는데, 똑같은 과정을 반복하는 것과 같죠? 🤔 반복문도 마찬가지로, 특정 코드 블록이 **여러 번 실행되는** 작전들을 담당합니다.  저희 C#에는 세 가지 주요 반복문 챔피언들이 있습니다!

* **for 루프**: 딱 정해진 횟수만큼 반복하는 '숫자 전문가'!
* **while 루프**: 특정 조건이 만족할 때까지 반복하는 '조건 파악의 달인'!
* **do-while 루프**: 최소 한 번은 실행되고, 조건 확인 후 다음 실행 여부를 결정하는 '끈기 있는 친구'!

### 🚀 for 루프: 깔끔하고 정확한 반복 전문가!

**💡 초보자 폭풍 질문!** "왜 for 루프만 쓰면 안 되나요? 🤔"

진짜 좋은 질문이에요! 각 루프는 다른 상황에 적합하죠.  for 루프는 정해진 반복 횟수가 확실할 때 최고의 선택이랍니다 💪!

예를 들어, 숫자 목록을 순회하며 각각 출력하고 싶다고 가정해볼까요? 🤔


```csharp
// 1부터 5까지 출력하는 for 루프 예시
for (int i = 1; i <= 5; i++) // "i" 변수 초기화, 조건 확인, 증가
{
    Console.WriteLine(i); // 각 번호를 콘솔에 출력
}
```

🚀 **코드 분석**:

* `int i = 1;`:  'i'라는 변수를 정의하고 처음에는 1로 초기화! 이 변수가 우리 반복 카운터 역할을 합니다.
* `i <= 5;`: 'i' 값이 5보다 작거나 같으면 루프가 계속됩니다. 조건 확인은 항상 중요해요!
* `i++`:  각 실행 후 'i' 값이 1씩 증가합니다. 이렇게 반복하며 마지막에 i = 5를 만족하면 루프 종료 😎
* `Console.WriteLine(i);`:  현재 'i' 값을 콘솔 창에 출력하는 행동입니다.

**🚨 실무주의보**:

for 루프는 딱 정해진 반복 횟수가 불분명한 상황에서는 적합하지 않아요! 그때는 while 루프나 do-while 루프를 사용해야 합니다. 😉


### 🚀 while 루프: 조건을 확인하며 지속적인 탐험!

**💡 초보자 폭풍 질문!** "for 루프와 어떤 차이가 있죠?" 🤔

좋은 질문입니다!  while 루프는 **특정 조건이 참(True)일 때만 실행되는 반복문** 입니다.


```csharp
// 사용자 입력을 받아 짝수인지 확인하는 while 루프 예시
int number;
Console.Write("숫자를 입력하세요: ");
number = int.Parse(Console.ReadLine()); // 사용자가 입력한 값을 정수로 변환

while (number % 2 == 0) // "number"가 짝수인지 확인
{
    Console.WriteLine($"{number}는 짝수입니다.");
    Console.Write("다음 숫자를 입력하세요: ");
    number = int.Parse(Console.ReadLine());
}
```

🚀 **코드 분석**:

* `int number;`:  'number'라는 정수 변수를 선언합니다.
* `while (number % 2 == 0)`: 'number'가 2로 나누어떨어질 때(짝수) 루프가 계속됩니다. 조건이 거짓이 되면 루프 종료!
* `Console.WriteLine($"{number}는 짝수입니다.");`:  해당 숫자가 짝수인 경우 출력합니다.
* `number = int.Parse(Console.ReadLine());`: 사용자가 다시 입력한 값을 'number'에 저장합니다.

**🚨 실무주의보**:

while 루프의 조건이 항상 참으로 유지되면 **무한루프**가 발생하여 프로그램이 멈출 수 있습니다! 주의하세요!



### 🚀 do-while 루프: 적어도 한 번은 실행해볼 거야!

do-while 루프는 반복문 중 가장 '끈기 있는' 친구죠!  조건 확인은 **마지막에** 해서, 반드시 최소 한 번은 코드가 실행됩니다.


```csharp
// 사용자 입력을 받아 홀수인지 확인하는 do-while 루프 예시
int number;
Console.Write("숫자를 입력하세요: ");
number = int.Parse(Console.ReadLine());

do
{
    Console.WriteLine($"{number}는 홀수입니다.");
    Console.Write("다음 숫자를 입력하세요: ");
    number = int.Parse(Console.ReadLine());
} while (number % 2 != 0); // "number"가 짝수인 경우 루프 종료
```

🚀 **코드 분석**:


* `do {...} while (조건)`:  do-while 구문의 패턴입니다. '도'는 반드시 실행, '하면서'는 조건 확인!
* 나머지는 while 루프와 유사하며, 홀수 입력을 받았을 때만 루프가 종료됩니다.

**🚨 실무주의보**:

do-while 루프 역시 조건이 항상 참으로 유지될 경우 무한루프 발생 가능성이 있습니다! 주의하세요!

오늘 C# 반복문의 세계에 함께 여행해 보았습니다.

* while 루프는 조건 확인,
