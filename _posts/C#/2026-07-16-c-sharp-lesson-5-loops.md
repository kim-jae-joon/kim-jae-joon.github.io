## 5강: 반복문: for 루프, while 루프 - 코드로 말하는 '지루함'과 '무한 가능성'!

👋 **사랑하는 코딩초보님들!** 👋  저는 C#를 5년 동안 아끼고 바탕으로 하는 당신에게 반복문의 신비로운 세계를 알려드릴 겁니다. 지금까지 배운 것을 이용해서 코드로 말을 해주거나, 무한히 가능성을 열어줄 매력적인 도구가 될 거예요!

###  1. 반복문이란? - '지루함'은 이제 끝나요!

👨‍💻 개발자는 항상 같은 작업을 여러 번 반복해야 하는 경우가 많습니다. 🍔햄버거를 만드는 것처럼, 코드에서도 같은 작업을 여러 번 실행하고 싶을 때, **반복문**이 등장해 '지루함'을 날려 버립니다!

`for 루프`, `while 루프` 두 가지 주인공들이 이러한 반복적인 작업을 돕는 데 최고의 실력을 발휘하는 거예요. 🔥

###  2. for 루프: 마법 같은 정확성 - '나열된 가능성'으로 접근!

💡 **초보자 폭풍 질문!** 🤔 "for 루프는 무엇일까?"

`for 루프`는 특정 횟수만큼 반복 작업을 실행하는 주인공입니다. 마치 햄버거를 만들 때 한 조각씩 쌓아가듯이, 정해진 규칙에 따라 작업을 반복합니다.

**🔑 for 루프의 구성 요소:**
1. **초기화:** 작업 시작 전, 변수 초기값 설정 (계획!)
2. **조건:** 일정 조건 충족 시 반복 (햄버거 재료가 부족하지 않을 때까지)
3. **증분:** 각 반복 후에 변수 값 변경 (다음 층의 재료 추가)

**✨ for 루프 코드 예제:**

```C#
for (int i = 1; i <= 5; i++) // 초기화: 숫자 변수 i를 1로 설정, 조건: i가 5보다 작거나 같을 때까지 반복, 증분: i에 1씩 추가
{
  Console.WriteLine(i);   // 현재 값을 출력 (햄버거 재료 쌓기)
}
```

* **설명:** 위 코드는 `for` 루프를 사용하여 1부터 5까지의 숫자를 출력합니다. 각 반복에서 `i` 변수에 `1`씩 증가하며, 조건이 만족될 때까지 계속 반복됩니다.
💡 이렇게 "정해진 가능성"을 담은 코딩 방식입니다!

###  3. while 루프: '지금'의 명령 - 끊임없는 감시와 행동!

🚨 **실무 주의보:** `while` 루프는 조심스럽게 사용해야 합니다! 조건이 계속 참인 경우, 무한루프가 발생하여 프로그램이 멈출 수 있습니다. (😨)

💡 **초보자 폭풍 질문!** 🤔 "while 루프는 어떻게 작동할까?"

`while 루프`는 특정 조건이 참일 때만 반복 작업을 실행하는 주인공입니다. 마치 '지금'의 상황에 따라 행동하는 것처럼, 계속해서 조건을 확인하며 반복합니다. 🏃‍♀️

**🔑 while 루프의 구성 요소:**
1. **조건:** 코드 블럭이 실행될 조건 (상황 판단)

```C#
while (i <= 5) // 조건: i가 5보다 작거나 같을 때까지 반복
{
  Console.WriteLine(i);   // 현재 값 출력
  i++;                  // 증가시키기
}
```


* **설명:** 위 코드는 `while` 루프를 사용하여 `i` 변수가 5보다 작거나 같을 동안 계속해서 값을 출력하고 증가시킵니다. 조건이 참인 동안 반복되며, 조건이 거짓이 되면 루프가 종료됩니다.
