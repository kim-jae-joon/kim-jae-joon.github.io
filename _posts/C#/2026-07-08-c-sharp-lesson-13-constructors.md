##  13강: 생성자 - 객체 초기화 : 코드의 건설 시작! 🎉

안녕하세요, 여러분! 저는 **5년 차 C# 개발자**로서 오늘은 코딩의 신성한 공간, **객체**를 만드는 비밀병기인 **생성자**에 대해 알아보겠습니다. 🤫  이 주제는 객체지향 프로그래밍에서 가장 중요하고 기본적인 개념 중 하나라니깐요!

물론 이전 강의들에서 이미 변수와 타입 등의 기초를 배웠을 거죠? 그렇다면 생성자 이야기까지 하면 훨씬 더 멋진 프로그램을 만들 수 있는 초점을 맞출 수 있겠죠? 🤔  오늘은 코드의 정령이 되고, 객체를 건설하는 마법사가 될 준비를 함께 해 보세요! ✨

### 객체는 무엇일까요? 🤔

생각보다 쉬운 개념이에요. 사실 우리 주변에서 만나는 모든 것들을 '객체'로 생각해 볼 수 있어요. 예를 들어, 자동차, 책, 사람은 모두 객체입니다. 각각 **특정 기능**을 가지고 있으며, 이러한 기능들은 **데이터(변수)**와 **작업을 수행하는 방법(함수)**으로 구성되어 있습니다. 🚗📚👨

### 생성자: 객체의 건설자! 🚀

이제부터 우리 C# 세계에서는 객체를 만들기 위한 특별한 도구, 바로 **생성자**에 대해 알아보겠습니다! 생성자는 코드가 실행될 때 새로운 객체를 만드는 작업을 담당하는 멋진 함수랍니다. 😎

예를 들어, 자동차를 만들고 싶다면 차량의 색깔, 모델, 연료 등 정보를 필요로 합니다. 이러한 정보들을 생성자가 받아들여 자동차 객체를 완성해 줄 수 있겠죠?  汽车처럼! 🏎️✨

### 생성자의 구조: 코드 정석을 보여드릴게요! 📚

생성자는 클래스 이름과 동일한 이름으로 선언되고, `()` 안에 필요한 정보(파라미터)를 받아들이도록 구성됩니다. 🚀  이 파라미터들은 객체를 만들 때 입력되는 데이터들입니다. 💪

```csharp
public class 자동차 
{
    // 속성 (데이터): 색깔, 모델, 연료 등
    string 색깔; // 자동차의 색깔
    string 모델; // 자동차의 모델명
    string 연료; // 자동차가 사용하는 연료

    // 생성자: 새로운 자동차 객체를 만드는 함수!
    public 자동차(string 색깔, string 모델, string 연료) 
    {
        this.색깔 = 색깔;  // 생성자에서 this 키워드로 속성 값을 설정
        this.모델 = 모델;
        this.연료 = 연료;
    }
}

```

**💡 초보자 폭풍 질문!** 🤔

* `public`은 무슨 의미일까요?
*  `색깔`, `모델`, `연료`와 같은 변수들이 **속성**이라고 불리는 이유는 무엇일까요?

### 생성자의 활용: 코드 실력 향상! 🚀

생성자를 사용하면 객체 초기화 작업을 간결하게 만들고 코드 가독성을 높일 수 있습니다. 💥

```csharp
// 새로운 자동차 객체 생성
자동차 나의자동차 = new 자동차("빨간색", "티코", "가솔린"); 

Console.WriteLine(나의자동차.색깔); // 출력: 빨간색
Console.WriteLine(나의자동차.모델); // 출력: 티코
```

이렇게 코드를 작성하면, 새로운 자동차 객체가 생성되고, 색깔, 모델, 연료 값을 설정합니다. 마치 Lego 블록처럼! 다만 C#에서는 '블록'은 코딩 규칙에 맞는 코드 조각인 거죠!

### 여러 가지 생성자: 더욱 강력한 제어권을 가지세요!  💪

생성자는 파라미터의 개수와 타입에 따라 다양하게 만들 수 있습니다. 예를 들어, 자동차 객체를 만드는 데 필요한 정보가 달라질 때 유용합니다. 

* **기본 생성자:** 생략이 가능하다는 특징을 가지고 있어 편리하게 객체를 초기화할 수 있는 방법입니다.


```csharp
public class 자동차 
{
    // ... (속성 정의) ...

    public 자동차() // 기본 생성자
    {
        색깔 = "흰색";
        모델 = "기본 모델";
        연료 = "휘발유";
    }


    // ... (나머지 생성자) ...
} 



```

###  🚨 실무주의보!🚨

* 생성자 이름은 클래스 이름과 동일해야 합니다.
* 생성자는 `return` 문을 사용할 수 없습니다. 생성자의 목적은 객체를 생성하는 것입니다.
