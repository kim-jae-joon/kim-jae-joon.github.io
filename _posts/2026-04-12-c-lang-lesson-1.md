## 💡 C언어 1강: 변수와 자료형, 이제 헷갈리지 마세요!

### 1. 변수 (Variable): 데이터를 담는 상자!

프로그래밍에서 '변수'란 무엇일까요? 어렵게 생각할 필요 없습니다. 여러분 주변의 **서랍장**, **사물함**, 아니면 택배 박스라고 생각해 보세요. 📦

상자마다 이름표가 붙어있고, 그 안에 뭔가 담겨 있죠?
프로그램 안에서 데이터를 잠시 저장해 두는 **'메모리 공간의 이름표'**가 바로 변수입니다. 이 이름표가 있기 때문에 우리는 어떤 데이터를 어디에 저장했는지, 그리고 나중에 다시 꺼내 쓸 수 있는지 알 수 있어요.

변수의 3가지 핵심 요소는 이렇습니다.

*   **이름 (Name):** 상자 겉에 붙은 이름표 (예: `age`, `score`, `name`)
*   **값 (Value):** 상자 안에 담긴 내용물 (예: `30`, `95`, `"철수"`)
*   **자료형 (Data Type):** 상자에 어떤 종류의 내용물을 담을지 알려주는 정보 (예: 숫자인지, 글자인지)

---

#### 📌 변수 선언과 초기화: 상자를 준비하고 내용물을 넣자!

컴퓨터에게 "나 이런 이름의 서랍장 쓸 거야!"라고 미리 알려주는 과정을 **'변수 선언'**이라고 합니다.
선언할 때는 어떤 종류의 데이터를 담을 건지(`자료형`)와 상자의 이름(`변수이름`)을 지정해 줘야 해요.

```c
// 변수 선언 기본 문법
자료형 변수이름;
```

**예시:**
```c
int number; // 'number'라는 이름으로 정수(int)를 저장할 공간을 만들게요!
```

**여기서 잠깐! 🚨**
변수를 선언만 하면 그 안에는 이전에 사용했던 알 수 없는 '쓰레기 값'이 들어있을 수 있습니다. 마치 비어있는 줄 알았던 서랍에 잊고 있던 이상한 물건이 들어있는 것처럼요.
그래서 보통 선언과 동시에 깨끗한 값으로 **'초기화'**하는 습관을 들이는 게 좋습니다.

```c
// 변수 선언과 동시에 초기화 문법
자료형 변수이름 = 초기값;
```

**예시:**
```c
int age = 30; // 'age'라는 이름의 정수형 변수를 만들고, 30으로 초기화!
```

이제 여러분은 데이터를 담을 상자를 만들고, 그 안에 초기값까지 넣어줄 수 있게 되었습니다. 멋지죠? 😉

### 2. 자료형 (Data Type): 어떤 종류의 데이터를 담을 것인가?

그럼 아까부터 계속 등장하는 **'자료형'**은 뭘까요?
자료형은 변수가 어떤 종류의 데이터를 저장할 것인지 컴퓨터에게 알려주는 '정보'입니다. 숫자를 담을 상자인지, 글자를 담을 상자인지, 아니면 소수점을 가진 숫자를 담을 상자인지 말이죠.

컴퓨터는 이 자료형을 보고 "아, 이 상자에는 정수가 들어갈 테니 이만큼의 공간을 준비해야겠군!" 하고 예측하고 효율적으로 메모리를 할당합니다.

C언어에서 가장 많이 사용되는 주요 자료형들을 살펴볼게요.

| 자료형 (Type) | 설명 (Description)                               | 예시 (Example)                               | `printf` 형식 지정자 |
| :------------ | :----------------------------------------------- | :------------------------------------------- | :------------------- |
| `int`         | **정수 (Integer)**: 소수점이 없는 숫자.          | `10`, `-5`, `100000`                         | `%d`                 |
| `char`        | **문자 (Character)**: 한 글자. 따옴표(') 사용. | `'A'`, `'a'`, `'!'`, `'5'`                   | `%c`                 |
| `float`       | **실수 (Floating-point)**: 소수점이 있는 숫자. (단정밀도) | `3.14`, `-0.5`, `123.45f` (뒤에 `f` 붙이는 경우 많음) | `%f`                 |
| `double`      | **실수 (Double floating-point)**: `float`보다 두 배 더 정밀한 실수. (배정밀도) | `3.141592`, `0.00001`                        | `%lf` (입력 시) / `%f` (출력 시) |

**닥터 C의 꿀팁! ✨**
실수를 다룰 때는 특별한 경우가 아니라면 `float`보다는 **`double`**을 사용하는 것이 일반적입니다. `double`이 더 넓은 범위의 수를 표현하고, 정밀도도 높아서 오류 발생 확률이 적거든요!

### 3. 변수 선언, 초기화, 그리고 출력 (feat. `printf`)

자, 이제 배운 내용을 바탕으로 직접 코드를 짜보고, 그 결과물을 화면에 띄워볼 시간입니다!
우리가 만든 변수 안에 뭐가 들어있는지 궁금하시죠? C언어의 강력한 출력 함수인 `printf`를 이용하면 변수 안의 값을 화면에 짠! 하고 보여줄 수 있습니다.

`printf` 함수를 사용할 때는 해당 변수의 자료형에 맞는 **'형식 지정자(Format Specifier)'**를 사용해야 해요. 위 표에 정리해 드렸으니 참고하세요!

```c
#include <stdio.h> // printf 함수를 사용하기 위해 필요한 헤더 파일입니다.

int main() { // 모든 C 프로그램은 main 함수에서 시작합니다.
    // 1. int (정수) 변수 선언 및 초기화
    int student_age = 20; // 학생의 나이를 20으로 설정
    int score = 95;       // 점수를 95점으로 설정

    // 2. char (문자) 변수 선언 및 초기화
    char grade = 'A'; // 학점을 'A'로 설정 (작은따옴표 사용!)

    // 3. float (실수 - 단정밀도) 변수 선언 및 초기화
    // 실수 리터럴은 기본적으로 double로 취급되므로, float형임을 명시하기 위해 'f'를 붙여주는 경우가 많습니다.
    float pi_float = 3.141592f; // 원주율을 float으로 설정

    // 4. double (실수 - 배정밀도) 변수 선언 및 초기화
    double gravity = 9.80665; // 중력 가속도를 double으로 설정

    printf("--- 변수와 자료형 첫 실습! ---\n"); // 제목 출력 (\n은 줄바꿈)

    // int 변수 출력
    printf("학생의 나이: %d세\n", student_age); // %d는 int형 변수의 값을 출력합니다.
    printf("시험 점수: %d점\n", score);

    // char 변수 출력
    printf("학점: %c\n", grade); // %c는 char형 변수의 값을 출력합니다.

    // float 변수 출력
    printf("원주율 (float): %f\n", pi_float); // %f는 float/double형 변수의 값을 출력합니다.
    printf("원주율 (float, 소수점 2자리): %.2f\n", pi_float); // %.2f는 소수점 둘째 자리까지 출력합니다.

    // double 변수 출력
    printf("중력 가속도 (double): %f\n", gravity); // double도 %f로 출력합니다.
    printf("중력 가속도 (double, 소수점 4자리): %.4f\n", gravity); // %.4f는 소수점 넷째 자리까지 출력합니다.

    return 0; // 프로그램이 성공적으로 종료되었음을 운영체제에 알립니다.
}
```

**[실행 결과 예시]**
```
--- 변수와 자료형 첫 실습! ---
학생의 나이: 20세
시험 점수: 95점
학점: A
원주율 (float): 3.141592
원주율 (float, 소수점 2자리): 3.14
중력 가속도 (double): 9.806650
중력 가속도 (double, 소수점 4자리): 9.8067
```

어때요? 여러분이 입력한 값이 화면에 예쁘게 출력되죠? 신기하지 않나요?! 🎉

### 4. `sizeof()` 연산자로 변수 크기 확인하기

컴퓨터는 '메모리'라는 저장 공간을 사용한다고 했죠? 각 자료형마다 메모리를 차지하는 크기가 다릅니다. 이는 컴퓨터 환경(32비트/64비트 시스템 등)에 따라 조금씩 다를 수 있지만, 일반적으로 정해진 규칙이 있어요.

C언어에서는 `sizeof()` 연산자를 사용하면 각 자료형이나 변수가 얼마만큼의 메모리를 차지하는지 직접 확인해볼 수 있습니다. 단위는 **바이트(Byte)**입니다.

```c
#include <stdio.h> // printf 함수를 사용하기 위해 필요한 헤더 파일

int main() {
    // 각 자료형의 크기를 확인
    printf("--- 자료형별 메모리 크기 (Byte) ---\n");
    printf("int의 크기: %zu 바이트\n", sizeof(int));     // %zu는 sizeof 연산자의 결과를 출력할 때 사용합니다.
    printf("char의 크기: %zu 바이트\n", sizeof(char));
    printf("float의 크기: %zu 바이트\n", sizeof(float));
    printf("double의 크기: %zu 바이트\n", sizeof(double));

    // 변수의 크기도 직접 확인할 수 있어요!
    int my_number = 100;
    char initial = 'D';
    double big_value = 1234.5678;

    printf("\n--- 선언된 변수의 메모리 크기 (Byte) ---\n");
    printf("my_number 변수의 크기: %zu 바이트\n", sizeof(my_number));
    printf("initial 변수의 크기: %zu 바이트\n", sizeof(initial));
    printf("big_value 변수의 크기: %zu 바이트\n", sizeof(big_value));

    return 0;
}
```

**[실행 결과 예시]** (시스템 환경에 따라 다를 수 있음)
```
--- 자료형별 메모리 크기 (Byte) ---
int의 크기: 4 바이트
char의 크기: 1 바이트
float의 크기: 4 바이트
double의 크기: 8 바이트

--- 선언된 변수의 메모리 크기 (Byte) ---
my_number 변수의 크기: 4 바이트
initial 변수의 크기: 1 바이트
big_value 변수의 크기: 8 바이트
```

보통 `int`는 4바이트, `char`는 1바이트, `float`는 4바이트, `double`은 8바이트를 차지하는 것을 볼 수 있습니다. `double`이 `float`보다 더 큰 메모리를 사용하기 때문에 더 정확하고 넓은 범위의 실수를 표현할 수 있는 거예요! 🧐

## 🚀 1강 마무리: 복습만이 살 길이다!

자, 어떠셨나요? C언어 첫 강의, '변수와 자료형'!
오늘은 변수가 무엇인지, 어떤 종류의 자료형이 있는지, 그리고 이들을 어떻게 선언하고 초기화하며 화면에 출력하는지 배워봤습니다.

**핵심 요약!**
*   **변수:** 데이터를 저장하는 이름 붙은 메모리 공간.
*   **자료형:** 변수에 어떤 종류의 데이터를 담을지 알려주는 정보. (`int`, `char`, `float`, `double` 등)
*   **선언:** `자료형 변수이름;`
*   **초기화:** `자료형 변수이름 = 값;`
*   **출력:** `printf("형식지정자", 변수이름);`

오늘 배운 내용은 C언어의 가장 기본 중의 기본이자, 앞으로 여러분이 만들 모든 프로그램의 토대가 됩니다. 마치 집을 짓기 위한 튼튼한 주춧돌과 같아요!
