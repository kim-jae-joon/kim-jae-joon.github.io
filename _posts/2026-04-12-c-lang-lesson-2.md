## 🚀 C언어 2강: 개발의 뼈대! 조건문(if)과 반복문(for) 기초 마스터하기

### 1. 프로그램의 '판단력'을 키워주는 **조건문 (Conditional Statement) - `if`**

우리가 일상생활에서 "만약 ~라면, 이렇게 하고, 그렇지 않으면 저렇게 하자"라고 말하는 것처럼, 프로그램도 특정 조건에 따라 다른 동작을 하도록 만들 때 사용하는 문법이 바로 `if` 문입니다.

#### 1.1. `if` 문의 기본 구조: "만약 ~라면"

가장 기본적인 `if` 문은 특정 조건이 `참(true)`일 때만 코드를 실행합니다.

**✅ 기본 문법**

```c
if (조건식) {
    // 조건식이 참(true)일 때 실행할 코드
}
```

-   **`조건식`**: 결과가 `참(0이 아닌 값)` 또는 `거짓(0)`으로 나오는 표현식입니다.
-   **`{}` (중괄호)**: 조건식이 참일 때 실행될 코드 블록입니다. 만약 실행할 코드가 한 줄이라면 중괄호를 생략할 수도 있지만, 가독성과 잠재적인 오류 방지를 위해 항상 사용하는 것을 권장합니다!

**💻 예제 코드: 성인 인증하기**

```c
#include <stdio.h> // 표준 입출력 함수를 사용하기 위한 헤더 파일

int main() {
    int age = 20; // 나이 변수 선언 및 초기화

    // 만약 나이가 18 이상이면 "성인입니다." 출력
    if (age >= 18) {
        printf("당신은 성인입니다.\\n");
    }

    printf("프로그램이 종료됩니다.\\n"); // if문 실행 여부와 관계없이 항상 출력

    return 0; // 프로그램 성공 종료
}
```

**🔍 실행 결과**

```
당신은 성인입니다.
프로그램이 종료됩니다.
```

`age`가 20이므로 `age >= 18` 조건은 `참`이 되어 `if` 블록 안의 코드가 실행됩니다.

#### 1.2. `if-else` 문: "만약 ~라면, 아니면"

조건식이 `참`일 때와 `거짓`일 때 각각 다른 동작을 하도록 만들고 싶을 때 `else` 키워드를 사용합니다.

**✅ 기본 문법**

```c
if (조건식) {
    // 조건식이 참(true)일 때 실행할 코드
} else {
    // 조건식이 거짓(false)일 때 실행할 코드
}
```

**💻 예제 코드: 짝수/홀수 판별하기**

```c
#include <stdio.h>

int main() {
    int number = 7; // 정수 변수 선언 및 초기화

    // number를 2로 나눈 나머지가 0이면 짝수, 아니면 홀수
    if (number % 2 == 0) { // '%'는 나머지 연산자입니다.
        printf("%d는 짝수입니다.\\n", number);
    } else {
        printf("%d는 홀수입니다.\\n", number);
    }

    return 0;
}
```

**🔍 실행 결과**

```
7는 홀수입니다.
```

`number`가 7이므로 `7 % 2 == 0` 조건은 `거짓`이 되어 `else` 블록 안의 코드가 실행됩니다.

#### 1.3. `if-else if-else` 문: "만약 ~라면, 아니면 만약 ~라면, 그것도 아니면"

여러 가지 조건을 순차적으로 검사하여 각 조건에 맞는 동작을 하고 싶을 때 `else if`를 사용합니다.

**✅ 기본 문법**

```c
if (조건식1) {
    // 조건식1이 참일 때 실행할 코드
} else if (조건식2) {
    // 조건식1이 거짓이고, 조건식2가 참일 때 실행할 코드
} else if (조건식3) {
    // 조건식1, 2가 거짓이고, 조건식3이 참일 때 실행할 코드
} else {
    // 모든 조건식이 거짓일 때 실행할 코드 (선택 사항)
}
```

**💻 예제 코드: 학점 계산기**

```c
#include <stdio.h>

int main() {
    int score = 85; // 점수 변수 선언 및 초기화

    if (score >= 90) {
        printf("당신의 학점은 A입니다.\\n");
    } else if (score >= 80) { // 90점 미만이면서 80점 이상
        printf("당신의 학점은 B입니다.\\n");
    } else if (score >= 70) { // 80점 미만이면서 70점 이상
        printf("당신의 학점은 C입니다.\\n");
    } else { // 70점 미만
        printf("재수강이 필요합니다.\\n");
    }

    return 0;
}
```

**🔍 실행 결과**

```
당신의 학점은 B입니다.
```

`score`가 85이므로 첫 번째 `if (score >= 90)`은 거짓입니다. 다음 `else if (score >= 80)`은 참이므로 해당 블록의 코드가 실행되고, 나머지 `else if`와 `else`는 검사하지 않고 건너뜁니다.

### 💡 잠!깐! 조건에는 뭐가 들어가나요? (비교 연산자와 논리 연산자)

`if` 문의 조건식에는 주로 변수나 상수를 비교하는 **비교 연산자**와 여러 조건을 묶어주는 **논리 연산자**가 사용됩니다.

#### 1. 비교 연산자 (Comparison Operators)

두 값을 비교하여 참(1) 또는 거짓(0)을 반환합니다.

| 연산자 | 의미                | 예시         | 설명                                            |
| :----- | :------------------ | :----------- | :---------------------------------------------- |
| `==`   | 같다 (equal to)     | `a == b`     | `a`와 `b`가 같으면 참                           |
| `!=`   | 같지 않다 (not equal to) | `a != b`     | `a`와 `b`가 다르면 참                           |
| `>`    | 보다 크다 (greater than) | `a > b`      | `a`가 `b`보다 크면 참                           |
| `<`    | 보다 작다 (less than) | `a < b`      | `a`가 `b`보다 작으면 참                         |
| `>=`   | 크거나 같다 (greater than or equal to) | `a >= b`     | `a`가 `b`보다 크거나 같으면 참                  |
| `<=`   | 작거나 같다 (less than or equal to) | `a <= b`     | `a`가 `b`보다 작거나 같으면 참                  |

#### 2. 논리 연산자 (Logical Operators)

여러 개의 조건을 조합하여 하나의 참/거짓 값을 만듭니다.

| 연산자 | 의미                | 예시            | 설명                                              |
| :----- | :------------------ | :-------------- | :------------------------------------------------ |
| `&&`   | 논리 AND (Logical AND) | `(A && B)`      | `A`와 `B` 모두 참이면 참. 둘 중 하나라도 거짓이면 거짓 |
| `||`   | 논리 OR (Logical OR)   | `(A || B)`      | `A`와 `B` 중 하나라도 참이면 참. 모두 거짓일 때만 거짓 |
| `!`    | 논리 NOT (Logical NOT) | `(!A)`          | `A`가 참이면 거짓, 거짓이면 참                   |

**💻 예제: AND, OR 활용**

```c
#include <stdio.h>

int main() {
    int age = 25;
    int isStudent = 1; // 1은 true, 0은 false를 의미 (C언어에서는 0이 아닌 모든 수가 true)

    // 나이가 18세 이상이고 학생이면 할인 적용
    if (age >= 18 && isStudent == 1) {
        printf("학생 할인이 적용됩니다.\\n");
    }

    // 나이가 65세 이상이거나 7세 미만이면 무료 입장
    if (age >= 65 || age < 7) {
        printf("무료 입장입니다.\\n");
    } else {
        printf("입장료를 지불해야 합니다.\\n");
    }

    return 0;
}
```

### 2. 프로그램의 '반복적인 일 처리'를 담당하는 **반복문 (Loop Statement) - `for`**

프로그램을 만들다 보면 똑같은 작업을 여러 번 해야 할 때가 정말 많습니다. 예를 들어 "안녕하세요"를 10번 출력하거나, 1부터 100까지의 숫자를 모두 더하는 것 같은 작업이죠. 이럴 때 일일이 코드를 작성하는 건 너무 비효율적입니다. 바로 이때 `for` 반복문이 등장합니다!

#### 2.1. `for` 문의 기본 구조

`for` 문은 정해진 횟수만큼 또는 특정 조건이 만족하는 동안 코드 블록을 반복 실행합니다.

**✅ 기본 문법**

```c
for (초기화; 조건식; 증감식) {
    // 반복될 코드
}
```

-   **`초기화`**: 반복문이 시작될 때 단 한 번 실행됩니다. 주로 반복 횟수를 세는 변수를 선언하고 초기화합니다.
-   **`조건식`**: 매 반복마다 이 조건식이 참인지 검사합니다. 조건식이 `참(true)`이면 반복을 계속하고, `거짓(false)`이면 반복문을 종료합니다.
-   **`증감식`**: 한 번의 반복(iteration)이 끝날 때마다 실행됩니다. 주로 반복 변수의 값을 증가시키거나 감소시켜 조건식에 영향을 줍니다.
-   **`{}` (중괄호)**: 반복될 코드 블록입니다. 조건문과 마찬가지로 한 줄일 경우 생략 가능하지만, 항상 사용하는 것을 권장합니다!

**💻 예제 코드: "안녕하세요" 5번 출력하기**

```c
#include <stdio.h>

int main() {
    // i를 0으로 초기화 -> i가 5보다 작은 동안 -> i를 1씩 증가시키며 반복
    for (int i = 0; i < 5; i++) {
        printf("%d번째: 안녕하세요!\\n", i + 1);
    }

    printf("반복문 종료.\\n");

    return 0;
}
```

**🔍 실행 결과**

```
1번째: 안녕하세요!
2번째: 안녕하세요!
3번째: 안녕하세요!
4번째: 안녕하세요!
5번째: 안녕하세요!
반복문 종료.
```

-   `int i = 0;`: `i`는 0으로 시작합니다.
-   `i < 5;`: `i`가 5보다 작을 때까지 반복합니다. 즉, `i`가 0, 1, 2, 3, 4일 때 총 5번 반복됩니다.
-   `i++`: 한 번 반복할 때마다 `i`의 값이 1씩 증가합니다. (i++는 i = i + 1과 같습니다. i--는 i = i - 1과 같습니다.)

**💻 예제 코드: 1부터 10까지의 합 구하기**

```c
#include <stdio.h>

int main() {
    int sum = 0; // 합계를 저장할 변수 초기화

    // i를 1부터 10까지 1씩 증가시키며 반복
    for (int i = 1; i <= 10; i++) {
        sum = sum + i; // 또는 sum += i; 와 같습니다.
        printf("현재까지의 합: %d (더해진 수: %d)\\n", sum, i); // 각 단계 확인
    }

    printf("1부터 10까지의 총합: %d\\n", sum);

    return 0;
}
```

**🔍 실행 결과**

```
현재까지의 합: 1 (더해진 수: 1)
현재까지의 합: 3 (더해진 수: 2)
현재까지의 합: 6 (더해진 수: 3)
현재까지의 합: 10 (더해진 수: 4)
현재까지의 합: 15 (더해진 수: 5)
현재까지의 합: 21 (더해진 수: 6)
현재까지의 합: 28 (더해진 수: 7)
현재까지의 합: 36 (더해진 수: 8)
현재까지의 합: 45 (더해진 수: 9)
현재까지의 합: 55 (더해진 수: 10)
1부터 10까지의 총합: 55
```

### 💡 엉클 코딩의 특급 팁! (초보자들이 자주 하는 실수)

1.  **`if` 문 조건식 끝에 세미콜론(;)은 금물!**
    ```c
    if (age >= 18); { // <--- 여기에 세미콜론이 있으면 안 돼요!
        printf("성인입니다.\\n");
    }
    ```
    세미콜론은 문장의 끝을 의미하기 때문에 `if (age >= 18);` 이 자체가 하나의 빈 문장이 되어 버리고, `printf` 문은 조건과 상관없이 항상 실행됩니다.
2.  **`==` (같다)와 `=` (대입)의 혼동**
    ```c
    int num = 5;
    if (num = 10) { // <--- num == 10 으로 써야 합니다!
        printf("num은 10입니다.\\n");
    }
    ```
    `num = 10`은 `num`에 10을 **대입**하는 연산이고, 이 연산의 결과는 대입된 값(10)입니다. C언어에서는 0이 아닌 모든 숫자를 `참`으로 간주하므로, `if (num = 10)`은 항상 참이 되어 `if` 블록이 실행됩니다. 의도치 않은 결과를 초래할 수 있으니 조심하세요!
3.  **무한 루프 조심! (`for` 문)**
    `for` 문의 조건식이 항상 참이 되면, 프로그램은 영원히 반복되는 **무한 루프(Infinite Loop)**에 빠지게 됩니다.
    ```c
    for (int i = 0; i < 5; i--) { // i++ 대신 i-- 로 실수하면 i는 계속 작아져서 i < 5는 항상 참!
        printf("무한 반복!\\n");
    }
    ```
    프로그램이 멈추지 않을 때는 `Ctrl + C` 키를 눌러 강제 종료할 수 있습니다.
4.  **`for` 문 범위 오류 (Off-by-one Error)**
    `for (int i = 0; i < 5; i++)`는 0, 1, 2, 3, 4 (총 5번) 반복입니다.
    `for (int i = 1; i <= 5; i++)`는 1, 2, 3, 4, 5 (총 5번) 반복입니다.
    시작 값과 조건식을 정확히 설정해야 원하는 횟수만큼 반복할 수 있습니다.

## 📝 오늘 배운 내용 정리!

오늘 우리는 프로그램의 흐름을 제어하는 두 가지 강력한 도구, **조건문 `if`**와 **반복문 `for`**를 깊이 있게 다뤄봤습니다.

-   **`if` 문**: 프로그램이 특정 조건에 따라 다르게 행동하도록 '판단'하게 만듭니다. `if`, `if-else`, `if-else if-else` 구조를 통해 다양한 조건 분기가 가능합니다.
-   **`for` 문**: 정해진 횟수만큼 또는 특정 조건이 만족하는 동안 코드를 '반복' 실행하게 만듭니다. `초기화`, `조건식`, `증감식`의 3단계로 이루어져 있습니다.
-   **비교 연산자**(`==`, `!=`, `>`, `<`, `>=`, `<=`)와 **논리 연산자**(`&&`, `||`, `!`)는 조건식을 구성하는 데 필수적입니다.

이 두 가지를 잘 활용하면, 여러분의 프로그램은 훨씬 더 똑똑하고 효율적으로 동작하게 될 거예요. 마치 사람의 생각처럼요!
