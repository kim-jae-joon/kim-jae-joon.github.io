## 🚀 3강: 프로그램의 흐름 제어: 조건문과 반복문

우리가 일상생활에서 어떤 결정을 내리거나, 특정 행동을 여러 번 반복하는 것처럼, 컴퓨터 프로그램도 상황에 따라 다른 행동을 하거나, 똑같은 행동을 여러 번 해야 할 때가 있습니다. 이걸 가능하게 해주는 것이 바로 **'흐름 제어(Control Flow)'** 문법입니다.

크게 두 가지 갈래로 나눌 수 있어요.

1.  **조건문 (Conditional Statements):** "만약 ~라면 이렇게 해라, 그렇지 않으면 저렇게 해라!" 처럼, 특정 조건에 따라 프로그램의 실행 흐름을 바꾸는 문법입니다.
2.  **반복문 (Loop Statements):** "이것을 10번 반복해라!" 또는 "이 조건이 참인 동안 계속 반복해라!" 처럼, 특정 코드 블록을 여러 번 실행하는 문법입니다.

자, 그럼 우리 함께 C 언어의 강력한 흐름 제어 마법을 하나씩 파헤쳐 볼까요? ✨

### 1. 조건문: 프로그램, 이제 스스로 판단하고 결정합니다!

프로그램이 똑똑하게 판단하려면 '조건'을 주고, 그 조건이 맞는지(참인지), 틀리는지(거짓인지)에 따라 다른 길을 가도록 해야 합니다.

C 언어에서 조건문은 크게 `if`, `if-else`, `else if`, 그리고 `switch` 문이 있습니다.

#### 1.1. `if` 문: '만약에 ~라면'

가장 기본적인 조건문입니다. `if` 뒤에 오는 **조건이 참(True)일 때만** 특정 코드 블록을 실행합니다.

**💡 잠시만요! C 언어에서 '참'과 '거짓'은 어떻게 표현될까요?**
*   **참 (True):** `0이 아닌 모든 값` (보통 `1`을 많이 사용하지만, `-5`, `100` 등 어떤 0이 아닌 숫자도 참으로 간주됩니다.)
*   **거짓 (False):** `0` (오직 숫자 `0`만이 거짓으로 간주됩니다.)

```c
#include <stdio.h> // printf 함수를 사용하기 위해 포함합니다.

int main() {
    int score = 95; // 학생의 점수

    // 만약 점수가 90점 이상이라면 '합격입니다!'를 출력
    if (score >= 90) {
        printf("합격입니다!\n"); // 조건이 참이므로 이 줄이 실행됩니다.
    }

    int age = 15;
    // 만약 나이가 20세 이상이라면 '성인입니다.'를 출력 (이 코드는 실행되지 않습니다.)
    if (age >= 20) {
        printf("성인입니다.\n"); // 조건이 거짓이므로 이 줄은 실행되지 않습니다.
    }

    printf("프로그램 종료.\n");
    return 0;
}
```

**실행 결과:**
```
합격입니다!
프로그램 종료.
```

**설명:** `score >= 90`은 `95 >= 90`이므로 참(True)이 됩니다. 따라서 `if` 블록 안의 `printf` 문이 실행됩니다. 반면, `age >= 20`은 `15 >= 20`이므로 거짓(False)이 되고, `if` 블록 안의 `printf` 문은 실행되지 않습니다.

#### 1.2. `if-else` 문: '만약 ~라면 이것, 그렇지 않으면 저것'

`if` 문의 조건이 참일 때와 거짓일 때, 각각 다른 코드를 실행하고 싶을 때 사용합니다. 마치 두 갈래 길에서 한 길만 선택하는 것과 같아요.

```c
#include <stdio.h>

int main() {
    int temperature = 28; // 현재 온도

    // 만약 온도가 25도 이상이라면 '더워요!' 출력, 그렇지 않으면 '시원해요!' 출력
    if (temperature >= 25) {
        printf("더워요! 에어컨을 켜세요.\n"); // 조건이 참이므로 이 줄이 실행됩니다.
    } else {
        printf("시원해요! 산책하기 좋은 날씨네요.\n"); // 조건이 거짓일 때 이 줄이 실행됩니다.
    }

    int num = 10;
    // num이 홀수인지 짝수인지 판별
    if (num % 2 == 1) { // num을 2로 나눈 나머지가 1이면 홀수
        printf("%d은(는) 홀수입니다.\n", num);
    } else { // 나머지가 0이면 짝수
        printf("%d은(는) 짝수입니다.\n", num); // 이 줄이 실행됩니다.
    }

    return 0;
}
```

**실행 결과:**
```
더워요! 에어컨을 켜세요.
10은(는) 짝수입니다.
```

**설명:** `temperature >= 25`는 `28 >= 25`이므로 참(True)이 되어 `if` 블록이 실행됩니다. `num % 2 == 1`은 `10 % 2 == 1`이므로 `0 == 1`이 되어 거짓(False)이 됩니다. 따라서 `else` 블록이 실행됩니다.

#### 1.3. `else if` (다중 조건): '여러 갈래 길에서 딱 하나만!'

여러 개의 조건 중에서 하나를 선택해야 할 때 사용합니다. 위에서부터 차례대로 조건을 검사하며, **가장 먼저 참이 되는 조건의 코드 블록만 실행하고 전체 조건문을 빠져나옵니다.**

```c
#include <stdio.h>

int main() {
    int grade = 85; // 학생의 점수

    // 점수에 따라 학점을 부여하는 프로그램
    if (grade >= 90) {
        printf("학점: A\n");
    } else if (grade >= 80) { // grade가 90 미만이면서 80 이상일 때
        printf("학점: B\n"); // 이 줄이 실행됩니다.
    } else if (grade >= 70) { // grade가 80 미만이면서 70 이상일 때
        printf("학점: C\n");
    } else { // 위 모든 조건에 해당하지 않을 때
        printf("학점: F\n");
    }

    int hour = 14; // 현재 시각

    // 시간에 따라 다른 메시지 출력
    if (hour < 12) {
        printf("좋은 아침입니다!\n");
    } else if (hour < 18) { // hour가 12 이상이면서 18 미만일 때
        printf("즐거운 오후입니다!\n"); // 이 줄이 실행됩니다.
    } else { // hour가 18 이상일 때
        printf("편안한 저녁 보내세요!\n");
    }

    return 0;
}
```

**실행 결과:**
```
학점: B
즐거운 오후입니다!
```

**설명:** `grade = 85`일 때, 첫 번째 `if (grade >= 90)`은 거짓입니다. 다음 `else if (grade >= 80)`은 `85 >= 80`이므로 참이 되어 해당 블록이 실행되고, 나머지 `else if`와 `else`는 건너뛰게 됩니다. **순서가 중요해요!**

#### 1.4. `switch` 문: '특정 값에 따라 다른 동작을!'

주로 하나의 변수(정수형 또는 문자형)의 **값에 따라** 여러 경우 중 하나를 선택하여 실행할 때 `if-else if` 문보다 가독성이 좋고 효율적일 수 있습니다.

**핵심:** `case` 뒤에 오는 값과 `switch`의 변수 값이 일치하면 해당 `case` 블록이 실행됩니다. **반드시 `break`를 사용하여 해당 `switch` 문을 빠져나와야 합니다.** `break`가 없으면 일치하는 `case` 이후의 모든 `case` 블록이 조건과 상관없이 계속 실행되는 '폴스루(fall-through)' 현상이 발생합니다.

```c
#include <stdio.h>

int main() {
    int day = 3; // 요일을 나타내는 숫자 (1:월, 2:화, ..., 7:일)

    printf("오늘은 ");
    switch (day) {
        case 1:
            printf("월요일");
            break; // switch 문을 여기서 빠져나갑니다.
        case 2:
            printf("화요일");
            break;
        case 3: // day가 3과 일치하므로 이 case가 실행됩니다.
            printf("수요일");
            break; // break가 있으므로 여기서 switch 문을 빠져나갑니다.
        case 4:
            printf("목요일");
            break;
        case 5:
            printf("금요일");
            break;
        default: // 위에 해당하는 case가 없을 때 실행됩니다.
            printf("주말");
            break;
    }
    printf("입니다.\n");

    char menu = 'C'; // 사용자가 선택한 메뉴

    printf("선택하신 메뉴는 ");
    switch (menu) {
        case 'A':
            printf("아메리카노");
            break;
        case 'B':
            printf("카페라떼");
            break;
        case 'C': // menu가 'C'와 일치하므로 이 case가 실행됩니다.
            printf("카푸치노");
            break;
        default:
            printf("알 수 없는 메뉴");
            break;
    }
    printf("입니다.\n");

    return 0;
}
```

**실행 결과:**
```
오늘은 수요일입니다.
선택하신 메뉴는 카푸치노입니다.
```

**설명:** `day`가 `3`이므로 `case 3`이 실행되고, `break` 덕분에 다른 `case`들은 실행되지 않고 `switch` 문을 빠져나옵니다. 만약 `break`가 없었다면, `수요일` 다음으로 `목요일`, `금요일`, `주말`까지 모두 출력될 뻔 했겠죠? `default`는 `if-else if` 문의 마지막 `else`와 같은 역할로, 어떤 `case`에도 해당하지 않을 때 실행됩니다.

### 2. 반복문: 똑같은 일을 여러 번!

컴퓨터는 지루해하지 않고 똑같은 일을 수없이 반복하는 데 탁월합니다. 우리가 프로그래밍할 때 이 반복의 힘을 빌리는 경우가 아주 많습니다.

C 언어에는 크게 `while`, `do-while`, `for` 세 가지 반복문이 있습니다.

#### 2.1. `while` 문: '조건이 참인 동안 계속 반복!'

가장 기본적인 반복문입니다. `while` 뒤에 오는 **조건이 참(True)인 동안** 코드 블록을 계속해서 반복 실행합니다. 조건을 먼저 검사하고 실행합니다.

**🚨 중요: 무한 루프 조심!**
`while` 문의 조건이 **항상 참**이 되면, 프로그램은 영원히 반복되는 무한 루프(Infinite Loop)에 빠지게 됩니다. 이렇게 되면 프로그램이 멈추지 않으니, 반복문 안에서 조건을 거짓으로 만들 수 있는 로직을 꼭 포함해야 합니다.

```c
#include <stdio.h>

int main() {
    int count = 0; // 반복 횟수를 세는 변수

    printf("while 반복 시작!\n");
    while (count < 5) { // count가 5보다 작은 동안 반복
        printf("현재 count: %d\n", count);
        count = count + 1; // count 값을 1씩 증가 (이것이 없으면 무한 루프!)
    }
    printf("while 반복 종료!\n");

    // 사용자로부터 'q'를 입력받을 때까지 계속 입력받는 예제
    char input_char;
    printf("\n'q'를 입력하면 종료됩니다.\n");
    while (1) { // 1은 항상 참이므로 일단 무한 루프처럼 시작합니다.
        printf("문자를 입력하세요: ");
        scanf(" %c", &input_char); // ' %c' 앞의 공백은 이전 입력 버퍼를 비워줍니다.

        if (input_char == 'q') {
            printf("q가 입력되어 종료합니다.\n");
            break; // 반복문을 강제로 빠져나갑니다.
        }
        printf("입력된 문자: %c\n", input_char);
    }
    return 0;
}
```

**실행 결과:**
```
while 반복 시작!
현재 count: 0
현재 count: 1
현재 count: 2
현재 count: 3
현재 count: 4
while 반복 종료!

'q'를 입력하면 종료됩니다.
문자를 입력하세요: a
입력된 문자: a
문자를 입력하세요: b
입력된 문자: b
문자를 입력하세요: q
q가 입력되어 종료합니다.
```

**설명:** 첫 번째 `while` 문은 `count`가 0부터 시작하여 4까지 5번 반복됩니다. `count`가 5가 되는 순간 `count < 5` 조건은 거짓이 되어 반복문이 종료됩니다. 두 번째 `while (1)`은 `break` 문을 사용하여 특정 조건(`input_char == 'q'`)이 만족되면 반복문을 빠져나오는 예제입니다.

#### 2.2. `do-while` 문: '일단 한 번은 실행하고, 조건 검사!'

`while` 문과 비슷하지만, 코드 블록을 **최소한 한 번은 실행한 후** 조건을 검사하여 계속 반복할지 결정합니다.

```c
#include <stdio.h>

int main() {
    int num;

    // 사용자로부터 1부터 10 사이의 숫자를 입력받을 때까지 반복
    do {
        printf("1부터 10 사이의 숫자를 입력하세요: ");
        scanf("%d", &num);
    } while (num < 1 || num > 10); // num이 1보다 작거나 10보다 크면 다시 반복

    printf("올바른 숫자 %d를 입력했습니다.\n", num);

    int i = 5;
    do {
        printf("do-while 안의 i: %d\n", i);
        i++;
    } while (i < 5); // i가 5부터 시작했으므로 조건은 거짓이지만, 한 번은 실행됩니다.

    return 0;
}
```

**실행 결과:**
```
1부터 10 사이의 숫자를 입력하세요: 0
1부터 10 사이의 숫자를 입력하세요: 11
1부터 10 사이의 숫자를 입력하세요: 5
올바른 숫자 5를 입력했습니다.
do-while 안의 i: 5
```

**설명:** `do-while`은 조건이 거짓이더라도 무조건 `do` 블록을 한 번 실행합니다. 첫 번째 예제에서는 사용자가 올바른 숫자를 입력할 때까지 계속 입력을 받습니다. 두 번째 예제에서는 `i`가 5로 시작했기 때문에 `i < 5` 조건은 처음부터 거짓이지만, `do` 블록 안의 코드가 한 번 실행되는 것을 볼 수 있습니다.

#### 2.3. `for` 문: '정해진 횟수만큼 반복할 때!'

특정 횟수만큼 반복하거나, 일정한 규칙(증가, 감소)을 가지고 반복할 때 가장 많이 사용되는 반복문입니다.

`for` 문은 세 가지 요소로 구성됩니다: `(초기화; 조건; 증감식)`

1.  **초기화 (Initialization):** 반복문이 시작될 때 **딱 한 번** 실행됩니다. 보통 반복 횟수를 세는 변수를 선언하고 초기값을 지정합니다.
2.  **조건 (Condition):** 매 반복 시작 전에 조건을 검사합니다. 조건이 참(True)이면 코드 블록을 실행하고, 거짓(False)이면 반복문을 종료합니다.
3.  **증감식 (Increment/Decrement):** 한 번의 반복이 끝날 때마다 실행됩니다. 보통 초기화 변수의 값을 증가시키거나 감소시켜 조건을 변화시킵니다.

```c
#include <stdio.h>

int main() {
    printf("for 반복 시작!\n");
    // 초기화: i = 0 (변수 i를 0으로 초기화)
    // 조건: i < 5 (i가 5보다 작은 동안 반복)
    // 증감식: i++ (반복 후 i를 1씩 증가)
    for (int i = 0; i < 5; i++) {
        printf("for 루프 안의 i: %d\n", i);
    }
    printf("for 반복 종료!\n");

    // 1부터 10까지의 합 계산
    int sum = 0;
    for (int i = 1; i <= 10; i++) {
        sum += i; // sum = sum + i;
    }
    printf("1부터 10까지의 합: %d\n", sum); // 55

    // 역순으로 5부터 1까지 출력
    for (int j = 5; j > 0; j--) { // j--는 j = j - 1과 같습니다.
        printf("역순: %d\n", j);
    }

    return 0;
}
```

**실행 결과:**
```

for 루프 안의 i: 0
for 루프 안의 i: 1
for 루프 안의 i: 2
for 루프 안의 i: 3
for 루프 안의 i: 4
for 반복 종료!
1부터 10까지의 합: 55
역순: 5
역순: 4
역순: 3
역순: 2
역순: 1
```

**설명:** `for` 문은 특히 반복 횟수가 명확하거나 일정한 규칙으로 변하는 경우에 아주 편리합니다. 첫 번째 예제는 `i`가 0부터 4까지 5번 반복하며 출력하고, 두 번째 예제는 1부터 10까지 숫자를 더합니다. 세 번째 예제는 `j`를 1씩 감소시키며 역순으로 출력합니다.

#### 2.4. `break`와 `continue`: 반복문의 특별한 제어

반복문 안에서 특별한 상황이 발생했을 때 반복문의 흐름을 제어하는 두 가지 유용한 키워드가 있습니다.

*   **`break;`**: 현재 실행 중인 반복문(가장 가까운 `for`, `while`, `do-while` 문)을 **즉시 종료하고** 반복문 다음 코드로 넘어갑니다. (위 `switch` 문에서 보셨죠?)

*   **`continue;`**: 현재 실행 중인 반복의 **나머지 부분을 건너뛰고**, 다음 반복(next iteration)을 즉시 시작합니다. 즉, 현재 반복만 건너뛰고 다음 반복으로 넘어갑니다.

```c
#include <stdio.h>

int main() {
    printf("--- break 예제 ---\n");
    for (int i = 1; i <= 10; i++) {
        if (i == 5) {
            printf("i가 5가 되어 반복문을 종료합니다.\n");
            break; // i가 5가 되면 반복문을 즉시 종료합니다.
        }
        printf("현재 숫자: %d\n", i);
    }

    printf("\n--- continue 예제 ---\n");
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) { // i가 짝수이면
            printf("짝수 %d는 건너뜁니다.\n", i);
            continue; // 아래 코드를 실행하지 않고 다음 반복으로 넘어갑니다.
        }
        printf("홀수: %d\n", i); // 홀수일 때만 이 줄이 실행됩니다.
    }

    return 0;
}
```

**실행 결과:**
```
--- break 예제 ---
현재 숫자: 1
현재 숫자: 2
현재 숫자: 3
현재 숫자: 4
i가 5가 되어 반복문을 종료합니다.

--- continue 예제 ---
홀수: 1
짝수 2는 건너뜁니다.
홀수: 3
짝수 4는 건너뜁니다.
홀수: 5
짝수 6는 건너뜁니다.
홀수: 7
짝수 8는 건너뜁니다.
홀수: 9
짝수 10는 건너뜁니다.
```

**설명:**
`break` 예제에서는 `i`가 5가 되는 순간, 그 뒤의 `printf("현재 숫자: %d\n", i);`는 실행되지 않고 `for` 문 전체가 종료됩니다.
`continue` 예제에서는 `i`가 짝수일 때, `continue`를 만나면 `printf("홀수: %d\n", i);`를 건너뛰고 다음 반복으로 바로 넘어가는 것을 볼 수 있습니다.
