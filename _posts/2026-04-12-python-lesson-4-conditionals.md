# 4강: 파이썬 기초: 조건문

안녕하세요! 이번 강의에서는 파이썬에서 가장 기본적인 문법 중 하나인 **조건문**에 대해 알아보겠습니다. 조건문을 통해 프로그램의 흐름을 제어하고 다양한 상황에 따라 적절한 동작을 할 수 있습니다. 초보자들이 쉽게 이해할 수 있도록 단계별로 설명드리겠습니다.

## 1. `if` 문

가장 기본적인 조건문인 `if` 문부터 살펴보겠습니다. `if` 문은 특정 조건이 참(True)일 때만 코드 블록을 실행합니다.

### 예제 코드
```python
# 숫자를 입력 받습니다.
number = 10

# if 문으로 조건을 검사합니다.
if number > 5:
    print("10은 5보다 큽니다.")
```

**실행 결과**
```
10은 5보다 큽니다.
```

## 2. `else` 문

`else` 문은 `if` 조건이 거짓(False)일 때 실행되는 코드 블록입니다.

### 예제 코드
```python
# 숫자를 입력 받습니다.
number = 3

# if-else 문으로 조건을 검사합니다.
if number > 5:
    print("3은 5보다 큽니다.")
else:
    print("3은 5보다 작거나 같습니다.")
```

**실행 결과**
```
3은 5보다 작거나 같습니다.
```

## 3. `elif` 문

여러 개의 조건을 검사할 때 사용하는 `elif` 문입니다. `elif`는 "else if"의 약자로, 여러 가지 조건 중 하나가 참일 때 해당 코드 블록이 실행됩니다.

### 예제 코드
```python
# 숫자를 입력 받습니다.
score = 85

# if-elif-else 문으로 조건을 검사합니다.
if score >= 90:
    print("A 등급입니다.")
elif score >= 80:
    print("B 등급입니다.")
elif score >= 70:
    print("C 등급입니다.")
else:
    print("D 등급입니다.")
```

**실행 결과**
```
B 등급입니다.
```

## 4. 중첩된 조건문

조건문 안에 다른 조건문을 사용할 수 있습니다. 이를 **중첩된 조건문**이라고 합니다.

### 예제 코드
```python
# 날씨와 옷 선택
weather = "비"
temperature = 20

# 중첩된 if 문으로 조건을 검사합니다.
if weather == "비":
    print("우산을 챙기세요.")
    if temperature > 15:
        print("가벼운 옷이 좋습니다.")
    else:
        print("두꺼운 옷이 필요합니다.")
else:
    print("우산을 안 챙기세요.")
```

**실행 결과**
```
우산을 챙기세요.
가벼운 옷이 좋습니다.
```

## 5. 논리 연산자

조건문에서 여러 조건을 함께 검사할 때 사용하는 **논리 연산자**(`and`, `or`, `not`)에 대해 알아보겠습니다.

### 예제 코드
```python
# 나이와 취업 여부
age = 25
employed = True

# 논리 연산자를 사용한 조건문
if age >= 18 and employed:
    print("투표권이 있습니다.")
else:
    print("투표권이 없습니다.")
```

**실행 결과**
```
투표권이 있습니다.
```
