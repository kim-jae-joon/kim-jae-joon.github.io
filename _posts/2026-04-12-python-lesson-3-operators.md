# 3강: 파이썬 기초: 연산자 🔥🚀

안녕하세요, 여러분! 이번에는 정말 대박인 파이썬의 기본부터 살펴볼게요. 이 강의에서는 우리가 일상 생활에서 무작정 사용하는 '연산자'라는 친구들을 하나씩 소개하고, 어떻게 쓰고 활용할 수 있는지 알려드리겠습니다. 🎉

## 1️⃣ 숫자 연산자 (산술 연산자) 😎

먼저 눈에 띄는 것이 바로 '숫자 연산자'입니다. 우리가 휴대폰이나 학교에서 배운 덧셈, 뺄셈, 곱셈, 나눗셈 같은 것들이죠.

### (1-1) 덧셈 (+)
두 수를 더합니다.
```python
# 3과 5를 더해보자!
result = 3 + 5
print(result) # 8이 출력됩니다!
```
**설명:** `+` 기호는 두 숫자를 더하는 역할을 합니다. 결과는 변수 `result`에 저장되며, `print()` 함수로 그 값을 확인할 수 있습니다.

### (1-2) 뺄셈 (-)
첫 번째 수에서 두 번째 수를 빼줍니다.
```python
# 10에서 3을 뺀 결과는?
diff = 10 - 3
print(diff) # 7이 나오겠죠?
```
**설명:** `-` 기호는 첫 번째 숫자에서 두 번째 숫자를 빼주는 역할을 합니다. 예를 들어, `10 - 3`은 `10`에서 `3`을 뺀 결과인 `7`이 나옵니다.

### (1-3) 곱셈 (*)
두 수를 곱합니다.
```python
# 4와 5를 곱해보세요!
product = 4 * 5
print(product) # 20이 출력됩니다!
```
**설명:** `*` 기호는 두 숫자를 곱하는 역할을 합니다. 예를 들어, `4 * 5`는 `4`와 `5`를 곱한 결과인 `20`이 나옵니다.

### (1-4) 나눗셈 (/)
첫 번째 수를 두 번째 수로 나눕니다.
```python
# 18을 3으로 나눠봅시다!
quotient = 18 / 3
print(quotient) # 6.0이 출력됩니다! 주의할 점은 이건 실수 결과입니다.
```
**설명:** `/` 기호는 첫 번째 숫자를 두 번째 숫자로 나누는 역할을 합니다. 예를 들어, `18 / 3`은 `18`을 `3`으로 나눈 결과인 `6.0`이 나옵니다. 주의할 점은 이 경우 항상 실수(`float`) 타입으로 반환됩니다.

## 2️⃣ 나머지 연산자 (나눗셈 연산자) 🤔

여기서 특별한 것을 하나 소개해 드리겠습니다! 우리가 흔히 '나머지'라고 부르는 것들이 바로 **나머지 연산자**입니다. 

### (2-1) 나머지 연산자 (%)
두 수를 나누고 그 나머지를 반환합니다.
```python
# 7을 3으로 나눈 나머지는?
remainder = 7 % 3
print(remainder) # 1이 출력됩니다!
```
**설명:** `%` 기호는 두 숫자를 나누었을 때의 나머지를 반환하는 역할을 합니다. 예를 들어, `7 % 3`은 `7`을 `3`으로 나눈 나머지인 `1`이 나옵니다.

### (2-2) 정수 나눗셈 연산자 (//)
두 수를 나누고 그 몫만 반환합니다.
```python
# 9를 4로 나눈 몫은?
quotient_int = 9 // 4
print(quotient_int) # 2가 출력됩니다!
```
**설명:** `//` 기호는 두 숫자를 나누었을 때의 정수 부분 (몫)만 반환하는 역할을 합니다. 예를 들어, `9 // 4`은 `9`을 `4`로 나눈 몫인 `2`가 나옵니다.

## 3️⃣ 비교 연산자 (비교 연산자) 🧐

다음으로는 **비교 연산자**입니다. 서로 다른 두 값이나 변수를 비교할 때 사용합니다.

### (3-1) 등호 (==)
두 값이 같은지 확인합니다.
```python
# 5와 5가 같나요?
is_equal = 5 == 5
print(is_equal) # True가 출력됩니다!
```
**설명:** `==` 기호는 두 값이 서로 같은지를 비교하고, 같으면 `True`, 다르면 `False`를 반환합니다.

### (3-2) 부등호 (!=)
두 값이 다른지 확인합니다.
```python
# 5와 10이 다르나요?
is_not_equal = 5 != 10
print(is_not_equal) # True가 출력됩니다!
```
**설명:** `!=` 기호는 두 값이 서로 다름지를 비교하고, 다르면 `True`, 같으면 `False`를 반환합니다.

### (3-3) 크기 비교 연산자 (>와 <)
두 값 중 큰 것과 작은 것을 판단합니다.
```python
# 7이 5보다 큰가요?
is_greater = 7 > 5
print(is_greater) # True가 출력됩니다!

# 4가 8보다 작나요?
is_less = 4 < 8
print(is_less) # True가 출력됩니다!
```
**설명:** `>`와 `<` 기호는 두 값을 비교하여, 왼쪽 값이 오른쪽 값보다 크면 `True`, 작으면 `False`를 반환합니다.

### (3-4) 크거나 같은 것, 작거나 같은 것 (>=와 <=)
두 값 중 큰 것과 작은 것, 같음도 판단합니다.
```python
# 6이 5보다 크거나 같은가요?
is_greater_or_equal = 6 >= 5
print(is_greater_or_equal) # True가 출력됩니다!

# 3이 10보다 작거나 같은가요?
is_less_or_equal = 3 <= 10
print(is_less_or_equal) # True가 출력됩니다!
```
**설명:** `>=`와 `<=` 기호는 두 값을 비교하여, 왼쪽 값이 오른쪽 값보다 크거나 같으면 `True`, 작거나 같으면 `False`를 반환합니다.

## 4️⃣ 논리 연산자 (논리 연산자) 💡

마지막으로 가장 중요하고 재미있는 **논리 연산자**를 소개해 드리겠습니다. 이 친구들은 여러 개의 조건을 하나로 합치거나, 서로 반대되는 값을 만들어 낼 수 있습니다.

### (4-1) AND 논리 연산자
두 가지 조건이 모두 참일 때만 `True`를 반환합니다.
```python
# 3이 5보다 작고 5가 7보다 작은가요?
is_and_true = (3 < 5) and (5 < 7)
print(is_and_true) # True가 출력됩니다!
```
**설명:** `and`는 두 개의 조건을 모두 만족해야 `True`를 반환합니다. 위 예제에서는 `3 < 5`와 `5 < 7`이 모두 참이기 때문에 최종적으로 `True`가 출력됩니다.

### (4-2) OR 논리 연산자
두 가지 조건 중 하나라도 참일 때 `True`를 반환합니다.
```python
# 10이 5보다 크거나, 3과 3이 같은가요?
is_or_true = (10 > 5) or (3 == 3)
print(is_or_true) # True가 출력됩니다!
```
**설명:** `or`는 두 개의 조건 중 하나라도 참이면 `True`를 반환합니다. 위 예제에서는 `10 > 5`가 참이기 때문에 최종적으로 `True`가 출력됩니다.

### (4-3) NOT 논리 연산자
조건을 반대로 바꿉니다.
```python
# 8이 2보다 큰 것을 부정해보세요!
is_not_true = not (8 > 2)
print(is_not_true) # False가 출력됩니다!
```
**설명:** `not`은 조건을 반대로 바꿉니다. 위 예제에서는 `8 > 2`가 참이지만, `not`을 붙여서 최종적으로 `False`가 출력됩니다.

## 💡 초보자 폭풍 질문!

- **질문:** 왜 나눗셈의 결과는 항상 실수(`float`) 타입인가요? 🤔
  - **답변:** 이는 파이썬의 기본 동작 방식 때문입니다. 나눗셈 연산자 `/`는 항상 정확한 소수점까지 계산하여 실수를 반환합니다. 만약 정수 결과만 원한다면, `//` 연산자를 사용하면 됩니다.

- **질문:** 논리 연산자의 순서가 중요한가요? 🤔
  - **답변:** 네, 그렇습니다! 예를 들어 `(A and B) or C`와 `A and (B or C)`는 결과가 다릅니다. 이런 경우 괄호를 사용하여 원하는 순서대로 계산할 수 있습니다.

## 🚨 실무주의보!

- **주의사항:** 나머지 연산자 `%`는 두 번째 숫자가 0인 경우 에러가 발생합니다. 항상 나누어지는 수가 0이 아님을 확인해야 합니다.
  
  ```python
  # 잘못된 예제 - 에러 발생!
  result = 10 % 0 # ZeroDivisionError: division by zero

  # 올바른 예제 - 조건 추가 필요
  divisor = 3
  if divisor != 0:
      result = 10 % divisor
      print(result) # 1이 출력됩니다!
  else:
      print("나누는 수가 0이면 안 됩니다!") 
  ```

- **고급 팁:** 논리 연산자를 사용할 때, 코드의 가독성을 높이기 위해 괄호를 적절히 사용하는 것이 중요합니다. 예를 들어 `(A and B) or C`보다는 `((A and B) or C)`와 같이 괄호로 묶어두면 더 명확하게 이해할 수 있습니다.
