**3강: 변수와 자료형 - 숫자, 문자열, 불 🔥**

**변수 📚**

변수란, 데이터를 저장할 수 있는 상징적인 이름입니다. 예를 들어, "나의 나이"라고 할 때, 이 변수에 30이라는 값을 저장한다면, 나의 나이 = 30이 됩니다. 💡

```python
my_age = 30
print(my_age)  # 출력: 30
```

위 코드에서 `my_age`는 변수 이름입니다. `=`은 할당 연산자로, 왼쪽 변수에 오른쪽 값을 저장합니다.

**자료형 📈**

자료형은 데이터가 어떤 형태를 가지는지 나타내는 것입니다. 파이썬에는 여러 자료형이 있습니다.

### 숫자 💯

숫자 자료형에는 정수(int)와 실수(float)가 있습니다.

```python
num1 = 10  # 정수
num2 = 3.14  # 실수
print(num1 + num2)  # 출력: 13.14
```

위 코드에서 `num1`은 정수, `num2`는 실수입니다. 두 값을 더한 결과는 13.14가 됩니다.

### 문자열 🔗

문자열 자료형은 문자를 저장하는 데 사용됩니다.

```python
name = "이름"
print(name)  # 출력: 이름
```

위 코드에서 `name`은 문자열입니다. 이 값을 `print()` 함수로 출력하면, "`이름`"이 됩니다.

### 불 👀

불 자료형은 True 또는 False를 나타냅니다.

```python
is_admin = True
print(is_admin)  # 출력: True
```

위 코드에서 `is_admin`은 불 자료형입니다. 이 값은 True가 됩니다.

**실무 활용 🚀**

변수와 자료형을 이해하면, 코드 작성이 훨씬 더 쉬워집니다! 예를 들어, 쇼핑몰에 상품의 가격과 이름을 저장할 때, 아래와 같이 변수와 자료형을 사용할 수 있습니다.

```python
price = 1000  # 정수
name = "스마트폰"  # 문자열

print(f"{name}의 가격은 {price}원입니다.")  # 출력: 스마트폰의 가격은 1000원입니다.
```

위 코드에서 `price`는 정수, `name`은 문자열 자료형입니다. 이 값을 `f-string`으로 출력하면, "`스마트폰의 가격은 1000원입니다.`"가 됩니다.
