## 🚀 3강: 흐름을 제어하라: 조건문과 반복문 마스터하기

프로그램은 기본적으로 위에서 아래로 한 줄씩 순서대로 실행됩니다. 하지만 실제 세상은 그렇게 단순하지 않죠? "만약 ~라면 이렇게 하고, 그렇지 않다면 저렇게 해"라든지, "이 작업을 10번 반복해"와 같은 복잡한 요구사항들이 많습니다. 이런 요구사항들을 해결해 주는 것이 바로 '흐름 제어'입니다.

크게 두 가지 갈래로 나눠서 살펴볼 거예요.

1.  **조건문 (Conditional Statements):** 프로그램이 '선택'하게 합니다. (예: `if`, `else if`, `else`, `switch`)
2.  **반복문 (Loop Statements):** 프로그램이 '반복'하게 합니다. (예: `while`, `for`, `do-while`)

준비되셨나요? 출발! 🚀

### 1. 조건문: 프로그램, 선택의 기로에 서다 🚦

조건문은 마치 우리 인생의 선택과 같아요. "비가 오면 우산을 쓰고, 안 오면 그냥 나간다" 처럼, 어떤 '조건'이 참(True)이냐 거짓(False)이냐에 따라 다른 코드를 실행하게 합니다.

C 언어에서 '참'은 0이 아닌 모든 값(주로 1)으로, '거짓'은 0으로 표현됩니다.

#### 1.1. `if` 문: 가장 기본적인 선택

가장 기본적인 조건문입니다. "만약 ~라면, 이렇게 해!" 와 같습니다.

**✨ 문법:**

```c
if (조건) {
    // 조건이 참일 때 실행할 코드
}
```

*   `조건` 부분에는 비교 연산자나 논리 연산자를 사용한 식이 들어갑니다. 이 식이 참(0이 아닌 값)이면 중괄호 `{}` 안의 코드가 실행됩니다.
*   만약 실행할 코드가 한 줄이라면 중괄호 `{}`는 생략할 수 있지만, 초보자 때는 명확하게 써주는 습관을 들이는 것을 추천합니다!

**💻 예제 코드: 합격 여부 확인하기**

```c
#include <stdio.h> // printf 함수를 사용하기 위해 포함합니다.

int main() {
    int score = 85;

    printf("점수: %d\n", score);

    // 만약 점수가 60점 이상이라면 합격입니다.
    if (score >= 60) {
        printf("축하합니다! 합격입니다.\n");
    }

    printf("프로그램 종료.\n");
    return 0; // 프로그램이 성공적으로 종료되었음을 나타냅니다.
}
```

**🔍 실행 결과:**

```
점수: 85
축하합니다! 합격입니다.
프로그램 종료.
```

`score`가 85이므로 `score >= 60` 조건은 참이 되어 "축하합니다! 합격입니다."가 출력됩니다. 만약 `score`를 50으로 바꾸면 해당 문구는 출력되지 않을 거예요.

#### 1.2. `if-else` 문: 양자택일의 순간

`if` 문이 "만약 ~라면"이라면, `if-else` 문은 "만약 ~라면 이렇게 하고, 그렇지 않다면 저렇게 해!" 와 같습니다. 둘 중 하나는 무조건 실행됩니다.

**✨ 문법:**

```c
if (조건) {
    // 조건이 참일 때 실행할 코드
} else {
    // 조건이 거짓일 때 실행할 코드
}
```

**💻 예제 코드: 성인 / 미성년자 판별**

```c
#include <stdio.h>

int main() {
    int age = 17;

    printf("나이: %d세\n", age);

    // 나이가 19세 이상이면 성인, 그렇지 않으면 미성년자
    if (age >= 19) {
        printf("당신은 성인입니다.\n");
    } else {
        printf("당신은 미성년자입니다.\n");
    }

    printf("프로그램 종료.\n");
    return 0;
}
```

**🔍 실행 결과:**

```
나이: 17세
당신은 미성년자입니다.
프로그램 종료.
```

`age`가 17이므로 `age >= 19` 조건은 거짓이 되어 `else` 블록 안의 코드가 실행됩니다.

#### 1.3. `if-else if-else` 문: 다양한 경우의 수 처리

여러 가지 조건을 순서대로 검사하고 싶을 때 사용합니다. "만약 첫 번째 조건이 참이면 이걸 하고, 그게 아니라 두 번째 조건이 참이면 저걸 하고, 그것도 아니라면 다른 걸 해!" 와 같습니다.

**✨ 문법:**

```c
if (조건1) {
    // 조건1이 참일 때 실행할 코드
} else if (조건2) {
    // 조건1이 거짓이고 조건2가 참일 때 실행할 코드
} else if (조건3) {
    // 조건1, 조건2가 거짓이고 조건3이 참일 때 실행할 코드
} else {
    // 위의 모든 조건이 거짓일 때 실행할 코드 (선택 사항)
}
```

*   `else if`는 여러 개 붙일 수 있습니다.
*   `else`는 맨 마지막에 선택적으로 붙여서, 위의 모든 조건이 거짓일 때 실행될 코드를 지정합니다.

**💻 예제 코드: 학점 계산기**

```c
#include <stdio.h>

int main() {
    int score; // 점수를 저장할 변수

    printf("점수를 입력하세요 (0-100): ");
    scanf("%d", &score); // 사용자로부터 점수를 입력받습니다.

    printf("입력된 점수: %d점\n", score);

    if (score >= 90) {
        printf("당신의 학점은 A입니다.\n");
    } else if (score >= 80) { // 90점 미만이면서 80점 이상일 때
        printf("당신의 학점은 B입니다.\n");
    } else if (score >= 70) { // 80점 미만이면서 70점 이상일 때
        printf("당신의 학점은 C입니다.\n");
    } else if (score >= 60) { // 70점 미만이면서 60점 이상일 때
        printf("당신의 학점은 D입니다.\n");
    } else { // 60점 미만일 때
        printf("당신의 학점은 F입니다.\n");
    }

    printf("학점 계산 프로그램 종료.\n");
    return 0;
}
```

**🔍 실행 결과 (예시):**

```
점수를 입력하세요 (0-100): 77
입력된 점수: 77점
당신의 학점은 C입니다.
학점 계산 프로그램 종료.
```

이 코드는 위에서부터 순서대로 조건을 검사하여, 하나의 조건이라도 참이 되면 해당 블록을 실행하고 나머지 `else if`나 `else` 블록은 건너뛰게 됩니다.

#### 1.4. `switch` 문: 깔끔한 다중 선택

`if-else if` 문은 여러 조건을 검사하는 데 유용하지만, 특정 변수의 '값'이 정확히 무엇이냐에 따라 다른 동작을 하고 싶을 때는 `switch` 문이 더 간결하고 가독성이 좋습니다.

**✨ 문법:**

```c
switch (변수 또는 식) {
    case 값1:
        // 변수/식의 결과가 값1과 같을 때 실행할 코드
        break; // break를 꼭 넣어주세요!
    case 값2:
        // 변수/식의 결과가 값2와 같을 때 실행할 코드
        break;
    // ... 더 많은 case
    default:
        // 어떤 case와도 일치하지 않을 때 실행할 코드 (선택 사항)
        break; // default에도 break를 넣어주는 것이 좋습니다.
}
```

*   `switch` 괄호 안에는 정수형 변수 또는 정수형으로 평가될 수 있는 식이 들어갑니다.
*   `case` 뒤에는 해당 변수가 가질 수 있는 '상수' 값이 옵니다.
*   `break` 문이 매우 중요합니다! `break`가 없으면 일치하는 `case` 이후의 코드들이 줄줄이 실행되는 '폴스루(fall-through)' 현상이 발생합니다.
*   `default`는 `else`와 같이, 어떤 `case`와도 일치하지 않을 때 실행됩니다.

**💻 예제 코드: 오늘의 운세 뽑기 (간단 버전)**

```c
#include <stdio.h>
#include <stdlib.h> // rand(), srand() 함수를 위해 포함
#include <time.h>   // time() 함수를 위해 포함

int main() {
    // 난수(랜덤 숫자) 생성 초기화
    srand(time(NULL)); 
    // 0부터 2까지의 랜덤 숫자를 생성 (0, 1, 2 중 하나)
    int fortune = rand() % 3; 

    printf("오늘의 운세는...!\n");

    switch (fortune) {
        case 0:
            printf("대길! 모든 일이 술술 풀릴 거예요!\n");
            break; // 여기서 switch 문을 빠져나옵니다.
        case 1:
            printf("길! 평범하지만 행복한 하루가 될 겁니다.\n");
            break;
        case 2:
            printf("소길! 작은 행운이 당신을 기다려요.\n");
            break;
        default: // 혹시 모를 상황을 대비하여 (여기서는 실행될 일 없음)
            printf("알 수 없는 운세입니다.\n");
            break;
    }

    printf("운세 프로그램 종료.\n");
    return 0;
}
```

**🔍 실행 결과 (예시, 실행할 때마다 달라짐):**

```
오늘의 운세는...!
길! 평범하지만 행복한 하루가 될 겁니다.
운세 프로그램 종료.
```

**🚨 `break`의 중요성:**

만약 `break`를 빼먹으면 어떻게 될까요?
`case 0:` 다음에 `break`가 없다면, `fortune`이 0일 때 "대길!"을 출력한 후에 바로 `case 1:`의 "길!" 마저 출력하게 됩니다. 이것을 **폴스루(fall-through)**라고 부르는데, 때로는 의도적으로 사용하기도 하지만 초보자에게는 실수하기 쉬운 부분이므로 항상 `break`를 잊지 마세요!

#### 1.5. `if-else if` vs `switch` 비교

| 특징          | `if-else if` 문                                   | `switch` 문                                           |
| :------------ | :------------------------------------------------ | :---------------------------------------------------- |
| **조건 유형** | 복잡한 조건식 (크다, 작다, 논리 연산 등)           | 특정 변수/식의 '정확한 값' 일치 여부                 |
| **변수 타입** | 모든 자료형 (정수, 실수, 문자열 등)               | 주로 정수형 (char, short, int, long)                  |
| **가독성**    | 조건식이 복잡해지면 가독성이 떨어질 수 있음        | 값에 따른 분기가 명확하여 가독성이 좋음              |
| **유연성**    | 매우 유연하며 다양한 조건 처리에 적합             | 특정 값 일치에만 사용 가능하여 유연성이 떨어짐       |
| **주요 사용처** | 범위, 복합 조건, 논리 판단이 필요한 경우          | 메뉴 선택, 요일 판별, 에러 코드 처리 등               |

### 2. 반복문: 프로그램, 지루함을 모르는 자동화 🔄

반복문은 특정 코드를 여러 번, 또는 어떤 조건이 만족될 때까지 계속해서 실행하게 만듭니다. 우리는 지루해서 못하는 반복 작업을 컴퓨터는 지치지 않고 정확히 해냅니다. 이것이 바로 프로그래밍의 강력함 중 하나죠!

#### 2.1. `while` 문: 조건이 참인 동안 계속

`while` 문은 "어떤 조건이 참인 동안, 이 코드를 계속 반복해!" 와 같습니다. 조건이 거짓이 되는 순간 반복을 멈춥니다.

**✨ 문법:**

```c
while (조건) {
    // 조건이 참인 동안 반복해서 실행할 코드
    // 이 안에 조건을 변화시켜 언젠가는 거짓이 되게 해야 합니다!
}
```

*   `조건`이 참일 때마다 중괄호 `{}` 안의 코드를 실행합니다.
*   가장 중요한 것은 반복문 내에서 `조건`이 언젠가는 거짓이 될 수 있도록 변화를 주어야 한다는 것입니다. 그렇지 않으면 **무한 루프(Infinite Loop)**에 빠지게 됩니다!

**💻 예제 코드: 1부터 5까지 세기**

```c
#include <stdio.h>

int main() {
    int count = 1; // 카운트 시작 값

    printf("while 문으로 1부터 5까지 셉니다:\n");

    while (count <= 5) { // count가 5 이하인 동안 반복
        printf("%d\n", count);
        count = count + 1; // count 값을 1 증가시킵니다. (조건을 변화시킴)
    }

    printf("반복 종료. 최종 count: %d\n", count);
    return 0;
}
```

**🔍 실행 결과:**

```
while 문으로 1부터 5까지 셉니다:
1
2
3
4
5
반복 종료. 최종 count: 6
```

`count`가 1부터 시작하여 5까지 출력되고, `count`가 6이 되는 순간 `count <= 5` 조건이 거짓이 되어 반복문이 종료됩니다.

#### 2.2. `do-while` 문: 최소 한 번은 실행

`while` 문과 비슷하지만, `do-while` 문은 무조건 한 번은 코드를 실행한 후에 조건을 검사하여 반복 여부를 결정합니다. "일단 한 번 해보고, 그 다음부터는 조건이 맞으면 계속 해!" 와 같습니다.

**✨ 문법:**

```c
do {
    // 최소 한 번은 실행되고, 조건이 참인 동안 반복해서 실행할 코드
} while (조건); // 세미콜론(;)을 잊지 마세요!
```

*   `while` 문과 달리 `do` 블록의 코드가 먼저 한 번 실행됩니다.
*   그 후에 `while` 괄호 안의 `조건`을 검사합니다. 조건이 참이면 다시 `do` 블록으로 돌아가 코드를 실행하고, 거짓이면 반복을 종료합니다.

**💻 예제 코드: 올바른 숫자 입력받기**

```c
#include <stdio.h>

int main() {
    int num;

    printf("do-while 문으로 양수만 입력받습니다.\n");

    do {
        printf("양수를 입력하세요: ");
        scanf("%d", &num); // 사용자로부터 숫자를 입력받습니다.

        if (num <= 0) {
            printf("잘못된 입력입니다. 양수를 다시 입력해주세요.\n");
        }
    } while (num <= 0); // num이 0보다 작거나 같으면 계속 반복

    printf("올바른 양수 %d를 입력하셨습니다.\n", num);
    printf("프로그램 종료.\n");
    return 0;
}
```

**🔍 실행 결과 (예시):**

```
do-while 문으로 양수만 입력받습니다.
양수를 입력하세요: -5
잘못된 입력입니다. 양수를 다시 입력해주세요.
양수를 입력하세요: 0
잘못된 입력입니다. 양수를 다시 입력해주세요.
양수를 입력하세요: 10
올바른 양수 10를 입력하셨습니다.
프로그램 종료.
```

이 예제에서는 사용자가 양수를 입력할 때까지 계속 숫자를 입력받습니다. `do` 블록 덕분에 적어도 한 번은 `scanf`가 실행되어 사용자에게 입력을 요구할 수 있습니다.

#### 2.3. `for` 문: 반복 횟수가 명확할 때 최고!

`for` 문은 반복 횟수가 명확할 때 가장 많이 사용되는 반복문입니다. 반복에 필요한 '초기화', '조건', '증감' 세 가지 요소를 한 줄에 명시하여 코드를 간결하게 만듭니다.

**✨ 문법:**

```c
for (초기화; 조건; 증감식) {
    // 조건이 참인 동안 반복해서 실행할 코드
}
```

*   **초기화 (Initialization):** 반복문이 시작될 때 딱 한 번 실행됩니다. 주로 반복 횟수를 세는 변수(카운터 변수)를 선언하고 초기화합니다.
*   **조건 (Condition):** 매 반복이 시작되기 전에 이 조건을 검사합니다. 조건이 참이면 중괄호 안의 코드를 실행하고, 거짓이면 반복을 종료합니다.
*   **증감식 (Increment/Decrement):** 중괄호 안의 코드가 한 번 실행될 때마다 이 식이 실행됩니다. 주로 카운터 변수의 값을 증감시켜 `조건`이 언젠가는 거짓이 되도록 만듭니다.

**💻 예제 코드: 1부터 10까지의 합 구하기**

```c
#include <stdio.h>

int main() {
    int sum = 0; // 합계를 저장할 변수
    int i;       // 반복문 카운터 변수

    printf("for 문으로 1부터 10까지의 합을 구합니다.\n");

    for (i = 1; i <= 10; i++) { // i는 1부터 시작해서 10까지 1씩 증가
        sum = sum + i; // sum에 i 값을 더합니다.
        printf("현재 i: %d, 현재까지의 합: %d\n", i, sum);
    }

    printf("1부터 10까지의 총합: %d\n", sum);
    printf("프로그램 종료.\n");
    return 0;
}
```

**🔍 실행 결과:**

```

현재 i: 8, 현재까지의 합: 36
```

`for` 문은 `while` 문으로도 구현할 수 있지만, 위와 같이 반복 횟수가 정해져 있거나 특정 범위 내에서 반복해야 할 때 가독성 좋고 깔끔하게 코드를 작성할 수 있게 해줍니다.

### 3. `break`와 `continue`: 반복문의 특별한 제어

조건문과 반복문을 배웠으니, 반복문 안에서 좀 더 세밀하게 흐름을 제어하는 두 가지 키워드 `break`와 `continue`도 알아볼까요?

#### 3.1. `break`: 반복문 즉시 탈출!

`break` 문은 반복문(`for`, `while`, `do-while`)이나 `switch` 문 안에서 사용되며, `break`를 만나는 순간 해당 반복문이나 `switch` 문을 즉시 종료하고 다음 코드를 실행합니다.

**💻 예제 코드: 특정 숫자 찾기**

```c
#include <stdio.h>

int main() {
    int i;
    int target = 7; // 찾고 싶은 숫자

    printf("1부터 10까지의 숫자 중 %d를 찾습니다.\n", target);

    for (i = 1; i <= 10; i++) {
        printf("%d ", i);
        if (i == target) {
            printf("\n%d를 찾았습니다! 반복을 멈춥니다.\n", target);
            break; // target을 찾았으므로 for 문을 즉시 종료
        }
    }

    printf("반복문 바깥 코드 실행.\n");
    return 0;
}
```

**🔍 실행 결과:**

```
1부터 10까지의 숫자 중 7를 찾습니다.
1 2 3 4 5 6 7 
7를 찾았습니다! 반복을 멈춥니다.
반복문 바깥 코드 실행.
```

`i`가 7이 되는 순간 `if (i == target)` 조건이 참이 되어 `break`가 실행되고, `for` 문이 즉시 종료됩니다. 만약 `break`가 없었다면 10까지 계속 숫자를 출력했을 거예요.

#### 3.2. `continue`: 이번 반복만 건너뛰고 다음으로!

`continue` 문은 반복문(`for`, `while`, `do-while`) 안에서 사용됩니다. `continue`를 만나는 순간, 현재 실행 중인 반복의 나머지 부분을 건너뛰고 바로 다음 반복(다음 조건 검사 및 증감식 실행)으로 넘어갑니다.

**💻 예제 코드: 짝수만 출력하기**

```c
#include <stdio.h>

int main() {
    int i;

    printf("1부터 10까지의 숫자 중 짝수만 출력합니다:\n");

    for (i = 1; i <= 10; i++) {
        if (i % 2 != 0) { // i가 홀수라면 (2로 나누어 나머지가 0이 아니라면)
            continue;    // 현재 반복을 건너뛰고 다음 i 값으로 넘어갑니다.
        }
        printf("%d\n", i); // 홀수일 경우 이 코드는 실행되지 않습니다.
    }

    printf("반복문 종료.\n");
    return 0;
}
```

**🔍 실행 결과:**

```
1부터 10까지의 숫자 중 짝수만 출력합니다:
2
4
6
8
10
반복문 종료.
```

`i`가 홀수일 때는 `if (i % 2 != 0)` 조건이 참이 되어 `continue`가 실행되고, `printf("%d\n", i);` 코드는 실행되지 않고 `for` 문의 다음 반복으로 넘어가 `i`가 1 증가합니다.
