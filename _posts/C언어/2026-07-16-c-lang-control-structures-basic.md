## 🚀 5강: C언어 기초 마스터! 🤯 제어 구조 - if, for, while 문 🌌

안녕하세요, 코딩 탐험가 여러분! 오늘은 C언어의 핵심 무기, **제어 구조**에 대해 짜릿한 여정을 떠나볼게요!  만약 코딩을 게임처럼 생각한다면, 제어 구조는 레벨 업을 위한 **마력의 주문서**라고 할 수 있어요! 🧙‍♂️

**그럼, 마법 주문을 하나씩 풀어보도록 하죠!**

### 🧙‍♂️ 조건문: if, if-else 문 - 선택의 마법! ✨

**"만약 ~ 이라면, 이런 행동을 해라!"** 이보다 더 시원한 주문이 있을까요? 바로 **조건문**이죠! 코드가 특정 조건을 충족할 때만 특정 행동을 수행하도록 만드는 마법입니다.

**예시 1: 간단한 숫자 비교**

```c
#include <stdio.h>

int main() {
    int score = 85; // 학생 점수 예시

    // 만약 점수가 90점 이상이라면, "장학금 수혜!" 출력
    if (score >= 90) {
        printf("장학금 수혜!\n");
    } else { // 점수가 90점 미만이면
        printf("축하합니다! 합격했어요!\n"); 
    }
    return 0;
}
```

**코드 분석 🧙**

1. `int score = 85;` : 학생 점수를 변수 `score`에 저장합니다. 마치 마법사가 주문을 펼치기 전에 준비하는 것처럼요!
2. `if (score >= 90)` : 점수가 90점 이상인지 확인하는 마법 주문! 괄호 안 조건이 참이면 그 안 코드가 실행됩니다.
3. `printf("장학금 수혜!\n");` : 조건이 참이면 "장학금 수혜!"라는 메시지를 화면에 출력합니다. 마치 마법의 힘이 작동하는 것처럼 보이죠?
4. `else { printf("축하합니다! 합격했어요!\n"); }` : 조건이 거짓이면 (점수가 90점 미만이면) 다른 메시지를 출력합니다. 마법의 선택!

**💡 초보자 폭풍 질문!**

- 만약 여러 조건을 동시에 확인해야 할 때는 어떻게 할까요? 🤔

**답변:** 여러 조건을 확인하려면 `if-else if-else` 구조를 사용할 수 있어요! 각 조건을 순차적으로 확인하고 맞는 조건의 코드를 실행합니다.

**예시 2: 나이별 등급 결정**

```c
#include <stdio.h>

int main() {
    int age = 22; // 사용자 나이 입력

    if (age < 18) {
        printf("청소년 등급!\n");
    } else if (age >= 18 && age < 60) {
        printf("성인 등급!\n");
    } else { // age >= 60
        printf("Senior 등급!\n");
    }
    return 0;
}
```

### ⚙️ 반복문: for, while, do-while 문 - 무한 반복의 힘! 🔁

반복문은 마치 **마법의 주문을 반복해서 부르는 것**과 같아요! 특정 작업을 여러 번 수행할 때 필수적인 주문이죠!

**예시 3: for 문 - 정확한 반복 횟수!**

```c
#include <stdio.h>

int main() {
    for (int i = 0; i < 5; i++) { // i는 0부터 시작해서 5 미만까지 증가
        printf("안녕하세요! %d번째 인사!\n", i);
    }
    return 0;
}
```

**코드 분석 🧙**

1. `for (int i = 0; i < 5; i++)`: 마법 주문 시작! 
    - `int i = 0;`: 마법의 변수 `i`를 0으로 초기화합니다. 마치 마법의 시작점!
    - `i < 5;`:  `i`가 5보다 작은 동안만 주문을 반복합니다.
    - `i++`: 각 반복 후 `i` 값을 1 증가시켜 다음 반복을 준비합니다. 마치 마법의 힘을 모으는 것처럼!
2. `printf("안녕하세요! %d번째 인사!\n", i);`: 각 반복마다 인사 메시지를 출력합니다.

**💡 실무주의보**

- `for` 문은 반복 횟수가 미리 알려져 있을 때 유용합니다. 예를 들어, 배열의 모든 요소를 처리할 때 유용합니다.

**예시 4: while 문 - 조건 만족까지 계속!**

```c
#include <stdio.h>

int main() {
    int count = 0;
    while (count < 3) { // count가 3 미만일 동안 계속 반복
        printf("카운트: %d\n", count);
        count++; // 조건을 만족시키기 위해 count 증가
    }
    return 0;
}
```

**코드 분석 🧙**

1. `int count = 0;`: 마법의 변수 `count`를 초기화합니다. 마치 주문을 시작하기 전 준비 자세!
2. `while (count < 3)`: `count`가 3보다 작은 동안만 주문을 반복합니다. 마치 조건이 풀릴 때까지 마법을 계속하는 것처럼!
3. `printf("카운트: %d\n", count);`: 각 반복마다 현재 `count` 값을 출력합니다.
4. `count++;`: 각 반복 후 `count`를 1 증가시켜 조건을 만족시키도록 합니다. 마법 주문의 힘을 모으는 것!

**예시 5: do-while 문 - 최소한 한 번 실행!**

```c
#include <stdio.h>

int main() {
    int guess = 5;
    int secretNumber = 7;

    do { // 최소한 한 번 실행!
        printf("비밀 숫자를 맞춰보세요! (입력: %d)\n", guess);
        // 사용자 입력 처리 (여기서는 예시로 guess 값을 변경하지 않음)
        guess++; // 다음 추측을 위해 guess 증가
    } while (guess < secretNumber); // guess가 secretNumber보다 작으면 계속 반복

    printf("정답! 비밀 숫자는 %d였습니다.\n", secretNumber);
    return 0;
}
```

**코드 분석 🧙**

1. `do { ... } while (condition);`:  `do-while`은 최소한 한 번은 코드 블록을 실행하고, 조건이 참이면 다시 반복합니다. 마치 게임에서 첫 시도는 무조건 성공하는 마법 같죠?
