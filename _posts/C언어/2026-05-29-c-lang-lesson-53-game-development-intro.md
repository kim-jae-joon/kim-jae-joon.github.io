### 💻 초기 프로그래밍: "동전줍기 게임" 시작

우리의 "동전줍기 게임"은 아주 간단한 구조로 만들어 보겠습니다.


####  1️⃣ C 프로그램 구성

C 프로그램은 일반적으로 다음과 같은 구조를 가지고 있습니다. 😎

```c
#include <stdio.h> // 입력/출력 함수 활용을 위한 헤더파일 

int main() { // 프로그램 시작 지점
    printf("동전줍기 게임 시작!\n"); // 화면에 메시지 출력
    // 나머지 코드는 여기에 작성!
    return 0; // 프로그램 종료 시 알림
}
```

* **#include <stdio.h>**:  프로그램이 입력과 출력을 처리하기 위한 함수들을 사용할 수 있도록 허용하는 헤더 파일입니다. 💻

* **int main()**:  C 프로그램의 실행 시작 지점인 `main` 함수입니다. 모든 프로그램은 이 함수에서부터 시작됩니다! 🚀

* **printf("동전줍기 게임 시작!\n");**:  화면에 "동전줍기 게임 시작!"라는 메시지를 출력하는 코드입니다. 📢

* **return 0;**: 프로그램이 정상적으로 종료되었음을 알리는 문장입니다. 👍

#### 2️⃣ 동전의 위치 표현

```c
int main() {
    printf("동전줍기 게임 시작!\n");
    int coin_location = 5; // 동전은 처음부터 5번째 위치에 있습니다!
    // 나머지 코드는 여기에 작성!
    return 0; 
}
```

* **int coin_location = 5;**: `coin_location` 변수를 선언하고, 게임 내 동전의 위치를 나타내기 위해 "5"라는 값을 저장합니다.  🪙

#### 3️⃣ 플레이어 입력 받기

```c
int main() {
    printf("동전줍기 게임 시작!\n");
    int coin_location = 5;
    int player_guess;
    printf("행운을 빌게요! 동전의 위치를 맞혀 보세요 (1~10): ");
    scanf("%d", &player_guess); // 사용자 입력 받기!
    // 나머지 코드는 여기에 작성!
    return 0; 
}
```

* **int player_guess**: `player_guess` 변수를 선언하여 사용자가 입력한 동전 위치를 저장할 공간을 만듭니다. 🧠

* **scanf("%d", &player_guess);**:  사용자의 입력 값을 받아 `player_guess` 변수에 저장합니다. %d는 정수형 데이터를 의미합니다.



**💡 초보자 폭풍 질문!**: 
"scanf 함수가 무엇인지 알고 계신가요? 🤔  다음 강좌에서는 사용자 입력을 처리하는 방법에 대해 자세히 알아보겠습니다."

🚨 **실무주의보:** `scanf()` 함수는 주의해서 사용해야 합니다. 사용자가 문자를 입력하면 프로그램이 오류를 발생시킬 수 있습니다! 👀



#### 4️⃣ 결과 출력

```c
int main() {
    printf("동전줍기 게임 시작!\n");
    int coin_location = 5;
    int player_guess;
    printf("행운을 빌게요! 동전의 위치를 맞혀 보세요 (1~10): ");
    scanf("%d", &player_guess); 

    if (coin_location == player_guess) {
        printf("축하합니다! 당신은 동전을 찾았습니다!\n"); // 동전이 찾으면 성공 메시지 출력
    } else {
        printf("아쉽네요! 다음에 다시 시도해 보세요.\n"); // 동전을 못 찾으면 실패 메시지 출력
    }

    return 0; 
}
```


* **if (coin_location == player_guess)** : 사용자가 입력한 위치와 실제 동전의 위치가 같은지 확인하는 조건문입니다.

* **printf("축하합니다! 당신은 동전을 찾았습니다!\n");**  동전을 맞추면 성공 메시지를 출력합니다. 🎉
* **printf("아쉽네요! 다음에 다시 시도해 보세요.\n");**: 동전을 맞추지 못하면 실패 메시지를 출력합니다. 😥
