## 5강: C언어 기초 마스터! 🎨 **함수 작성 & 호출: 코딩의 마법고리를 다뤄보자!**

안녕하세요, 코딩 모험가 여러분! 오늘은 🌟 **C언어의 핵심 무기 중 하나인 함수**를 향해 짜릿한 여정을 떠나볼게요! 함수라는 건 마치 코딩 세계의 마법사 같아요. 복잡한 작업을 깔끔하게 분해해서 처리해주는 능력이 있거든요. 처음엔 "이게 왜 필요할까?" 싶겠지만, 믿으세요! 함수 없이는 제대로 된 소프트웨어를 만드는 게 거의 불가능해요.

### 🤔 함수, 왜 필요할까?

상상해봐요. 거대한 건물을 짓는다고 가정해봅시다. 각 방을 개별적으로 설계하고 따로 따로 시공해야 한다면, 굉장히 복잡하고 시간이 오래 걸리겠죠? 하지만 건축가가 각 방을 미리 디자인한 **모듈**을 만들어두면, 그 모듈들을 조립하듯이 쉽게 건물을 완성할 수 있어요!

**C언어에서 함수도 똑같아요!** 특정 작업을 수행하는 **모듈** 같은 역할을 해요. 반복되는 코드를 함수로 만들면 코드의 재사용성과 가독성을 훨씬 높일 수 있어요. 마치 레고 블록처럼!

### 💡 함수 작성하기: 기본 레시피

함수를 작성하는 방법을 쉽게 이해할 수 있도록 단계별로 살펴볼게요.

#### 1. 함수 선언 (Declaration)

함수를 처음 사용하기 전에 **함수 선언**이 필요해요. 이건 마치 메뉴를 미리 보는 것과 같아요!

```c
// 정수를 입력받아 제곱값을 반환하는 함수 선언
int square(int num);  // 타입: int, 이름: square, 매개변수: int num
```

- **`int`**: 함수가 반환하는 값의 타입을 말해요. 여기서는 정수를 반환하죠.
- **`square`**: 함수의 이름이에요. 직관적으로 이해할 수 있게 지어주는 게 좋아요.
- **`int num`**: 함수에 전달되는 입력 매개변수예요. 여기서는 정수 `num`을 받을 예정이에요.

#### 2. 함수 정의 (Definition)

실제로 함수의 동작을 정의하는 부분이에요. 선언 뒤에 따라오는 코드 블록이죠!

```c
// 함수 정의
int square(int num) {  // 함수 이름과 매개변수 일치
    int result = num * num;  // 계산: 제곱
    return result;  // 결과 반환
}
```

- **`int square(int num) { ... }`**: 함수 정의 블록 시작! 이름과 매개변수가 일치해야 해요.
- **`int result = num * num;`**: 여기서 실제로 작업이 일어나요. 숫자를 제곱하는 거죠.
- **`return result;`**: 계산 결과를 호출한 곳으로 되돌려 보내요.

### 💡 함수 호출: 마법의 주문

함수를 작성했으니 이제 호출해서 사용해봅시다! 함수 호출은 마치 마법사가 주문을 외우는 것 같아요.

```c
#include <stdio.h>

int main() {
    int number = 5;  // 사용할 숫자 지정
    int squared = square(number);  // 함수 호출: 제곱 계산
    printf("%d의 제곱은 %d입니다.\n", number, squared);  // 결과 출력
    return 0;
}
```

- **`int squared = square(number);`**: `square` 함수를 호출하고 결과를 `squared` 변수에 저장해요.
- **`printf`**: 계산 결과를 화면에 보여주는 역할을 해요. 마치 마법처럼 결과가 나타나는 거죠!

### 🏆 다양한 함수 타입: 실전에서 활용하기

함수의 힘을 더 잘 이해하기 위해 몇 가지 다른 타입의 함수 구조를 살펴볼게요!

#### 반복문과 함수의 만남: 반복 작업 단순화

반복 작업을 함수로 묶어보세요. 예를 들어, 숫자 목록의 합계를 구하는 함수를 만들어볼게요.

```c
// 숫자 리스트의 합계를 반환하는 함수
int sumList(int numbers[], int size) {
    int total = 0;  // 합계 초기화
    for (int i = 0; i < size; i++) {  // for문으로 반복
        total += numbers[i];  // 각 숫자 더하기
    }
    return total;  // 합계 반환
}

int main() {
    int numbers[] = {1, 2, 3, 4, 5};  // 숫자 배열
    int nSize = sizeof(numbers) / sizeof(numbers[0]);  // 배열 크기 계산
    int total = sumList(numbers, nSize);  // 함수 호출
    printf("합계는 %d입니다.\n", total);
    return 0;
}
```

- **`for`문**: 배열의 각 요소를 차례대로 처리해요. 반복 작업을 간결하게 처리할 수 있죠.
- **`sizeof`**: 배열의 크기를 계산하는 데 유용해요.

#### 조건에 따른 동작: `if` 문과 함수 결합

조건에 따라 다른 동작을 수행하는 함수도 만들어볼게요.

```c
// 나이에 따른 인사말을 반환하는 함수
void greet(int age) {
    if (age < 19) {
        printf("청소년이시네요! 🎓");  // 청소년 인사말
    } else if (age >= 19 && age <= 60) {
        printf("성인이시네요! 👨‍💼");  // 성인 인사말
    } else {
    printf("어르신이시네요! 👵👴");  // 어르신 인사말
    }
}

int main() {
    int userAge = 25;  // 사용자 나이 설정
    greet(userAge);  // 함수 호출
    return 0;
}
```

- **`if-else` 문**: 다양한 조건에 따라 다른 동작을 수행해요. 사용자 경험을 더욱 풍부하게 만들어줘요.

### 🚨 실무주의보: 함수 작성 시 주의사항

1. **명확한 이름**: 함수 이름은 그 역할을 명확히 나타내야 해요. `"calculateSomething"` 보다 `"calculateUserAge"`가 훨씬 좋죠!
2. **매개변수 관리**: 매개변수의 타입과 개수는 신중하게 결정해야 해요. 잘못된 타입은 오류의 원인이 될 수 있어요.
3. **리턴 타입 일관성**: 함수가 반환하는 값의 타입은 일관성을 유지하세요. 예를 들어, 모든 함수가 정수를 반환하도록 하거나 명확한 타입을 지정하세요.

### 💡 초보자 폭풍 질문! 🤔

**Q1:** 함수 내부에서 `main()` 함수로 값을 어떻게 전달할 수 있을까요?
- **A1:** 매개변수를 통해 전달해요! 함수 호출 시 인자를 전달하면 `main()`에서 그 값을 사용할 수 있어요. 예를 들어, `square(5)`에서 `5`가 매개변수로 전달되죠.

**Q2:** 함수를 너무 많이 만들면 코드가 복잡해지지 않나요?
- **A2:** 적당한 수준의 함수 분할은 오히려 코드의 가독성을 높여줘요. 너무 작은 단위로 나누면 오히려 복잡해질 수 있으니, 기능별로 적절히 묶는 것이 중요해요.
