# 1강: Rust C 소개 및 설치 가이드 - 초보자도 쉽게! 🤓✨

안녕하세요, 멋진 코딩 여정을 함께할 친구들! 오늘은 여러분의 새로운 코딩 파트너, **Rust**에 대해 이야기해볼게요. 저는 5년 차 Rust 개발자로서, 이 언어가 얼마나 멋진지 그리고 왜 지금 배우는 게 딱 맞는지 알려드리려고 해요. 준비됐나요? 그럼 출발해볼게요!

## Rust란 무엇인가요? 🤯

Rust는 시스템 프로그래밍에서 빛을 발하는 현대적인 프로그래밍 언어예요. 마치 슈퍼히어로처럼, Rust는 안정성과 성능을 동시에 잡는 데 특화되어 있죠. 특히 메모리 관리가 자동화되어 있어 초보자들도 메모리 누수나 버퍼 오버플로우 같은 문제로 고생할 필요가 없어요! 이걸 이해하기 쉽게 비유하자면, Rust는 **로봇 청소기** 같아요. 여러분의 집(프로젝트)을 깨끗하게 청소하면서도 안전하게 작동하게 돕는 거죠.

### Rust의 핵심 특징들
- **소유권 시스템**: 메모리 안전성을 보장하는 강력한 메커니즘이에요. 마치 장난감 창고를 관리하는 것처럼요!
- **성능**: C나 C++ 수준의 성능을 자랑해요. 빠른 게임 엔진이나 임베디드 시스템에서 딱이죠!
- **소유권과 라이프타임**: 복잡해 보이지만, 실제로는 안전한 코딩을 돕는 훌륭한 도구예요.

## Rust 설치하기: 초보자 가이드 🛠️

Rust를 설치하는 건 생각보다 간단해요. 아래 단계를 따라 해보세요!

### 1. 운영 체제 확인
먼저, 사용 중인 운영 체제가 무엇인지 확인해보세요:
- **Windows**, **macOS**, **Linux** 중 하나일 거예요.

### 2. Rust 설치 도구 다운로드
**[Rust 공식 웹사이트](https://www.rust-lang.org/tools/install)**로 이동해서 **"Install Rust using Rustup"** 옵션을 선택하세요. 이 과정은 Rustup이라는 설치 도구를 통해 이루어지는데, 이 도구 덕분에 최신 버전의 Rust를 쉽게 관리할 수 있어요.

#### 단계별 가이드:
1. **터미널 또는 명령 프롬프트 열기**:
   - **Windows**: `Win + R`, `cmd` 입력 후 엔터
   - **macOS/Linux**: 터미널 앱 열기

2. **Rustup 설치 스크립트 실행**:
   ```bash
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```
   - 이 스크립트를 실행하면 Rustup이 설치되며, Rust와 Cargo(Rust의 패키지 매니저)가 함께 설치됩니다.

3. **설치 완료 확인**:
   ```bash
   rustc --version
   cargo --version
   ```
   위 명령어를 실행하면 설치가 성공적으로 이루어졌는지 확인할 수 있어요. 버전 정보가 표시되면 설치 완료!

### 코드 예제: 기본 Rust 프로그램

#### 1. `main.rs` 파일 생성 및 작성
```rust
// 파일 이름: main.rs
fn main() {
    // 이 부분이 프로그램의 시작점이에요!
    println!("Hello, Rust World!"); // 여러분의 이름을 넣어보세요!
}
```
- **설명**:
  - `fn main() { ... }`: 프로그램의 진입점 역할을 해요. 마치 집의 대문처럼요!
  - `println!("메시지")`: 화면에 메시지를 출력해요. 마치 친구에게 편지를 쓰는 것 같죠!

#### 2. 컴파일 및 실행
터미널에서 아래 명령을 실행하세요:
```bash
cargo new hello_rust --bin
cd hello_rust
# main.rs 파일을 위의 코드로 바꿔주세요
# 다시 컴파일
cargo build --release
# 실행
cargo run
```
- **설명**:
  - `cargo new hello_rust --bin`: 새로운 프로젝트 생성
  - `cd hello_rust`: 프로젝트 디렉토리로 이동
  - `cargo build --release`: 최적화된 버전으로 컴파일
  - `cargo run`: 컴파일된 프로그램 실행

### 다양한 제어 구조 살펴보기 🧩

#### 반복문: `for`, `while`, `do-while`
반복문은 마치 매일 반복되는 루틴처럼 생각하면 돼요!

##### 1. `for` 문 예제
```rust
// 파일 이름: numbers.rs
fn main() {
    for i in 0..5 { // 0부터 4까지 반복
        println!("현재 숫자: {}", i);
    }
}
```
- **설명**:
  - `for i in 0..5`: `i`가 0부터 4까지 증가하면서 반복
  - `println!`: 각 반복마다 숫자 출력

##### 2. `while` 문 예제
```rust
// 파일 이름: countdown.rs
fn main() {
    countdown: loop { // 무한 루프 시작
        println!("카운트다운: 5");
        let n = 5;
        let mut i = n;
        while i > 0 {
            i -= 1;
            println!("{}", i);
            if i == 0 { break countdown; } // 루프 종료 조건
        }
        println!("끝!");
        break countdown; // 루프 종료
    }
}
```
- **설명**:
  - `loop {}`: 무한 루프 생성
  - `while i > 0`: 조건에 따라 반복
  - `break countdown;`: 특정 조건에서 루프 탈출

##### 3. Rust에서 `do-while` 구조는 직접 지원하지 않지만, `while` 문으로 구현 가능
```rust
// 파일 이름: do_while_like.rs
fn main() {
    let mut i = 0;
    loop {
        println!("현재 숫자: {}", i);
        i += 1;
        if i >= 5 { break; } // 최소 한 번 실행 후 조건 검사
    }
}
```
- **설명**:
  - `loop {}`: 무한 루프 시작 후 조건 검사
  - `if 조건 { break }`: 조건 만족 시 루프 탈출
