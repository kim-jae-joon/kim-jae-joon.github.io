반복문은 크게 **for**, **while**, **do-while** 세 가지 유형으로 나뉩니다. 각각의 특징과 사용법을 자세히 알아보고, 이 마법 같은 기술을 활용하여 코드를 간결하고 효율적으로 작성하는 법을 배워봅시다! 💻

### 1. for 반복문: "정해진 횟수만큼 반복" 🚀

`for` 루프는 정해진 횟수만큼 코드 블럭이 실행되는 반복문입니다. 마치 책갈피를 이용하여 특정 페이지부터 특정 페이지까지 텍스트를 읽는 것과 같습니다.

**💡 'for' 루프의 구조:**
```c
for (초기화; 조건식; 증감식) {
    // 반복문 실행 코드 블럭
}
```


**> 각 부분은 무엇을 의미할까요? 🤔** 
- **초기화**: 루프가 시작하기 전에 한 번만 실행되는 초기화 문장입니다. 변수를 설정하는 역할이죠.  🔥예시: `int i = 0;` 
- **조건식**: 루프가 계속 실행될지 여부를 결정하는 조건문입니다. `참`일 경우 루프는 계속되고, `거짓`이면 종료됩니다.  🔑예시: `i < 10;` - 변수 `i` 가 10보다 작으면 루프 실행!
- **증감식**: 각 반복문마다 실행되는 코드 블럭 이후에 수행되는 문장입니다. 일반적으로 변수 값을 수정하는 역할을 합니다. 🚀예시: `i++;` - 변수 `i` 를 1씩 증가시킵니다.

**🚨 실무주의보**: 조건식이 항상 참이거나 거짓으로만 구성된다면 루프는 무한히 돌고 프로그램이 계속 실행되어 문제 발생할 수 있습니다! (예: `for (int i = 0; ; ) { ... }`)



**⚡️ 코드 예제:**

```c
#include <stdio.h>

int main() {
    for (int i = 0; i < 5; i++) {   // 초기화: i = 0, 조건식: i < 5, 증감식: i++
        printf("반복 %d번째!\n", i); // 반복문 실행 코드 블럭 - 숫자 출력!
    }

    return 0;
}
```


### 2. while 반복문: "조건이 참인 동안 반복" 🚀

`while` 루프는 특정 조건식이 참일 때만 코드 블럭을 실행하는 반복문입니다. 마치 원탁에 자리잡은 사람들이 이야기를 나눌 때, 누군가 말할 때까지 기다리고 그 순간부터 이야기 하는 것과 같습니다. 🗣️

**💡 'while' 루프의 구조:**
```c
while (조건식) {
    // 반복문 실행 코드 블럭
}
```



**> 조건식이 항상 참으로 설정되어 있으면 무한 루프가 발생할 수 있습니다. 주의하세요!⚠️**


**⚡️ 코드 예제:**

```c
#include <stdio.h>

int main() {
    int number = 0;

    while (number < 5) {  // 조건식: number < 5
        printf("숫자 %d 입니다.\n", number); // 반복문 실행 코드 블럭 - 숫자 출력!
        number++;      // 증감식 - 변수 `number` 를 1씩 증가시킴.
    }

    return 0;
}
```


### 3. do-while 반복문: "한 번은 반드시 실행, 조건에 따라 계속" 🚀🚀🚀

`do-while` 루프는 조건식이 참인 동안 코드 블럭을 실행하는 반복문이며, **반드시 한 번은 코드 블럭을 실행**합니다. 마치 첫 번째 이야기 🗣️를 하지 않고 "원탁에 자리잡은 사람들이 이야기를 나눌 때" 와 같은 상황과 다르게,  '무엇이든 한번은 해보자!' 라는 용감한 도전이죠!

**💡 'do-while' 루프의 구조:**
```c
do {
    // 반복문 실행 코드 블럭
} while (조건식); 
```


 **⚡️ 코드 예제:**

```c
#include <stdio.h>

int main() {
    int number = 0;

    do {
        printf("숫자 %d 입니다.\n", number); // 반복문 실행 코드 블럭 - 숫자 출력!
        number++;      // 증감식 - 변수 `number` 를 1씩 증가시킴.
    } while (number < 5);  // 조건식: number < 5

    return 0;
}
```
