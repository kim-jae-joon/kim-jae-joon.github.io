**8강: 함수 선언과 호출 - 프로그램의 재사용성 🔥**

안녕하세요! 여러분, 오늘은 아주 특별한 주제를 다루게 됩니다. 바로 **함수 선언과 호출**입니다! 😎 이 부분을 이해하기 위해서는, 왜 우리가 이러한 기능을 사용해야 하는지, 그리고 어떻게 이것이 프로그램의 재사용성을 높이는지에 대해 알아야 합니다.

### 함수 선언과 호출이 뭐죠?

함수 선언은 **어떤 기능을 수행하는 코드 블록**을 정의합니다. 예를 들어, "정수를 더하기"라는 기능을 수행하는 코드 블록을 만들면 그것을 하나의 함수라고 부릅니다. 함수 선언을 통해 우리는 한 번에 프로그램을 구성할 수 있습니다. 📝

함수 호출은 **이미 작성된 함수를 다시 사용**해서 특정 기능을 수행합니다. 예를 들어, "정수를 더하기"라는 함수가 이미 정의되어 있다면, 이 함수를 호출하면 "정수가 있는지" 파악하고, 그 다음에 2 + 3 이라는 연산을 수행한다고 할 수 있습니다.

### 함수 선언의 필요성

함수 선언이 필요한 이유는, **코드의 재사용성**을 높이기 위해서입니다. 예를 들어, "정수를 더하기"라는 기능은 다른 프로그램에서도 사용할 수 있습니다. 만약 코드가 길다면, 이는 매우 유용하게 사용될 것입니다.

### 함수 선언과 호출 예제 1

```c
#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int main() {
    printf("%d\n", add(2, 3));
    return 0;
}
```

해당 코드는 다음과 같이 작동합니다.

1. `add` 함수가 선언됩니다. 이 함수는 두 개의 인자 `a`와 `b`를 받고, 그 둘을 더한 결과를 반환합니다.
2. `main` 함수에서 `add` 함수를 호출합니다. 인자는 2와 3입니다.
3. `printf` 함수를 사용하여 결과가 출력됩니다.

### 함수 선언과 호출 예제 2

```c
#include <stdio.h>

int max(int a, int b) {
    if (a > b)
        return a;
    else
        return b;
}

int main() {
    printf("%d\n", max(3, 5));
    return 0;
}
```

해당 코드는 다음과 같이 작동합니다.

1. `max` 함수가 선언됩니다. 이 함수는 두 개의 인자 `a`와 `b`를 받고, 그 중 더 큰 값을 반환합니다.
2. `main` 함수에서 `max` 함수를 호출합니다. 인자는 3과 5입니다.
3. `printf` 함수를 사용하여 결과가 출력됩니다.

### 실무 활용

함수 선언과 호출은 매우 많은 분야에서 사용됩니다. 예를 들어, **데이터베이스**를 관리할 때, 데이터베이스에 대한 기능을 수행하는 함수를 정의하고 호출합니다.

```c
#include <sqlite3.h>

void connectDB() {
    // DB 연결 코드
}

void selectData() {
    // SELECT 문으로 데이터 조회 코드
}

int main() {
    connectDB();
    selectData();
    return 0;
}
```

### 초보자 폭풍 질문!

*   함수 선언과 호출을 사용했을 때, 프로그램의 크기가 줄어듭니다.
*   함수를 여러 번 호출하면 그 다음에 수행되는 코드가 반복적으로 실행됩니다.

#### 🚨 실무주의보

*   함수 선언과 호출은 매우 중요한 개념입니다. 하지만, **함수 이름**이 잘못되면 프로그램의 성능이 저하될 수 있습니다.
*   따라서, 함수 이름을 항상 **자연어로 정의**하는 것이 중요합니다.

### 요약

함수 선언과 호출은 프로그램의 재사용성을 높이고, 코드의 크기를 줄입니다. 우리는 함수를 여러 번 호출하면 그 다음에 수행되는 코드가 반복적으로 실행됩니다. 하지만, 함수 이름을 잘못 정의하면 프로그램의 성능이 저하될 수 있습니다.
