# 2강: C언어 기초 - 기본 데이터 타입 소개: 코드로 배우는 C 언어의 매력!

## 기본 데이터 타입이란?

기본 데이터 타입이란 프로그래밍에서 자주 사용되는 가장 기본적인 데이터 형태들을 말해요. C언어에서는 숫자와 문자를 다루는 데 필요한 핵심적인 타입들을 제공해요. 이걸 알아두면 코드를 쓸 때 훨씬 더 유연하게 움직일 수 있어요!

### 정수 타입 (Integer Types)

먼저 **정수 타입**에 대해 살펴볼까요? 숫자만 다룰 때 주로 사용해요. 다양한 크기와 범위를 가진 정수 타입들이 있어요.

#### 1. `int` 타입

- **설명**: 가장 일반적인 정수 타입이에요. 대부분의 시스템에서 32비트(4바이트) 크기로 취급되지만, 시스템에 따라 달라질 수 있어요.
- **예제 코드**:
  ```c
  #include <stdio.h>

  int main() {
      int age = 25;       // 나이를 나타내는 정수 변수 선언 및 초기화
      int maxScore = 100; // 최고점수를 나타내는 정수 변수 선언 및 초기화
      printf("나이: %d\n", age);    // 나이 출력
      printf("최고 점수: %d\n", maxScore); // 최고점수 출력
      return 0;
  }
  ```
  - **해설**: `int` 타입은 정수 값을 저장하는데 사용되죠. 여기서 `age`와 `maxScore`는 각각 나이와 최고점수를 정수로 표현해요. `printf` 함수를 통해 화면에 출력해요.

#### 2. `short int` 타입

- **설명**: `short int`는 `int`보다 작은 범위를 가진 타입이에요. 주로 메모리 사용량을 줄이기 위해 사용되죠.
- **예제 코드**:
  ```c
  #include <stdio.h>

  int main() {
      short int smallNumber = 1000;  // 메모리 효율성을 위해 작은 숫자 사용
      printf("작은 숫자: %hd\n", smallNumber); // 'hd' 포맷 지정자 사용
      return 0;
  }
  ```
  - **해설**: `short int`는 메모리를 절약하고자 작은 범위의 정수를 다룰 때 유용해요. 여기서 `printf`에서 `%hd`는 짧은 정수를 위한 포맷 지정자예요.

#### 3. `long int` 타입

- **설명**: `long int`는 더 큰 범위의 정수를 다루는 데 사용되죠. 특히 큰 숫자를 다룰 때 필수적이에요.
- **예제 코드**:
  ```c
  #include <stdio.h>

  int main() {
      long int largeNumber = 1234567890L; // 큰 숫자를 다룰 때 사용 (L 접미사로 long int임을 명시)
      printf("큰 숫자: %ld\n", largeNumber); // 'ld' 포맷 지정자 사용
      return 0;
  }
  ```
  - **해설**: `largeNumber` 변수는 큰 숫자를 저장하고, `printf`에서 `%ld`는 긴 정수 타입을 위한 포맷 지정자예요. `L` 접미사는 C 언어에서 긴 정수를 명시적으로 표시하는 데 사용됩니다.

### 실수 타입 (Floating Point Types)

숫자 중에서 소수점이 있는 경우에는 **실수 타입**을 사용해요. 정밀도와 범위에 따라 여러 타입이 있어요.

#### 1. `float` 타입

- **설명**: 일반적인 부동 소수점 타입이에요. 대부분의 경우에서 충분한 정확도를 제공해요.
- **예제 코드**:
  ```c
  #include <stdio.h>

  int main() {
      float pi = 3.14159f;  // 소수점 포함 숫자를 다룰 때 사용 (f 접미사로 float임을 명시)
      printf("원주율: %f\n", pi); // 'f' 포맷 지정자 사용
      return 0;
  }
  ```
  - **해설**: `float` 타입은 소수점이 있는 숫자를 다룰 때 적합해요. `f` 접미사는 이 변수가 `float` 타입임을 명확히 해요. `%f` 포맷 지정자는 부동 소수점 숫자를 출력할 때 사용해요.

#### 2. `double` 타입

- **설명**: `float`보다 더 높은 정밀도를 제공하는 타입이에요. 과학 계산이나 정밀한 수치 작업에 주로 사용해요.
- **예제 코드**:
  ```c
  #include <stdio.h>

  int main() {
      double gravity = 9.81;  // 중력 가속도 같이 정확한 값을 다룰 때 사용
      printf("중력 가속도: %lf\n", gravity); // 'l' 포맷 지정자 사용
      return 0;
  }
  ```
  - **해설**: `double` 타입은 `float`보다 더 정밀한 실수 계산이 필요할 때 사용해요. `%lf` 포맷 지정자는 `double` 타입의 숫자를 출력할 때 사용합니다.

### 문자 타입 (Character Types)

마지막으로 **문자 타입**에 대해 알아볼게요. 이건 간단하지만 중요한 타입이에요.

#### 1. `char` 타입

- **설명**: 문자 하나를 저장하는 데 사용되죠. ASCII 코드를 기반으로 작동해요.
- **예제 코드**:
  ```c
  #include <stdio.h>

  int main() {
      char initial = 'J';  // 문자 변수 선언 및 초기화
      printf("캐릭터: %c\n", initial); // 문자 출력
      return 0;
  }
  ```
  - **해설**: `char` 타입은 단일 문자를 다룰 때 사용해요. 여기서 `'J'`는 대문자 'J'를 나타내고, `%c` 포맷 지정자로 화면에 출력해요.
