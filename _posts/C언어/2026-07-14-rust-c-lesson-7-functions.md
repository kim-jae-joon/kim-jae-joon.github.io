### 7강: 함수 작성 및 호출 - 코딩의 마법, 당신의 손길로 펼쳐보기

안녕하세요, 코딩의 세계에서 이제는 제법 익숙해진 여러분! 오늘은 코딩에서 가장 중요한 도구 중 하나인 **함수**에 대해 깊이 들어가 보도록 하겠습니다. 함수는 마치 주방에서의 레시피와 같다고 생각하면 쉽죠. 레시피를 따라하면 매번 맛있는 요리를 만들 수 있듯이, 함수를 잘 이해하고 활용하면 복잡한 코드도 간결하게 만들 수 있습니다. "진짜 신기하죠? 이걸 알고 나면 프로그래밍이 훨씬 쉬워질 거예요!"

#### 함수의 기본 개념

함수는 특정 작업을 수행하기 위한 **재사용 가능한 코드 블록**입니다. 쉽게 말해, 당신이 자주 사용하는 동작을 모듈화해서 필요할 때마다 호출할 수 있게 만드는 것이죠. 이는 코드의 가독성을 높이고, 유지보수를 용이하게 해줍니다. **"이거 모르면 큰일 납니다!"**  함수 없이 코딩하면 코드가 여기저기 흩어져서 복잡해지고, 수정하기 어려워집니다.

##### 함수의 구조

함수는 일반적으로 다음과 같은 구조를 갖습니다:

```rust
// 함수 정의 예시: 간단한 인사말 함수
fn greet(name: &str) -> String {
    // 함수 내부 코드
    format!("안녕하세요, {}님!", name)
}

fn main() {
    // 함수 호출 예시
    let message = greet("우주");
    println!("{}", message); // 출력: 안녕하세요, 우주님!
}
```

###### 코드 해설

1. **함수 정의 (`fn greet(name: &str) -> String`)**
   - `fn` 키워드로 함수 시작 표시.
   - `greet`: 함수의 이름입니다. 여기서는 사용자 이름을 받아 인사말을 만드는 함수입니다.
   - `name: &str`: 매개변수 타입 선언. `&str`은 문자열 슬라이스를 의미합니다 (읽기 전용 문자열 참조).
   - `-> String`: 반환 타입을 명시합니다. 이 경우, 함수가 `String` 타입을 반환합니다.

2. **함수 내부 코드**
   - `format!("안녕하세요, {}님!", name)`: 문자열 포맷팅을 사용해 인사말을 생성합니다. `{}`는 플레이스홀더로, `name` 변수가 들어갑니다.

3. **함수 호출**
   - `let message = greet("우주");`: 함수 `greet`를 호출하고 결과를 `message` 변수에 저장합니다.
   - `println!("{}", message);`: 저장된 인사말을 출력합니다.

#### 다양한 함수 유형과 활용법

##### 반복문과 함수의 조화

반복문은 함수와 함께 쓰일 때 더욱 강력해집니다. 예를 들어, 숫자 리스트에서 특정 조건을 만족하는 요소를 찾는 함수를 만들어 보겠습니다.

###### 예시 1: `for` 문을 이용한 리스트 필터링

```rust
fn find_even_numbers(numbers: &[i32]) -> Vec<i32> {
    let mut evens = Vec::new();
    for number in numbers {
        if number % 2 == 0 {
            evens.push(*number); // 숫자를 벡터에 추가
        }
    }
    evens
}

fn main() {
    let numbers = vec![1, 2, 3, 4, 5, 6];
    let even_numbers = find_even_numbers(&numbers);
    println!("짝수들: {:?}", even_numbers); // 출력: 짝수들: [2, 4, 6]
}
```

###### 코드 해설

1. **함수 정의**:
   - `find_even_numbers` 함수는 정수 슬라이스를 받아 짝수만 벡터로 반환합니다.
   - `for number in numbers`: 반복문으로 리스트의 각 요소를 순회합니다.
   - `if number % 2 == 0`: 짝수인지 확인하고, 짝수라면 `evens` 벡터에 추가합니다.

##### 조건문과 함수의 결합

조건에 따라 다른 동작을 수행하는 함수도 만들 수 있습니다. 다양한 조건문 구조를 활용해 보세요.

###### 예시 2: `if-else` 문 활용

```rust
fn process_grade(score: i32) -> String {
    if score >= 90 {
        format!("당신의 등급은 A입니다!")
    } else if score >= 70 {
        format!("당신의 등급은 B입니다!")
    } else {
        format!("당신의 등급은 C입니다!")
    }
}

fn main() {
    let grade_message = process_grade(85);
    println!("{}", grade_message); // 출력: 당신의 등급은 B입니다!
}
```

###### 코드 해설

1. **함수 정의**:
   - `process_grade` 함수는 점수를 받아 등급을 반환합니다.
   - `if score >= 90`: 점수가 90 이상이면 "A" 등급을 반환합니다.
   - `else if score >= 70`: 그렇지 않으면 "B" 등급을 반환합니다.
   - `else`: 그 외의 경우 "C" 등급을 반환합니다.

##### 중첩 함수와 콜백 활용

함수 내에서 다른 함수를 호출하는 경우도 있습니다. 콜백 함수를 활용하면 더욱 유연한 코드를 작성할 수 있습니다.

###### 예시 3: 중첩 함수와 콜백

```rust
fn perform_operation(numbers: &[i32], operation: fn(i32, i32) -> i32) -> i32 {
    let mut result = numbers[0]; // 첫 번째 숫자로 초기화
    for &number in numbers.iter().skip(1) {
        result = operation(result, number); // 각 숫자에 대해 연산 수행
    }
    result
}

fn add(a: i32, b: i32) -> i32 {
    a + b
}

fn main() {
    let numbers = vec![1, 2, 3, 4];
    let sum = perform_operation(&numbers, add); // 모든 숫자 더하기
    println!("합계: {}", sum); // 출력: 합계: 10
}
```

###### 코드 해설

1. **함수 정의**:
   - `perform_operation`: 숫자 리스트와 연산 함수를 매개변수로 받아 결과를 반환합니다.
   - `operation(result, number)`: 반복문 내에서 각 숫자에 대해 연산을 적용합니다.
   - `add`: 단순히 두 수를 더하는 함수입니다.

### 💡 초보자 폭풍 질문!

**질문 1:** 함수를 정의할 때 `fn` 키워드는 왜 필요한가요?
- **답변:** `fn` 키워드는 Rust 컴파일러에게 "이 부분이 함수의 정의임"을 알려주는 역할을 합니다. 이를 통해 코드의 구조를 명확하게 하여 오류를 줄이고 가독성을 높입니다.

**질문 2:** 함수 내부에서 반환 타입을 명시하는 이유는 무엇인가요?
- **답변:** 반환 타입을 명시하면 컴파일러가 타입 안전성을 보장하고, 함수의 출력을 명확하게 이해할 수 있게 해줍니다. 이는 오류를 미리 잡아낼 수 있게 도와줍니다.
