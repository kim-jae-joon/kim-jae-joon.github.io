## 3강: C언어 기초 - 자료형 이해: 코드의 비타민, 데이터 타입에 대해 알아보자!

안녕하세요, 코딩 초보자 여러분! 🚀 오늘은 **C언어의 비타민**이라고 할 수 있는 **자료형 이해**에 대해 함께 탐험해볼게요. 혹시 지금까지 코드를 쓰면서 숫자나 문자가 갑자기 어디로 사라지거나 이상하게 변하는 걸 느낀 적 있나요? 🤔 바로 이때 **자료형**의 중요성이 빛을 발하는 순간이에요!

### 왜 자료형이 중요할까? 🤔

**자료형**이란 컴퓨터에 데이터를 저장할 때 사용하는 공간의 종류와 크기를 결정하는 것과 같아요. 마치 우리가 음식을 먹을 때 어떤 그릇에 담아 먹는지에 따라 맛이 달라질 수 있는 것처럼, 코드에서도 데이터 타입이 달라지면 결과가 달라질 수 있어요. 그럼 실제 사례를 통해 자세히 살펴보죠!

### 1. 정수형 (Integer Types) - 숫자의 세계로 떠나보자!

#### a. `int` 타입
- **설명**: 가장 기본적인 정수형 타입이에요. 주로 일상에서 자주 쓰이는 숫자들이 여기에 속해 있어요. 예를 들어, 나이, 점수, 카운트 등이 여기에 해당해요.
- **예제**:
  ```c
  #include <stdio.h>

  int main() {
      int age = 25; // 나이를 저장하는 변수
      int score = 95; // 시험 점수 저장
      printf("나이: %d, 점수: %d\n", age, score); // 출력: 나이: 25, 점수: 95
      return 0;
  }
  ```
  - **코드 해설**:
    - `int age = 25;`: 변수 `age`에 정수값 25를 저장해요.
    - `printf("나이: %d, 점수: %d\n", age, score);`: `%d`는 정수형 데이터를 출력할 때 사용하는 포맷 문자에요. 따라서 `age`와 `score`의 값이 정수 형태로 출력되요.

#### b. `short`, `long` 타입
- **설명**: 더 큰 숫자나 더 작은 숫자를 다루고 싶을 때 사용해요.
  - `short`: 보통 16비트 (2바이트)로, 메모리 사용을 절약할 때 유용해요.
  - `long`: 보통 32비트 (4바이트)로, 큰 숫자를 다룰 때 적합해요.
- **예제**:
  ```c
  #include <stdio.h>

  int main() {
      short smallNumber = 32767; // 짧은 정수형 예시
      long largeNumber = 1234567890; // 긴 정수형 예시
      printf("작은 숫자: %hd, 큰 숫자: %ld\n", smallNumber, largeNumber);
      return 0;
  }
  ```
  - **코드 해설**:
    - `short smallNumber = 32767;`: 짧은 정수형 변수에 큰 정수를 저장하는 예지만, 실제로는 범위를 넘어서 오류가 발생할 수 있으니 주의해야 해요.
    - `printf("작은 숫자: %hd, 큰 숫자: %ld\n", smallNumber, largeNumber);`: 각각 `short`와 `long` 타입을 포맷 문자 `%hd`, `%ld`로 출력해요.

### 2. 실수형 (Floating Point Types) - 정확한 소수점 계산으로 정밀하게!

#### a. `float` 타입
- **설명**: 소수점이 있는 숫자를 다룰 때 사용해요. 예를 들어, 기온이나 환율 등이 여기에 해당해요.
- **예제**:
  ```c
  #include <stdio.h>

  int main() {
      float temperature = 23.5; // 기온 예시
      printf("현재 온도: %.2f°C\n", temperature); // 소수점 둘째 자리까지 출력
      return 0;
  }
  ```
  - **코드 해설**:
    - `printf("현재 온도: %.2f°C\n", temperature);`: `.2f`는 소수점 아래 두 자리까지 반올림하여 출력하도록 지시해요.

#### b. `double` 타입
- **설명**: `float`보다 더 정밀한 실수형 데이터를 다룰 때 사용해요. 과학적 계산이나 금융 계산 등에서 주로 활용되요.
- **예제**:
  ```c
  #include <stdio.h>

  int main() {
      double pi = 3.1415926535; // 원주율 예시
      printf("원주율: %.10f\n", pi); // 소수점 10자리까지 출력
      return 0;
  }
  ```
  - **코드 해설**:
    - `printf("원주율: %.10f\n", pi);`: `.10f`는 소수점 아래 열 자리까지 정밀하게 출력해요. 너무 많은 자리수를 출력하면 오버플로우가 발생할 수 있으니 주의해야 해요.

### 3. 문자형 (Character Types) - 알파벳과 숫자의 비밀 병기!

#### a. `char` 타입
- **설명**: 문자 하나를 저장하는 데 사용해요. 특히 문자열을 다룰 때 매우 중요해요.
- **예제**:
  ```c
  #include <stdio.h>

  int main() {
      char initial = 'J'; // 알파벳 문자 저장
      printf("이름의 첫 글자: %c\n", initial); // 출력: 이름의 첫 글자: J
      return 0;
  }
  ```
  - **코드 해설**:
    - `printf("이름의 첫 글자: %c\n", initial);`: 문자형 데이터는 `%c` 포맷 문자를 사용해요.

### 💡 초보자 폭풍 질문! 💡
- **질문**: 자료형을 잘못 선택하면 어떤 문제가 발생할까요?
  - **답변**: 자료형을 잘못 선택하면 **오버플로우(Overflow)**나 **데이터 손실**이 발생할 수 있어요. 예를 들어, `int`로 큰 숫자를 저장하려고 하면 범위를 넘어서 오류가 생길 수 있고, `float` 대신 `double`을 사용해야 하는 정밀한 계산에서는 정확성이 떨어질 수 있어요. 항상 데이터의 크기와 필요한 정밀도를 고려해야 해요!

### 🚨 실무주의보 🚨
- **주의사항**: 메모리 효율성도 중요해요. 큰 데이터 타입을 사용하면 메모리 사용량이 증가할 수 있으니, 필요한 최소한의 타입을 선택하는 습관을 들이세요. 작은 프로젝트에서는 큰 차이가 없지만, 대규모 시스템에서는 중요한 요소가 될 수 있어요.
