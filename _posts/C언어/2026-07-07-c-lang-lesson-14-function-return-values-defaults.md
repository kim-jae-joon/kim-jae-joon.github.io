**14강: 함수 반환값과 기본값, 이해하고 실천하기! 🤯**

안녕하세요! 오늘부터 우리는 C언어의 함수와 그 안에서 일어나는 MAGIC 🔮을 익혀볼게요! 😊

**함수 기본 개념**

함수는 일종의 수식입니다. 다른 값으로 계산되는 규칙이나 알고리즘을 말합니다. 이 규칙이나 알고리즘을 함수로 만들어 줘야 코드를 쉽고 직관적으로 이해할 수 있어요.

```c
int add(int a, int b) {
    return a + b;
}
```

위의 예제에서 `add`함수는 `a`와 `b` 두 개의 변수가 들어오면 그 합을 반환해 주는 함수입니다. 💡

**함수 반환값**

함수의 가장 중요한 기능 중 하나가 바로 **반환값**입니다! 🎁

함수가 실행된 결과를 반환하여 우리가 쉽게 사용할 수 있게 하는 거예요.

```c
int add(int a, int b) {
    // 함수의 로직을 처리합니다.
    return a + b; // 반환값이 되는 식입니다. 😉
}
```

위의 예제에서 `add`함수는 그저 두 개의 숫자를 더하고 그 결과를 반환해 주기만 하면 돼요! 🤯

### 함수에 기본값을 설정하는 방법

몇몇 경우에는 함수가 실행될 때 항상 몇 개의 매개변수가 들어와야 하는 상황이 생깁니다. 이럴 때 우리는 **기본값**을 설정할 수 있어요.

```c
int add(int a = 1, int b = 2) {
    return a + b;
}
```

위의 예제에서 `add`함수는 매개변수가 두 개가 들어와야 한다는 조건을 걸었습니다. 만약 매개변수가 하나도 안 들으면, 기본값으로 `a`는 1, `b`는 2가 들어가게 됩니다! 🌟

### 실무에서는 어떻게 활용할까? 🚨 실무주의보

```c
// 예시: 계산기 만들기
int calculator(int num) {
    return num * 10;
}

int main() {
    int result = calculator(2); // 결과는 20이 나옵니다! 🎉
    printf("%d\n", result);
    return 0;
}
```

위의 예제에서 `calculator`함수는 숫자를 받아서 그 숫자에 10을 곱해주는 함수입니다. 이 함수의 반환값은 `result`라는 변수에 저장되어, 마지막으로 출력됩니다! 🎯

**마무리:**

오늘은 C언어의 함수와 기본값 그리고 실무 활용에 대해 배웠습니다! 💪

함수는 일종의 수식이며, 가장 중요한 기능 중 하나가 바로 **반환값**입니다.

기본값을 설정하는 방법을 통해 매개변수가 반드시 들어와야 하는 상황에서 편리하게 사용할 수 있게 해줍니다.

마지막으로, 실무에서는 함수를 다양한 곳에 활용하며, 코드의 재사용성을 높일 수 있습니다!
