## 🚀 14강: 제네릭 프로그래밍, 코드의 마법사가 되어보자!

안녕하세요, 🎉 초보 코딩 마법사 여러분! 오늘은 **제네릭 프로그래밍**이라는 마법의 세계로 여러분을 안내할 거예요. 마치 알라딘의 마법램프를 발견한 것처럼, 이 개념 하나로 코드의 효율성과 유연성이 하늘을 찌를 듯이 향상될 거예요! 🤯

### 💡 개념 설명: 제네릭이란 무엇인가요?

제네릭 프로그래밍이란, **특정 데이터 타입에 국한되지 않고 여러 타입을 유연하게 다룰 수 있는 프로그래밍 기법**을 말해요. 쉽게 말해, **한 가지 틀 안에서 다양한 종류의 물건을 넣고 뺄 수 있는 만능 상자**라고 생각하면 됩니다. 이렇게 하면 코드를 한 번 작성하면 여러 상황에서 재사용할 수 있어 시간과 노력을 크게 절약할 수 있어요!

#### 기본 원리와 필요성

1. **다양성 확보**: 동일한 로직을 다양한 데이터 타입에 적용할 수 있어요. 예를 들어, 정수와 문자열을 모두 처리하는 함수를 만들 수 있죠!
   
   ```rust
   // 제네릭 타입 매개변수 정의 예시
   fn print_item<T>(item: T) {
       println!("Item: {:?}", item);
   }

   // 다양한 타입 사용
   let number = 42;
   let text = String::from("안녕하세요!");

   print_item(number);    // 출력: Item: 42
   print_item(text);      // 출력: Item: "안녕하세요!"
   ```

   **해설**: 여기서 `<T>`는 타입 매개변수로, 함수 내에서 `item` 변수가 어떤 타입이든 받아 처리할 수 있게 해줍니다. `println!` 매크로는 타입에 상관없이 값을 출력할 수 있어요.

2. **코드 재사용성 향상**: 동일한 로직을 여러 타입에 적용하므로 코드 중복을 줄이고 유지보수를 쉽게 만들어요.

### 💡 실제 적용 사례: 제네릭을 활용한 다양한 예제

#### 1. **벡터 내 모든 요소 합계 계산**

   제네릭을 사용하면 정수 벡터든 문자열 벡터든 상관없이 합계를 계산할 수 있어요!

   ```rust
   // 제네릭 함수로 벡터 요소 합계 계산
   fn sum_elements<T: std::ops::Add<Output = T>>(vec: Vec<T>) -> T {
       let mut total = T::default();  // 타입에 맞는 기본값 초기화
       for element in vec {
           total = total + element;   // 덧셈 연산
       }
       total
   }

   // 정수 벡터 예시
   let numbers = vec![1, 2, 3, 4];
   println!("Sum of numbers: {}", sum_elements(numbers));  // 출력: Sum of numbers: 10

   // 문자열 길이 합계 예시 (각 문자의 아스키 코드 값 합산)
   let strings = vec![String::from("안녕"), String::from("하세요")];
   println!("Sum of string lengths: {}", sum_elements(strings.iter().map(|s| s.len() as i32).collect()));
   // 출력: Sum of string lengths: 9 (아스키 코드 값 합산)
   ```

   **해설**:
   - `T: std::ops::Add<Output = T>`는 타입 `T`가 덧셈 연산을 지원해야 함을 의미합니다.
   - `T::default()`는 타입에 맞는 기본 값을 초기화합니다 (예: `0` for `i32`, `Vec::new()` 등).
   - `map`과 `collect`를 사용해 벡터 내 각 문자열 길이를 정수로 변환하여 합산합니다.

#### 2. **다차원 배열 처리**

   다차원 배열을 처리할 때도 제네릭을 활용하면 편리해요!

   ```rust
   // 제네릭 다차원 배열 처리 함수
   fn print_matrix<T>(matrix: &[[T; 3]; 3]) {
       for row in matrix {
           for element in row {
               println!("{:?} ", element);
           }
           println!("---");  // 행 구분자
       }
   }

   // 정수 행렬 예시
   let int_matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];
   print_matrix(&int_matrix);
   // 출력:
   // 1  2  3  ---
   // 4  5  6  ---
   // 7  8  9  ---

   // 문자열 행렬 예시
   let str_matrix = [["사과", "바나나", "체리"], ["토마토", "오이", "멜론"], ["딸기", "포도", "키위"]];
   print_matrix(&str_matrix);
   // 출력:
   // 사과 바나나 체리 ---
   // 토마토 오이 멜론 ---
   // 딸기 포도 키위 ---
   ```

   **해설**:
   - `&[[T; 3]; 3]`는 3x3 크기의 2차원 배열을 타입 `T`로 구성합니다.
   - 이중 반복문을 통해 각 요소를 출력합니다.

#### 3. **조건부 데이터 처리**

   제네릭을 활용해 다양한 타입에 따라 다른 처리를 할 수도 있어요!

   ```rust
   // 타입에 따른 조건부 처리 예시
   fn process_data<T>(data: T) -> String {
       match data {
           i32(x) => format!("정수: {}", x),
           String(s) => format!("문자열: {}", s),
           _ => "지원하지 않는 타입입니다.".to_string(),
       }
   }

   let num = 42;
   let text = String::from("코딩의 세계");

   println!("{}", process_data(num));    // 출력: 정수: 42
   println!("{}", process_data(text));   // 출력: 문자열: 코딩의 세계
   ```

   **해설**:
   - `match` 문을 사용해 타입 패턴 매칭을 수행합니다.
   - `_`는 지원하지 않는 타입에 대한 기본 케이스를 처리합니다.
