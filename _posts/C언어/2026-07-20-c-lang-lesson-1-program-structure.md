### 1강: 프로그램 구성 및 실행 - 코드의 기초 다지기

**✅ 첫걸음:** **프로그램? 🤔 어떤 게 바로 "프로그램"일까요? 🤔**
단순히 우리가 사용하는 스마트폰이나 게임 같은 거들은 무슨 말인지 알아맞추셨나요? 그거, 바로 프로그램이죠! 📱🎮 C 언어는 이런 프로그램들을 만들기 위한 도구라고 생각하면 좋아요.

**✅ 두 번째 걸음:** **함수 (Function) - 프로젝트의 미니 블록 형태 🚀**
C 언어에서는 작업을 작게 나눠서 처리하는 "함수"라는 개념이 있습니다. 마치 레시피에서 재료를 넣고 요리하는 과정처럼, 함수는 프로그램의 일부를 담당하는 작은 코드 조각입니다. 💪

```c
#include <stdio.h>

int main() { // 시작! 이것이 프로그램 실행되는 주요 부분이에요!
    printf("Hello, World!\n"); // '프린트'라는 함수를 이용해서 화면에 'Hello, World!' 출력합니다.
    return 0; // 작업 완료 신호!

}
```


* **#include <stdio.h>** : 프로그램이 필요한 도구들을 불러옵니다.  'stdio.h'는 화면에 글자를 출력하는 데 필요한 도구들을 가지고 있습니다. 🛠️

* **int main() { ... }**: 모든 C 프로그램의 시작점입니다! 프로그램 실행을 알리는 역할인데, 여기에 코드가 들어갑니다.
* **printf("Hello, World!\n");**: 'printf' 함수는 화면에 글자를 출력하는 기능을 가지고 있습니다. `"Hello, World!\n"`는 출력할 문구이며, `\n`은 줄바꿈을 의미합니다.

* **return 0;**: 프로그램이 정상적으로 실행되었음을 알리는 코드입니다.


###  2. 코딩의 기본 요소 - 변수 (Variable) 🚀🚀🚀
변수는 우리가 프로그램에서 사용할 데이터를 담기 위한 공간인 거예요. 이름도 지어서 데이터를 저장하는 칸이라고 생각하면 좋아요!

```c
int age = 25; // 'age'라는 이름의 변수에 25를 저장합니다. int는 정수형 자료형입니다.

float price = 19.99; // 'price'라는 이름의 변수에 19.99를 저장합니다. float는 실수형 자료형입니다.
```



* **int** : 정수 (숫자)를 저장하는 변수 입니다.
* **float**: 소수점을 가진 수를 저장하는 변수입니다.

###  3. 조건문 - 프로그램의 선택과 결정 💡💡💡
조건문은 특정 조건이 충족되면 실행되는 코드 블록을 선택하도록 하는 역할을 합니다. 마치 "만약 날씨가 좋아요면, 공원에 가서 산책" 하고 "안 그래도, 집에서 영화를 보자" 처럼 선택하는 것과 같죠!

```c
#include <stdio.h>

int main() {
    int temperature = 28; // 온도 변수

    if (temperature > 30) {
        printf("매우 더운 날씨네요! 😎"); 
    } else if (temperature > 25) {
        printf("날씨가 좋습니다! 😊 ");  
    } else {
        printf("추울 수 있습니다. 😔 ");
    }

    return 0;
}
```



* **if**: 특정 조건이 맞으면 코드 블럭을 실행합니다.
* **else if**: 첫 번째 조건이 거짓일 때, 추가적인 조건을 확인합니다.
* **else**: 모든 조건이 거짓일 경우 실행되는 코드 블럭입니다.

###  4. 반복문 - 코드를 반복해서 실행하기 🔁🔁

반복문은 특정 조건이 충족될 때까지 코드를 계속 반복하여 실행하는 문장입니다. 마치 "1부터 10까지의 수를 더하기" 같은 작업을 할 때 유용하게 사용됩니다.

```c
#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) { // 변수 'i'가 1부터 5까지 증가하며 반복합니다.
        printf("%d ", i);  // 각 반복에서 현재 값을 출력합니다.
    }

    return 0;
}
```



* **for**: 시작값, 종료 조건, 증감 조건을 설정하여 반복문을 실행합니다.
