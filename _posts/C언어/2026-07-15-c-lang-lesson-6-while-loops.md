**6강: while 반복문 - 조건에 따라 반복 수행! 🔥**

안녕하세요, Coding World 여러분! 이번에는 가장 유용하고 필수적인 C언어 문법 중 하나인 `while` 반복문을 배워보겠습니다. 💡

**왜 필요한가? 🤔**

다들 알 정도로, 우리의 프로그램은 수많은 조건과 상황을 처리해야 합니다. 하지만 이 모든 것을 한 번에 계산하는 것이 얼마나 어려운지 상상만 해도 머리가 아파질 것입니다! 🔥

이때 `while` 반복문을 이용하면, 특정 조건이 만족할 때까지 반복적으로 실행되도록 할 수 있습니다. 그야말로 신기하죠?! 🤯

**while 반복문 구조**

```c
while(조건) {
    // 코드 블록
}
```

`while` 반복문의 기본 구조는 간단합니다. `조건`을 평가하여 TRUE인 경우, 내부의 코드 블록이 실행됩니다.

**예제 1: 사용자 입력 수만큼 출력하기 📝**

```c
#include <stdio.h>

int main() {
    int num;

    printf("숫자를 입력하세요: ");
    scanf("%d", &num);

    while(num > 0) {
        printf("%d\n", num);
        num--;
    }

    return 0;
}
```

위 코드는 사용자에게 숫자를 입력받고, 그 수만큼 반복적으로 숫자를 출력합니다.

```c
// 1. 조건 evaluation (num > 0)
while(num > 0) {
    // 2. TRUE case - 코드 블록 실행
    printf("%d\n", num);
    // 3. 코드 블록의 마지막에서 num-- 연산 수행
}

// 이후, while 반복문이 종료되며 while문의 scope가 끝나게 됩니다.
```

**예제 2: 구구단 출력하기 📊**

```c
#include <stdio.h>

int main() {
    int num;

    printf("숫자를 입력하세요: ");
    scanf("%d", &num);

    while(num <= 9) {
        for(int i = 1; i <= 10; i++) {
            printf("%d x %d = %d\n", num, i, num * i);
        }
        num++;
    }

    return 0;
}
```

위 코드는 사용자에게 숫자를 입력받고, 그 수까지의 구구단을 출력합니다.

**while 반복문 활용**

`while` 반복문을 다양하게 활용할 수 있습니다. 예를 들어, 다음과 같은 경우에도 사용할 수 있습니다:

*   **데이터 읽기/쓰기**: 파일이나 네트워크에서 데이터를 읽어오거나 쓰는 것을 반복적으로 처리할 때.
*   **알고리즘 수행**: 특정 알고리즘을 수행하는 동안 반복적으로 실행해야 하는 코드 블록이 필요할 때.
