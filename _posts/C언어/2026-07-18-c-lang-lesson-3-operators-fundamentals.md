# 3강: C언어 기초 - 연산자 이해하기: 코드 속 마법의 주문을 배우다!

안녕하세요, 코딩 초보자 여러분! 오늘은 **C언어의 마법사**가 되어 여러분을 **연산자의 신비로운 세계**로 안내할게요. 이건 진짜 신기하죠? 컴퓨터가 말하는 언어를 이해하는 순간, 여러분은 마치 마법을 부리는 마법사 같아질 거예요! 🧙‍♂️✨

## 연산자란 무엇인가요? 🧙‍♂️✨

연산자란 간단히 말해 **수식을 만들어 연산을 수행하는 핵심 도구**라고 생각하면 돼요. 예를 들어, 여러분이 좋아하는 **치킨을 주문할 때** 연산자는 바로 **가격 계산**을 담당하죠! 🍗💰

### 기본 연산자들

#### 1. 산술 연산자 (Arithmetic Operators)
산술 연산자는 숫자를 다루는 마법사들이에요! 주요 연산자들을 하나씩 살펴볼게요.

##### 덧셈 (+)
```c
#include <stdio.h>

int main() {
    int num1 = 10;  // 숫자 10을 변수 num1에 할당
    int num2 = 5;   // 숫자 5를 변수 num2에 할당
    int sum = num1 + num2;  // 덧셈 연산 수행: num1 + num2 = sum
    printf("합계는 %d입니다.\n", sum);  // 결과 출력
    return 0;
}
```
**코드 분석:**
- `num1 + num2` 부분에서 `+` 연산자가 두 숫자를 더해요. 결과는 `sum` 변수에 저장됩니다.
- `printf` 함수로 결과를 화면에 출력합니다. **진짜 신기하죠?** 단순히 `+` 하나로 숫자들이 상호작용해요!

##### 뺄셈 (-)
```c
#include <stdio.h>

int main() {
    int difference = 10 - 5;  // 뺄셈 연산 수행: 10 - 5 = difference
    printf("차이는 %d입니다.\n", difference);  // 결과 출력
    return 0;
}
```
**코드 분석:**
- `10 - 5`에서 `-` 연산자가 첫 번째 숫자에서 두 번째 숫자를 뺍니다. 결과는 `difference` 변수에 저장됩니다.

##### 곱셈 (*) 과 나눗셈 (/)
```c
#include <stdio.h>

int main() {
    int product = 10 * 5;  // 곱셈 연산 수행: 10 * 5 = product
    float quotient = 10.0 / 5;  // 나눗셈 연산 수행: 10 / 5 = quotient (실수 결과)
    printf("곱셈 결과: %d, 나눗셈 결과: %.2f\n", product, quotient);
    return 0;
}
```
**코드 분석:**
- `10 * 5`는 곱셈 연산자 `*`를 통해 결과를 `product`에 저장합니다.
- `10.0 / 5`는 실수 나눗셈을 수행하여 `quotient`에 저장합니다. 여기서 `.0`은 나눗셈 결과가 실수로 처리되도록 합니다.

### 비교 연산자 (Comparison Operators)
비교 연산자는 두 값 사이의 관계를 판단하는 마법사들이에요!

#### 1. `==` (같음)
```c
#include <stdio.h>

int main() {
    int a = 10;
    int b = 10;
    if (a == b) {  // 두 값이 같은지 확인
        printf("a와 b는 같아요!\n");
    } else {
        printf("a와 b는 다릅니다.\n");
    }
    return 0;
}
```
**코드 분석:**
- `a == b`는 `a`와 `b`가 같은지 비교합니다. 만약 같다면 "a와 b는 같아요!"를 출력합니다.

#### 2. `!=` (다름)
```c
#include <stdio.h>

int main() {
    int x = 10;
    int y = 20;
    if (x != y) {  // 두 값이 다른지 확인
        printf("x와 y는 다릅니다.\n");
    } else {
        printf("x와 y는 같아요.\n"); // 이건 거의 안 일어날 상황이에요!
    }
    return 0;
}
```
**코드 분석:**
- `x != y`는 `x`와 `y`가 다른지 확인합니다. 다르다면 "x와 y는 다릅니다."를 출력합니다.

#### 3. `>`, `<`, `>=`, `<=` (크기 비교)
```c
#include <stdio.h>

int main() {
    int num1 = 15;
    int num2 = 25;
    if (num1 > num2) {  // num1이 num2보다 큰지 확인
        printf("num1이 num2보다 큽니다.\n");
    } else if (num1 < num2) {  // num1이 num2보다 작은지 확인
        printf("num1이 num2보다 작습니다.\n");
    } else {
        printf("num1과 num2는 같습니다.\n");
    }
    return 0;
}
```
**코드 분석:**
- `num1 > num2`는 `num1`이 `num2`보다 큰지 확인합니다. 결과에 따라 적절한 메시지를 출력합니다.
