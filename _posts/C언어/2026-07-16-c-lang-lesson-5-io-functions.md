## 🚀 5강: 입출력 왕초보 탈출! printf와 scanf 마스터하기

**"아, 입력 받으려면 scanf... 출력하려면 printf...?"** 

개발 세계에 발을 들여놓은 지 벌써 5년! 🎉  저는 코딩의 달콤한 맛을 알려준 너희의 믿음직스러운 선배, **코드 마스터 C**입니다. 오늘은 프로그래밍에서 가장 기본적이면서도 엄청나게 중요한 친구들을 만나볼게요. 바로 **입출력 함수의 왕, printf와 scanf**입니다! 

**진짜 신기하죠? 이 두 친구 없이는 프로그램이 말도 못 하고, 사용자와 소통도 할 수 없어요!** 마치 사람과의 대화처럼 프로그램도 입력을 받아야 정보를 이해하고, 결과를 출력해야 사용자에게 피드백을 줄 수 있답니다.

### 🤔 출력의 마법사: printf

**printf**는 마치 똑똑한 스피커 같아요. 컴퓨터 안에 숨어있는 생각들을 아름답고 정돈된 문구로 멋지게 출력해주는 역할을 합니다.

**기본 구조:**

```c
#include <stdio.h> //  printf를 사용하기 위한 헤더 파일 포함

int main() {
    printf("안녕하세요! 당신의 이름은 무엇입니까? "); // 문자열 출력
    char 이름[50]; // 이름을 저장할 공간 준비
    scanf("%s", 이름); // 사용자 입력 받기 (다음 강의에서 자세히!)
    printf("만나서 반가워요, %s!", 이름); // 입력받은 이름으로 환영 메시지 출력
    return 0;
}
```

**코드 해부 타임!**

1. **`#include <stdio.h>`:**  printf 함수를 사용하기 위한 필수 문법! 마치 도구 상자를 열어주는 것처럼요.
2. **`printf("안녕하세요! 당신의 이름은 무엇입니까? ");`:** 컴퓨터 화면에 텍스트를 보여주는 마법 주문! 큰따옴표 안에 문자열을 넣으면 출력됩니다. 마치 성우가 대사를 읽어주는 것처럼요.
3. **`char 이름[50];`:** 사용자가 입력할 이름을 저장할 빈 상자를 만드는 거예요. 크기는 50글자까지 허용합니다. 
4. **`scanf("%s", 이름);`:**  이제 사용자에게 질문을 던지고 답을 듣는 시간! `scanf`는 키보드 입력을 받아 준비한 상자 (`이름`)에 담아줍니다.
5. **`printf("만나서 반가워요, %s!", 이름);`:**  입력받은 이름을 이용해 더욱 개인화된 환영 메시지를 출력! `%s`는 `이름` 변수에 저장된 문자열을 대체하는 마법 주문입니다. 마치 이름으로 인사하는 것처럼 친근하게 느껴지죠?

**다양한 출력 마법!**

`printf`는 다양한 포맷 지정자를 사용해 더욱 풍성한 출력을 가능하게 합니다. 몇 가지 예시를 살펴볼게요.

* **숫자 출력:**

```c
int 나이 = 25;
printf("당신의 나이는 %d살입니다.\n", 나이); // %d는 정수 출력
```

* **실수 출력:**

```c
float 몸무게 = 65.5;
printf("당신의 몸무게는 %.2f kg입니다.\n", 몸무게); // %.2f는 소수점 둘째 자리까지 출력
```

* **여러 데이터 출력:**

```c
int 점수 = 90;
char 과목 = "수학";
printf("과목: %s, 점수: %d\n", 과목, 점수); // 여러 데이터 연결
```

**💡 초보자 폭풍 질문!**  
**Q: `%d`와 `%f`의 차이는 무엇인가요?**
**A: `%d`는 정수(정수형 변수)를 출력할 때 사용하고, `%f`는 소수점이 있는 실수(float형 변수)를 출력할 때 사용합니다. 숫자의 종류에 따라 적절한 포맷 지정자를 선택해야 정확한 출력이 가능해요!**

### 🎯 사용자와 소통하는 마법사: scanf

**scanf**는 사용자의 말을 컴퓨터가 이해할 수 있도록 듣는 역할을 합니다. 마치 똑똑한 비서처럼 입력받은 데이터를 변수에 안전하게 저장해줍니다.

**기본 구조:**

```c
#include <stdio.h>

int main() {
    printf("숫자를 입력하세요: ");
    int 숫자; // 숫자를 저장할 변수 선언
    scanf("%d", &숫자); // 사용자 입력 받기 (& 연산자 중요!)
    printf("입력하신 숫자는 %d입니다.\n", 숫자);
    return 0;
}
```

**코드 해부 타임!**

1. **`scanf("%d", &숫자);`:** 여기서 핵심은 `&` 연산자입니다! 변수의 주소를 전달하여 scanf가 값을 직접 변수에 저장하도록 합니다. 마치 비서에게 서류를 전달하는 것과 같죠!

**다양한 입력 마법!**

scanf는 다양한 데이터 타입에 맞춰 포맷 지정자를 사용합니다. 몇 가지 예시를 살펴볼게요.

* **문자 입력:**

```c
char 문자;
scanf("%c", &문자); // 문자 하나 입력
printf("입력하신 문자는 %c입니다.\n", 문자);
```

* **실수 입력:**

```c
float 금액;
scanf("%f", &금액); // 소수점 포함 입력
printf("입력하신 금액은 %.2f원입니다.\n", 금액);
```

* **여러 데이터 입력:**

```c
int 나이, 키;
char 이름[50];
scanf("%d %s %f", &나이, 이름, &키); // 여러 데이터 동시 입력
printf("나이: %d, 이름: %s, 키: %.2f\n", 나이, 이름, 키);
```

**🚨 실무주의보**  
**scanf는 입력 데이터의 유효성을 직접 검증하지 않습니다. 잘못된 형식의 입력이 들어올 경우 프로그램 오류가 발생할 수 있으니 주의해야 합니다! 입력 검증 로직을 추가하는 것이 안전합니다.**

### 마무리: 입출력 마스터가 되는 비결

printf와 scanf는 마치 프로그래밍의 **쌍방향 소통 채널**입니다. 사용자와 프로그램 사이의 대화를 가능하게 하는 핵심적인 도구죠! 이 두 친구를 잘 이해하고 활용하면, 더욱 생생하고 인터랙티브한 프로그램을 만들 수 있을 거예요.
