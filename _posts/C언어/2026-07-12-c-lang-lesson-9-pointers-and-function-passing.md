###  🎉 Ponteer: 명칭이 뭔진 모르겠지만 중요한건 알아두세요!

Ponteer는 "포인터"라는 이름과 다름없이, 다른 변수를 가리키는 변수라고 생각하면 편하게 이해할 수 있어요. 예를 들어, 아래 그림처럼 당신의 집주소가 적힌 카드를 주고받는 것과 같다는 식으로 생각해보세요.

[그림: 사람 A가 '집'이라는 변수에 주소값을 넣고, 사람 B에게 집 주소가 적힌 카드를 전달하는 그림]

만약 당신이 사람 A에게 "내 집 주소"라고 말하면, 사람 A는 당신의 집 주소를 가지고 있는 새로운 변수(Ponteer)를 만들어서, 그 주소값을 가리키도록 설정합니다. 이때, 사람 A가 가지고 있는 새로운 변수는 사람 B에게 전달될 수 있습니다!

🚨 **실무주의보**: C언어에서는 Ponteer를 직접 명시적으로 선언해야 합니다.  C 언어의 주인공은 "주소값"이기 때문에 좀 더 신중하게 다뤄야 한다는 점을 기억하세요! 💡


### 함수 전달 방식: 변수만 보내면 되죠? 🤔

함수 전달 방식은 함수를 호출할 때, 해당 함수에 필요한 정보들을 "변수"로 전달하는 방법이라고 할 수 있습니다.  여기서 Ponteer는 이러한 변수들을 효과적으로 전달하고 사용하는 데 중요한 역할을 합니다!

💡 **초보자 폭풍 질문!**: 그럼 주소값만 보내면 되지 않나요? 함수 내부에서 직접 값을 바꿀 수도 있잖아요!

정말 좋은 질문입니다! 💯 하지만, 함수가 변수를 수정하지 않는 것이 원칙이라는 것을 기억해야 합니다. Ponteer를 사용하면 함수는 원본 데이터에 영향을 미치지 않고 새로운 값을 생성하는 방식으로 작동합니다. 그렇기 때문에 깨끗하고 안전한 코드 작성이 가능해지는 거죠! 😎

###  💻 실제 코드 예시: Ponteer와 함수 전달 방식 활용하기

아래 코드 예시는 "Ponteer와 함수 전달 방식"을 직접 적용하여 사용하는 방법을 보여줍니다.



```c
#include <stdio.h>

// 변수를 가리키는 포인터를 받는 함수 정의
void swapValues(int *a, int *b) {
    int temp = *a; // Ponteer를 사용하여 a 변수의 값을 임시 변수에 저장
    *a = *b;      // b 변수의 값을 a 변수의 주소값으로 할당
    *b = temp;     // 임시 변수의 값을 b 변수의 주소값으로 할당 
}

int main() {
    int num1 = 10, num2 = 20; // 두 개의 정수 변수 선언 및 초기화

    printf("num1: %d, num2: %d\n", num1, num2);  // 함수 호출 전 값 출력

    swapValues(&num1, &num2);     // Ponteer를 이용하여 num1과 num2의 주소값을 함수에 전달
   
    printf("num1: %d, num2: %d\n", num1, num2);  // 함수 호출 후 값 출력

    return 0;
}
```



**설명:**


* `swapValues` 함수는 두 개의 포인터 (`int *a`, `int *b`)를 받습니다. 이 포인터들은 각각 `num1`과 `num2` 변수의 주소를 가리키고 있습니다.

* 함수 내부에서 `*a`와 `*b`를 사용하여 두 변수에 저장된 값을 읽고,  `temp` 변수에 저장합니다. 그 다음에는 `*a = *b`,  `*b = temp`를 통해 두 변수의 값을 교환합니다.
    

### 🥇 Ponteer와 함수 전달 방식의 장점


C언어에서 "Ponteer와 함수 전달 방식"을 사용하는 것은 여러 가지 장점이 있습니다!


1. **메모리 효율성**: 데이터를 직접 복사하지 않고 주소값만 전달하기 때문에 메모리를 효율적으로 사용할 수 있습니다. 
2. **데이터 공유**: 함수 내부에서 원본 변수에 영향을 미치지 않고 값을 변경하여 다른 함수들과 데이터를 공유하는 효과적인 방법이 됩니다!

3. **코드 재사용성**: 동일한 코드를 다양한 상황에서 재사용할 수 있습니다.  Ponteer와 함수 전달 방식은  C 언어의 가장 강력하고 유연한 기능 중 하나로, 복잡한 프로그램 구현에 필수적인 도구가 됩니다! 😎
