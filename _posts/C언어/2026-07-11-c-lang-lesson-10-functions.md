## 🔥 **10강: 함수 - 선언 및 정의, 프로그래밍 마법사가 되자!** 🚀

안녕하세요, C 언어 일타 최고 강사인 저 '코딩은재미'를 만나게 되셨네요! 🤩 오늘부터 함수 이야기를 시작할 거랍니다. 🤫 함수는 프로그램을 좀 더 효율적이고 읽기 쉬운 모습으로 만들어주는 프로그래밍 마법 ✨ 

함수라니? 어떤 걸 의미하는지는 이 강의가 끝나면 당신도 "마술처럼 이해되네!"라고 외치실 정도로 신나는 경험이 될 거예요. 😎   
 
### 🎉 함수란 무엇일까요?

직접 이야기하기전에, 그림으로 설명해 드릴게요! ✨  상상해보세요. '만두' 만드는 과정을 프로그래밍처럼 세분화해서 생각해 보겠습니다.

1. **재료 챙기기**:  
   - 채소 손질 (🥬🧅🥕)
   - 돼지고기를 갈아서 준비하기 (🥓🥩)
2. **만두피 말려주기** 🥟
3. **필요한 만큼 반죽 만들기** 🥣
4. **반죽에 재료 채우고 접는 행위**  👨‍🍳
5. **냄비에 물붓고 끓여서 요리하기**  🍲

이처럼 '만두'를 만들어내는 과정은 여러 단계로 나눠볼 수 있습니다. 각 단계마다 특정 작업을 하는 것과 같죠? 🤔 프로그램도 마찬가지! 복잡한 프로그램은 여러 작고 쉬운 함수들을 모아서 이루어집니다. 🎉

💡 **함수**는 하나의 특정 작업을 담당하는 프로그래밍 블록이에요.  
마치 '만두' 만드는 과정에서 각 단계를 나타내는 것처럼, 프로그램도 여러 함수들을 연결하여 완전한 코드를 만들 수 있죠! 🤯

### 🚀 함수 선언: 지금부터 알고 있는 명령어로 "함수라는 것을 말하는 법"을 배워볼까요?

예를 들어, '만두' 만들기를 위한 각 단계에 이름을 붙여보세요. 

1.  **재료 준비**: `get_ingredients()`
2.  **만두피 준비**: `prepare_dough()`
3. **만두 채우기**: `fill_dumplings()`
4. **끓이기**: `cook_dumplings()`

함수 선언은 이런 "작업 이름"을 정하는 과정입니다. 📑 함수의 이름에 따라 어떤 작업이 수행되는지 알 수 있습니다!  C 언어에서 함수 선언 문법은 다음과 같습니다:

```c
반환형 변수명(매개변수1, 매개변수2...) {
    // 함수 내부 코드 작성
}
```

> 💡 **반환형**: 함수가 실행된 후에 어떤 값을 돌려줄지 나타냅니다. (ex: `int`, `void`)
> > *`int`: 정수형 값을 반환할 때 사용합니다.
> > *`void`: 아무것도 반환하지 않을 때 사용합니다.

> 💡 **변수명**: 함수의 이름입니다. 명확하고 간결하게 지어주세요! (ex: `calculate_sum()`, `print_message()`)
> >  함수 이름은 알파벳, 숫자, _(_밑줄)으로 시작하며, 이를 포함하여 구성될 수 있습니다.

> 💡 **매개변수**: 함수가 작업을 하기 위해 필요한 입력 값입니다. (ex: `number1`, `number2`)
> > *괄호 안에 ,(콤마)로 구분하여 매개변수를 정의합니다.

### 💪 함수 정의: 마지막으로, "함수가 실제로 어떤 작업을 하는지" 알려주는 시간!


```c
#include <stdio.h> 

// 함수 선언 (이름만 작성!)
int add_numbers(int num1, int num2);

int main() {

    int result = add_numbers(5, 3); // 함수 호출!

    printf("합계: %d\n", result); // 결과 출력

    return 0;
}

// 함수 정의 (함수 이름과 선언이 같아야 합니다!)
int add_numbers(int num1, int num2) {
    int sum = num1 + num2; // 두 수를 더하여 합계 계산!
    return sum; // 계산된 합계 값을 반환합니다.
}

```


> 💡  `main()` 함수는 프로그램이 시작되는 가장 첫 번째 함수입니다. `add_numbers()` 함수를 호출하고, 결과값을 변수 `result`에 저장합니다.



**자, 이제 우리가 만들었던 '만두'의 각 단계 함수들을 활용해서 '실제 만두를 만들어볼까요?' 🤔  함수를 사용하면 코드 작성이 쉬워지고 재사용 가능해져서 정말 효율적인 프로그래밍을 할 수 있답니다! 🤩**
