### 1.  비트: C 언어의 가장 기본적인 단위! 🧱

처음부터 비밀스럽고 어려운 설명은 지금부터 하지 않겠습니다! 😎 C 언어에서 모든 데이터는 **비트(bit)**라는 깜짝 작은 블록들로 구성되어 있습니다. 비트는 '1' 또는 '0' 이 두 가지 값만을 가질 수 있는 마법 같은 존재죠. 🧙‍♂️

예를 들어, 온도가 25°C인 경우, 컴퓨터에서 이 정보는 여러 비트들이 조합된 형태로 저장됩니다! 🌡️ 각 비트마다 특정한 의미를 가지고 있어서 정확하고 효율적인 데이터 처리가 가능해집니다.

### 2. 비트 조작 연산자: 마법의 손길을 이용하여 비트들을 다듬어봐요! ✨
비트 조작 연산자는 바로 이 '1'과 '0'로 이루어진 비트들을 조작하는 데 사용되는 특별한 기능들입니다.  🪄 

* **& (AND 연산자):** 두 수의 각 비트가 모두 '1'이면 결과도 '1', 그렇지 않으면 '0'을 출력합니다. 🧐 예를 들어, 5(101)와 3(011)의 AND 연산은 1(001)입니다!

```c
#include <stdio.h>

int main() {
    int a = 5; // 101 (Binary)
    int b = 3; // 011 (Binary)

    int result = a & b; // 각 비트가 모두 1이면 결과는 1, 아니면 0 출력
    printf("a & b = %d\n", result);  // Output: a & b = 1
    return 0;
}
```

* **| (OR 연산자):** 두 수의 각 비트 중 하나라도 '1'이면 결과도 '1', 모두 '0'이면 '0'을 출력합니다. 🤔 예를 들어, 5(101)와 3(011)의 OR 연산은 7(111)입니다!

```c
#include <stdio.h>

int main() {
    int a = 5; // 101 (Binary)
    int b = 3; // 011 (Binary)

    int result = a | b; // 하나라도 1이면 결과는 1, 모두 0이면 0 출력
    printf("a | b = %d\n", result);  // Output: a | b = 7
    return 0;
}
```

* **^ (XOR 연산자):** 두 수의 각 비트가 다르면 '1', 같으면 '0'을 출력합니다. 🤔 5(101)와 3(011)의 XOR 연산은 6(110)입니다!

```c
#include <stdio.h>

int main() {
    int a = 5; // 101 (Binary)
    int b = 3; // 011 (Binary)

    int result = a ^ b; // 비트가 다르면 1, 같으면 0 출력
    printf("a ^ b = %d\n", result);  // Output: a ^ b = 6
    return 0;
}
```

* **~ (NOT 연산자):** 각 비트의 값을 반대로 바꿉니다. '1'은 '0', '0'은 '1'이 되는 거예요! 🤩 예를 들어, ~5(101)는 -6(-110)입니다!

```c
#include <stdio.h>

int main() {
    int a = 5; // 101 (Binary)

    int result = ~a; // 각 비트의 값을 반대로 바꿈
    printf("~a = %d\n", result);  // Output: ~a = -6 (-110 in Binary)
    return 0;
}
```
