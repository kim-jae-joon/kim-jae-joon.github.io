**11강: 연산자와 연산**

🔥 오늘은 정말 기대가 된 강의 중 하나야! 연산자와 연산이라는 주제를 처음 접하는 분들이 궁금해 할만한 내용들을 다룰 거야. 💡 연산자, 연산... 말도 안 되는 단어 같지만, 이해하면 코딩이 정말 즐겁다!

**연산자의 개념**

연산자는 뭐가 하는 것일까? 🔍 간단하게 말하면, 데이터를 조작하고 변환하는 역할을 하는 거야. 🎉 예를 들어, `a + b`는 두 수를 더한다는 뜻이고, `c - d`는 두 수를 뺀다는 뜻이야.

### 연산자의 종류

연산자는 크게 다음과 같은 몇 가지로 분류되는데, 이 중 일부를 살펴보면 좋을 것 같아!

#### **1. 산술 연산자**

- `+`: 더하기 (ex: `a + b`)
- `-`: 빼기 (ex: `c - d`)
- `*`: 곱하기 (ex: `e * f`)
- `/`: 나누기 (ex: `g / h`)

```c
// 산술 연산자 예제
int a = 10;
int b = 5;
int result1 = a + b;    // 15
int result2 = a - b;    // 5
int result3 = a * b;    // 50
int result4 = a / b;    // 2
```

### **2. 할당 연산자**

할당 연산자는 한 변수에 다른 값이나 결과를 저장하는 역할을 하는 거야!

- `=`: 등호 (ex: `a = 10`)
- `+=`: 더하기 할당 (ex: `a += b;`은 `a = a + b;`과 같다!)
- `-=`: 빼기 할당 (ex: `c -= d;`은 `c = c - d;`과 같다!)

```c
// 할당 연산자 예제
int x = 10;
x += 5;     // x는 이제 15로 바뀌었다!
```

### **3. 비교 연산자**

비교 연산자는 두 값의 상대적인 크기를 판단하는 역할을 하는 거야!

- `==`: 같다 (ex: `a == b`)
- `!=`: 다르다 (ex: `c != d`)
- `<` : 작다 (ex: `e < f`)
- `>` : 크다 (ex: `g > h`)
- `<=` : 작거나 같다 (ex: `i <= j`)
- `>=` : 크거나 같다 (ex: `k >= l`)

```c
// 비교 연산자 예제
int a = 10;
int b = 5;
bool isEqual = a == b;    // isEqual은 false로 설정된다.
bool isGreater = a > b;   // isGreater는 true로 설정된다.
```

### **4. 논리 연산자**

논리 연산자는 조건을 평가하는 역할을 하는 거야!

- `&&`: 그리고 (ex: `a && b`)
- `||`: 또는 (ex: `c || d`)

```c
// 논리 연산자 예제
bool a = true;
bool b = false;
bool isTrue = a && b;    // isTrue는 false로 설정된다.
```

### **5. 단항 연산자**

단항 연산자는 하나의 값을 취해 다른 값으로 변환하는 역할을 하는 거야!

- `!` : 부정 (ex: `!a`)
- `++`: 전치 증가 (ex: `++a`)
- `--`: 전치 감소 (ex: `--b`)

```c
// 단항 연산자 예제
int a = 10;
bool isFalse = !a;    // isFalse는 false로 설정된다.
```

💡 이것들은 거의 모든 프로그램에서 사용되는 기본적인 연산자들입니다. 코딩을 시작하는 초보자는 꼭 이런 기본기를 익혀야 합니다! 😊

**연산의 중요성**

연산은 데이터를 처리하고 변환하여 의미 있는 정보로 만드는 핵심 과정이야. 🔍 그리고 코딩도 결국 이렇게 데이터를 다루는 것에요! 👨‍💻 연산을 잘 이해하고 사용하면, 더 효과적으로 코딩을 하고 프로그램을 만들 수 있어요.

### **연산의 예시**

- 계산기: 입력된 두 수를 더한 결과를 출력하는 프로그램
```c
// 간단한 계산기 프로그램
#include <stdio.h>

int main() {
    int num1, num2;
    printf("두 숫자를 입력하세요:\n");
    scanf("%d %d", &num1, &num2);
    printf("%d + %d = %d\n", num1, num2, num1 + num2);
    return 0;
}
```

- 자산 관리 프로그램: 사용자의 자산을 계산하여 표시하는 프로그램
```c
// 간단한 자산 관리 프로그램
#include <stdio.h>

int main() {
    int asset;
    printf("자산을 입력하세요:\n");
    scanf("%d", &asset);
    printf("합계: %d\n", asset * 1000); // 예시 코드, 실제에는 더 복잡하게 해야 함.
    return 0;
}
```
