**구조체: 자료형을 구조적으로 정의하는 마스터스**

안녕하세요, 여러분! 오늘은 구조체에 대해 배울 겁니다. 구조체는 데이터를 저장하고 조작하기 위한 기본적인 빌딩 블록입니다. 이 강좌에서 우리는 구조체의 개념과 사용법을 배우게 될 것입니다. 그리고 나서, 실무에서 어떻게 적용할 수 있는지도 살펴볼 것입니다.

### 구조체란 무엇인가?

구조체는 관련된 데이터를 하나의 단위로 묶어주는 자료형입니다. 예를 들어, 학생의 정보를 저장하는 경우, 이름, 학번, 전화번호 등과 같은 다양한 데이터가 필요합니다. 이때, 구조체를 사용하면 이러한 데이터를 한꺼번에 관리할 수 있습니다.

```c
struct Student {
    char name[20];
    int studentID;
    char phoneNum[15];
};
```

위의 코드에서는 `Student`라는 이름으로 구조체를 정의하고 있습니다. 구조체 내부에는 `name`, `studentID`, `phoneNum` 등의 필드가 존재합니다.

### 구조체 필드

구조체 필드는 구조체 내부에 있는 데이터의 단위입니다. 필드는 타입과 크기를 지정할 수 있으며, 이름을 부여할 수도 있습니다.

```c
struct Student {
    char name[20]; // 이름은 20자까지 입력할 수 있습니다.
    int studentID; // 학번은 정수형인데, 최소 1개, 최대 8개의 자리 수입니다.
    char phoneNum[15]; // 전화번호는 10~14자의 숫자 문자열입니다.
};
```

위의 예에서 `name` 필드는 문자열이기 때문에 `char[]` 타입을 사용했습니다. `studentID` 필드는 정수형인데, 크기는 `short int`와 동일합니다.

### 구조체 인스턴스

구조체 인스턴스는 구조체를 실제로 생성한 것입니다. 예를 들어, 다음과 같이 `Student` 구조체의 인스턴스를 만들 수 있습니다:

```c
struct Student student1 = {"홍길동", 20120001, "010-1234-5678"};
```

이 코드에서는 `student1`이라는 이름으로 `Student` 구조체 인스턴스를 생성했습니다. 필드에 값을 넣는 것을 "**구조체 초기화**"라고 합니다.

### 구조체 메소드

구조체 메소드는 구조체 내부의 함수입니다. 구조체 메소드를 사용하면 구조체 데이터를 쉽게 조작할 수 있습니다.

```c
void printStudent(struct Student student) {
    printf("이름: %s\n", student.name);
    printf("학번: %d\n", student.studentID);
    printf("전화번호: %s\n", student.phoneNum);
}
```

위의 코드에서는 `printStudent`라는 이름으로 구조체 메소드를 정의했습니다. 이 메소드는 구조체 인스턴스를 매개변수로 받고, 필드 값을 출력합니다.

### 실무 활용

구조체는 다양한 프로그래밍 상황에서 사용할 수 있습니다. 예를 들어, 학생 정보 관리 시스템을 만들 때, 구조체를 사용하면 데이터 관리가 더 쉬워집니다.

```c
int main() {
    struct Student student1 = {"홍길동", 20120001, "010-1234-5678"};
    printStudent(student1);
    
    return 0;
}
```

이 코드에서는 `student1`이라는 이름으로 구조체 인스턴스를 생성하고, `printStudent` 메소드를 호출합니다.

### 마무리

구조체는 프로그래밍에서 중요한 자료형입니다. 관련된 데이터를 하나의 단위로 묶어주는 역할을 하며, 실무 활용에 편리합니다. 이러한 개념과 코드 예제를 통해 여러분은 구조체를 효과적으로 사용하는 것을 배울 수 있었습니다.
