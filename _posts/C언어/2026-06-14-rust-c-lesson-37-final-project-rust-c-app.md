### **1단계: 프로젝트 틀 만들기**

먼저, 기본적인 Rust 프로젝트 구조를 설정해 봅시다. 마치 멋진 집을 짓기 전에 기초 공사를 하는 것과 같죠!

**코드 예제 1: 프로젝트 디렉토리 구조**

```rust
# 프로젝트 폴더 구조 예시
my_quiz_manager/
├── src/
│   └── main.rs  // 핵심 코드 파일
├── Cargo.toml // 프로젝트 설정 파일
```

* **Cargo.toml:** Rust 프로젝트의 필수 파일입니다. 프로젝트 이름, 버전, 의존성 등을 정의합니다.

```toml
[package]
name = "my_quiz_manager"
version = "0.1.0"
edition = "2021"
```

* **src/main.rs:** 애플리케이션의 주요 로직이 들어가는 파일입니다.

### **2단계: 사용자 입력과 퀴즈 생성**

이제 사용자로부터 단어를 입력받아 간단한 퀴즈를 생성해 보겠습니다. 마치 마법사가 주문을 외듯이 코드를 통해 퀴즈를 만들어내는 거죠! 🧙‍♂️

**코드 예제 2: 사용자 입력 및 퀴즈 생성**

```rust
use std::io; // 입출력 처리를 위한 모듈

fn main() {
    println!("비밀 단어를 입력하세요: "); // 사용자에게 입력 요청

    let mut input = String::new(); // 사용자 입력을 저장할 변수
    io::stdin().read_line(&mut input).expect("입력 실패!"); // 입력 받기

    let secret_word = input.trim(); // 공백 제거

    println!("퀴즈: {}는 무엇일까요?", secret_word); // 퀴즈 출력

    // ... (정답 확인 및 점수 계산 로직 추가)
}
```

* **`use std::io;`:** 입출력 작업을 위한 표준 라이브러리를 가져옵니다.
* **`io::stdin().read_line(&mut input)`:** 사용자 키보드 입력을 읽어 `input` 변수에 저장합니다.
* **`input.trim()`:** 사용자 입력 앞뒤 공백을 제거합니다.

**다양한 퀴즈 생성 방식** 🎉

* **`for` 루프:** 여러 단어를 저장한 리스트에서 무작위로 단어를 선택하여 퀴즈를 만들 수 있습니다. (예: `for` 루프를 이용해 단어 목록을 순회하며 퀴즈 생성)
* **`match` 문:** 입력 단어에 따라 다른 퀴즈 유형을 제공할 수 있습니다. (예: `match` 문으로 입력 단어에 따라 난이도 조절)
* **`if` 문:** 특정 조건에 따라 퀴즈 난이도를 조절할 수 있습니다. (예: 입력 단어 길이에 따라 난이도 변경)

### **3단계: 정답 확인 및 점수 관리**

사용자 답변을 받아 정답 여부를 판단하고 점수를 누적해 보세요! 마치 게임 마스터처럼 정확하게 판정하고 결과를 알려주는 역할을 수행하는 거죠! 🎉

**코드 예제 3: 정답 확인 및 점수 계산**

```rust
use std::io;

fn main() {
    // ... (이전 코드 생략)

    println!("정답을 입력하세요 (예/아니오): ");
    let mut guess = String::new();
    io::stdin().read_line(&mut guess).expect("입력 실패!");

    let guess = guess.trim().to_lowercase(); // 입력 소문자 변환

    if guess == "예" && secret_word == "apple" {
        println!("정답입니다! +10점");
        // 점수 변수 선언 및 업데이트 로직 추가
    } else {
        println!("틀리셨습니다. 다음 퀴즈로 넘어갑니다.");
        // 다음 퀴즈로 넘어가는 로직 추가
    }

    // ... (스코어 출력 및 저장 로직 추가)
}
```

* **`guess.trim().to_lowercase()`:** 사용자 입력을 공백 제거 후 소문자로 변환하여 정확한 비교를 위해 처리합니다.
* **`if guess == "예" && secret_word == "apple"`:** 정답 여부를 확인하고 점수를 부여하는 로직입니다.

**✨ 점수 관리 꿀팁 ✨**

* **변수 활용:** 점수를 저장할 변수를 선언하고, 정답 시 점수를 누적합니다. (예: `let score = 0;` 초기화 후 정답 시 `score += 10;`)
* **파일 저장:** `std::fs` 모듈을 사용하여 점수를 파일에 저장하고, 게임 종료 시 불러와 지속적으로 유지할 수 있습니다. (예: `std::fs::write()` 함수 활용)

### **4단계: 데이터 저장 및 로딩 (선택 사항)**

더욱 풍성한 경험을 위해 퀴즈 데이터를 파일에 저장하고 불러올 수 있는 기능을 추가해 보세요! 마치 게임 속 아이템을 저장하고 불러오는 것처럼, 사용자 데이터를 영구적으로 보관할 수 있습니다.

**코드 예제 4: 간단한 데이터 저장 및 로딩**

```rust
use std::fs::{File, OpenOptions}; // 파일 입출력
use std::io::{Write, Read};

fn save_data(data: &str) {
    let mut file = OpenOptions::new().write(true).create(true).open("quiz_data.txt").expect("파일 열기 실패!");
    writeln!(file, "{}", data).expect("쓰기 실패!");
    println!("데이터 저장 완료!");
}

fn load_data() -> String {
    let mut file = File::open("quiz_data.txt").expect("파일 열기 실패!");
    let mut contents = String::new();
    file.read_to_string(&mut contents).expect("읽기 실패!");
    contents
}

fn main() {
    // ... (이전 코드 생략)

    // 퀴즈 데이터 저장 예시
    save_data(format!("퀴즈: {}는 무엇일까요? 정답: 사과", secret_word));

    // 로딩 예시 (게임 초기화 시 사용)
    let loaded_data = load_data();
    println!("로드된 데이터: {}", loaded_data);
}
```

* **`OpenOptions::new().write(true).create(true).open("quiz_data.txt")`:** 파일을 쓰기 모드로 열거나 없으면 생성합니다.
* **`writeln!(file, "{}", data)`:** 파일에 데이터를 씁니다.
* **`file.read_to_string(&mut contents)`:** 파일 내용을 문자열로 읽습니다.
