### **1. 논리 연산자란 무엇일까요?** 🧐

우리가 일상에서 사용하는 '그리고', '또는', '아니'와 같은 단어들이 있잖아요? 🤔 이런 말들로 생각을 연결하고, 진실인지 거짓인지 판단하는 것을 논리 연산이라고 해요. C 언어에서는 이러한 논리를 구현하기 위해 **AND, OR, NOT** 세 가지 논리 연산자를 사용해요! 💪

### 2. AND (그리고) 연산자 (&) - 동의자가 모두 참일 때만 승리하는 용감한 전략 🚀

`&` 연산자는 두 값이 **모두 참(true)**일 때만 **참(true)**을 반환해요. 마치 둘 다 성적이 최고인 학생들끼리 만나서 '우리 함께 이걸 해냈다!'라고 외치는 것과 같은 느낌이에요! 🎉  

```c
#include <stdio.h>

int main() {
  int x = 1, y = 1; // x와 y 모두 참(true) 
  int result = (x & y); // AND 연산자를 사용하여 두 값을 연결합니다.
  printf("result: %d\n", result); // 출력 결과는 1 (참) 입니다.

  return 0;
}
```


💡 이 예제에서 `x`와 `y`가 모두 1이므로, AND 연산자는 참으로 판단하여 `result`에 1을 저장합니다. 마치 두 친구가 함께 승리한 것과 같아요! 🤩

### 3. OR (또는) 연산자 (|) - 적어도 하나라도 참일 때면 이기는 상황 😜

`|` 연산자는 두 값 중 **적어도 하나가 참(true)**이면 **참(true)**을 반환해요! 마치 한 번 시험에 합격하면 되는 상황과 같아요. 😎 하지만, 두 친구가 모두 합격했을 때 더욱 기쁘지? 🎉

```c
#include <stdio.h>

int main() {
  int x = 1, y = 0; // x는 참(true), y는 거짓(false)
  int result = (x | y); // OR 연산자를 사용하여 두 값을 연결합니다.
  printf("result: %d\n", result); // 출력 결과는 1 (참) 입니다.

  return 0;
}
```


💡 `x`가 참(true)이므로, OR 연산자는 결과값을 참으로 판단하여 `result`에 1을 저장합니다. 하나라도 참이면 된다는 것을 알 수 있죠? 👍

### 4. NOT (아니) 연산자 (!) - 진실의 반전, 거짓에게 '진짜!'를 선물하는 마법 ✨

`!` 연산자는 값이 **참(true)**일 때 **거짓(false)**으로, **거짓(false)**일 때 **참(true)**로 변환해요. 💥 마치 마법사의 손길처럼 세상을 뒤집어버리는 거라고 생각하면 됩니다!


```c
#include <stdio.h>

int main() {
  int x = 1; // x는 참(true)
  int result = !x; // NOT 연산자를 사용하여 값을 반전합니다.
  printf("result: %d\n", result); // 출력 결과는 0 (거짓) 입니다.

  return 0;
}
```


💡 `x`가 참(true)이므로, NOT 연산자는 결과값을 거짓으로 판단하여 `result`에 0을 저장합니다. 마법사처럼 세상을 뒤집어버린 것과 같아요! ✨

### 🤔  실무 중 팁 - 논리 연산자는 '진실'이 어떻게 바뀌는지 시각화해 보세요!

* **AND**: 두 친구가 서로 손잡고 있다. (두 값 모두 참일 때만 진행!)
* **OR**: 한 친구만 손을 내밀면 돼. 적어도 하나라도 '참'이 되면 된다. 😎
* **NOT**: 거울 앞에서 자신의 모습을 보는 것과 같다! 반전된 세계가 나타나게 된다.
