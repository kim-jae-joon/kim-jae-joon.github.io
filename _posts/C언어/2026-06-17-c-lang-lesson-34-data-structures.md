### 🤔 데이터 구조란? (간단히 설명)

컴퓨터는 사람처럼 생각하지 못해요. 대신, 우리가 준 '지시'를 따라서 일하는 중독성 강력한 로봇 같은 존재예요! 🤖  그런데, 우리가 그 로봇에게 원하는 작업을 하도록 알려주기 위해서는 **"데이터 구조"**라는 도구가 필요하거든요. 🤔

마치 '장바구니'처럼 다양한 물건들을 효율적으로 정리하고 관리하는 방식이죠! 🧺  자, 그런 "장바구니"의 종류에 대해 알아볼게요.


### 🔥 리스트 - 모든 것을 담을 수 있는 장백곰 🚀

가장 기본적인 데이터 구조 중 하나인 **리스트**는 마치 무한히 펼쳐지는 '연결고리'처럼 작동합니다! 🔗  각 원소를 이어서 연결하고, 앞에서부터 하나씩 접근할 수 있어요.

```c
#include <stdio.h>
int main() {
    // 리스트는 변수로 선언되어야 합니다. 각 요소는 같은 데이터 타입을 가질 수 있습니다.
    int numbers[5] = {1, 2, 3, 4, 5};

    printf("첫 번째 요소: %d\n", numbers[0]); // 출력 결과: 첫 번째 요소: 1
    return 0;
}
```

* ** `int numbers[5]`**: 'numbers'라는 이름으로 5개의 정수를 저장할 수 있는 리스트를 선언! 💪  마치 빵장판에 5개의 구멍이 생겼다라고 생각하면 이해하기 쉬워요. 😎
* **`numbers[0]`**: 'numbers' 리스트에서 첫 번째 요소(인덱스가 0인 곳)를 선택해서 읽는 법!

**💡 초보자 폭풍 질문!**  왜 인덱스는 0부터 시작할까요? 🤔  컴퓨터의 머리가 아주 정리정돈되게 작동하기 때문에, 첫 번째 공간을 0번으로 부여하는 것이 편리하거든요!

### 🚀 스택 - '나중에 필요할' 것들을 모아놓는 책장 🚀

스택은 **"LIFO (Last-In, First-Out)"** 방식으로 데이터를 관리하는 구조예요. 마치 뜨개질처럼 최근에 추가한 요소만 먼저 사용하게 되는 거예요! 🧶  
새로운 요소를 추가하면 '윗단'에 올라가고, 원하는 요소를 가져오려면 가장 위의 요소부터 삭제해야 하죠.

```c
#include <stdio.h>
int main() {
    // 스택은 함수 호출을 돕기 위해 사용됩니다. (마치 괄호처럼)
    char stack[5] = {}; // 공백으로 채워진 스택 선언!
    int top = -1;  // 시작 위치는 '아래'로 설정

    stack[++top] = 'A'; // 최상단에 'A' 추가! 
    stack[++top] = 'B'; // B를 추가하면서 스택 크기가 커지죠.
    printf("스택의 가장 위 요소: %c\n", stack[top]);  // 출력 결과: 스택의 가장 위 요소: B

    return 0;
}
```

* **`stack[++top]`**: 스택에 'A'를 추가하는 부분이죠. 'top' 값을 증가시켜서 새로운 위치를 만들고, 그곳에 데이터를 저장하게 됩니다!
* **`printf("스택의 가장 위 요소: %c\n", stack[top])`**:  "top" 위치에 있는 데이터를 출력하는 부분이죠. 마치 스택 위쪽부터 꺼내서 보는 것과 같아요!


### ♻️ 큐 - 줄 서서 기다리는 사람들처럼 원칙적인 순서대로 🚀

큐는 **"FIFO (First-In, First-Out)"** 방식으로 데이터를 관리하는 구조예요. 마치 버스 대기 시스템처럼, 들어온 순서대로 나가도록 되어있답니다! 🚌

```c
#include <stdio.h>
int main() {
    // 큐는 데이터의 추가와 제거를 정해진 순서대로 수행합니다. (리스트처럼)

    int queue[5] = {}; // 공백으로 채워진 큐 선언!
    int front = 0;  // 앞쪽 인덱스를 나타냅니다.
    int rear = -1;   // 뒤쪽 인덱스를 나타냅니다.


    queue[++rear] = 10; // 큐에 숫자 10 추가!
    queue[++rear] = 20; // 다시 추가하면서 뒷 부분이 이동합니다.

    printf("큐의 앞쪽 요소: %d\n", queue[front]);  // 출력 결과: 큐의 앞쪽 요소: 10


    return 0;
}
```

* **`queue[++rear]`**: 큐에 데이터를 추가하는 부분이죠. 'rear' 값을 증가시켜서 새로운 위치를 만들고, 그곳에 데이터를 저장하게 됩니다!

### 🚀  실무에서 활용되는 주요 데이터 구조 🤔

리스트, 스택, 큐는 각각 고유한 특징을 가지고 있어서 다양한 프로그래밍 문제 해결에 사용됩니다.

* **스택**: 함수 호출 순서 관리, 재귀 함수 실행, Undo/Redo 기능 구현 등에 활용!
* **리스트**: 데이터를 저장하고 정렬하는 데 적합하며, 알고리즘 구현에도 자주 사용됨! (ex: 숫자 정렬)
* **큐**: 작업 스케줄링, 프린터 대기열, 네트워크 통신 등에 활용되는데요.
