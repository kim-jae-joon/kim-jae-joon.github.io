**23강: 구조체의 배열 - 자료형 다루기**

안녕하세요! 이번 강의에서는 여러분이 한번에 다루는 수 많은 자료형들 중에서도 특히나 주목할 만한 **구조체의 배열**을 배워보겠습니다. 💡

### 1️⃣ 왜 구조체의 배열을 공부해야 할까요?

어떤 경우에선 단순히 변수를 선언하고 사용하면 충분하지만, 다른 때는 여러 가지 자료형들의 묶음으로 처리해야 합니다. 이럴 때가 바로 **구조체**와 **배열**의 등장입니다. 구조체란 말 그대로 하나의 변수로 여러 개의 데이터를 저장하는 방식을 말합니다. 그리고 이것을 다루기 위해선 **배열**이 필요해집니다.

하지만, 여기서 중요한 것은 **구조체가 배열에 들어가는 것**이다! 🚀

### 2️⃣ 구조체란 무엇일까요?

구조체는 하나의 변수로 여러 개의 데이터를 저장할 수 있는 자료형입니다. 단순히 말로는 예를 들어, 사람의 이름, 나이, 주소 이렇게 세 가지 정보가 필요하다면, 각각의 변수를 선언해주기보다는 구조체를 사용하는 것이 더 효율적일 것입니다.

```c
struct Person {
    char name[20];
    int age;
    char address[100];
};
```

### 3️⃣ 구조체의 배열은 어떻게 처리할까요?

여기서 중요한 것은, 우리는 구조체 하나하나에 각각 다른 이름을 줄 수 있습니다! 예를 들어, 다음과 같이 표현할 수 있습니다.

```c
struct Person {
    char name[20];
    int age;
    char address[100];
};

int main() {
    struct Person p1, p2; // p1과 p2라는 두 명의 사람 구조체
    return 0;
}
```

### 4️⃣ 구조체를 다루는 방법

구조체에는 여러 가지 방법으로 데이터에 접근할 수 있습니다. 예를 들어, 위에 선언된 `p1`에 대한 이름을 `name`이라는 변수에 저장한 값을 출력하고 싶다면 어떻게 될까요? 그럼 아래와 같은 코드를 쓸 수 있습니다.

```c
printf("%s\n", p1.name);
```

### 5️⃣ 실무에서는 어떻게 적용할까?

실무에서 이런 방법으로 사용하는 예시가 아주 많습니다. 예를 들어, 다음과 같이 학생들의 이름, 나이, 점수를 저장할 때는 구조체와 배열을 이용하여 관리할 수 있습니다.

```c
struct Student {
    char name[20];
    int age;
    float score;
};

int main() {
    struct Student students[5]; // 최대 5명의 학생 정보를 저장할 수 있는 구조체의 배열

    for (int i = 0; i < 5; i++) {
        scanf("%s %d %f", &students[i].name, &students[i].age, &students[i].score);
    }

    for (int i = 0; i < 5; i++) {
        printf("이름: %s, 나이: %d, 점수: %.2f\n", students[i].name, students[i].age, students[i].score);
    }

    return 0;
}
```
