**3강: 데이터 형식과 변수 선언** 🔥

안녕하세요, 여러분! 오늘은 C 언어에서 가장 기초적인 개념을 배울 것입니다. 데이터 형식과 변수 선언이라는 주제로, 초보자들도 쉽게 이해할 수 있도록 가르쳐드리겠습니다.

### 데이터 형식이란? 🤔

데이터 형식이란, 컴퓨터가 데이터를 처리하고 저장하는 방식을 결정짓는 개념입니다. 예를 들어, 숫자 5는 어떻게 저장되고 처리될까요? 이 질문을 해결하기 위해서, C 언어에서는 다양한 데이터 형식을 제공합니다.

#### 정수형 (int)

정수형은 가장 기본적인 데이터 형식 중 하나입니다. 1, 2, 3, ..., -1, -2, -3 ...과 같은 숫자를 표현할 때 사용합니다.

```c
int num = 10;
```

위 코드에서 `num` 변수는 정수형으로 선언되었으며, 초기값은 10입니다. 이 코드를 실행하면 `num`의 값을 출력해 볼 수 있습니다.

#### 실수형 (float)

실수형은 소수점이 있는 숫자를 표현할 때 사용합니다.

```c
float pi = 3.14;
```

위 코드에서 `pi` 변수는 실수형으로 선언되었으며, 초기값은 3.14입니다.

#### 문자열형 (char)

문자열형은 한 글자를 저장할 때 사용합니다.

```c
char name[5] = "hello";
```

위 코드에서 `name` 변수는 문자열형으로 선언되었으며, 초기값은 `"hello"`입니다.

### 변수 선언이란? 📝

변수 선언이란, 데이터 형식을 지정하고 변수 이름을 지어주는 과정입니다. 변수 이름은 일반적으로 대문자로 시작하고, `_`로 연결하여 작성합니다.

#### 변수 이름의 규칙

*   첫 번째 문자는 대문자여야 합니다.
*   이후의 문자는 소문자 또는 `_`로 시작할 수 있습니다.
*   `_`를 사용할 때, `_`가 연속으로 반복되지 않아야 합니다.

### 예제: 데이터 형식과 변수 선언을 활용하는 코드 💻

```c
#include <stdio.h>

int main() {
    // 정수형
    int num = 10;
    printf("정수형 : %d\n", num);

    // 실수형
    float pi = 3.14;
    printf("실수형 : %.2f\n", pi);

    // 문자열형
    char name[5] = "hello";
    printf("문자열형 : %s\n", name);

    return 0;
}
```

위 코드에서는 `printf()` 함수를 사용하여 데이터 형식을 출력합니다. `%d`는 정수형, `%.2f`는 실수형 2자리까지 출력, `%s`는 문자열형을 출력하는 포맷입니다.

### 실무 활용: 🔒

데이터 형식과 변수 선언은 C 언어에서 매우 중요합니다. 잘못된 데이터 형식을 사용하면 오류가 발생할 수 있습니다. 예를 들어, 정수형 값을 실수형 변수에 할당하려고 하면, 컴파일 에러가 발생할 것입니다.

```c
// 오류: int는 float 타입에 할당할 수 없음!
float pi = 10;
```

이처럼 데이터 형식과 변수 선언은 매우 중요하기 때문에, 잘못된 코드를 작성하는 것을 피해야 합니다.

### ❓ 초보자 폭풍 질문! 💡

*   C 언어에서 데이터 형식을 지정할 때, 왜 `int`를 사용해야 합니까?
*   실수형을 사용할 때는 왜 소수점이 반드시 필요하나요?

**답:** 🤔

데이터 형식을 지정할 때 `int`를 사용하는 이유는, 정수형은 정수 값을 표현하기 때문에, 이에 적합한 데이터 형식입니다. 실수형을 사용할 때는 소수점이 반드시 필요합니다. 왜냐하면 실수형은 소수점이 있는 숫자를 표현하기 때문입니다.

### ❗️ 실무주의보! 🚨

데이터 형식을 잘못 지정하거나, 변수 이름을 잘못 지어주면 오류가 발생할 수 있습니다. 예를 들어, 정수형 값을 문자열형 변수에 할당하려고 하면, 오류가 발생할 것입니다.

```c
// 오류: int는 char 타입에 할당할 수 없음!
char name = 10;
```

이처럼 데이터 형식을 잘못 지정하는 것을 피해야 합니다.
