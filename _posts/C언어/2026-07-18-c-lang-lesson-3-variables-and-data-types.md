## 🔥🚀 C 언어 여정, 첫걸음: 변수와 자료형 (int, float, char)!

안녕하세요! 저는 대한민국 최고의 C 언어 강사이고, 15년차 시니어 개발자이기도 합니다! 😎 오늘부터 함께 C 언어를 배우기 시작하겠습니다. 탈출하고 싶은 이 코드 왕관을 위해서죠! 👑

처음 C언어를 접하는 분들을 위한 완벽 가이드라인입니다. 변수와 자료형에 대한 이해는 C 언어의 기본이라고 할 수 있습니다.  지금부터 변수와 자료형 중 가장 중요한 세 가지, **int, float, char** 에 대해 알아보겠습니다!

### 💡  "변수가 뭔지?" 🤔 초보자 폭풍 질문!

`변수`는 코드에서 데이터를 저장하는 이름입니다. 예를 들어, 당신의 이름을 담고 싶다면 "이름"이라는 변수를 만들 수 있습니다. 그런데 이 이름은 단순히 표시되는 것일 뿐 실제로 이름을 담아놓기 위해서는 공간이 필요합니다!  

코드에서는 이 공간을 **변수**라고 부르고, 데이터가 들어갈 **"상자"**와 같다고 생각하면 됩니다. 😉


### 🎉 **int: 정수의 왕관!** 👑

`int`는 **정수(Integer)**를 나타내는 자료형입니다.  음수, 0 또는 양수 같은 딱딱한 정수 값을 저장하는 데 사용합니다. 

* 예시: 사람 수, 나이, 가격 등이 `int` 형으로 표현됩니다.
* 최대 저장 가능한 값은 컴퓨터의 성능에 따라 다릅니다. 일반적으로 큰 수를 저장할 수 있죠!


```c
#include <stdio.h>

int main() {
  // 사람의 나이를 저장하는 변수
  int age = 25; // 25라는 정수 값을 'age' 변수에 저장합니다!
  printf("나이는 %d 입니다.\n", age); // "나이는 25 입니다." 를 화면에 출력합니다.
  return 0;
}
```

> **주의:** `int`는 마치 주머니 같아요, 정수만 담고 있어야 합니다!  小数점이 있는 값은 저장할 수 없으니까요!

### 🌊 **float: 실수의 바다** 🐠

`float`는 **실수(Floating-Point)**를 나타내는 자료형입니다. 소수점을 가진 값을 저장하는 데 사용합니다.


* 예시: 온도, 높이, 무게 등이 `float` 형으로 표현됩니다.
* `int`보다 더 많은 수의 자릿수까지 정확하게 담을 수 있죠!

```c
#include <stdio.h>

int main() {
  // 물체의 무게를 저장하는 변수 (소수점 포함)
  float weight = 78.5; // 78.5라는 실수 값을 'weight' 변수에 저장합니다!
  printf("무게는 %.2f 입니다.\n", weight); // "무게는 78.50 입니다." 를 화면에 출력합니다.
  return 0;
}
```

> **꿀팁:** `%.2f`는 소수점 이하 2자리까지 표시하는 형식입니다. 😉


### 🔤 **char: 문자의 세상** 🎉

`char`는 **문자(Character)**를 나타내는 자료형입니다. 한 글자를 저장하는 데 사용합니다.

* 예시: 이름, 성, 알파벳 등이 `char` 형으로 표현됩니다.
* 특수 기호나 문자가 포함된 '사랑' 같은 단어도 여러 개의 `char`로 만들 수 있습니다!


```c
#include <stdio.h>

int main() {
  // 사용자 이름을 저장하는 변수
  char initial = 'J'; // 'J'라는 문자를 'initial' 변수에 저장합니다!
  printf("이름의 첫 글자는 %c 입니다.\n", initial); // "이름의 첫 글자는 J 입니다." 를 화면에 출력합니다.
  return 0;
}```

> **주의:** `char`는 문자가 들어가는 상자라고 생각하세요! 숫자나 기타 다른 데이터를 담으면 에러가 발생할 수 있습니다.
