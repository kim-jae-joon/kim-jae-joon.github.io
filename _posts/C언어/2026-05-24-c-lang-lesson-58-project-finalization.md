##  🔑 포트폴리오, 당신의 가치를 보여주는 열쇠🗝️

포트폴리오는 프로그래머로서의 실력과 경험을 보여줄 수 있는 가장 중요한 도구입니다! 💪 
기업들은 단순히 이론적인 지식만으로는 채용 결정을 내리지 않습니다. 오늘부터 시작하여 프로젝트를 통해 얻은 실무 경험을 담아보세요! 😎

###  🎉 포트폴리오 제작의 장점 🎉

* **실력 증명:**  자신의 코딩 능력을 구체적으로 보여줄 수 있습니다.
* **적극성 표현:**  취업 시기, 새로운 기술에 대한 적극적인 배우는 자세를 보여줍니다.
* **프로젝트 경험 축적:**   실제 프로젝트 환경에서 문제 해결 능력과 팀워크를 키울 수 있습니다.

### 🚀 포트폴리오 제작 전략 🚀

1.  **간단한 프로젝트부터 시작:** 처음엔 조금 어려운 프로젝트보다 간단한 것을 선택하세요! 🤔
2. **자신의 관심 분야 활용:** 게임, 웹사이트, 애플리케이션 등 자신이 좋아하는 분야를 골라보세요! 💖
3. **깃허브 활용:**  프로젝트 소스 코드를 공유하여 전문가들에게 피드백을 받고 네트워킹을 구축하세요! 🤝

## 💪 프로젝트 구현, 이제부터 실력 UP시작!

### 🤔 어떤 프로젝트를 만들까? 💡

* **계산기:**  숫자 연산기능(덧셈, 뺄셈, 곱셈, 나눗셈)을 구현하여 기본적인 알고리즘 이해도를 높여보세요!
    
* **텍스트 게임:** 텍스트 입력 및 출력으로 이루어진 간단한 게임 (예: 추측게임, 사다리 타기 등)을 만들면서 문자열 처리와 조건문에 익숙해지세요! 😎

* **숫자 맞추기 게임:**   컴퓨터가 임의로 정수를 생성하고 사용자가 숫자를 입력하여 맞추는 게임을 구현하는 방법입니다.
    *  코드 설명: C언어에서는 `srand()` 함수를 사용하여 임의 수 발생 시스템을 설정한 후, `rand()` 함수를 통해 무작위 정수를 생성합니다.   

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
  int number = rand() % 100 + 1; // 컴퓨터가 임의로 1~100 사이의 숫자를 생성합니다.
  int guess;
  printf("1에서 100 사이의 숫자를 맞춰보세요!\n");

  do {
    printf("입력: ");
    scanf("%d", &guess);

    if (guess < number) {
      printf("더 크게! \n");
    } else if (guess > number) {
      printf("더 작게! \n");
    } else {
      printf("정답입니다!\n");
    }
  } while (guess != number);

  return 0;
}
```

* **사용자 정보 입력 및 출력:** 사용자 이름, 나이, 주소 등을 입력받아 화면에 출력하는 프로그램을 만들어 데이터 처리와 변수 활용에 대해 이해합니다.
