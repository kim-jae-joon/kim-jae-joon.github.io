## 🚀 4강: C언어 기초 탐험대! 🚀  제어의 마법: if-else & 반복의 신비 (for 루프)

안녕하세요, 코딩 새싹 여러분! 👋  오늘은 **제어 구조**라는 마법의 세계로 여러분을 안내할게요.  이제부터 여러분은 **프로 프로그램 조종사**가 될 준비 완료! 🤖✨

**진짜 신기하죠?** 컴퓨터는 단순히 명령을 따라하는 기계처럼 보이지만, 실제로는 우리가 **만약... 한다면... 아니면...** 와 같은 논리적인 생각을 할 수 있도록 도와주는 **제어 구조**를 사용해서 훨씬 지능적인 행동을 합니다. 마치 로봇이 "**지금 비가 온다! → 우산 챙기기**" 와 같은 판단을 내리는 것과 같죠!

### 🧩 if-else: 선택지의 마법사

**if-else** 문은 컴퓨터에게 **'만약 특정 조건이 맞으면 이렇게, 아니면 이렇게!'** 라는 선택지를 제시하는 마법 지팡이입니다. 마치 신호등처럼, 상황에 따라 다른 행동을 취하게 만드는 거죠!

**💡 핵심 개념:**

* **조건문 (Condition):** 특정 조건이 참(True)인지 거짓(False)인지 판단합니다.
* **괄호 (Parentheses):** 조건을 명확하게 감싸주세요!
* **명령어 블록 (Block):**  if 문 또는 else 문 뒤에 오는 코드들을 그룹화합니다. **들여쓰기 필수!**  C언어는 엄격한 규칙쟁이니까요 😉

**✨ 예시 코드 & 해체:**

```c
#include <stdio.h>

int main() {
    int score = 85;  // 학생 점수 저장

    // 만약 점수가 90 이상이면 'A등급', 아니면 'B등급'
    if (score >= 90) {  // 조건: 점수가 90 이상인가?
        printf("축하합니다! A등급!\n"); // 참일 경우 실행
    } else {           // 조건이 거짓일 경우
        printf("B등급입니다.\n"); // 실행
    }

    return 0;
}
```

**코드 분석:**

1. **`int score = 85;`**: 학생의 점수를 변수 `score`에 저장합니다. 마치 점수를 주머니에 넣는 것처럼요!
2. **`if (score >= 90) { ... }`**: 점수가 90 이상인지 확인합니다. 
    * **`score >= 90`**:  이 부분이 **조건문**입니다.
    * **괄호 안 코드 블록**: 점수가 90 이상이면 "축하합니다! A등급!"을 출력합니다.
3. **`else { ... }`**: 만약 점수가 90 미만이면 실행되는 코드 블록입니다.
4. **`printf(...)`**: 텍스트를 화면에 출력하는 함수입니다. 마치 마법 지팡이로 메시지를 쏘는 것처럼요!

**💡 초보자 폭풍 질문!**

* **Q: 들여쓰기는 왜 중요한가요?**
* **A:** C언어는 들여쓰기 규칙을 통해 코드 블록을 구분합니다. 들여쓰기가 일치하지 않으면 컴파일러가 오류를 발생시킬 수 있어요. 규칙을 지키는 것이 안전 운전과 같아요!

**다양한 조건문 활용**

* **다중 조건 (AND, OR):** 여러 조건을 결합하여 더 복잡한 판단을 할 수 있습니다. 예를 들어, `if (score >= 90 && grade >= 'B') { ... }` 처럼 사용할 수 있습니다.
* **switch 문:** 여러 개의 선택지 중 하나를 선택하는 데 유용합니다. (다음 강의에서 자세히 살펴볼게요!)

### 🕹️ 반복의 마법: for 루프 & while 루프

**반복**은 프로그래밍에서 핵심 마법 중 하나입니다! 특정 작업을 **여러 번** 수행해야 할 때 빛을 발합니다. 마치 로봇이 "**같은 동작을 10번 반복!**" 하는 것처럼요!

**🌟 for 루프: 정확한 반복 횟수 미리 알고 있을 때**

for 루프는 시작, 조건, 증가/감소 단계를 명확하게 정의하여 정확한 횟수만큼 반복하는 데 탁월합니다. 마치 레시피를 따라 정확한 재료를 정확히 넣는 것과 같죠!

**✨ 예시 코드 & 해체:**

```c
#include <stdio.h>

int main() {
    int i; // 카운터 변수

    // 1부터 5까지 숫자 출력 (for 루프)
    for (i = 1; i <= 5; i++) { // 초기값, 조건, 증가식
        printf("%d\n", i); // 반복시마다 i 값 출력
    }

    return 0;
}
```

**코드 분석:**

1. **`int i;`**: 반복 횟수를 추적할 변수 `i`를 선언합니다. 마치 달리기 타이머처럼요!
2. **`for (i = 1; i <= 5; i++) { ... }`**:
    * **`i = 1`**: 루프 시작 시 `i`를 1로 초기화합니다.
    * **`i <= 5`**: `i`가 5보다 작거나 같을 동안 루프가 계속됩니다.
    * **`i++`**: 각 반복 후 `i`를 1 증가시킵니다. 마치 계단을 한 계단씩 오르는 것처럼!
3. **`printf("%d\n", i);`**: 현재 `i` 값을 출력합니다.

**🚨 실무주의보:**

* **무한 루프 주의:** `조건`이 항상 참인 경우 루프가 멈추지 않고 계속 실행될 수 있습니다 (**위험 경고!** 디버깅 필수!).

**🤸️ while 루프: 예측 불가능한 반복**

while 루프는 **'조건이 참일 때까지 계속'** 실행되는 루프입니다. 예상치 못한 상황에서도 계속해서 작업을 수행해야 할 때 유용합니다. 마치 탐험가가 **'특정 신호를 받을 때까지 계속 탐사!'** 하는 것과 같습니다.

**✨ 예시 코드 & 해체:**

```c
#include <stdio.h>

int main() {
    int num = 0;
    // 사용자가 5 이하의 숫자를 입력할 때까지 반복
    while (num <= 5) { // 조건: num이 5 이하인가?
        printf("숫자를 입력하세요 (5 이하): ");
        scanf("%d", &num); // 사용자 입력 받기
        printf("입력하신 숫자는 %d입니다.\n", num);
    }
    printf("그만뒀습니다!\n");
    return 0;
}
```

**코드 분석:**

1. **`int num = 0;`**: 사용자 입력을 저장할 변수 `num`을 초기화합니다.
2. **`while (num <= 5) { ... }`**: 사용자가 5 이하의 숫자를 입력할 때까지 루프가 계속됩니다.
3. **`scanf("%d", &num);`**: 사용자로부터 숫자 입력을 받습니다. 마치 마법사가 주문을 외우듯이!
4. **출력문**: 입력받은 숫자를 보여줍니다.

**다양한 반복 구조 활용**

* **do-while 루프:** while 루프와 비슷하지만 조건 검사를 루프 **끝부분**에서 합니다. 처음 실행 시 조건이 거짓이어도 루프가 한 번 실행됩니다. (다음 강의에서 자세히 알아볼게요!)

**💡 초보자 폭풍 질문!**

* **Q: for 루프와 while 루프의 차이점은 무엇인가요?**
* **A:**  **for 루프**는 반복 횟수를 **미리 알고 있을 때** 사용하기 좋습니다 (예: 1부터 10까지 반복). **while 루프**는 **조건이 변할 때**까지 반복해야 할 때 유용합니다 (예: 사용자 입력에 따른 반복).
