## 🔥12강: 구조체 - C언어의 건축 블록, 이제 제대로 이해할 시간!🔥

안녕하세요! 저는 대한민국 최고의 C 언어 강사(자신감 UP!)이자 15년 차 시니어 개발자 **'코딩천재 김철수'**입니다. 😎 오늘은 C 언어 프로그래밍을 위한 필수 요소인 **구조체**에 대해 깊이 있게 알아보겠습니다!

구조체는 여러 변수를 하나의 단위로 묶어서 관리하는 데 사용되는, 마치 Lego 블록처럼 프로그램을 더욱 효율적이고 유연하게 만들어주는 신기한 도구입니다. 🤯 그렇다면 구조체는 정확히 어떻게 작동할까요?  

### 🧱  구조체의 기본 개념: 정보를 통합하는 마법! ✨

C 언어에서 변수들은 특정 자료 유형에 따라 저장 공간과 처리 방식이 다릅니다. 예를 들어, 숫자는 정수 또는 실수로 저장되고, 문자열은 순서대로 연결된 글자들로 저장됩니다. 🤔 하지만 어떤 프로그램에서는 여러 가지 다른 자료 유형을 하나의 단위로 사용해야 하는 경우가 생길 수 있습니다. 

예를 들어,  '사람' 정보를 저장하고 싶다고 생각해봅시다. 사람에게는 이름(문자열), 나이(정수), 키(실수)와 같은 여러 가지 정보가 필요합니다. 이런 정보들을 각각 별개의 변수로 관리한다면 코드가 복잡해지고, 프로그램의 유연성도 떨어집니다. 😔

바로 여기서 **구조체**가 등장하는 것입니다! 구조체는  다양한 자료 유형을 하나의 단위로 결합하여 '사람'과 같은 객체를 표현할 수 있게 해줍니다. 🤩 즉, 구조체는 정보를 효율적으로 통합하고 관리하는 데 필수적인 도구입니다.

###  📝 구조체 선언: 우리 블록을 만드세요!

C 언어에서 구조체를 정의하려면 `struct` 키워드를 사용합니다. 구조체 이름은 자신만의 특별한 이름으로 만들 수 있습니다. 🦸‍♀️ 각 속성(필드)에는 자료 유형이 명시되어야 합니다. 이제 마치 블록을 만드는 것처럼 우리 자신의 구조체를 선언할 준비가 되겠죠?

```c
struct 사람 {
    char 이름[50];  // 이름, 최대 50글자까지 저장 가능!
    int 나이;        // 나이는 정수로 표현됩니다.
    float 키;       // 키는 소수점을 가지고 있는 실수로 표현합니다.
};
```

이 코드를 살펴보면 `struct` 키워드로 구조체 이름을 '사람'으로 선언하고, 쉼표(,)로 분리된 세 가지 속성인 '이름', '나이', '키'를 정의했습니다. 각 속성은 저장하는 자료 유형에 따라 'char', 'int', 'float' 등으로 표현됩니다.

### 🚀 구조체 초기화: 블록을 채우는 시간!

구조체를 선언한 후에는 실제 데이터를 저장하여 사용할 수 있습니다. 이 과정을 **초기화**라고 합니다. 💡 초보자들은 이 부분에서 주의가 필요합니다! 🚨


```c
struct 사람 민수;  // '사람' 구조체를 하나 만들었으니, 민수라는 이름으로 불러서 사용할 수 있습니다!

민수.이름 = "민수";   // 민수님의 이름을 저장합니다.
민수.나이 = 25;       // 민수님의 나이는 25세입니다.
민수.키 = 178.5;    // 민수님의 키는 178.5cm 입니다.

```

여기서 `민수`라는 변수가 '사람' 구조체를 하나 나타냅니다. 그리고 점(.) 연산자를 사용하여 각 속성을 접근하여 값을 설정합니다. 

### 🔍 구조체 활용: 정보를 얻고 분석하기!

이제 마지막으로, 구조체를 이용해 프로그램에서 데이터를 읽어오거나 처리하는 방법을 살펴보겠습니다. 😄


```c
printf("민수님의 이름은 %s입니다.\n", 민수.이름); // 민수님의 이름 출력!
printf("나이는 %d세이고 키는 %.1f cm입니다.\n", 민수.나이, 민수.키); 
```

이 코드에서 `printf` 함수를 사용하여 구조체 속성에 저장된 값을 읽어와 화면에 출력합니다.
