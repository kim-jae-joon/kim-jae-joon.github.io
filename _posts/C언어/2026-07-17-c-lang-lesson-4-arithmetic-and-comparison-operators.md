### 산술 연산자: 컴퓨터의 심장! ❤️‍🔥

산술 연산자는 **간단한 계산**을 담당하는 핵심 요소입니다. 마치 우리가 식탁 위에서 저녁을 만들 때 쓰는 요리 도구처럼, C언어 코드를 통해 숫자를 다루고 원하는 결과물을 얻게 해줍니다!

> **💡 초보자 폭풍 질문!**
>  "C언어로 계산해도 정말 기본적인 연산이죠? 저는 이것까지 못하니...!"
>   걱정 마세요! C언어는 우리가 일상에서 사용하는 간단한 수학 연산을 완벽하게 지원합니다. 이 강의를 통해 컴퓨터에게 원하는 계산을 명령하고, 놀라운 결과를 얻도록 가르쳐 드릴게요! 🚀

**1️⃣ 덧셈 (+)**: 두 숫자를 합치는 연산입니다.
> ```c
> int a = 5;
> int b = 3;
> int sum = a + b; // sum은 8이라는 값을 가집니다.
> ```

**2️⃣ 뺄셈 (-)**: 두 숫자를 빼는 연산입니다.
> ```c
> int a = 10;
> int b = 4;
> int difference = a - b; // difference은 6이라는 값을 가집니다.
> ```

**3️⃣ 곱셈 (*)**: 두 숫자를 곱하는 연산입니다.
> ```c
> int a = 2;
> int b = 7;
> int product = a * b; // product은 14이라는 값을 가집니다.
> ```

**4️⃣ 나눗셈 (/)**: 두 숫자를 나누는 연산입니다. **실수** 결과가 발생할 수 있습니다.
> ```c
> int a = 10;
> int b = 3;
> float quotient = (float)a / b; // quotient은 3.3333...이라는 값을 가집니다.
> ```

**5️⃣ 나머지 (%)**: 두 숫자를 나누었을 때, 나머지를 계산하는 연산입니다.
> ```c
> int a = 10;
> int b = 3;
> int remainder = a % b; // remainder는 1이라는 값을 가집니다.
> ```


### 비교 연산자: 논리적인 판단에 최고! 🧐

비교 연산자는 두 값을 비교하여 **참(true) 또는 거짓(false)** 을 결과로 반환합니다. 마치 컴퓨터에게 "정답이 맞니?" 라고 물어보는 것과 같아요! 💡

> **🚨 실무주의보!**
>  비교 연산자는 C언어 프로그램의 논리흐름을 제어하는 중요한 역할을 합니다. 조건문(if-else)이나 반복문(for, while)에서 자주 사용되어, 정확한 비교를 통해 원하는 결과를 얻을 수 있도록 도와줍니다! 💪

**1️⃣ 등호 연산자 (==)**: 두 값이 같으면 **true**, 다르면 **false** 를 반환합니다.
> ```c
> int a = 5;
> int b = 5;
> if (a == b) { // true
>   printf("두 값은 같습니다.\n");
> } else { // false
>   printf("두 값이 다릅니다.\n");
> }
> ```

**2️⃣ 불등호 연산자 (!=)**: 두 값이 서로 다르면 **true**, 같으면 **false** 를 반환합니다.
> ```c
> int a = 10;
> int b = 5;
> if (a != b) { // true
>   printf("두 값은 다릅니다.\n");
> } else { // false
>   printf("두 값이 같습니다.\n");
> }
> ```

**3️⃣ 큰것보다 연산자 (>)**: 첫 번째 값이 두 번째 값보다 크면 **true**, 그렇지 않으면 **false** 를 반환합니다.
> ```c
> int a = 15;
> int b = 8;
> if (a > b) { // true
>   printf("a는 b보다 큽니다.\n");
> } else { // false
>   printf("a는 b보다 작거나 같습니다.\n");
> }
> ```

**4️⃣ 작은것보다 연산자 (<)**: 첫 번째 값이 두 번째 값보다 작으면 **true**, 그렇지 않으면 **false** 를 반환합니다.
> ```c
> int a = 3;
> int b = 10;
> if (a < b) { // true
>   printf("a는 b보다 작습니다.\n");
> } else { // false
>   printf("a는 b보다 크거나 같습니다.\n");
> }
> ```

**5️⃣ 큰것 또는 연산자 (>=)**: 첫 번째 값이 두 번째 값보다 크거나 같으면 **true**, 그렇지 않으면 **false** 를 반환합니다.
> ```c
> int a = 7;
> int b = 7;
> if (a >= b) { // true
>   printf("a는 b보다 크거나 같습니다.\n");
> } else { // false
>   printf("a는 b보다 작습니다.\n");
> }
> ```

**6️⃣ 작은것 또는 연산자 (<=)**: 첫 번째 값이 두 번째 값보다 작거나 같으면 **true**, 그렇지 않으면 **false** 를 반환합니다.
> ```c
> int a = 2;
> int b = 5;
> if (a <= b) { // true
>   printf("a는 b보다 작거나 같습니다.\n");
> } else { // false
>   printf("a는 b보다 큽니다.\n");
> }
> ```
