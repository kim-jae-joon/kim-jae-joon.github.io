### 5강: 조건문 - 만약 당신의 코드가 선택의 마법사가 된다면, `if` 문은 당신의 비결이야!

안녕하세요, 코딩 초보자 친구들! 오늘은 정말 특별한 주제로 여러분을 찾아가는 **"만약 당신의 코드가 선택의 마법사가 된다면, `if` 문은 당신의 비결"** 강의에 오신 것을 환영합니다! 🌟 오늘은 우리가 매일 사용하면서도 때로는 신비로운 힘을 가진 `if` 문에 대해 깊이 들어갈 거예요. 마치 마법 주문처럼 코딩 세계에서 결정을 내리는 데 핵심 역할을 하는 `if` 문을 배워볼게요.

#### 🚀 왜 `if` 문이 필요할까?

만약 여러분이 매일 아침 커피를 선택할 때, `"오늘은 라떼인가?"` 아니면 `"아아로 에너지 충전?"` 중 하나를 고르는 상황을 상상해보세요. 이럴 때 `if` 문이 바로 당신의 개인적인 마법사 역할을 합니다. 컴퓨터에게도 똑같은 결정 과정이 필요해요. `if` 문은 코드에게 "이 조건이 맞다면 이걸 해봐!" 라는 마법 주문을 내리는 거죠.

#### 💡 기본 구조: 마법 주문 "if"

`if` 문은 가장 기본적인 형태로 시작해볼게요.

```rust
fn main() {
    let score = 85; // 오늘의 시험 점수

    // 마법 주문! 만약 점수가 90점 이상이면...
    if score >= 90 {
        println!("축하해! 우수 학생이네요!"); // 마법 효과 발동!
    } else {
        println!("조금 더 노력하면 될 거예요!"); // 마법 효과 불발!
    }
}
```

**코드 설명:**
- `let score = 85;` : 변수 `score`에 점수를 할당합니다.
- `if score >= 90 { ... }` : 만약 `score`가 90 이상이면 코드 블록 안의 내용이 실행됩니다.
- `else { ... }` : 그렇지 않으면 `else` 블록 안의 내용이 실행됩니다.

#### 💡 `if` 문 심화: 다양한 마법 조합

##### 1. **`if-else` 문: 두 가지 길**

`if` 문이 한 가지만 선택하는 마법사라면, `if-else` 문은 두 가지 길을 열어주는 마법사입니다.

```rust
fn main() {
    let age = 22; // 당신의 나이

    if age >= 18 {
        println!("성인이시네요! 투표할 수 있어요!"); // 성인 길
    } else {
        println!("아직 미성년자입니다. 좀 더 기다려보세요!"); // 미성년자 길
    }
}
```

**코드 설명:**
- 여기서 `age`가 18세 이상이면 첫 번째 메시지가 출력되고, 그렇지 않으면 두 번째 메시지가 나옵니다.

##### 2. **`if` 문으로 반복하기: `for` 루프와 함께**

`if` 문을 반복문과 결합하면 좀 더 복잡한 로직을 구현할 수 있어요. 예를 들어, 숫자 리스트에서 특정 조건을 만족하는 요소만 필터링해보겠습니다.

```rust
fn main() {
    let numbers = vec![10, 20, 30, 40, 50]; // 숫자 리스트

    for number in numbers {
        if number % 2 == 0 { // 짝수인지 확인
            println!("{}는 짝수예요!", number); // 짝수 출력
        } else {
            println!("{}는 홀수예요!", number); // 홀수 출력
        }
    }
}
```

**코드 설명:**
- `for number in numbers` 루프를 돌면서 각 숫자를 확인합니다.
- `if number % 2 == 0` : 숫자가 짝수인지 확인하고, 짝수라면 해당 메시지를 출력합니다.
- `else` 블록은 홀수인 경우를 처리합니다.

##### 3. **`switch` 문 (Rust에서는 `match` 사용)**

Rust에서는 `switch` 문을 `match` 문으로 구현합니다. 여러 가지 경우의 수를 쉽게 다룰 수 있어요.

```rust
fn main() {
    let day = "월"; // 오늘의 날짜 (월, 화, 수, 목, 금, 토, 일)

    match day {
        "월" => println!("새로운 한 주가 시작되었어요!"),
        "화" => println!("중간 주가 왔어요!"),
        "수" => println!("중간 주를 넘어서"),
        "목" => println!("거의 주말이에요!"),
        "금" => println!("불금이에요!"),
        "토" | "일" => println!("주말이에요! 휴식을 즐기세요!"),
        _ => println!("잘못된 날짜 입력이에요!"), // 기본 케이스
    }
}
```

**코드 설명:**
- `match day` 문을 통해 `day` 변수의 값을 여러 경우에 따라 다르게 처리합니다.
- `|`는 `또는`을 의미하고, `_`는 모든 경우를 처리하는 기본 케이스입니다.

#### 💡 초보자 폭풍 질문!

**질문 1:** `if` 문에서 `else if`는 언제 쓰나요?

**답변:** `else if`는 첫 번째 `if` 조건이 거짓일 때 추가적인 조건을 검사할 때 유용해요. 예를 들어, 나이에 따라 다른 메시지를 출력할 때 사용할 수 있어요:

```rust
fn main() {
    let age = 25;

    if age < 18 {
        println!("미성년자세요!");
    } else if age >= 18 && age < 60 {
        println!("성인이시네요!");
    } else {
        println!("Senior 시민이세요!");
    }
}
```

**질문 2:** `match` 문은 어떤 상황에서 더 유용한가요?

**답변:** `match` 문은 여러 가지 가능한 결과를 명확하게 구분하고 처리해야 할 때 특히 유용해요. 예를 들어, 다양한 상태 코드를 처리하는 시스템 로그 분석이나 상태 기반 로직에서 활용됩니다.
