## 26강: 함수적 프로그래밍 개념 -  코딩, 생각하는 방식이 달라진다! 🚀💡

안녕하세요! 대한민국 최고의 C언어 일타 강사, 바로 저입니다 🔥😎. 오늘은 C언어를 배우는 여러분에게 혁명적인 개념을 소개할 거예요!  맞춰보시죠? 🤔 '함수적 프로그래밍'이라는 말에 대해 알고 계신가요?

**함수적 프로그래밍**은 단순히 함수만 쓰는 스타일이 아니에요.  ✨ 이 개념은 코드를 **마치 미술 작품처럼, 독립적인 블록들을 연결하여 완성하는 것과 같습니다!**. 각 블록은 하나의 기능을 담고 있어서, 짜릿한 합성 작품처럼 아름답고 효율적인 코드가 만들어지죠. 🤩

### 함수적 프로그래밍? 무슨 소리일까요?! 🤔

말만 들어봐도 복잡하게 느껴질 수 있겠지만,  진짜 쉬워요! 🔥 걱정 마세요, 저는 '함수'라는 친구를 통해 이해해보도록 할게요.

> 함수: 어떤 일을 하는 프로그램의 작은 부분이죠! 예를 들어, 사과를 먹는 행동에 집중할 수 있는 함수가 있다고 가정해볼까요? 🍎 함수적 프로그래밍에서는 여러 개의 '사과먹기' 함수들을 모아, 복잡한 식사 (예: 사과 + 바나나 + 우유) 를 만들 수 있습니다.

### 🔥  함수를 활용하면 뭔가 이렇게 좋을까요?


1. ** 코드 재사용성 UP!**: 이미 만든 함수를 다른 곳에 사용할 수 있어 시간과 노력을 절약할 수 있어요. 효율적인 개발은 우리의 생명줄이지 않나요?😎
2. **독립적 블록 구성**: 각 함수는 하나의 명확한 기능을 가지고 있기 때문에 코드 구조가 깔끔하고 이해하기 쉽죠.  마치 레시피처럼, 단계별로 명확하게 나눠져 있어요! 😉
3. **버그 수정 용이**: 코드 블록들이 독립적이므로 한 부분의 문제가 다른 곳에 영향을 미칠 확률이 적어집니다.  🐛 버그는 사라지고 우리는 조용히 개발할 수 있죠?

### 함수적 프로그래밍, 실제로 어떻게 쓰나요? 🤔


C언어에서 함수를 사용하는 방법은 아주 간단해요! ⬇️

```c
// 이 함수는 두 정수를 더하는 기능을 합니다.
int add(int a, int b) {
  return a + b; // 합계 값을 반환합니다.
}

int main() {
  int result = add(5, 3); // 5와 3를 더하고 결과를 변수 'result'에 저장합니다.
  printf("덧셈 결과: %d\n", result); // 8을 출력합니다.
  return 0;
}
```

* `int add(int a, int b)` : 이 부분은 함수의 이름(`add`)과 입력값 (`a`와 `b`)의 자료형(`int`)을 정의하는 부분입니다.
* `{ return a + b; }`: 함수 내부에서 계산된 합계 값을 `return` 문을 통해 출력합니다. 
* `int main() { ... }`:  프로그램 실행 시작점이며, `add` 함수를 호출하여 결과를 출력하는 부분입니다.
