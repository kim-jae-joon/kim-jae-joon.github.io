## 6강: C언어 심화 🚀 - 함수 정의와 호출, 코딩의 마법 지팡이!

안녕하세요, 코딩의 세계에서 이제 막 탐험을 시작하신 용감한 주니어 개발자 여러분! 👋  오늘은 **함수 정의와 호출**에 대해 깊이 있게 탐험해볼게요. 쉽게 말해, 함수는 코드의 '마법 지팡이'라고 생각하면 돼요. 복잡한 작업을 간단한 명령으로 해결할 수 있게 해주는 놀라운 도구죠! 🪄

### 함수의 신비로운 세계로 들어가기 🌌

**"함수란 무엇일까요?"** 라고 궁금해하실 거예요. 함수는 재사용 가능한 코드 블록이라고 생각하면 됩니다. 마치 우리가 일상에서 자주 사용하는 도구 상자 같은 거죠! 필요할 때마다 꺼내서 사용하면 되니까 코드 정리도 훨씬 수월해지고 유지보수에도 큰 도움이 됩니다.

#### 개념 설명: 함수의 핵심 원리

- **정의**: 함수를 만드는 과정을 **함수 정의**라고 합니다. 이때, 우리는 함수의 이름, 입력 매개변수, 그리고 실행할 코드 블록을 정합니다.
- **호출**: 이미 정의된 함수를 실행시키는 것을 **함수 호출**이라고 합니다. 마치 마법 지팡이를 휘두르는 것과 같아요! 🪄

### 함수 정의: 마법 지팡이 만드는 법 🪄

함수를 정의할 때 가장 중요한 부분은 **함수 헤더**와 **함수 본문**입니다. 헤더는 함수의 룰북 같은 거고, 본문은 실제 마법을 부리는 부분이에요.

#### 예제 1: 간단한 함수 정의

```c
#include <stdio.h>

// 함수 정의: "안녕하세요!" 메시지 출력하는 함수
void greet() {
    printf("안녕하세요!\n"); // 마법 지팡이 휘두르며 인사 메시지 출력
}

int main() {
    greet(); // 함수 호출: 마법 지팡이를 휘두르는 순간!
    return 0;
}
```

**코드 해설:**
- **void greet()**: `void` 타입의 함수 `greet`를 정의합니다. `void`는 반환값이 없음을 의미합니다.
- **printf("안녕하세요!\n");**: 함수 내부에서 `printf`를 사용해 메시지를 출력합니다.
- **greet();**: `main` 함수에서 `greet` 함수를 호출하여 마법을 실행합니다.

#### 예제 2: 매개변수 활용

함수에 입력값을 전달해 더 유연하게 사용할 수 있어요!

```c
#include <stdio.h>

// 함수 정의: 이름과 나이를 받아 인사 메시지 출력하는 함수
void introduce(char* name, int age) {
    printf("안녕하세요, %s님! %d살이시군요.\n", name, age);
}

int main() {
    introduce("철수", 25); // 함수 호출: 철수 씨를 소개하는 마법!
    return 0;
}
```

**코드 해설:**
- **char* name, int age**: 함수 매개변수로 이름과 나이를 받아옵니다.
- **printf("안녕하세요, %s님! %d살이시군요.\n", name, age);**: 전달된 값을 활용해 맞춤형 메시지를 출력합니다.

### 다양한 함수 호출 방법 🤩

#### 반복문과 함수 조합: 재귀 호출 예시

함수는 반복적으로 호출될 수 있어요. **재귀 호출**은 함수가 자기 자신을 호출하는 멋진 기법입니다.

```c
#include <stdio.h>

// 재귀 함수 정의: 숫자의 팩토리얼 계산
int factorial(int n) {
    if (n == 1) { // 기저 조건: n이 1이면 결과는 1
        return 1;
    } else {
        return n * factorial(n - 1); // 재귀 호출: 마법의 연쇄 반응!
    }
}

int main() {
    int num = 5;
    printf("%d의 팩토리얼은 %d\n", num, factorial(num)); // 함수 호출
    return 0;
}
```

**코드 해설:**
- **if (n == 1)**: 기저 조건으로 함수 호출을 멈추는 역할을 합니다.
- **return n * factorial(n - 1);**: 재귀적으로 자신을 호출하며 결과를 계산합니다.

#### 조건문과 함수 활용: 실용적 예제

조건문과 함수를 함께 사용하면 더 유연한 코드를 작성할 수 있어요.

```c
#include <stdio.h>

// 함수 정의: 숫자가 짝수인지 확인
int isEven(int num) {
    return (num % 2 == 0); // 짝수 판별
}

int main() {
    int number = 10;
    if (isEven(number)) { // 함수 호출을 통한 조건 검사
        printf("%d는 짝수입니다.\n", number);
    } else {
        printf("%d는 홀수입니다.\n", number);
    }
    return 0;
}
```

**코드 해설:**
- **isEven(int num)**: 숫자가 짝수인지 판별하는 함수입니다.
- **if (isEven(number))**: 함수 호출 결과를 조건문에 활용하여 결과를 출력합니다.

### 💡 초보자 폭풍 질문! 🚀

**Q1: 함수에서 `void` 타입을 사용하는 이유는 무엇인가요?**  
**A1:** `void` 타입은 함수가 값을 반환하지 않는다는 것을 의미합니다. 예를 들어, 단순히 메시지를 출력하거나 작업을 수행하는 함수에서는 결과값이 필요하지 않으므로 `void`를 사용합니다. 이렇게 하면 코드의 명확성이 높아지고, 반환값에 대한 불필요한 고민을 피할 수 있어요!

**Q2: 함수 호출 시 매개변수를 어떻게 전달하나요?**  
**A2:** 매개변수는 함수 정의 시 지정된 순서대로 호출 시 값을 넣어 전달합니다. 예를 들어, `introduce("철수", 25)`에서 `"철수"`는 이름 매개변수에, `25`는 나이 매개변수에 각각 전달됩니다. 함수 호출 시 순서와 타입이 일치해야 합니다!
