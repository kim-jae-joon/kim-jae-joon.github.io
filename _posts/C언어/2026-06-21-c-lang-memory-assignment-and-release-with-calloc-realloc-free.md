**30강: C언어 메모리 할당과 해제**

안녕하세요, 여러분! 오늘은 정말 중요한 주제인 **C언어 메모리 할당과 해제** 에 대해 배워 보겠습니다. 이 장에서는 메모리의 기본 개념부터 malloc(), free() 함수까지 다양한 예제를 통해 이해할 수 있을 것입니다.

**메모리 할당의 필요성**

프로그램을 실행하면, 운영체계가 프로그램에게 일정량의 메모리를 할당합니다. 하지만, 이러한 할당된 메모리가 얼마나 많은지 우리는 모릅니다. 그래서, 우리만 필요한 만큼 메모리를 할당하고 사용해야 합니다.

**malloc() 함수**

`malloc()` 함수는 동적 메모리 할당을 위한 표준函수입니다. 이 함수는 malloc.h 헤더파일에서 선언되며, 다음과 같은 형식을 가집니다.

```c
void* malloc(size_t size);
```

*size_t*은 unsigned int형의 크기 변수를 의미합니다.

`malloc()` 함수는 메모리를 할당하기 위한 인자를 하나만 받습니다. 이 인자는 할당할 메모리 크기를 나타냅니다. 반환값은 할당된 메모리의 시작 주소를 가지는 포인터입니다.

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    int* p = (int*)malloc(sizeof(int));
    if (p == NULL) { // malloc 에러시 NULL 리턴
        printf("메모리 할당 실패\n");
        return 1;
    }
    
    *p = 10; // 할당된 메모리에 값 저장
    printf("%d\n", *p);
    free(p); // 할당한 메모리 해제
    
    return 0;
}
```

위의 예제에서, `malloc()` 함수는 sizeof(int)크기 만큼의 메모리를 할당합니다. 그 후, 할당된 메모리에 10을 저장하고 출력합니다.

**free() 함수**

`free()` 함수는 동적 할당된 메모리 해제를 위한 표준함수입니다. 이 함수는 free.h 헤더파일에서 선언되며, 다음과 같은 형식을 가집니다.

```c
void free(void* ptr);
```

*size_t*은 unsigned int형의 크기 변수를 의미합니다.

`free()` 함수는 포인터 인자 하나만 받습니다. 이 포인터 인자는 할당된 메모리의 시작 주소를 가지는 포인터입니다.

위의 예제에서, `free()` 함수는 할당한 메모리를 해제합니다.

**동적 메모리 할당과 해제**

C언어에서 동적으로 할당하고 해제하는 방법은 다음과 같습니다.

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    int* p1 = (int*)malloc(sizeof(int));
    *p1 = 10;
    
    printf("%d\n", *p1);
    
    int* p2 = (int*)malloc(sizeof(int));
    *p2 = 20;
    
    printf("%d\n", *p2);
    
    free(p1); // 할당한 메모리 해제
    free(p2); // 할당한 메모리 해제
    
    return 0;
}
```

위의 예제에서, 두 개의 메모리를 동적으로 할당하고, 그 후에 메모리를 해제합니다.

**포인터와 동적 메모리**

포인터를 사용하여 메모리에 접근할 수 있습니다. 다음과 같은 코드가 가능합니다.

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    int* p = (int*)malloc(sizeof(int));
    *p = 10;
    
    printf("%d\n", *p);
    
    // 포인터 증가 연산자 (*p) -> *(p+1)
    int* q = p + 1; // p에 1을 더하는 것과 같음
    *q = 20;
    
    printf("%d\n", *p); // 10 출력
    printf("%d\n", *q); // 20 출력
    
    free(p);
    free(q);
    
    return 0;
}
```

위의 예제에서, 포인터를 증가 연산자(*)로 사용하여 메모리에 접근할 수 있습니다.

**메모리 해제의 중요성**

메모리는 프로그램이 실행되기 위해 할당되는 공간입니다. 하지만, 이러한 메모리가 얼마나 많은지 우리는 모릅니다. 그래서, 동적으로 할당된 메모리를 해제하는 것은 매우 중요합니다. 위의 예제에서, 할당한 메모리를 해제하지 않으면 메모리 누수가 발생하고 프로그램이 비정상적으로 종료될 수 있습니다.

**동적 메모리 할당과 해제 실습**

다음은 C언어에서 동적 메모리 할당과 해제를 실습하는 예제입니다.

```c
#include <stdio.h>
#include <stdlib.h>

int main() {
    int* p1 = (int*)malloc(sizeof(int));
    *p1 = 10;
    
    printf("%d\n", *p1);
    
    free(p1); // 할당한 메모리 해제
    
    return 0;
}
```

위의 예제에서, 할당된 메모리를 해제합니다.
