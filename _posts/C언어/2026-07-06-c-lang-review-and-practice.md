**15강: 정리와 실습 - 개념 마무리 및 연습문제 풀이 🔥**

안녕하세요, 여러분! 😊 오늘까지 함께 공부하신 분들 모두 환영입니다! 💕 우리 모두가 알고 싶은 것들이 많을 때, 하나하나 알려드릴게요. 그럼 개념 마무리를 하면서 실무 활용과 연관된 예제를 만들어 보겠습니다!

### 개념 정리

1.  **데이터 형식의 이해**

    -   **정수(Integer)**: 양의 정수는 0 이상, 음의 정수는 -1부터, 실수는 소수로 표현되는 데이터 형식을 말합니다.
    
    ```c
int a = 10;  // 정수 10을 담는 변수 a
    
float b = 10.5;  // 실수 10.5를 담는 변수 b
```
    
2.  **문자열(String)**: 여러 문자를 연결해 하나의 문자열로 표현됩니다.
    
    ```c
char* name = "홍길동";  // 문자열로 선언하는 이름
    
printf("%s", name);  // %s 형식에 따라 홍길동을 출력합니다.
```
    
3.  **배열(Array)**: 같은 데이터 타입의 여러 변수를 하나의 이름으로 관리할 수 있는 자료구조입니다.

    ```c
int numbers[5] = {1, 2, 3, 4, 5};  // 5개의 정수 배열 생성
    
for (int i = 0; i < 5; i++) {
        printf("%d ", numbers[i]);
}
```
    
### 실무 활용 예제

#### 예제 1: 정수 데이터 형식의 사용 🔑

```c
#include <stdio.h>

int main() {
    int year = 2024;
    printf("현재 년도: %d\n", year);
    return 0;
}
```

*   위 코드는 `printf()` 함수를 통해 `year` 변수에 저장된 정수 값이 출력됩니다. 🔓

#### 예제 2: 문자열 데이터 형식의 사용 📝

```c
#include <stdio.h>

int main() {
    char* name = "홍길동";
    printf("이름: %s\n", name);
    return 0;
}
```

*   위 코드에서는 `printf()` 함수를 통해 `%s` 형식에 따라 `name` 변수에 저장된 문자열이 출력됩니다. 📚

#### 예제 3: 배열 데이터 형식의 사용 👀

```c
#include <stdio.h>

int main() {
    int numbers[5] = {1, 2, 3, 4, 5};
    printf("정수 배열: ");
    for (int i = 0; i < 5; i++) {
        printf("%d ", numbers[i]);
    }
    printf("\n");
    return 0;
}
```

*   위 코드에서는 `for` 루프를 사용하여 `numbers` 변수에 저장된 정수 배열의 모든 요소가 출력됩니다. 💻


    ```c

printf("이름: %s\n", name);
```

3.  **배열 데이터 형식으로 정수 배열 5개를 선언하고 각 요소에 1부터 5까지의 값을 입력하세요.**
    
