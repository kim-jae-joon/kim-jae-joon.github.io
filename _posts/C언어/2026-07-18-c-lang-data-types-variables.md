## 3강: C언어 기초 마스터! 🤖 데이터 타입과 변수, 이제 당신의 손안에!

**안녕하세요, 코딩 탐험가 여러분!** 오늘은 C언어의 기초 중 기초를 탐험하는 짜릿한 여정에 함께 떠나볼게요! 바로 **데이터 타입과 변수 선언**에 대해 알아보는 시간입니다. 

**"데이터 타입? 변수? 이게 뭐냐고요? 내 코드가 갑자기 갑자기 춤추기 시작하는 마법 주문 같은 거예요!"**  ( 😉 농담이지만, 꽤 중요하답니다! 😉)

컴퓨터는 세상 모든 것을 숫자로 이해하는 걸까요?  **맞아요!** 결국 컴퓨터는 0과 1의 마법으로 모든 것을 표현하죠. 그래서 우리는 데이터를 저장할 때 어떤 종류의 숫자를 사용할지 알려줘야 해요. 그게 바로 **데이터 타입**이랍니다!

### 데이터 타입: 숫자의 변신 마법! 🪄

데이터 타입은 마치 변신 로봇처럼 숫자가 다양한 형태로 나타날 수 있도록 돕는 역할을 합니다. 몇 가지 핵심 타입을 소개할게요.

#### 1. **정수형 (Integer Types)**

숫자를 그대로 저장할 때 쓰는 타입이에요. 마치 숫자 마법사의 기본 무기 같은 존재죠!

* **`int`:** 가장 기본적인 정수형입니다. 일상생활에서 흔하게 쓰이는 정수들을 저장하는데 좋아요. 예를 들어, 학생들의 점수나 게임 캐릭터의 레벨처럼요!

    ```c
    int age = 25; // 나이를 저장하는 변수 'age'에 25를 할당
    ```

    * **코드 해부:** 여기서 `int`는 변수 타입을 지정하고, `age`는 변수 이름을, `25`는 저장할 정수 값을 나타냅니다. 마치 마법 주문처럼 `age`라는 공간에 숫자 25를 정확하게 넣어주는 거예요!

* **`short int`, `long int`:**  `int`보다 더 작거나 큰 범위의 정수를 저장할 때 사용합니다. 
    * 예시: 게임에서 플레이어의 위치 좌표를 저장할 때, `short int`를 사용하면 메모리를 효율적으로 사용할 수 있습니다.

#### 2. **부동 소수점형 (Floating-Point Types)**

소수점이 있는 숫자, 즉 실수를 다룰 때 쓰는 타입이에요. 마치 정교한 측정 도구처럼 정확한 값을 표현합니다.

* **`float`:**  일반적인 실수 표현에 사용됩니다. 가격 계산이나 물리학 공식에서 자주 등장하죠!

    ```c
    float price = 19.99; // 상품 가격을 저장하는 변수 'price'에 19.99 할당
    ```

    * **코드 해부:** `float` 타입으로 `price` 변수에 금액 19.99를 저장합니다. 소수점까지 정확하게 표현 가능하다는 게 핵심 포인트예요!

* **`double`:** `float`보다 더 큰 범위와 정밀도를 가진 실수를 다룰 때 사용합니다. 과학 계산이나 정밀한 금융 모델링에 적합합니다.

    ```c
    double circumference = 3.14159 * diameter; // 원의 둘레 계산
    ```

    * **코드 해부:** 여기서 `double` 타입의 `circumference` 변수는 `diameter` 값을 이용해 원의 둘레를 계산하고 저장합니다. 정밀한 계산 결과를 위해서 `double`을 선택했네요!

#### 3. **기타 유용한 타입**

* **`char`:** 문자 하나를 저장하는 데 사용됩니다. 마치 작은 문자 주머니 같죠!

    ```c
    char initial = 'J'; // 이름의 약자 'J'를 저장
    ```

    * **코드 해부:** `char` 타입의 `initial` 변수에 문자 'J'를 넣어 저장합니다. 컴퓨터는 이를 ASCII 코드로 이해하죠.

* **`boolean`:**  참(true) 또는 거짓(false) 값을 나타냅니다. 조건문에서 자주 쓰이는 타입이에요.

    ```c
    boolean isValid = true; // 유효성 검사 결과를 저장
    ```

### 변수 선언: 빈 공간을 마법의 창고로! 🧙‍♂️

데이터 타입을 알았으니, 이제 그 타입에 맞는 **변수**를 만들어줘야 합니다. 변수는 컴퓨터 메모리의 빈 공간을 마법의 창고처럼 활용하는 거예요!

**변수 선언**은 마치 "이 공간에 어떤 종류의 데이터를 보관할지 정해줘!"라고 명령하는 것과 같습니다.

**예시:**

```c
int studentScore; // 정수형 변수 'studentScore' 선언 (아직 값은 없음!)
float averageGrade = 85.5; // 부동 소수점 변수 'averageGrade'에 값 할당
char favoriteColor = 'R'; // 문자 변수 'favoriteColor'에 'R' 저장
```

* **코드 해부:**
    1. `int studentScore;`: 정수형 변수 `studentScore`를 선언했지만, 아직 값은 넣지 않았어요. 마치 빈 상자를 준비한 것처럼요! 나중에 코드에서 `studentScore = 90;` 처럼 값을 넣어줄 거예요.
    2. `float averageGrade = 85.5;`:  바로 값을 할당해서 평균 학점을 저장했습니다. 마치 상자에 물건을 넣는 것처럼요!
    3. `char favoriteColor = 'R';`: 좋아하는 색깔 'R'을 문자 변수에 담았습니다.

**💡 초보자 폭풍 질문!**

* **Q: 변수를 선언했는데 값을 나중에 넣으면 괜찮나요?**
* **A:** 네, 맞아요! 변수 선언은 빈 공간을 준비하는 것과 같습니다. 나중에 `studentScore = 90;` 처럼 값을 할당하면 됩니다. 이렇게 하면 프로그램이 변수에 데이터를 안전하게 저장하고 다룰 수 있습니다.

### 실무에서 데이터 타입과 변수 활용: 실제 예시 탐험! 🌎

데이터 타입과 변수는 코딩에서 **기본이자 필수** 요소입니다. 몇 가지 실제적인 예시를 통해 얼마나 중요한지 살펴보겠습니다.

**1. 간단한 계산기 프로그램:**

```c
#include <stdio.h>

int main() {
    double num1, num2, sum;

    printf("첫 번째 숫자를 입력하세요: ");
    scanf("%lf", &num1); // 사용자로부터 실수 입력 받음

    printf("두 번째 숫자를 입력하세요: ");
    scanf("%lf", &num2); // 사용자로부터 실수 입력 받음

    sum = num1 + num2; // 두 숫자를 더함 (double 타입으로 결과 저장)

    printf("합계는 %.2lf 입니다.\n", sum); // 소수점 두 자리까지 출력

    return 0;
}
```

* **코드 해부:** 
    * `double num1, num2, sum;`:  실수 계산을 위해 `double` 타입의 변수들을 선언합니다. 
    * `scanf("%lf", &num1);`: 사용자 입력을 받을 때 `double` 타입 변수에 저장하기 위해 `%lf` 형식 지정자를 사용합니다.
    * `sum = num1 + num2;`: 두 실수 변수를 더하고 결과를 `sum`에 저장합니다.

**2. 간단한 출석 체크 시스템:**

```c
#include <stdio.h>

int main() {
    int studentID, numberOfStudents = 30;
    int presentCount = 0;

    printf("학생 번호를 입력하세요 (종료하려면 -1 입력): ");
    
    while (1) { // 무한 루프를 이용해 여러 학생 입력 가능
        scanf("%d", &studentID);

        if (studentID == -1) { // 종료 조건
            break;
        }

        // 출석 처리 로직 (예시: studentID가 0~numberOfStudents 범위 내라면 출석 처리)
        if (studentID >= 0 && studentID <= numberOfStudents) {
            presentCount++;
            printf("학생 %d님, 출석 성공!\n", studentID);
        } else {
            printf("잘못된 학생 번호입니다.\n");
        }
    }

    printf("총 출석자 수: %d명\n", presentCount);
    return 0;
}
```

* **코드 해부:**
    * `int studentID, numberOfStudents = 30, presentCount = 0;`: 학생 ID와 출석 인원 수를 저장할 정수형 변수들을 선언합니다.
    * `while (1)` 루프: 여러 학생의 출석을 입력받습니다. `studentID == -1` 조건으로 루프를 종료합니다.
    * **조건문 활용:** `if (studentID >= 0 && studentID <= numberOfStudents)` 로 유효한 학생 ID인지 확인하고 출석 처리를 합니다.

### 🚨 실무 주의보! 💥

**데이터 타입 불일치 오류**: 주의하세요! 변수에 저장하려는 값의 데이터 타입이 선언된 변수 타입과 다를 경우 컴파일 오류가 발생할 수 있습니다. 예를 들어, `int` 변수에 `double` 값을 할당하려고 하면 문제가 생길 수 있어요. 항상 데이터 타입을 신중하게 선택하고 확인하세요!
