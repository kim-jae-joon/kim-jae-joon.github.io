### 🔌  socket: 네트워크 연결의 열쇠 🔑


C 언어에서 네트워킹은 **소켓(socket)** 이라는 개념을 통해 가능합니다. 소켓은 서로 정보를 주고받을 수 있는 통신 채널이라고 생각해주세요!  

> 예를 들어, 당신과 친구가 전화통으로 대화하는 것처럼, 💻 두 개의 컴퓨터는 소켓을 통해 데이터를 주고받을 수 있습니다.

### 🔨  C 언어와 socket: 함께 만드는 네트워킹 애플리케이션 💪

소켓 프로그래밍을 시작하기 전, C 언어의 **`socket()` 함수**에 대해 알아볼 필요가 있습니다. 💻 이 함수를 사용하면 새로운 소켓 객체를 생성하는 데 쓰입니다.  


```c
#include <sys/socket.h> // 네트워킹 헤더 파일
#include <netinet/in.h> // IP 주소와 포트 번호 설정에 사용되는 헤더 파일

int main() {
    int sockfd = socket(AF_INET, SOCK_STREAM, 0); // 소켓 생성 함수 호출! AF_INET: IPv4 주소를 사용하는 소켓, SOCK_STREAM: TCP 연결을 사용하는 소켓, 0: 기본 설정 활용

    if (sockfd == -1) {
        perror("socket() error");
        exit(1);
    }
    printf("Socket created successfully! Socket file descriptor: %d\n", sockfd);
    return 0;
}
```

> **설명:**
> * `#include <sys/socket.h>` : 네트워킹 관련 함수를 사용하기 위한 헤더 파일입니다.
> * `#include <netinet/in.h>` : IP 주소와 포트 번호 설정에 필요한 헤더 파일입니다.
> * `int sockfd = socket(AF_INET, SOCK_STREAM, 0);`:  `socket()` 함수를 이용해 IPv4 주소(`AF_INET`), TCP 연결 (`SOCK_STREAM`)을 사용하는 소켓 객체를 생성합니다. `sockfd` 변수에 생성된 소켓 파일 디스크립터가 저장됩니다.
> * `perror("socket() error");` : 소켓 생성 실패 시 오류 메시지를 출력하고 프로그램을 종료합니다.

### 🗺️ 네트워킹 주요 개념 이해 🚀

네트워킹의 기초를 다지는 것은 매우 중요합니다! C 언어에서 네트워크 프로그래밍을 하는 데 필요한 주요 개념들을 알아봅시다:

* **IP 주소:** 인터넷에 연결된 장치(컴퓨터, 서버 등)를 식별하는 고유 번호입니다. 숫자와 점(.)으로 구성되어 있습니다. (예: 192.168.0.1)
* **포트 번호:** 서비스 또는 응용 프로그램을 구분하기 위해 사용되는 번호입니다. TCP/UDP 프로토콜에서 소켓 주소를 식별하는 데 사용됩니다. (예: 80 - 웹 서버, 443 - HTTPS 웹 서버)
