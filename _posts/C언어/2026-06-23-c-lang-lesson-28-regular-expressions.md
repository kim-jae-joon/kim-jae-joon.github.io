##  28강: 정규 표현식 (Regular Expressions) 활용하기 - 코드의 마법사가 되어 보세요! 😎

**🔥 흥미로운 질문이 있나요?** 🤔 만약 당신이 수많은 데이터 속에서 특정 정보를 찾고 싶다고 상상해보세요. 예를 들어, 웹 페이지에 있는 이메일 주소만 추출하고 싶거나, 사용자 입력 값 중 불법적인 문자를 제거하고 싶을 때! 😱  

**그때가 정규 표현식의 등장이야!** 🎉  정규 표현식은 코드에서 특정 패턴을 찾아내는 강력한 도구입니다. 마치 슈퍼 영웅처럼 🦸‍♀️ 데이터 속 숨겨진 보물들을 찾아낼 수 있어요!

### 1. 정규 표현식? 무슨 말이야?! 🤔

**> "코드를 다룰 때, 특정 문자나 패턴을 찾고 싶은 상황에 자주 처하게 되죠."**  - 어떤 사람의 말입니다. 이때 정규 표현식이 빛나는 순간이지요! ✨

바로 좀 더 자세히 알아볼까요? 🧐

* **일반적인 문자 검색:** "apple"과 같은 단어를 찾는 것은 쉽죠. 🍎  하지만, 특정 패턴이나 형태의 데이터를 찾으려면 코드가 복잡해질 수 있습니다. 예를 들어, 이메일 주소 형식 "name@domain.com"을 검색하는 경우입니다.

* **정규 표현식의 등장:** 정규 표현식은 괄호, 대괄호, 그리고 기타 특수 문자들을 사용하여 복잡한 패턴을 나타낼 수 있도록 합니다. 이를 통해 코드가 간결하고 효율적으로 작성될 수 있습니다!

### 2. 💥 정규 표현식의 세계로 들어서기! 🚀

정규 표현식은 여러 가지 기호와 문자를 사용하여 복잡한 패턴을 나타낼 수 있습니다. 

**💡 초보자에게 유용하게 활용되는 기본적인 정규 표현식입니다.**
* `.`: 임의의 한 글자를 나타냅니다. 예: "a."는 'aa', 'ab', 'ac' 등 여러 가지 패턴을 검색합니다.
* `*`: 직전 문자가 0개 이상 반복될 때 사용됩니다. 예: "a*"는 'aaa', '', 'a' 등 이어지는 'a'를 의미합니다.
* `+`: 직전 문자가 1개 이상 반복될 때 사용됩니다. 예: "a+"는 'aa', 'aaa', 'aaaa' 등 'a'가 여러 번 연속된 패턴을 검색합니다.

### 3.  🎉 실제 코드로 시연해 보자!

**예시 1: 이메일 주소 추출하기**

```c
#include <stdio.h>
#include <regex.h>

int main() {
    char text[] = "이메일 주소는 abc@domain.com이고, 다른 이메일은 xyz@example.net입니다.";
    regex_t regex; 
    regmatch_t match;

    // 정규 표현식을 정의합니다. (\w+@\w+\.\w+) 는 이메일 형식을 나타내는 패턴입니다.
    if (regcomp(&regex, "^(\\w+@\\w+\\.\\w+)$", REG_EXTENDED) != 0) { 
        printf("Regex compile error!\n");
        return 1;
    }

    // 문자열에서 정규 표현식과 일치하는 부분을 찾습니다.
    if (regexec(&regex, text, 1, &match, 0) == 0) {
        printf("찾은 이메일 주소: %s\n", text + match.rm_so); // match.rm_so는 일치하는 문자열의 시작 위치를 나타냅니다.

    } else {
        printf("이메일 주소를 찾지 못했습니다.\n");
    }

    return 0;
}
```

**해설:** 위 코드는 이메일 주소 패턴을 정의하여 "text" 문자열에서 일치하는 부분을 찾아 출력합니다.   `regcomp()` 함수는 정규 표현식을 정의하고 `regexec()` 함수는  정규 표현식이 지정된 문자열과 일치하는지 확인합니다. 

**💡 초보자가 이해하기 쉬운 설명:** 예를 들어, 'abc@domain.com' 이런 형태의 이메일 주소만 찾아 내고 싶을 때 사용해 보세요! 


**예시 2: URL에서 도메인 이름 추출하기**

```c
#include <stdio.h>
#include <regex.h>

int main() {
    char url[] = "https://www.example.com/page?id=123";
    regex_t regex;
    regmatch_t match;

    // 정규 표현식을 정의합니다. (?:http|ftp)와 같은 모양일때도 도메인 이름만 추출하고 싶습니다.
    if (regcomp(&regex, "^(https?://)?(?:www\\.)?([^/]+)", REG_EXTENDED) != 0) {
        printf("Regex compile error!\n");
        return 1;
    }

    // 문자열에서 정규 표현식과 일치하는 부분을 찾습니다.
    if (regexec(&regex, url, 1, &match, 0) == 0) {
        printf("도메인 이름: %s\n", url + match.rm_so); // 일치하는 문자열 추출

    } else {
        printf("도메인 이름을 찾지 못했습니다.\n");
    }

    return 0;
}
```


**해설:** 위 코드는 URL에서 도메인 이름만 추출하는 예제입니다.  `https?://` 부분은 HTTP 또는 HTTPS 프로토콜을 나타내며 `(?:www\\.)?` 는 'www.' 가 없는 경우도, 있는 경우도 모두 처리합니다.

### 4.  💪 정규 표현식 활용 실무 꿀팁! 🚀


* **데이터 분석:** 웹 페이지에서 특정 정보를 추출하거나, CSV 파일에서 데이터를 분리하는 데 유용합니다.
* **입력 검증:** 사용자 입력 값이 정해진 형태를 따르는지 확인하는 데 사용됩니다. 예를 들어, 이메일 주소, 휴대폰 번호, 날짜 등을 검증할 때 활용합니다.
* **텍스트 편집:** 특정 패턴의 문자를 찾아 바꾸거나 삭제하는 작업에 활용됩니다. 


**🚨 실무주의보:** 정규 표현식은 복잡하게 사용될 수 있으므로, 처음부터 너무 어려운 패턴을 사용하지 않고 단계적으로 배우는 것이 좋습니다!
