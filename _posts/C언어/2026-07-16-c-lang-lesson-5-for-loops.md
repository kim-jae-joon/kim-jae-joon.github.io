### 1. for 반복문의 개념

`for` 반복문은 프로그래밍에서 가장 많이 사용되는 문법 중 하나입니다. 간단하게 말하면, "반복적으로 어떤 작업을 수행하고 싶을 때"에 사용하는 문법입니다. 😊 하지만, 그보다는 더 깊은 의미를 가지고 있습니다.

`for` 반복문의 주된 목적은 **반복적인 작업을 자동화** 하는 것입니다. 예를 들어, 1부터 10까지의 숫자를 출력하고 싶다면? 🤔

```c
#include <stdio.h>

int main() {
    int i;
    for (i = 1; i <= 10; i++) {
        printf("%d\n", i);
    }
    return 0;
}
```

위 코드는 1부터 10까지의 숫자를 출력하는 간단한 예제입니다. 하지만, 이보다 더 유용한 점은, **반복문이 끝나면 반복 변수(여기서는 `i`)도 자동으로 초기화** 되는 것입니다! 🔁

### 2. for 반복문의 구조

`for` 반복문에는 총 세 가지 부분이 있습니다: **초기화**, **조건**, **증가**.

- 초기화: 반복 시작 전에 한번만 실행되는 코드입니다.
- 조건: 반복을 계속할지 말지를 결정하는 코드입니다. 조건이 거짓이면 반복은 끝납니다.
- 증가: 반복 중에 실행되는 코드입니다. 일반적으로 반복 변수를 증감시키거나 감소시킵니다.

위의 예제에서, 초기화는 `i = 1`로, 조건은 `i <= 10`으로, 증가는 `i++`으로 설정되어 있습니다. 😊

### 3. 실무 활용 - 1부터 100까지의 합 구하기

```c
#include <stdio.h>

int main() {
    int i;
    int sum = 0;
    for (i = 1; i <= 100; i++) {
        sum += i;
    }
    printf("1부터 100까지의 합: %d\n", sum);
    return 0;
}
```

위 코드는 1부터 100까지의 합을 구하는 예제입니다. `for` 반복문을 사용하여, 1부터 100까지의 숫자를 순서대로 더해가면서, 최종 합을 출력합니다.

### 4. 실무 활용 - 주사위 6면 roll하기

```c
#include <stdio.h>
#include <time.h>

int main() {
    srand(time(NULL));
    int num;
    for (num = 1; num <= 6; num++) {
        printf("주사위 %d: %d\n", num, rand() % 6 + 1);
    }
    return 0;
}
```

위 코드는 주사위를 6번 roll하는 예제입니다. `for` 반복문을 사용하여, 1부터 6까지의 숫자를 순서대로 출력합니다.

### 5. 실무 활용 - 학생 점수 평균 구하기

```c
#include <stdio.h>

int main() {
    int i;
    float sum = 0.0f;
    int count = 0;
    for (i = 1; i <= 10; i++) {
        printf("학생 %d의 점수: ", i);
        float score;
        scanf("%f", &score);
        sum += score;
        count++;
    }
    printf("점수 평균: %.2f\n", sum / count);
    return 0;
}
```

위 코드는 학생들의 점수를 입력받아, 점수 평균을 구하는 예제입니다. `for` 반복문을 사용하여, 학생의 수를 입력받고, 점수 평균을 계산합니다.
