### ##  🌟 'if' - 가장 기본적인 판단

`if` 문은 주로 **"만약 ~일 경우~!"** 라고 하는 조건을 검사합니다. 🔍 만약 조건이 참 (진리) 이라면 코드 블럭이 실행되고, 거짓이라면 코드 블럭은 스킵됩니다! 🚀

```c
#include <stdio.h>

int main() {
  int score = 85;

  if (score >= 90) { // 만약 점수가 90 이상이라면?
    printf("A등급 🎉"); 
  }

  return 0;
}
```

* 위 코드에서 `score` 변수의 값이 `90` 보다 크거나 같으면 `"A등급 🎉"` 가 출력됩니다.
* 반대로 `90` 보다 작으면 `if` 문 안에 있는 코드 블럭은 실행되지 않습니다! 😉 

### ## 🌟 'else if' - 여러 조건을 추가하기

`else if` 는 `if` 문 뒤에 이어서 사용하여 **더 많은 조건**들을 확인할 수 있게 해줍니다. 🤔 여러 가지 조건이 있을 때, 가장 적합한 코드를 실행하고 싶은데? 💥 바로 "else if"를 활용하면 돼요!

```c
#include <stdio.h>

int main() {
  int score = 78;

  if (score >= 90) { // 만약 점수가 90 이상이라면?
    printf("A등급 🎉");
  } else if (score >= 80) { // 아니면, 80점 이상이라면?
    printf("B등급 👍");
  } else if (score >= 70) { // 아니면, 70점 이상이라면?
    printf("C등급 😎");
  } else { // 나머지 경우?
    printf("개선 필요! 😥");
  }

  return 0;
}
```

* 이번 코드에서는 점수에 따라 A등급부터 개선필요까지 다양한 메시지를 출력합니다. 💡 `else if` 를 사용하여 여러 가지 조건을 순차적으로 검사하기 때문에, 가장 적합한 결과를 반환할 수 있죠!


### ## 🌟 'else' - 마지막 선택지

`else` 문은 **모든 조건이 거짓**일 때 실행되는 최종 코드 블럭입니다. 🔚 처음에는  조금 어려워 보여도, 실제로는 "아무리 다양한 조건들을 검사하더라도 딱 하나만 남았을 때 넣어주면 된다"라고 생각하면 쉬워집니다! 😉

```c
#include <stdio.h>

int main() {
  int age = 15;

  if (age >= 19) { 
    printf("성인입니다!");
  } else { // 나이가 19세 미만일 경우?
    printf("청소년입니다.");
  }

  return 0;
}
```

* `else` 문은, 위 예시처럼 모든 조건문이 거짓일 때, 단 한 번 실행되도록 해줍니다. 🚀
