## 변수란 무엇인가요? - 코딩의 보석 같은 존재

**변수**는 코딩 세계에서 가장 중요한 보석 중 하나입니다. 간단히 말해, 변수는 데이터를 저장하고 이름을 붙이는 마법 상자 같은 거죠. 예를 들어, 여러분이 좋아하는 숫자를 임시로 보관하거나 계산 결과를 저장할 때 쓰입니다. 

### 개념 이해하기

#### 1. 데이터 타입 이해하기

변수를 선언하기 전에, 어떤 종류의 데이터를 저장할지 정해야 합니다. C언어에서는 주로 다음과 같은 데이터 타입을 사용합니다:

- **정수형 (`int`)**: 예를 들어, 학생들의 점수를 저장할 때 사용합니다.
- **실수형 (`float`, `double`)**: 가격이나 비율 같은 값을 저장할 때 쓰입니다.
- **문자형 (`char`)**: 한 글자를 저장할 때 사용합니다.

#### 2. 변수 선언 예시

```c
#include <stdio.h>

int main() {
    // 정수형 변수 선언 및 초기화 예시
    int score = 85; // 점수를 저장할 변수 'score' 선언 및 초기값 85 설정
    float price = 19.99; // 가격을 저장할 변수 'price' 선언 및 초기값 설정
    char initial = 'J'; // 이름의 초기를 저장할 변수 'initial' 선언 및 초기값 설정

    // 출력 예시
    printf("점수: %d\n", score); // %d는 정수형 포맷 지정자
    printf("가격: %.2f\n", price); // %.2f는 소수점 두 자리까지 표시
    printf("초기: %c\n", initial); // %c는 문자 포맷 지정자

    return 0;
}
```

**해설:**
- `int score = 85;`: `score`라는 이름의 정수형 변수를 선언하고 초기값으로 85를 설정합니다.
- `float price = 19.99;`: `price`라는 이름의 실수형 변수를 선언하고 초기값으로 19.99를 설정합니다.
- `char initial = 'J';`: `initial`이라는 이름의 문자형 변수를 선언하고 'J'를 초기값으로 설정합니다.

### 다양한 상황에서의 변수 사용

#### 반복문과 변수

반복문을 활용하면서 변수의 힘을 체감할 수 있습니다. 예를 들어, 학생들의 점수를 모두 더해 평균을 구하는 코드를 만들어볼게요.

```c
#include <stdio.h>

int main() {
    int numberOfStudents = 5; // 학생 수
    int sum = 0; // 합계를 저장할 변수
    int average = 0; // 평균을 저장할 변수

    // 학생들의 점수를 더하는 반복문 예시
    for (int i = 0; i < numberOfStudents; i++) {
        int score; // 각 학생의 점수를 저장할 변수
        printf("학생 %d의 점수를 입력하세요: ", i + 1);
        scanf("%d", &score); // 사용자로부터 점수 입력 받기
        sum += score; // 합계에 점수 추가
    }

    // 평균 계산
    average = sum / numberOfStudents; // 주의: 정수 나눗셈으로 소수점 아래 버림

    printf("평균 점수: %d\n", average);

    return 0;
}
```

**해설:**
- `for (int i = 0; i < numberOfStudents; i++)`: 학생 수만큼 반복합니다.
- `int score;`: 각 학생의 점수를 저장할 변수 선언.
- `scanf("%d", &score);`: 사용자로부터 점수를 입력받아 변수에 저장합니다.
- `sum += score;`: 각 학생의 점수를 누적 합산합니다.
- 평균을 계산할 때는 정수 나눗셈을 사용하므로 소수점 아래는 버림됩니다.

### 조건문과 변수

조건문을 통해 변수를 활용하는 또 다른 예시를 살펴봅시다. 예를 들어, 사용자의 나이에 따라 등급을 결정하는 코드입니다.

```c
#include <stdio.h>

int main() {
    int age;
    char grade; // 등급 저장 변수 ('A', 'B', 'C')

    printf("나이를 입력하세요: ");
    scanf("%d", &age); // 사용자로부터 나이 입력 받기

    if (age >= 18 && age <= 25) {
        grade = 'A'; // 18세~25세 범위의 학생은 'A' 등급
    } else if (age > 25 && age <= 30) {
        grade = 'B'; // 26세~30세 범위의 학생은 'B' 등급
    } else {
        grade = 'C'; // 그 외는 'C' 등급
    }

    printf("등급: %c\n", grade);

    return 0;
}
```

**해설:**
- `if (age >= 18 && age <= 25)`: 나이가 18세에서 25세 사이인지 확인합니다.
- `grade = 'A';`: 조건이 맞으면 등급 'A'를 할당합니다.
- `else if (age > 25 && age <= 30)`: 다른 범위 조건 확인.
- `else`: 그 외의 경우 등급 'C'를 할당합니다.

**질문 1:** 변수를 선언할 때 타입을 반드시 지정해야 하나요?
- **답변:** 네, 맞습니다! 타입을 명시하지 않으면 컴파일러가 변수의 타입을 정확히 인식하지 못해 오류가 발생할 수 있습니다. 항상 정확한 타입을 지정해주세요.

**질문 2:** 변수 이름을 정할 때 어떤 규칙을 지켜야 하나요?
- **답변:** 변수 이름은 영어 알파벳, 숫자, 언더스코어(`_`)를 사용할 수 있지만, 숫자로 시작할 수는 없습니다. 예를 들어, `myScore`, `_studentID`, `total_points`는 유효하지만 `1st_position`은 허용되지 않습니다.

## 실무 주의보 🚨

**주의 사항:** 변수를 선언할 때 메모리 관리에 유의해야 합니다. 특히, 포인터를 사용할 때는 메모리 누수나 잘못된 접근을 피해야 합니다. 예를 들어, 동적으로 할당된 메모리는 `free()` 함수로 해제해야 합니다.

```c
int* dynamicArray = (int*)malloc(sizeof(int) * 10); // 메모리 동적 할당
// 사용 후
free(dynamicArray); // 메모리 해제
```

이런 기본 원칙을 잘 지켜나가면, 변수를 효과적으로 관리하며 안정적인 코드를 작성할 수 있습니다.
