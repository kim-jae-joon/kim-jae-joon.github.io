## 4강: C언어 기초 마스터하기 - 연산자 종류와 마법 같은 활용법!

안녕하세요, 코딩의 세계로 함께 탐험하고 있는 여러분! C언어의 마법사 **"코드 캡틴"**입니다. 오늘은 당신의 코딩 기초를 한층 더 단단하게 만들어줄, **"연산자"**의 세계로 뛰어들어볼게요. 🤯 이 강의는 당신이 C언어를 마치 숙련된 장인처럼 다루게 도와줄 거예요. 진짜 신기하죠? 

### 🚀 연산자란 무엇인가요?

연산자는 간단히 말해 **데이터를 다루는 도구**입니다. 덧셈, 뺄셈, 곱셈, 나눗셈 같은 기본적인 수학적 연산부터 논리적 판단까지 다양한 역할을 합니다. 이걸 이해하지 못하면 코딩은 마치 빵 굽기에서 반죽을 넣지 않는 것과 같아요! 🍞👎

### 연산자의 주요 종류

#### 1. **산술 연산자 (Arithmetic Operators)**

산술 연산자는 숫자를 다루는 데 필수적인 친구들입니다. 다양한 수학 연산을 수행하며, 코딩에서 가장 기본적이면서도 자주 쓰이는 연산자들입니다.

##### 예제 1: 덧셈과 뺄셈
```c
#include <stdio.h>

int main() {
    int num1 = 10; // 첫 번째 숫자 설정
    int num2 = 5;  // 두 번째 숫자 설정
    int sum, diff; // 결과 변수 선언

    // 덧셈 연산자 '+' 사용
    sum = num1 + num2;  // num1 + num2의 결과를 sum에 저장
    printf("덧셈 결과: %d + %d = %d\n", num1, num2, sum); // 출력: 덧셈 결과 출력

    // 뺄셈 연산자 '-' 사용
    diff = num1 - num2;  // num1 - num2의 결과를 diff에 저장
    printf("뺄셈 결과: %d - %d = %d\n", num1, num2, diff); // 출력: 뺄셈 결과 출력

    // **왜 이렇게 했는지?**
    // `+`는 두 숫자를 더하고, `-`는 첫 번째 숫자에서 두 번째 숫자를 뺍니다.
    // 이렇게 하면 간단한 계산이 가능해져요!
    return 0;
}
```

##### 예제 2: 곱셈과 나눗셈
```c
#include <stdio.h>

int main() {
    int product, quotient;
    int a = 20, b = 4; // 숫자 설정

    // 곱셈 연산자 '*' 사용
    product = a * b;  // a * b의 결과를 product에 저장
    printf("곱셈 결과: %d * %d = %d\n", a, b, product); // 출력: 곱셈 결과 출력

    // 나눗셈 연산자 '/' 사용 (정수 나눗셈 주의!)
    quotient = a / b;  // a / b의 결과를 quotient에 저장 (정수 나눗셈은 소수점 아래 버림)
    printf("나눗셈 결과: %d / %d = %d\n", a, b, quotient); // 출력: 나눗셈 결과 출력

    // **주의사항**
    // 정수 나눗셈은 소수점 아래를 버리므로 정확한 소수 결과를 원한다면 `float` 타입을 사용해야 합니다.
    return 0;
}
```

#### 2. **비교 연산자 (Comparison Operators)**

비교 연산자는 두 값 사이의 관계를 판단합니다. 코딩에서 조건문을 작성할 때 핵심적인 역할을 합니다.

##### 예제 3: `if` 문 활용
```c
#include <stdio.h>

int main() {
    int age = 25; // 사용자의 나이 설정

    // '==' (같음), '!=' (다름), '>', '<', '>=', '<=' 비교 연산자 사용
    if (age >= 18) {  // 나이가 18세 이상인지 확인
        printf("성인입니다!\n"); // 성인일 경우 출력
    } else {
        printf("미성년자입니다.\n"); // 미성년자일 경우 출력
    }

    // **실무주의보**
    // 조건문은 프로그램의 흐름을 제어하는 데 필수적이므로 정확하게 작성해야 합니다.
    // 예를 들어, `>=` 대신 `>`를 사용하면 나이가 정확히 18세인 경우에는 성인으로 분류되지 않습니다.
    return 0;
}
```

##### 예제 4: 복합 비교 연산자
```c
#include <stdio.h>

int main() {
    int score = 85; // 시험 점수 설정

    if (score > 90 || score >= 80) { // 점수가 90점 이상이거나 80점 이상인지 확인
        printf("우수 성적입니다!\n"); // 조건 만족 시 출력
    } else {
        printf("노력이 필요해요.\n"); // 조건 불만족 시 출력
    }

    // **왜 이렇게 했는지?**
    // 복합 비교 연산자 `||` (또는)을 사용하면 여러 조건을 동시에 체크할 수 있어요.
    // 이렇게 하면 더 유연한 조건 설정이 가능해집니다.
    return 0;
}
```

#### 3. **논리 연산자 (Logical Operators)**

논리 연산자는 주로 조건의 결합과 부정에 사용됩니다. `&&` (그리고), `||` (또는), `!` (부정)이 주요 친구들입니다.

##### 예제 5: 논리 연산자 활용
```c
#include <stdio.h>

int main() {
    int isRaining = 1; // 비가 오는지 여부 (1: 오는 중, 0: 안 옴)
    int hasUmbrella = 1; // 우산을 가지고 있는지 여부 (1: 있음, 0: 없음)

    // 논리 AND 연산자 '&&' 사용
    if (isRaining && hasUmbrella) {  // 비가 오면서 우산을 가지고 있는지 확인
        printf("우산 쓰세요!\n"); // 조건 만족 시 출력
    } else {
        printf("비가 오지 않거나 우산이 없네요.\n"); // 조건 불만족 시 출력
    }

    // 논리 OR 연산자 '||' 사용
    if (isRaining || !hasUmbrella) {  // 비가 오거나 우산이 없는지 확인
        printf("주의하세요! 비가 오는 상황!\n"); // 조건 만족 시 출력
    }

    // **왜 이렇게 했는지?**
    // 논리 연산자는 복잡한 조건을 간결하게 표현할 수 있게 해줍니다.
    // 예를 들어, 여러 조건이 동시에 충족되어야 하는 경우 `&&`를 사용하면 매우 유용해요.
    return 0;
}
```

### 💡 초보자 폭풍 질문! 💡

1. **Q: 정수 나눗셈 시 소수점 아래가 버리는 이유는 무엇인가요?**
   **A:** 정수 나눗셈은 결과를 정수로 반환하므로 소수점 아래는 버려집니다. 만약 소수점 아래 값도 필요하다면 `float` 또는 `double` 타입을 사용하여 실수 나눗셈을 수행해야 합니다.

2. **Q: 논리 연산자 `&&`와 `||`의 차이점은 무엇인가요?**
   **A:** `&&`는 두 조건 모두 참일 때만 참을 반환하고, `||`는 두 조건 중 하나라도 참이면 참을 반환합니다. 이를 통해 조건을 더 유연하게 조합할 수 있어요.

### 🚨 실무주의보 🚨

연산자를 적절히 활용하지 못하면 코드가 복잡해지고 오류가 생길 수 있어요. 특히 조건문에서 논리 연산자를 잘못 사용하면 예상치 못한 동작이 발생할 수 있으니 주의하세요! 예를 들어, `if (x && y)`에서 `x`가 `0`일 경우 `y`를 평가하지 않아도 됩니다. 이는 성능 최적화에도 도움이 되지만, 의도하지 않은 결과를 초래할 수 있으니 항상 주의 깊게 코드를 검토하세요.
