## 배열이란 무엇인가요? 🤔

**배열**이란 간단히 말해 **동일한 데이터 타입을 가진 여러 값들을 순서대로 저장하는 방법**입니다. 생각해보세요, 여러분의 앨범 속 사진들처럼 말이죠! 각각의 사진은 특별하지만 모두 앨범이라는 공통의 컨테이너 안에 들어있죠. 배열도 마찬가지입니다. 각각의 요소는 개별적이지만 하나의 큰 그룹 안에 묶여서 관리됩니다.

### 배열의 기본 구조

```c
#include <stdio.h>

int main() {
    // 정수형 배열 선언: 크기가 5인 정수 배열 `numbers` 생성
    int numbers[5];

    // 배열 요소에 값 할당
    numbers[0] = 10; // 첫 번째 요소에 10 할당
    numbers[1] = 20; // 두 번째 요소에 20 할당
    numbers[2] = 30; // 세 번째 요소에 30 할당
    numbers[3] = 40; // 네 번째 요소에 40 할당
    numbers[4] = 50; // 다섯 번째 요소에 50 할당

    // 배열 요소 출력
    printf("배열 요소 출력:\n");
    for (int i = 0; i < 5; i++) { // 0부터 4까지 반복
        printf("numbers[%d] = %d\n", i, numbers[i]);
    }

    return 0;
}
```

**코드 해설:**
- `int numbers[5];` : 크기가 5인 정수형 배열 `numbers`를 선언합니다. 인덱스는 0부터 시작하므로 최대 인덱스는 4입니다.
- `numbers[i] = value;` : 각 요소에 값을 할당합니다. 인덱스 `i`에 해당하는 위치에 값을 넣습니다.
- `for` 루프를 사용해 배열의 모든 요소를 출력합니다. `i`는 배열의 인덱스를 순회하며 `numbers[i]`에 접근합니다.

## 다양한 반복 구조 활용하기 🕹

배열을 다룰 때 다양한 반복 구조를 활용하면 더욱 유연하게 작업할 수 있습니다. 오늘은 **for 문**, **while 문**, 그리고 **do-while 문**을 통해 배열의 요소를 처리하는 방법을 살펴보겠습니다.

### 1. **for 문:** 가장 기본적인 반복 구조

```c
#include <stdio.h>

int main() {
    int scores[3] = {85, 90, 78}; // 배열 초기화와 동시에 값 할당

    printf("성적 리스트:\n");
    // 0부터 2까지 반복 (배열 크기 - 1까지)
    for (int i = 0; i < 3; i++) {
        printf("성적 %d: %d\n", i + 1, scores[i]); // 인덱스와 값 출력
    }

    return 0;
}
```

**코드 해설:**
- `for` 루프는 명확한 시작과 끝 조건을 제공합니다. 여기서는 배열의 크기가 3이므로 `i < 3` 조건을 사용합니다.

### 2. **while 문:** 조건에 따라 반복

```c
#include <stdio.h>

int main() {
    int count = 0;
    int grades[5]; // 크기가 5인 배열

    // 사용자로부터 점수 입력 받기
    printf("최대 5개의 점수 입력:\n");
    while (count < 5) {
        printf("점수 %d 입력: ", count + 1);
        scanf("%d", &grades[count]);
        count++; // 카운터 증가
    }

    printf("입력된 점수:\n");
    for (int i = 0; i < 5; i++) {
        printf("%d\n", grades[i]);
    }

    return 0;
}
```

**코드 해설:**
- `while` 루프는 사용자가 원하는 만큼 입력을 받을 수 있게 해줍니다. `count`가 5 미만일 때까지 반복됩니다.

### 3. **do-while 문:** 최소한 한 번은 실행되는 반복 구조

```c
#include <stdio.h>

int main() {
    int choice = 1;
    int numbers[3] = {1, 2, 3};

    do {
        printf("배열 요소 출력:\n");
        for (int i = 0; i < 3; i++) {
            printf("%d ", numbers[i]);
        }
        printf("\n또 보실래요? (1: 예, 다른 숫자: 아니요) ");
        scanf("%d", &choice);
    } while (choice == 1); // 선택이 1일 때까지 반복

    return 0;
}
```

**코드 해설:**
- `do-while` 루프는 최소한 한 번 실행되며, 사용자의 입력에 따라 조건을 만족할 때까지 계속 반복됩니다. 여기서는 사용자가 계속 보고 싶을 때까지 배열 요소를 출력합니다.

## 💡 초보자 폭풍 질문! 🧐

**질문 1:** 배열 인덱스는 왜 0부터 시작하나요?
- **답변:** 배열 인덱스는 컴퓨터 과학에서의 효율성을 위해 0부터 시작합니다. 이렇게 하면 배열의 시작 위치를 정확히 가리키고, 요소 접근이 훨씬 간편해집니다. 마치 책의 첫 페이지가 1장이 아닌 0장이라는 것과 비슷합니다!

**질문 2:** 배열 크기를 동적으로 변경할 수 있나요?
- **답변:** 기본적인 C에서는 배열 크기를 런타임에 동적으로 변경하는 것이 어렵습니다. 대신, **동적 메모리 할당**을 사용하는 `malloc`, `realloc`, `free` 함수를 통해 메모리를 유연하게 관리할 수 있습니다. 이는 좀 더 고급 주제이지만, 코딩 실력을 한층 업그레이드시키는 데 큰 도움이 됩니다!
