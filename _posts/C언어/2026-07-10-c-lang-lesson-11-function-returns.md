## 🔥11강: return 문 및 함수 호출 - 코드의 마법사가 될 준비!🔥

안녕하세요! C언어 일타 신입생들, 환영합니다~ 😎 오늘은 'return문'과 '함수 호출', 이 두 마법 같은 친구들을 소개하는데요. 🤯  이 튜토리얼을 보고나면 함수에 담긴 힘겨운 코드를 손쉽게 읽고 파악할 수 있게 될 거예요! 🚀

### 🔍 return 문: 코드의 결과물 집합자 🎉
`return문`은 함수가 계산한 결과 값을 돌려주는 마법 지팡이 같은 역할을 해요. 🧙‍♂️ 함수 안에서 작업을 한 후, `return` 명령어를 사용해서 그 결과값을 바깥 세계로 전달하는 거죠.

> 예시:
```c
int add(int a, int b) { // 함수 정의!
  int sum = a + b;    // a와 b를 더해서 합계 계산
  return sum;        // 계산된 합계 값을 '바깥'으로 돌려줌!
} 

int main() {
  int result = add(5, 3); // 함수 호출하여 결과값 저장!
  printf("결과: %d\n", result); // 출력 결과 확인!
  return 0;
}
```

* `add` 함수는 두 개의 정수를 입력받아 그 합을 계산하는데요. 🤔 
* `return sum;` 라는 부분이 마법 지팡이 역할을 하죠!  합계 값 (`sum`)를 `main` 함수로 돌려보내서 결과를 확인할 수 있게 해주는 거예요.

### 🧰 함수 호출: 코드의 작동 시스템 ⚙️
함수는 프로그램에서 특정 작업을 수행하는 '기능 단위'라고 생각하면 좋아요! 기능이라고 하니, 요즘 트렌드에 맞게 **API** 라고 부르면 더 좋겠네요. 😎

`함수 호출`은 이러한 함수들을 실행하고 그 결과를 사용하는 과정입니다. 💥 코드의 작동 시스템을 생각하면, 함수는 각기 다른 기계들이고, `호출`이란 기계가 하나씩 움직이는 순간이죠.

```c
#include <stdio.h>

int add(int a, int b) {
  return a + b;
}

int main() {
  int result = add(5, 3); // 함수 호출! 'add' 함수가 실행되어 8이 반환됩니다.
  printf("결과: %d\n", result); // 결과 출력!
  return 0;
}
```

### 🚀 실무주의보: return 문과 함수 호출의 강력한 연합 🔥

* **코드 재사용:** 함수를 여러 번 사용하면 코드를 반복 쓰지 않고, 효율적으로 활용할 수 있어요.  
* **독립적인 모듈:** 각 함수는 작은 프로그램처럼 독립적이며, 복잡한 문제를 쉽게 해결할 수 있도록 도와줍니다.
* **관리 용이성:** 코드가 분리되어 관리하기 훨씬 쉬워요!

### 🎁  함수 호출의 다양한 유형

`int main()` 함수 외에도 여러 종류의 함수를 만들고 호출할 수 있습니다! 🎉 다음은 예시입니다.


```c
// 함수 정의: 'square' 함수는 입력받은 값을 제곱하여 반환
int square(int num) { // 정수형 인자를 받아도 정수형 결과를 반환하는 함수
  return num * num; 
}

int main() {
  int result = square(5); // 'square' 함수 호출! 결과값은 25입니다.
  printf("제곱: %d\n", result);
  return 0;
}
```
