**17강: 변수와 데이터 형식 - 종합 이해**

안녕하세요! 여러분, 오늘은 C언어가 처음 배운다면 꼭 알아야 하는 개념을 배우게 됩니다. 변수와 데이터 형식이 무엇인지, 어떻게 사용하는지, 왜 필요한지도 함께 공부할 것입니다. 😊

변수란? 🤔
--------

변수는 데이터를 임의로 이름을 붙여서 저장하고 참조할 수 있게 해주는 메모리 공간입니다. 예를 들어, 여러분이 어떤 사람에게 이름을 묻는다고 합시다. 그 사람이 이름은 '장동건'이라고 하면, 우리는 그 이름을 기억하는 공간으로 생각됩니다. 하지만, 실제로 컴퓨터가 가지고 있는 메모리는 0과 1의 조합으로만 구성되어 있습니다. 그래서 우리는 이러한 0과 1의 조합에 이름을 붙여서 사용할 수 있게 하는 것이 변수입니다.

예를 들어, 우리가 int x; 라고 선언하면, 메모리에 공간이 할당되고 그 공간에 이름 'x'을 붙입니다. 이 때, x라는 이름은 정수형인지를 의미합니다.

```c
#include <stdio.h>

int main() {
    int x;
    printf("%d\n", sizeof(x));
    return 0;
}
```

위의 코드에서 `sizeof` 함수는 변수에 저장된 데이터의 크기를 리턴하는 함수입니다. 위의 예제에서는 변수 x의 크기는 4로 출력됩니다.

```markdown
### 크기 종류

- 정수형(int): 4byte
- 실수형(float): 4byte
- 문자형(char): 1byte
```

위와 같이, 데이터 형식에 따라 크기가 달라집니다. 위의 예제에서는 int x; 가 선언된 것과 같이, 정수형인데 메모리의 크기는 4byte입니다.

**변수 데이터 형식 종류** 🤔

C언어에서 변수를 사용하는 데에는 여러 가지 데이터 형식이 있습니다. 아래는 그 종류이다.

*   **정수형(int)**: 4byte, 정수를 저장할 때 사용
*   **실수형(float)**: 4byte, 실수를 저장할 때 사용
*   **문자형(char)**: 1byte, 한 글자만 저장할 때 사용
*   **문자열형(char\*)**: 문자와 함께 '\0'으로 끝나며, 여러 문자를 저장할 때 사용

위의 데이터 형식에 대해 예제로 알아보겠습니다.

```c
#include <stdio.h>

int main() {
    int a = 10;
    float b = 3.14f;
    char c = 'A';
    char str[] = "Hello, World!";
    
    printf("%d %f %c %s\n", a, b, c, str);
    
    return 0;
}
```

위의 예제에서는 int, float, char, char\*을 선언하고, 각각의 데이터를 저장했습니다. 위에서 선언한 변수는 메모리 공간에 할당되어 저장됩니다.

```markdown
### 크기 종류

- 정수형(int): 4byte
- 실수형(float): 4byte
- 문자형(char): 1byte
- 문자열형(char\*): 10byte (Hello, World!의 경우)
```

위와 같이, 데이터 형식에 따라 메모리의 크기가 달라집니다.

**포인터** 🚀

포인터는 주소로 참조할 수 있는 변수입니다. 포인터를 선언한 변수의 이름 앞에 '\*'를 붙여 사용합니다. 예를 들어, int \*x;라고 선언하면 x라는 이름으로 정수형의 메모리 공간을 할당하고 그 주소를 참조하는 변수가 됩니다.

```c
#include <stdio.h>

int main() {
    int a = 10;
    int *p = &a;
    
    printf("%d %p\n", a, p);
    
    return 0;
}
```

위의 예제에서 포인터 p는 주소로 정수형의 메모리 공간에 참조합니다. 위에서 사용한 & 연산자는 주소를 리턴하는 연산자입니다.

```markdown
### 포인터

- 포인터 선언: int \*p;
- 포인터에 데이터 할당: p = &a;
```

위와 같이, 포인터는 주소로 참조할 수 있는 변수입니다.
