##  C언어 일타: 파일 I/O - 파일 읽기 및 종료 (15강) 🚀

**안녕하세요! C언어 일타 최고의 선생님, 저게 누구냐? 맞아요! 🎸 신나는 코드와 함께 당신을 coding 스타로 만들어 줄 멋진 가르침을 전달하는 매력적인 강사, 그 자리에 서 있는 저를 만나세요!** 🔥

오늘은 **파일 I/O - 파일 읽기 및 종료** 에 대해 알아보겠습니다. 파일?  그건 프로그램이 먹고 살 수 있는 간판처럼, 데이터를 저장하는 공간이죠! 💻 그 안에 우리가 넣는 코드는 마치 요리 레시피와 같아서, 특정 순서로 이뤄져 있어요. 그런데, 이 파일을 읽어서 프로그램에서 사용하도록 해야지 말이죠! 😎  우리가 할 일은 바로, 이렇게 간편하게 파일을 열고 데이터를 가져오는 방법 배우기란 것입니다!

**1단계: 📚 파일 준비**

우선 우리가 쓰고 싶은 코드의 시작점인 "main" 함수 안에 파일 I/O 기능을 사용하는 헤더 파일 `<stdio.h>` 를 포함해야 합니다. 이 헤더 파일은 입력과 출력 작업, 특히 파일 처리를 위한 중요한 함수들을 제공하는 마법같은 지팡이라고 할 수 있습니다!  🪄

```c
#include <stdio.h> 

int main() {
    // ... 코드 작성 ...
}
```

**2단계: 💪 파일 열기 (fopen)**

`fopen` 함수를 사용하면 우리는 원하는 파일을 열고, 그 열린 파일에 대한 "File Pointer" 를 얻어낼 수 있습니다.  이 File Pointer는 파일을 열지 않았다면 쓸모없지만, 열었다면 마치 아침에 커피잔을 들려온 것처럼, 프로그램은 이제 바로 그 파일에서 내용을 읽고 사용할 수 있죠!

```c
FILE *fp = fopen("my_file.txt", "r"); // r: read 모드 (파일 읽기) 
```


* `FILE *fp`:  File Pointer 변수를 선언하고, 이 변수가 파일과 연결되어 있어야 합니다. 마치 우리가 가끔 쓰는 주소처럼!
* `"my_file.txt"`: 원하는 파일 이름을 지정합니다. 

**💡 초보자 폭풍 질문!**

> 'r' 이렇게 모드를 설정하는 건 왜 일까요? "w" 어떻게 사용하면 되는지 알려주세요? 🤔

**3단계: ✨ 파일에서 데이터 읽기 (fscanf)**

`fscanf` 함수를 사용하면, File Pointer를 통해 열린 파일에 있는 데이터를 읽어올 수 있습니다. `fscanf` 는 주로 "형식 지정자" 를 사용하여 데이터의 형태에 따라 적절히 읽어옵니다.  예를 들어, 문자열을 읽으려면 "%"c", 숫자를 읽으려면 "%d" 등과 같이 지정합니다!

```c
char text[100]; // 문자열 저장할 변수

fscanf(fp, "%s", text); // 파일에서 문자열 읽기
printf("파일에서 읽은 문자열: %s\n", text); 
```


* `text[100]`: 읽어온 문자열을 담을 배열입니다.

**🚨 실무주의보!**

>  `fscanf` 는 입력 데이터의 형태가 제대로 되지 않았을 때 오류를 일으킬 수 있습니다. 파일을 정확히 형식에 맞게 작성하는 것이 중요합니다. 😩


**4단계: 🚪 파일 종료 (fclose)**

파일 작업이 끝나면 `fclose` 함수를 사용하여 파일을 닫아야 합니다.  마치 커피잔을 마시고 깨끗하게 정리할 때처럼, 프로그램은 이렇게 파일을 관리해야 데이터 손실이나 오류가 생기지 않습니다! 🚀

```c
fclose(fp); // 파일 닫기
```



**5단계: 🎉 성공적으로 파일 읽기 완료!**

우리가 이제까지 배운 내용들을 코드에 적용하여, "my_file.txt" 파일에 있는 데이터를 읽어내는 간단한 프로그램을 작성해 보세요! 💪


```c
#include <stdio.h> 

int main() {
    FILE *fp = fopen("my_file.txt", "r"); // 파일 열기: 'r' 모드로
    if (fp == NULL) { // 파일이 존재하지 않으면 오류 처리!
        printf("파일을 열 수 없습니다!\n"); 
        return 1;  // 프로그램 종료 코드
    }

    char text[100]; 
    fscanf(fp, "%s", text); 
    printf("파일에서 읽은 문자열: %s\n", text); 

    fclose(fp); // 파일 닫기
    return 0;  // 프로그램 정상 종료 코드
}
```
