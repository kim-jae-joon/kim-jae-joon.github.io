### 🤔 기본 개념: 파일 입출력이란?
파일 입출력은 컴퓨터 프로그램이 데이터를 외부 저장 장치에 기록하고 읽어내는 과정을 말합니다. 쉽게 말해, 코드가 일기장이나 문서 파일처럼 데이터를 저장하고 불러올 수 있게 하는 거죠. 이 기능이 없으면 프로그램은 모든 데이터를 메모리에만 갇혀 있어야 합니다. 마치 비밀 일기가 컴퓨터 속에 갇혀 있는데, 읽거나 쓸 방법이 없는 것과 같죠! 😲

#### 📝 파일 입출력의 핵심 명령어
C 언어에서 파일 입출력을 담당하는 주요 함수들은 다음과 같습니다:

- **`fopen()`**: 파일을 열거나 생성합니다.
- **`fclose()`**: 파일을 닫습니다.
- **`fread()` / `fwrite()`**: 각각 파일에서 데이터를 읽고 파일에 데이터를 씁니다.
- **`fseek()`**: 파일 내에서 특정 위치로 이동합니다.

### ### 예제 1: 파일 열기와 기본 출력 - 간단한 일기 쓰기 🎉

가장 먼저, 파일을 어떻게 열고 기본적인 데이터를 쓸 수 있는지 살펴볼게요. 아래 코드는 "mydiary.txt"라는 파일을 생성하고 간단히 메시지를 쓰는 예제입니다.

```c
#include <stdio.h>

int main() {
    // 1. 파일 열기 (파일이 없으면 생성)
    FILE *file = fopen("mydiary.txt", "w");  // "w" 모드: 쓰기 전용, 파일 없으면 생성
    
    if (file == NULL) {
        printf("파일을 열 수 없습니다.\n");
        return 1;  // 에러 처리
    }
    
    // 2. 데이터 쓰기
    printf("오늘의 일기를 적어보세요:\n");
    fprintf(file, "안녕하세요, 코딩의 세계!\n");  // 파일에 텍스트 쓰기
    fprintf(file, "새로운 기능을 배우고 있어요.\n");
    
    // 3. 파일 닫기
    fclose(file);
    
    printf("일기 쓰기 완료! 파일 확인해보세요.\n");
    return 0;
}
```

**코드 해설:**
- **`fopen("mydiary.txt", "w")`**: "mydiary.txt" 파일을 생성하거나 열어 쓰기 모드로 설정합니다. `"w"`는 파일이 이미 존재하면 덮어쓰기 모드입니다.
- **`fprintf(file, "텍스트 문자열");`**: 파일 포인터 `file`을 통해 파일에 텍스트를 씁니다.
- **`fclose(file)`**: 파일 작업이 끝나면 반드시 닫아줍니다. 메모리 누수를 방지하기 위해 필수적입니다.

### ### 예제 2: 파일 읽기 - 일기 일기 복원 🗞️

다음으로는 파일에서 데이터를 읽는 방법을 알아보겠습니다. 이전에 작성한 "mydiary.txt" 파일을 다시 읽어보는 예제입니다.

```c
#include <stdio.h>

int main() {
    FILE *file = fopen("mydiary.txt", "r");  // 읽기 모드 "r"
    
    if (file == NULL) {
        printf("파일을 열 수 없습니다.\n");
        return 1;  // 에러 처리
    }
    
    char buffer[200];  // 읽어들일 버퍼 선언
    
    // 파일 내 데이터 읽기
    while (fgets(buffer, sizeof(buffer), file) != NULL) {  // 줄 단위로 읽기
        printf("%s", buffer);  // 화면에 출력
    }
    
    fclose(file);
    printf("일기 내용을 모두 확인했습니다!\n");
    return 0;
}
```

**코드 해설:**
- **`fopen("mydiary.txt", "r")`**: 파일을 읽기 모드로 열어줍니다.
- **`fgets(buffer, sizeof(buffer), file)`**: 파일에서 문자열을 줄 단위로 읽어들입니다. `buffer` 크기만큼 읽어들일 수 있습니다.
- **`while` 루프**: 파일의 끝까지 계속 읽어들입니다. `fgets`가 `NULL`을 반환하면 파일 읽기가 완료된 것입니다.

### ### 예제 3: 다양한 조건으로 데이터 쓰기 - 맞춤형 일기 📑

다양한 조건에 따라 데이터를 다르게 쓰는 방법도 알아볼까요? 예를 들어, 사용자의 입력에 따라 일기 내용을 다르게 저장할 수 있습니다.

```c
#include <stdio.h>
#include <stdlib.h>  // exit() 함수를 사용하기 위해 포함

int main() {
    FILE *file = fopen("customdiary.txt", "w");  // 파일 생성 및 쓰기 모드로 열기
    
    if (file == NULL) {
        printf("파일을 열 수 없습니다.\n");
        exit(1);  // 에러 처리: 프로그램 종료
    }
    
    char entry[100];
    char mood;
    
    printf("오늘 기분은? (happy/sad): ");
    scanf(" %c", &mood);  // 사용자 입력 받기 (공간 포함된 scanf 사용)
    
    if (mood == 'h') {
        fprintf(file, "행복한 하루를 보내고 있습니다!\n");
    } else if (mood == 's') {
        fprintf(file, "조금 슬픈 날이지만 극복할 수 있어요.\n");
    } else {
        fprintf(file, "오늘은 특별한 기분이네요!\n");
    }
    
    fclose(file);
    printf("일기 작성 완료!\n");
    return 0;
}
```

**코드 해설:**
- **`scanf(" %c", &mood)`**: 사용자로부터 한 글자 입력을 받습니다. `%c` 포맷 지정자는 문자 하나를 읽습니다.
- **`if-else` 구조**: 사용자의 기분에 따라 다른 메시지를 파일에 씁니다. 조건문을 활용해 다양한 시나리오를 처리할 수 있습니다.

### 💡 초보자 폭풍 질문! 🚨
- **Q**: 파일을 열 때 여러 모드가 있는데, 각각 어떤 상황에 써야 하나요?
  - **A**: 
    - `"r"`: 읽기 전용 (기존 데이터 읽기)
    - `"w"`: 쓰기 전용 (기존 데이터 덮어쓰기, 파일 없으면 생성)
    - `"a"`: 추가 전용 (파일 끝에 데이터 추가, 기존 데이터 덮어쓰지 않음)
    - `"r+"`: 읽고 쓰기 (파일을 동시에 읽고 쓸 수 있음)
    - `"w+"`: 읽기와 쓰기 (파일 생성 또는 덮어쓰기)
    - `"a+"`: 읽기와 추가 (파일 끝에 데이터 추가)
