### **2강: C언어 기초 - 코딩의 마법을 여는 문, C 언어의 기본 구조와 문법**

안녕하세요, 코딩의 초보자 여러분! 오늘은 C 언어의 세계로 여러분을 안내할 준비가 되었습니다. 🚀 **"C 언어"**라는 이름만 들어도 설레지 않나요? 이 언어는 컴퓨터의 뇌와 직접 대화하는 듯한 짜릿함을 선사합니다. 이제부터 여러분의 코딩 모험을 함께 시작해볼게요!

#### **# 1. C 언어의 기본 구조: Hello World에서 시작하기**

**🌟 핵심 개념: 프로그램의 시작**

가장 먼저 만나볼 것은 바로 **"Hello, World!"** 프로그램입니다. 이건 코딩의 문을 두드리는 첫걸음이죠!

**코드 예제 1: 가장 기본적인 "Hello World" 프로그램**

```c
#include <stdio.h>  // 입출력 기능을 사용하기 위한 헤더 파일 포함

int main() {        // 프로그램의 진입점인 main 함수 시작
    printf("Hello, World!\n");  // 문자열 출력 함수 printf를 사용하여 메시지 출력
    return 0;               // 프로그램 종료 시 0 반환 (정상 종료)
}
```

**🔍 코드 분석:**
- `#include <stdio.h>`: 이 줄은 우리가 사용할 함수들을 제공하는 라이브러리를 불러오는 역할을 합니다. 마치 주방에서 필요한 도구를 꺼내는 것과 같아요!
- `int main()`: 이 함수는 프로그램의 중심이자 시작점입니다. 모든 코드의 실행은 여기서부터 시작됩니다.
- `printf("Hello, World!\n");`: `printf` 함수는 화면에 텍스트를 출력합니다. `\n`은 줄바꿈을 의미해요. 마치 말하는 것처럼 메시지를 전달하는 거죠!
- `return 0;`: 프로그램이 잘 끝났음을 알려주는 코드입니다. 0은 일반적으로 성공적인 종료를 의미합니다.

**💡 초보자 폭풍 질문!**
- **Q: `#include <stdio.h>`는 왜 필요한가요?**
  - **A:** 이 헤더 파일은 `printf`와 같은 입출력 함수를 사용할 수 있게 해줍니다. 마치 주방에서 레시피를 가져오는 것과 같아요. 없이는 메시지를 출력할 수 없어요!

#### **# 2. 변수와 데이터 타입: 코드의 재료**

**🌟 핵심 개념: 정보 저장**

코딩에서 변수는 정보를 저장하는 컨테이너 같은 역할을 합니다. 다양한 종류의 데이터를 다루려면 적절한 타입을 선택해야 합니다.

**코드 예제 2: 변수 선언과 사용**

```c
#include <stdio.h>

int main() {
    int age = 25;       // 정수형 변수 age 선언 및 초기화
    double salary = 3000.50;  // 실수형 변수 salary 선언 및 초기화
    char initial = 'J';  // 문자형 변수 initial 선언 및 초기화

    printf("나이: %d\n", age);  // %d는 정수 포맷 지정자
    printf("월급: %.2f\n", salary);  // %.2f는 소수점 두 자리까지 표시
    printf("이름의 첫 글자: %c\n", initial);  // %c는 문자 출력

    return 0;
}
```

**🔍 코드 분석:**
- `int age = 25;`: 정수형 변수 `age`를 선언하고 25로 초기화합니다. 정수형은 정수를 저장하는 데 사용됩니다.
- `double salary = 3000.50;`: 실수형 변수 `salary`를 선언하고 3000.50으로 초기화합니다. 실수형은 소수점까지의 값을 저장합니다.
- `char initial = 'J';`: 문자형 변수 `initial`을 선언하고 'J'로 초기화합니다. 문자형은 단일 문자를 저장합니다.
- `printf` 함수에서 포맷 지정자(`%d`, `%.2f`, `%c`)는 각 변수 타입에 맞게 데이터를 출력하는 데 사용됩니다.

**🚨 실무주의보:**
- **실수형 타입 사용 시 주의**: 실수 연산에서는 부동소수점 오차가 발생할 수 있으니 정밀한 계산이 필요한 경우 주의가 필요합니다. 예를 들어, 금융 계산에서는 정수형을 사용하는 것이 더 안전할 수 있어요!

#### **# 3. 제어 구조: 코드의 흐름 제어**

**🌟 핵심 개념: 프로그램의 흐름 조절**

이제 코드의 흐름을 조절하는 기본 제어 구조를 알아볼게요. 반복문과 조건문이 여기에 포함됩니다.

**코드 예제 3: 반복문 - `for` 문**

```c
#include <stdio.h>

int main() {
    for (int i = 0; i < 5; i++) {  // 초기값, 조건, 증감식 정의
        printf("반복 번호: %d\n", i);
    }
    return 0;
}
```

**🔍 코드 분석:**
- `for (int i = 0; i < 5; i++)`: 이 구조는 세 부분으로 나뉩니다.
  - 초기값: `int i = 0` - 변수 `i`를 0으로 초기화합니다.
  - 조건: `i < 5` - 루프가 계속되는 조건입니다.
  - 증감식: `i++` - 루프가 한 번 돌 때마다 `i`가 1 증가합니다.

**코드 예제 4: 반복문 - `while` 문**

```c
#include <stdio.h>

int main() {
    int count = 0;
    while (count < 3) {  // 조건이 참인 동안 반복
        printf("카운트: %d\n", count);
        count++;  // 조건을 만족시키기 위해 값 증가
    }
    return 0;
}
```

**🔍 코드 분석:**
- `while (count < 3)`: 조건이 참일 때까지 반복합니다. 여기서는 `count`가 3 미만일 때까지 실행됩니다.
- `count++`: 각 반복마다 `count` 값을 1 증가시킵니다.

**코드 예제 5: 조건문 - `if-else` 문**

```c
#include <stdio.h>

int main() {
    int score = 85;
    if (score >= 90) {
        printf("장학금 수혜 자격!");
    } else if (score >= 70) {
        printf("우수 학생입니다.");
    } else {
        printf("개선 필요!");
    }
    return 0;
}
```

**🔍 코드 분석:**
- `if (score >= 90)`: 첫 번째 조건이 참이면 해당 코드 블록이 실행됩니다.
- `else if (score >= 70)`: 첫 번째 조건이 거짓이면 이 조건을 확인합니다.
- `else`: 위의 모든 조건이 거짓일 때 실행되는 기본 코드 블록입니다.

**💡 초보자 폭풍 질문!**
- **Q: 반복문과 조건문을 언제 어떻게 사용해야 하나요?**
  - **A:** 반복문은 일정한 작업을 여러 번 반복할 때 사용합니다 (예: 리스트 순회). 조건문은 특정 조건에 따라 다른 동작을 수행할 때 사용합니다 (예: 점수에 따른 등급 부여). 상황에 맞게 적절히 선택하면 됩니다!

#### **# 4. 함수: 코드의 모듈화**

**🌟 핵심 개념: 코드 재사용**

함수는 코드를 논리적으로 나누고 재사용 가능하게 만드는 핵심 요소입니다. 마치 레시피를 나눠 담아놓은 요리 상자와 같죠!

**코드 예제 6: 간단한 함수 예제**

```c
#include <stdio.h>

// 함수 선언: 매개변수와 반환값 정의
int greet(char name) {
    printf("안녕하세요, %s님!\n", name);  // 이름을 받아 환영 메시지 출력
    return 0;  // 함수의 결과 반환
}

int main() {
    greet("지수");  // 함수 호출
    return 0;
}
```

**🔍 코드 분석:**
- `int greet(char name)`: 함수 `greet`를 선언합니다. 매개변수 `char name`을 받아 환영 메시지를 출력하고 정수 값을 반환합니다.
- `greet("지수");`: `main` 함수에서 `greet` 함수를 호출하고 "지수"라는 이름을 전달합니다.

**🚨 실무주의보:**
- **함수의 효율성**: 함수를 잘 사용하면 코드의 가독성과 유지보수성이 크게 향상됩니다. 작은 기능 단위로 나누어 작성하는 것이 좋습니다.
