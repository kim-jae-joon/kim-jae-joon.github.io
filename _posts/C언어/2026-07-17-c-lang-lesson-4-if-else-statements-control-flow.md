**4강: if-else 문 - 조건문 및 제어문의 정석! 🔥**

안녕하세요, 여러분! 저는 오늘도 재미있게 시작할 거예요. 😊 이번에는 'if-else 문'에 대해서 이야기하겠습니다. 이게 바로 프로그램이 어떤 조건에 따라 다른 행동을 하게 만드는 마술적인 마법입니다. 🧙‍♂️

**왜 if-else 문이 필요한가요?**

조건문과 제어문을 사용할 때, 우리는 프로그램이 특정한 조건에 따라서만 코드를 실행시키고자 합니다. 이럴 때, if-else 문이 등장합니다! 💡 하지만, 왜 이게 필요했나요?

 예를 들어, 여러분은 수업 시간에 15분 이상 지났다면, 교재에서 공부하도록 말하는 거죠. 그러나, 만약 5분 안에는 식당으로 가야 하거나, 다른 동기는 급하게 물어볼 수도 있습니다. 이럴 때, 프로그램이 15분보다 더 많은지, 또는 적은지를 확인하고에 따라서 행동을 해야 합니다.

**if-else 문의 기본 구조**

if-else 문은 간단히 말해 '조건'과 '코드'를 연결하는 것입니다. 조건이 참일 때( if ) 코드가 실행되며, 그렇지 않으면 else 에 있는 코드가 실행됩니다!

```c
if (조건) {
    // 조건이 참일 때 수행할 코드
} else {
    // 조건이 거짓일 때 수행할 코드
}
```

**if-else 문의 예시**

### 1. 간단한 if-else문

예를 들어, 프로그램이 5 이상의 숫자를 입력받았을 때 "숫자는 5 이상입니다."라는 메세지를 출력하는 경우!

```c
#include <stdio.h>

int main() {
    int num;

    printf("숫자를 입력하세요: ");
    scanf("%d", &num);

    if (num >= 5) {
        printf("숫자는 %d 이상입니다.\n", num);
    } else {
        printf("숫자는 %d 이하입니다.\n", num);
    }

    return 0;
}
```

*   `scanf()` 함수를 사용하여 숫자를 입력받습니다. 
*   `if`문을 통해, 입력받은 숫자가 5 이상인지 확인합니다.
*   조건이 참일 때, 프로그램은 `printf()` 함수를 이용해 "숫자는 %d 이상입니다."라는 메세지를 출력합니다.

### 2. 중첩된 if-else문

만약 여러분의 프로그램에 여러 경우를 처리해야 한다면? 예를 들어, 여러분은 프로그램이 수학 성적을 입력받았을 때, 평균 점수가 A, B, C, D, F 여덟 등급으로 나눠지게 하려면 어떻게 하나요?

```c
#include <stdio.h>

int main() {
    int score;
    char grade;

    printf("점수를 입력하세요: ");
    scanf("%d", &score);

    if (score >= 90) {
        grade = 'A';
    } else if (score >= 80) {
        grade = 'B';
    } else if (score >= 70) {
        grade = 'C';
    } else if (score >= 60) {
        grade = 'D';
    } else {
        grade = 'F';
    }

    printf("당신의 등급은 %c입니다.\n", grade);

    return 0;
}
```

*   `if`문을 중첩적으로 사용하여, 점수가 90 이상인지 확인합니다. 
*   만약 조건이 참이면 'A'를 할당하고 아니라면 아래의 조건을 반복적으로 확인합니다.

### 3. 여러 조건에 대한 처리

만약 여러분이 프로그램에서 여러 조건에 따라서 다른 행동을 하게 만들고 싶다면? 예를 들어, 여러분은 "오늘 날짜가 요일인지, 토요일인지, 일요일인지"에 따라서 다른 메세지를 출력하시겠나요?

```c
#include <stdio.h>

int main() {
    char day[10];

    // 오늘 날짜와 요일을 입력받습니다.
    scanf("%s", day);

    if (day[0] == 'S' || day[0] == 'M') { // 월요일, 화요일
        printf("오늘은 평일반입니다.\n");
    } else {
        printf("오늘은 주말입니다.\n");
    }

    return 0;
}
```

*   `if`문에 여러 조건을 연결하여 확인합니다. 
*   만약 조건이 참이면 "오늘은 평일반입니다."라는 메세지를 출력하고, 아니라면 "오늘은 주말입니다."라는 메세지를 출력합니다.

이러한 예제를 통해 여러분은 if-else 문을 어떻게 활용할 수 있는지 이해하실 겁니다! 💡

**Q&amp;A**

*   **질문:** 조건문과 제어문은 어떤 차이가 있나요?
    *   **답:** 조건문은 프로그램이 특정 조건에 따라서 코드를 실행시키는 문법입니다. 이와 달리, 제어문은 프로그램의 흐름을 바꾸기 위해 사용하는 문법입니다.
*   **질문:** 여러 조건이 있으면 어떻게 처리해야 하나요?
    *   **답:** if-else 문을 중첩적으로 사용하거나, switch-case문을 이용하여 다중 조건을 처리할 수 있습니다.

**💡 초보자 폭풍 질문!**

*   Q: 조건문과 제어문의 기본 구조는 어떻게 될까요?
    A: 조건문은 `if`와 `else`를 이용하여 조건에 따라 코드를 실행시키고, 제어문은 프로그램의 흐름을 바꾸기 위해 `while`, `for`, `switch-case` 문을 사용합니다.
*   Q: 중첩된 if-else문을 사용하는 것보다 switch-case문을 사용하는 것이 좋나요?
    A: 중첩된 if-else문은 복잡한 로직에서 유용하지만, switch-case문은 가독성과 성능이 뛰어나기 때문에 추천드립니다!

**🚨 실무주의보!**

*   **주의:** 조건문과 제어문을 잘못 사용할 경우, 프로그램의 오류가 발생하여 안정적인 작동을 할 수 없습니다. 따라서, 이에 대한 이해와 정확한 구현이 필수입니다!
*   **주의:** 여러 조건에 대한 처리를 할 때, switch-case문보다는 중첩된 if-else문을 사용하는 것이 가독성과 유지보수 측면에서 유용합니다.
