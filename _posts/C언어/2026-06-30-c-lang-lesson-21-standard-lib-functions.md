### 📚 stdlib.h - C 언어의 모든 것을 한 번에! (혹시 이걸 보면 개발자가 될 만한 소재인가?)

stdlib.h는 "C 표준 라이브러리 헤더 파일"을 의미하며, `malloc()`, `free()`, `printf()`, `rand()` 등과 같은 많은 기능들을 제공합니다.  💥💥💥 쉽게 말해서 C 언어 개발의 필수품이라고 할 수 있답니다! 😉

### 🧠 이제부터 알아볼거리 - stdlib.h의 주요 기능 🚀🚀🚀

#### 1. 메모리 관리 기능 💪💪💪

- **malloc()**: 필요한 메모리를 동적으로 할당하는 함수입니다.
    ```c
    int *ptr = (int *) malloc(sizeof(int) * 5); // 5개의 int형 데이터를 저장할 수 있는 메모리 할당
    ```
    **✅  주의!**:  `malloc()` 함수가 성공적으로 메모리를 할당하지 못하면 `NULL`을 반환합니다. 따라서 메모리가 할당되었는지 확인해야 합니다!

- **free()**: 사용한 후 메모리를 해제하는 함수입니다. 
    ```c
    free(ptr); // ptr에 할당된 메모리 해제
    ```
    **⚠️  주의!**: `free()`를 호출하지 않으면 메모리가 유지되고, 이는 "메모리 누수"라는 문제를 일으킬 수 있습니다. 😨

#### 2. 출력 함수 😎😎😎

- **printf()**: 데이터를 화면에 출력하는 가장 기본적인 함수입니다! 
    ```c
    int age = 25;
    printf("나이는 %d 입니다.\n", age); // "나이는 25 입니다." 출력
    ```
   **✨  코드 해설**: `%d`는 정수형 데이터를 나타내며, `{age}`는 변수 `age`의 값을 대체합니다.

#### 3. 기타 유용한 함수 🎁🎁🎁

- **rand()**: 임의의 수를 생성하는 함수입니다! (게임 개발에 자주 사용되죠!)
   ```c
   int random_num = rand(); // 0부터 RAND_MAX 사이의 임의의 정수 생성
   ```
- **srand()**: `rand()` 함수의 초기값을 설정하는 함수입니다.

###  🚨 실무주의보 🚨

stdlib.h 라이브러리를 사용할 때 주의해야 할 점이 있습니다! ⚠️⚠️⚠️

- 메모리 누수: `free()` 함수를 호출하지 않고 메모리를 사용하면, 프로그램 자원을 과도하게 소비하고 오류를 유발할 수 있습니다.  
   **✅ 따라서, `malloc()`로 메모리를 할당한 후에는 반드시 `free()`를 통해 해제해야 합니다.** 

- 잘못된 자료형: 함수의 매개변수에 적합하지 않은 자료형을 사용하면 실행 오류가 발생할 수 있습니다.  
   **✅ 항상 함수의 설명과 활용 사례를 확인하고, 정확한 자료형을 사용하세요.**
