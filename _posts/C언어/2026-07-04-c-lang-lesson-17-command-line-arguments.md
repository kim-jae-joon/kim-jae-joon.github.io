### 🤔 명령줄 인자란? 뭘 해주는 힘든 일인가요?

명령줄 인자는 우리가 터미널이나 명령 프롬프트에서 프로그램을 실행할 때 입력하는 정보들입니다.  예를 들어, 다음과 같은 명령어를 보세요:

```bash
./my_program hello world
```

여기서 `hello world` 부분이 바로 **명령줄 인자**입니다! 우리의 프로그램 'my_program'을 실행할 때, 사용자가 추가로 전달하는 정보라고 생각하면 됩니다. 🚀

💡 초보자 폭풍 질문! ❓
> 명령줄에 입력해주는 것인데, 이 정보가 어떻게 프로그램 안으로 들어갈까요? 🤔


### 🔐 C 언어와의 만남: 인자를 받아 들이는 방법!

C언어에서 명령줄 인자를 사용하려면 `argc` (argument count) 와 `argv` (argument vector) 변수들을 활용합니다. 
   -  `argc`: 프로그램이 받은 인자 개수를 나타내는 정수형 변수입니다.
   - `argv`: 인자들이 저장된 문자열 배열로, 각 요소가 한 명령줄 인자를 나타냅니다.

C언어의 예제 코드를 통해 봅시다!

```c
#include <stdio.h>
#include <stdlib.h>

int main(int argc, char *argv[]) {
  printf("인자 개수: %d\n", argc); // 입력받은 인자 개수 출력

  for (int i = 0; i < argc; i++) {
    printf("인자 %d: %s\n", i, argv[i]); // 각 인자 출력
  }

  return 0;
}
```

**코드 설명:**

1. `#include <stdio.h>` 와 `#include <stdlib.h>`: 프로그램에 필요한 표준 입출력 함수와 메모리 관리 함수를 포함합니다. 💡 잊지 말아야 할 중요한 친구들!
2. `int main(int argc, char *argv[])`: C 언어의 시작점인 `main` 함수입니다.  주요 인자로 `argc`와 `argv`를 받습니다. 🚨 실무주의보: 이 함수는 프로그램이 실행될 때 자동으로 호출됩니다!
3. `printf("인자 개수: %d\n", argc);`: 입력받은 인자의 개수를 출력합니다.
4. `for (int i = 0; i < argc; i++)`:  `argc` 만큼 반복문을 돌려 각 인자를 처리합니다.
5. `printf("인자 %d: %s\n", i, argv[i]);`: 현재 인덱스 번호(`i`)와 해당 인자(`argv[i]`)를 출력합니다.

### 🚀 실제 활용 사례: 프로그램을 더 유연하게 만들기!

명령줄 인자 처리 기술은 C언어 프로그램의 유연성과 사용자 친화도를 크게 높여줍니다. 몇 가지 실생활 예시를 살펴보겠습니다!

* **텍스트 파일 생성:** 명령줄에서 파일 이름을 입력받아 그에 맞는 빈 파일을 생성하는 프로그램

    ```bash
    ./create_file my_document.txt
    ```
* **정렬 알고리즘 테스트:** 다양한 정렬 알고리즘의 성능을 비교하기 위해, 명령줄에서 어떤 알고리즘을 사용할지 지정하는 프로그램
    ```bash
    ./sorting_tester bubble sort 
    ```

* **데이터 분석 도구:**  명령줄에 데이터 파일 경로를 입력받아 분석하고 결과를 출력하는 프로그램
    ```bash
    ./data_analyzer input.csv
    ```
