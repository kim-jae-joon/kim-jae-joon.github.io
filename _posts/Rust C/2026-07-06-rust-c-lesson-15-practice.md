###  🧱  Rust 프로그램 구조 이해하기: 블록 기반 시스템 🚀

Rust는 **블록(block)** 개념을 사용하여 코드를 정리하고 논리적인 구조를 만들어줍니다. 🤔   다시 말하자면, Rust 코드는 하나의 큰 집처럼 여러 개의 방들이 연결되어 있는 구조입니다. 각 방은 특정 작업을 수행하는 **블록**과 같아요! 🏡

**예제:**

```rust
fn main() {
    println!("안녕하세요, 세계!"); // 1번 블록: 출력문 실행
    let a = 5;  // 2번 블록: 변수 선언 및 초기화
    if a > 3 {   // 3번 블록: 조건문 시작
        println!("a는 3보다 크군요!"); // 4번 블록: 조건 충족시 출력 문 실행
    } else {  // 5번 블록: 조건 미충족시 실행되는 부분
        println!("a는 3보다 작거나 같아요."); // 6번 블록: 출력문 실행
    }
}
```

💡 초보자 폭풍 질문! Rust 코드를 '집'으로 비유해서 설명해 주셨는데, 각 방이 무엇을 담고 있는지 더 자세히 알려주실 수 있나요? 🤔

**설명:** 위 예제는 프로그램의 가장 기본적인 구조를 보여줍니다. 
1) `fn main()` :  프로그램 시작 부분! 여기서 모든 작업이 진행됩니다. 2) `println!("안녕하세요, 세계!");`: 출력 블록으로 "안녕하세요, 세계!" 문자열을 화면에 출력합니다.
3) `let a = 5;` : 변수 선언 및 초기화 블록입니다. 변수 `a`에 값 5를 저장합니다. 4-6) `if`문: 조건 판별 블록으로, `a`가 3보다 큰지 확인하여 그에 따라 다른 작업을 수행합니다.

###  🔗 Rust 문법 연습: 함수와 반복문! 💪


#### 🎯 함수 활용하기: 코드 재사용의 경험 🚀

Rust에서 **함수(function)**는 특정 작업을 수행하는 독립적인 블록입니다.  예를 들어, '더하기'라는 기능만 담은 함수를 만들고 여러 곳에서 이 함수를 사용하면 코드가 훨씬 가독성이 좋고 재사용 가능해집니다!

**예제:**
```rust
fn add(x: i32, y: i32) -> i32 { // 함수 정의
    x + y // 함수 내부 작업: x와 y를 더하기
}

fn main() {
    let result = add(5, 3); // 함수 호출 및 결과 저장
    println!("결과: {}", result); // 출력문을 통해 결과값 보여주기
}
```

💡 초보자 폭풍 질문! Rust에 함수를 정의하는 방법은 어떻게 되나요? 🤔


**설명:** 위 예제에서는 `add`라는 함수가 두 개의 정수 `x`와 `y`를 입력받고, 그 합계 값을 반환합니다.  `main()`함수에서 `add(5, 3)`를 호출하여 결과값을 저장하고 출력문으로 보여줍니다.

#### 🔄  반복문 활용하기: 데이터 처리의 효율성 높이기 🚀

Rust에서 **반복문(loop)**은 특정 작업을 여러 번 반복하는 데 사용됩니다. 예를 들어, 리스트 안에 있는 모든 요소를 순회하면서 처리하거나, 주어진 조건이 충족될 때까지 계속 작업을 수행하는 등 다양한 상황에서 활용할 수 있습니다!

**예제:**
```rust
fn main() {
    let numbers = [1, 2, 3, 4, 5]; // 정수 배열 초기화

    for number in numbers.iter() { // 반복문 시작: numbers 배열의 모든 요소를 순회합니다.
        println!("숫자: {}", number); // 각 요소 출력
    }
}
```

💡 초보자 폭풍 질문! Rust에서 어떤 종류의 반복문을 사용할 수 있나요? 🤔


**설명:**  위 예제에서는 `for` 문을 사용하여 `numbers` 배열에 있는 모든 요소를 순회합니다. 각 요소는 `number` 변수에 저장되고, 이 값이 출력됩니다.
