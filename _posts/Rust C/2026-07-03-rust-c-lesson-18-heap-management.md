## 🔥 18강: Rust의 heap 관리 시스템 배우기 - 동적 메모리 할당과 해제 🚀

안녕하세요, 여러분! 😎 대한민국 최고의 Rust C 일타 강사, 그리고 15년 차 시니어 개발자 "rusty_master"입니다. 오늘은 **Rust의 동적 메모리 관리 시스템**에 대해 심도 있게 알아볼 거예요. 마치 한입 물면 달콤한 초콜릿처럼 매혹적인 Rust의 동적 메모리 활용법을 탐험하고, 이를 통해 코드를 더욱 효율적으로 만들어보도록 하죠! 🍫

### **1. 우리는 언제 동적 메모리를 사용할까요? 🤔**

Rust에서 동적 메모리는 프로그램이 실행되는 동안 필요에 따라 메모리를 할당하는 것을 의미합니다.  static 메모리와 달리, 전역 변수나 함수 지역 변수처럼 초기 크기가 정해진 메모리가 아닌 **실제로 필요한 만큼의 공간을 쓸 수 있는 놀라운 기능**이죠! 💪

💡 초보자 폭풍 질문!
> "동적 메모리? 그건 C언어에서만 쓰는 거 아니야?" 🤔

정말 좋은 질문입니다! 👍  C 언어에서는 동적 메모리를 직접 관리하는 것이 번거롭고, **메모리 누수나 실제로는 사용하지 않아도 할당된 메모리의 공간을 낭비할 수 있는 오류 발생 위험**이 있습니다. 하지만 Rust는 이러한 문제들을 효과적으로 해결하여 동적 메모리를 안전하고 편리하게 사용할 수 있도록 도와주죠! 🎉

### **2.  `Box` 와 `new()` : Rust의 동적 메모리 할당 🧰**

Rust에서 동적 메모리 할당을 위해 주로 사용하는 것은 **`Box`** 이라는 자료구조입니다.  
> `Box<T>`는 어떤 타입 (T) 의 데이터를 담고 있는 동적 메모리 영역을 나타냅니다! 📦

`new()` 함수와 함께 사용하여 Rust의 동적 메모리를 효율적으로 관리할 수 있습니다. ⚙️


```rust
let data = Box::new(10);  // 'Box<i32>' 타입으로 10을 담은 동적 메모리 할당
println!("data: {}", data); // 출력: data: 10

// 다른 자료형도 가능! 📦
let name = Box::new("rusty_master"); 
println!("name: {}", name);  // 출력: name: rusty_master
```

* **`Box::new(value)`**:  'value' 값을 담은 `Box<T>` 타입의 동적 메모리 영역을 생성합니다.


### **3. `drop()` : Rust의 자동 해제 기능 ✨**

Rust는 동적 메모리 할당과 해제를 간편하게 처리하는 **자동 소멸  Garbage Collection 시스템**을 구현하고 있습니다! 🤖


`Box` 를 사용하면 해당 객체가 더 이상 필요하지 않을 때 자동으로 메모리를 해제하여 리소스를 효율적으로 관리할 수 있습니다. 🚀

```rust
let data = Box::new(10); // 'Box<i32>' 타입으로 10을 담은 동적 메모리 할당

// ... 코드 실행 후 데이터 더 이상 사용하지 않을 경우, 자동 해제!
```


> `drop()`는 Rust의 모든 변수들이 소멸될 때 자동으로 호출되는 함수입니다.  'data' 가 접근 범위를 벗어나면 `Box<i32>` 객체에 있는 데이터가 삭제되어 메모리에서 제거됩니다!

### **4. 실무 주의 사항: 다운캐스팅과 보안 🛡️**

* **다운캐스팅**: `Box`는 다른 타입으로 변환될 수 있습니다. 하지만, 타입이 정확하지 않으면 에러 발생 가능성이 있으므로 주의해야 합니다!
> 예를 들어 `let box_num: Box<i32> = Box::new(10); let box_str: Box<String> = box_num;` 에서는 `Box<i32>` 가 `Box<String>`으로 변환될 수 없습니다.

* **보안**:  동적 메모리를 안전하게 사용하기 위해 Rust에서는 `Unsafe` 문법을 이용하여 메모리 직접 접근이 가능하지만, 굉장히 위험한 작업입니다! 🔥
>  `Unsafe` 코드는 잘못 사용하면 심각한 보안 취약점을 초래할 수 있으므로 주의해야 합니다.

### **5. Rust의 heap 관리 시스템: 간결하고 강력 🚀**

Rust의 heap 관리 시스템은 동적 메모리를 안전하게, 효율적으로 관리하여 C언어에서 발생하는 오류를 방지하면서도 프로그램 성능을 최대화합니다! 💪


* **소멸자**: 
`Box<T>`는 `drop()` 함수를 사용하여 자동으로 소멸되므로 메모리 누수 문제 해결에 기여합니다.

* **생성/해제:**
`Box::new()`와 `drop()`를 통해 동적 메모리를 관리하는 것이 간단하고 직관적입니다.


* **타입 안전성**: Rust의 타입 시스템은 동적 메모리 할당과 해제 과정에서 오류를 방지하며 코드를 더욱 안정적으로 만듭니다!
