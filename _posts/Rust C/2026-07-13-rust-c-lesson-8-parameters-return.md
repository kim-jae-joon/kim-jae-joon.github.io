## 🚀 8강: 매개변수와 반환값 - 코드 마법의 주문을 배우다!

안녕하세요, 코딩 모험가 여러분! 💫 오늘은 우리 코드 세계에서 정말 중요한 두 가지 마법 주문, **매개변수**와 **반환값**에 대해 깊이 탐험해 볼 시간입니다. 혹시 코딩이 처음이라 낯선 단어들이 많이 들리셨나요? 걱정 마세요! 저와 함께라면 이 마법 주문들을 쉽게 익힐 수 있습니다. 지금부터 따라가시다 보면, 코딩이 얼마나 재미있고 멋진 일인지 체감하실 거예요!

### 💡 개념부터 알아보기

**매개변수**는 마치 마법의 주문을 불러낼 때 필요한 재료들 같아요. 예를 들어, "**'불을 피우는 주문'**"을 부르려면 "**'나무', '부채질', '불씨'**"가 필요하겠죠? 마찬가지로 함수(함수는 코드의 작은 마법사들이라고 생각해 보세요!)가 일을 하기 위해서는 **매개변수**라는 정보들이 필요해요. 이 매개변수들은 함수 호출 시점에 제공됩니다.

**반환값**은 그 마법 주문이 완성되었을 때 나오는 결과물이에요. "**'나무'를 이용해 '불을 피우는 주문'을 불렀더니 '뜨거운 불꽃'**"이 나오는 것처럼요! 함수가 어떤 작업을 수행한 후, 그 결과를 돌려주는 것이 바로 반환값이랍니다.

### 🧙‍♂️ 매개변수와 반환값의 마법 실습

#### 예제 1: 간단한 덧셈 함수

```rust
// 함수 선언: 'add'라는 이름의 마법 주문을 만듭니다.
fn add(a: i32, b: i32) -> i32 {
    // 매개변수 'a'와 'b'를 받아서 더합니다.
    let result = a + b; // 여기서 'result'는 덧셈의 결과물을 임시 보관하는 마법 그릇입니다.
    
    // 마법 주문의 결과물을 돌려줍니다 (반환값).
    result
}

fn main() {
    // 매개변수를 넘겨주며 마법 주문을 부릅니다 ('5'와 '3').
    let sum = add(5, 3);
    
    // 결과를 출력합니다. "불을 피우는 주문에 성공했습니다! 결과는?"
    println!("결과는 {}입니다!", sum); // 출력: 결과는 8입니다!
}
```

**해설:**
- **`fn add(a: i32, b: i32) -> i32`**: 이 부분은 함수 `add`를 정의합니다. 여기서 `a`와 `b`는 매개변수이고, `-> i32`는 이 함수가 `i32` 타입의 값을 반환한다는 의미입니다.
- **`let result = a + b;`**: 매개변수 `a`와 `b`를 더한 결과를 `result` 변수에 저장합니다.
- **`return result;`**: 함수의 결과를 `result`로 반환합니다.

#### 예제 2: 다양한 형태의 함수 호출

##### 반복문 활용

```rust
fn multiply_list(numbers: Vec<i32>, multiplier: i32) -> Vec<i32> {
    let mut results = Vec::new(); // 결과를 담을 마법의 그릇 준비
    
    for number in numbers.iter() { // 반복문을 통해 각 숫자에 대해 작업
        let product = number * multiplier; // 매개변수를 활용한 마법 주문
        results.push(product); // 결과를 그릇에 담음
    }
    
    results // 마법 주문의 결과물 반환
}

fn main() {
    let numbers = vec![1, 2, 3, 4]; // 숫자들을 마법의 목록으로 준비
    let multiplied = multiply_list(numbers, 5); // 함수 호출: '5'를 곱하는 주문
    
    println!("결과 목록: {:?}", multiplied); // 출력: 결과 목록: [5, 10, 15, 20]
}
```

**해설:**
- **반복문 (`for`)**: `for number in numbers.iter()`로 각 숫자에 대해 반복적으로 작업을 수행합니다.
- **매개변수 활용**: `number * multiplier`에서 매개변수 `multiplier`를 사용하여 각 숫자를 곱합니다.
- **반환값**: 함수는 곱셈 결과를 담은 `results` 벡터를 반환합니다.

##### 조건문 활용

```rust
fn check_even(number: i32) -> bool {
    if number % 2 == 0 { // 마법 주문: '짝수인지 확인'
        true // 마법 주문 성공!
    } else {
        false // 마법 주문 실패!
    }
}

fn main() {
    let test_number = 7; // 테스트할 숫자
    let is_even = check_even(test_number); // 함수 호출: '7이 짝수인지 확인'
    
    if is_even {
        println!("{}은 짝수입니다!", test_number);
    } else {
        println!("{}은 홀수입니다!", test_number);
    }
}
```

**해설:**
- **조건문 (`if-else`)**: 함수 내에서 `number % 2 == 0` 조건을 통해 숫자가 짝수인지 확인합니다.
- **반환값**: 함수는 `true` 또는 `false`를 반환하여 결과를 알려줍니다.

### 🧑‍💻 실무주의보

**주의사항**: 매개변수를 잘못 설정하거나 반환값을 명시하지 않으면 함수가 제대로 작동하지 않거나 오류를 발생시킬 수 있어요! 예를 들어, `fn add() -> i32 { /* 매개변수 없이 덧셈 수행 */ }`는 컴파일 오류를 일으킬 거예요. 항상 매개변수와 반환값을 명확히 정의하세요!

### 💡 초보자 폭풍 질문!

**질문 예시**:
1. 매개변수와 반환값이 없는 함수는 어떻게 만들 수 있을까요?
   - **답변**: 매개변수와 반환값이 필요하지 않은 경우에는 `fn no_params_no_return() { /* 코드 수행 */ }`처럼 간단히 정의할 수 있습니다. 하지만 대부분의 실용적인 함수에서는 적어도 하나의 매개변수나 반환값을 가지는 것이 일반적입니다.
   
2. 매개변수 타입을 다양하게 사용할 수 있나요?
   - **답변**: 네, 당연히 가능합니다! `fn process_data(data: String, value: f64) -> Vec<i32> { /* 데이터 처리 로직 */ }`처럼 문자열, 부동소수점, 벡터 등 다양한 타입을 매개변수로 사용할 수 있습니다.
