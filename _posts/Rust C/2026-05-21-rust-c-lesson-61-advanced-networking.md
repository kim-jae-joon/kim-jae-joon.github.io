### 1. 클라이언트-서버 구조 개념

클라이언트-서버는 가장 기본적인 네트워크 애플리케이션 아키텍처입니다. 

> 클라이언트: 정보를 요청하고 서버로부터 데이터를 받아오는 프로그램
> 서버: 클라이언트의 요청을 처리하고 응답하는 프로그램

예를 들어, 웹 브라우저가 웹 사이트에 요청을 보내면서 HTML 코드와 이미지 등을 받아 렌더링하는 것은 바로 클라이언트-서버 구조의 활용입니다.  🤯

**🚨 실무주의보:**
> 여러분이 만들어온 Rust 애플리케이션은 단순히 웹 브라우저와 같은 인터페이스를 가진 클라이언트가 아니고, 다른 프로그램들과 네트워크 상에서 소통하는 더 복잡한 시스템일 수 있습니다. 


### 2. TCP/UDP 프로토콜 이해

네트워크 애플리케이션은 데이터 전송에 필요한 "규칙"인 프로토콜을 사용합니다. 주요 프로토콜 중 두 가지를 살펴보겠습니다:

> **TCP (Transmission Control Protocol):**
    * 데이터를 순서대로 보내고, 오류를 확인하여 정확하게 전달하는 것을 목표로 합니다. 🧐 
    * 장점: 안정성이 높음!  👍
    * 단점: 비교적 느림. 👎

> **UDP (User Datagram Protocol):**
    * 데이터를 빠르게 전송하고, 오류 확인 기능은 부족합니다.   💨
    * 장점: 속도가 빠름! 🚀
    * 단점: 데이터 손실 가능성이 높음. 😭


### 3. Rust로 네트워킹 프로그래밍하기 - `tokio` 라이브러리 활용

Rust에서는 `tokio`라는 강력한 라이브러리를 사용하여 비동기 네트워킹을 구현할 수 있습니다.  💪

> **비동기:** 하나의 작업이 완료되면 다음 작업에 바로 넘어가는 방식으로, 여러 작업을 동시에 처리 가능! ✨
> **`tokio`**: Rust에서 비동기 프로그래밍을 위한 최고의 라이브러리 중 하나.  💯

**🚨 실무주의보:**
> `tokio`를 처음 접하는 분들을 위해 다음 강좌에서는 `tokio` 기본 개념부터 차근차근 설명하고 예제 코드를 통해 이해를 돕겠습니다!  🙌



### 4. 클라이언트-서버 구조 설계

우리의 Rust 네트워크 애플리케이션은 클라이언트와 서버로 구성됩니다:

**클라이언트:**
* 사용자가 입력하는 데이터를 받아옵니다.
* 서버에 요청을 보냅니다. (예: "Hello world!" 메시지 전송)

**서버:**
* 클라이언트의 요청을 수신합니다.
* 처리된 응답을 클라이언트로 돌려줍니다. (예: "Hello, from server!" 메시지 반환)


### 5. 프로토콜 설계 - 명확한 소통 위한 규칙 정하기

클라이언트와 서버가 효율적이고 안정적으로 소통하기 위해서는 명확한 '프로토콜'을 정의해야 합니다. 프로토콜은 데이터 전송 방식, 포맷 등에 대한 규칙입니다. 🚀


**5.1. JSON 프로토콜 활용:**

JSON (JavaScript Object Notation)은 데이터를 표현하는 표준 형식으로, Rust에서도 널리 사용됩니다!  🔥

* 클라이언트가 서버에게 "Get latest news"와 같은 요청을 보낼 때는 JSON 형식의 메시지를 사용합니다.
* 서버는 뉴스 기사 목록을 JSON 형식으로 반환합니다.

**💡 예제 코드:**


```rust
// 클라이언트에서 JSON 데이터를 전송하는 예시
let request_data = json!({ "action": "get_latest_news" });
// ... API 요청 함수 호출하여 JSON 데이터 전달

// 서버가 JSON 데이터를 처리하는 예시
fn handle_request(data: &Json) {
    if data["action"] == "get_latest_news" {
        // 뉴스 기사 목록 가져와서 JSON 형식으로 반환
    }
}
```



**5.2. Rust의 Serialization/Deserialization 라이브러리 활용:**

Rust에는 `serde`라는 라이브러리가 있어, 구조체나 클래스를 JSON 데이터로 변환하고 다시 읽어올 수 있습니다! 🤯


### 6. 네트워크 애플리케이션 개발 - 실제 예시

이번 강좌에서는 간단한 "Hello World" 네트워크 애플리케이션을 만들고, `tokio` 라이브러리를 활용하여 비동기 프로그래밍 방식을 이해해 보겠습니다! 😎


```rust
// 서버 코드 (server.rs)
use tokio::net::TcpListener;

#[tokio::main]
async fn main() {
    let listener = TcpListener::bind("127.0.0.1:8080").await.unwrap(); 

    loop {
        let (mut socket, _) = listener.accept().await.unwrap();
        // 클라이언트 연결 수락 시 새로운 스레드 생성
        tokio::spawn(async move {
            // 데이터 수신 및 처리 코드...
        });
    }
}

```
