##  🔥 54강: C와 Rust 코드 유지 보수 전략: 버그 수정 및 기능 추가 🚀

**안녕하세요, 친구들! 👋 이번 강의에서는 우리가 사랑하는 C와 Rust 언어로 만든 코드를 깨끗하고 효율적으로 관리하기 위한 비법을 공개할 거예요! 😎 지금까지 배운 것들을 활용하여 버그 수정과 기능 추가 퍼펙트하게 달성하며, 프로 개발자의 길에 한 걸음 더 다가갈 수 있겠죠?  💡**


### **1. 코드 유지보수란 무엇일까요? 🤔**

> 코드 유지 보수는 건물을 리모델링하거나 옷을 주기적으로 세탁하는 것과 같아요! 처음에는 잘 만들었지만 시간이 지나면서 작은 문제들이 생길 수 있고, 새로운 기능 추가를 원할 때가 오죠.  🛠️

C와 Rust 언어는 강력하고 효율적인 언어인 만큼 장기적으로 유지 보수해야 하는 코드의 양도 많습니다. 🤔 그렇기 때문에 버그 수정 🐶 및 기능 추가 🚀 전략을 잘 익혀두면 개발 생활이 훨씬 수월해지겠죠?

### **2.  버그 수정: 디버깅 도구 활용하기 🕵️‍♀️**


C와 Rust는 단순한 문자를 통해 코드가 동작하는 원리, 변수의 값 변경 과정 등을 직접 확인할 수 있는 강력한 디버깅 도구를 제공합니다!  

* **C**: GDB (GNU Debugger)
> `gdb program_name` 실행 후 breakpoints 설정과 step-by-step execution으로 버그 원인 파악 가능! 🤩 


```c
#include <stdio.h>

int main() {
    int a = 10;  // 변수 a 초기화
    int b = a + 5; // a에 저장된 값을 b에 더함
    printf("%d\n", b); // b의 값 출력

    return 0;
}
```

* **Rust**: Cargo Debugger (cargo run --debug)


> `println!()` 함수를 통해 변수 값을 실시간으로 확인하는 기능과 breakpoints 설정이 가능합니다. 🚀


```rust
fn main() {
    let a = 10; // 변수 a 초기화
    let b = a + 5; // a에 저장된 값을 b에 더함
    println!("b = {}", b); // b의 값 출력

}
```

**💡 초보자 폭풍 질문!** 🤔  "디버깅 도구 사용법이 너무 어렵아요?" 걱정 마세요! 온라인 강좌나 자료들이 많으니, 틈틈이 연습하면 쉽게 배우실 수 있습니다. 👍

### **3. 기능 추가: 새로운 기능을 통합하기 🧩**



C와 Rust는 확장성이 뛰어나기 때문에 새롭고 유용한 기능을 코드에 seamless하게 통합할 수 있어요! 🤩


* **모듈화:**  코드를 작은 모듈로 분리하여 관리 가능해지며, 새로운 기능을 추가하는 부분을 효율적으로 수정하며 다른 부분에 영향을 최소화 할 수 있습니다. 👍
```c
// header file (my_module.h)
int calculate_sum(int a, int b);

// source file (my_module.c)
#include "my_module.h"
int calculate_sum(int a, int b) {
    return a + b; 
}


```

* **구조체 활용:** 데이터를 효율적으로 정리하고 관리하여 새로운 기능을 구현하기 용이해집니다! 💪

```rust
struct User {
    name: String,
    age: u32,
}

fn main() {
    let user = User { name: "John".to_string(), age: 30 }; //  새로운 사용자 정보 추가
    println!("Name: {}, Age: {}", user.name, user.age);
}
```




**🚨 실무주의보! 🚨** 기능 추가 시 코드의 전체적인 구조 및 설계에 영향을 미칠 수 있으므로 신중하게 검토하고 계획적으로 작업해야 합니다.

### **4.  코드 스타일 가이드: 눈으로 읽기 쉬운 코드 작성하기 👀**



깔끔한 코드는 개발자와 디버거 모두에게 기쁨을 선사합니다! 🥳


* **주석:** 코드의 목적과 동작 방식을 명확하게 설명하여 이해를 돕습니다. 👍

```c
/* 이 함수는 두 정수를 더해서 결과를 반환합니다 */
int add_numbers(int a, int b) {
    return a + b;
}
```

* **공간 활용:**  문장을 분리하고 코드 블록을 구분하여 눈으로 읽기 쉽도록 합니다. 😎
```rust
fn calculate_result() {
    let x = 5;
    let y = 10;

    let sum = x + y;
    println!("Sum: {}", sum);
}
```


### **5. 테스트 코드 작성:  bugs 없는 코드의 왕좌에 오르기👑**



테스트 코드는 버그를 발견하고 수정하는 데 필수적인 도구입니다! 💯

* C 언어에서는 `unittest` 프레임워크를 활용합니다. 🧪
* Rust 언어에서는 `cargo test` 명령어로 테스트 코드를 실행하여 프로그램의 정확성을 검증할 수 있습니다. 🚀
