## 🔥8강: 반복문 소개 - for 문과 while 문 사용하기🚀💡

###  🌳 for 문: 원하는 횟수만큼 돌아가는 친구!

`for 문`은 순서대로 특정한 횟수만큼 반복적인 작업을 수행하는 데에 사용됩니다. 마치 "1부터 10까지" 라고 말했듯이, for문은  **시작점**, **종료점**, 그리고 **단위 증가량** 를 정의하여 원하는 범위 안에서 코드를 반복 실행하게 합니다.

   💡 예제: 
```rust
for i in 1..=5 {
    println!("{}번째 루프입니다!", i);
}
```

*  `for i in 1..=5`: 이 부분이 for 문의 심장부예요! 'i' 라는 변수를 정의하고, 1부터 5까지 (포함) 반복합니다. 마치 수학에서 "i = 1"부터 시작해서 "i <= 5"가 되기 전까지 반복하는 것처럼 이해하면 좋아요!
* `println!("{}번째 루프입니다!", i);`:  각 반복 시마다 'i' 의 값을 출력하고, 현재 번호를 알려주는 코드예요.

   **실행 결과:** ⬇️
   ```
   1번째 루프입니다!
   2번째 루프입니다!
   3번째 루프입니다!
   4번째 루프입니다!
   5번째 루프입니다!
   ```

* **🚨 실무주의보:** `..=5` 를 사용하면 5까지 포함하는 반복이 되고, `1..5` 라고 하면 5는 제외되는 점 주의하세요! 😎


###  🔄 while 문: 조건 만족할 때까지 돌아가는 친구!

`while 문`은 특정 조건이 **참** 이 될 때까지 반복문을 실행하는 역할을 합니다. 마치 "지금 숙제를 완료했니?" 라고 물어보고, 아직 안 끝났다면 계속해서 할 수 있도록 유도하는 것과 같죠!

   💡 예제:
```rust
let mut count = 0;
while count < 5 {
    println!("{}번째 루프입니다!", count);
    count += 1; // count 증가시키는 코드!
}
```

*  `let mut count = 0;`: 변수 'count'를 만들고, 처음에는 0으로 초기화합니다. 'mut' 키워드는 이 변수를 변경할 수 있도록 허용하는 뜻이에요.
* `while count < 5 { ... }`:  'count' 가 5보다 작을 때까지 코드가 실행되는 구문입니다. 조건이 참인 동안 반복!
* `println!("{}번째 루프입니다!", count);`: 현재 'count' 의 값을 출력합니다.
* `count += 1;`: 'count' 를 1씩 증가시켜, 조건문의 결과에 영향을 미치도록 합니다.

   **실행 결과:** ⬇️

   ```
   0번째 루프입니다!
   1번째 루프입니다!
   2번째 루프입니다!
   3번째 루프입니다!
   4번째 루프입니다!
   ```

* **🚨 실무주의보:** 'count' 가 5가 되면 조건이 거짓이 되고, 반복문이 종료됩니다. 주의하세요!
