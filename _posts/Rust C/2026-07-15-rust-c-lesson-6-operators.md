## 🔥 Rust C 연산자, 아직 못 알아봤다면 지금 바로 배우자! (6강) 🚀

안녕하세요! 대한민국 최고의 Rust C 강사, "코딩 마법사" 김용희입니다! 😎 오늘은 흔히 볼 수 있는 '연산자' 에 대해서 깊이 있게 알아볼 거예요. 연산자는 코딩에서 **식품 재료처럼 다양한 값들을 섞고 조립하여 새로운 값을 만들어내는 매력적인 도구**야!  👨‍🍳

직접 예시를 보면 더 이해하기 좋아요! 🤔 다음 코드가 있다고 해봅시다.

```rust
let a = 10;
let b = 5;
let result = a + b;
println!("{} + {} = {}", a, b, result);
```

이 코드는 두 개의 변수 `a`와 `b`에 각각 10과 5를 저장하고, 이들을 더하여 `result` 변수에 저장하는 간단한 프로그램입니다. **`+`** 기호가 바로 연산자죠!  🔍


### 🎉 산술 연산자: 수를 다루는 석학들의 비밀무서!

먼저 가장 자주 쓰이는 산술 연산자부터 살펴보겠습니다! 💪

* **+ (덧셈):** 두 값을 더합니다. 예시처럼 `a + b` 를 통해 두 변수의 합계를 얻을 수 있습니다.
* **- (뺄셈):** 첫 번째 값에서 두 번째 값을 뺀 결과를 반환합니다.  예: `10 - 5 = 5`.
* **\* (곱셈):** 두 값을 곱하여 결과를 반환합니다. 예: `2 * 3 = 6`.

```rust
let num1 = 10;
let num2 = 5;
println!("{} + {} = {}", num1, num2, num1 + num2); // 출력: 10 + 5 = 15
println!("{} - {} = {}", num1, num2, num1 - num2); // 출력: 10 - 5 = 5
println!("{} * {} = {}", num1, num2, num1 * num2); // 출력: 10 * 5 = 50
```

* **/** (나눗셈):** 첫 번째 값을 두 번째 값으로 나눈 결과를 반환합니다. 예: `10 / 2 = 5`.  
* **% (나머지):** 두 값을 나누었을 때 남는 나머지를 반환합니다. 예: `10 % 3 = 1`


### 🤔 비교 연산자: 당신의 생각과 코드의 마음 맞추기!

비교 연산자는 두 값을 비교하여 **참 (true) 또는 거짓 (false)**을 나타내는 데 사용합니다. 🤔 이런 연산자를 활용하면 우리가 코딩하는 '마음'이 코드에 정확하게 전달되는지 확인할 수 있습니다!


* **== (같다):** 두 값이 같으면 `true`, 다르면 `false`를 반환합니다. 예: `10 == 10` 는 `true`.
* **!= (다름):** 두 값이 같지 않으면 `true`, 같으면 `false`를 반환합니다. 예: `5 != 10` 는 `true`.


```rust
let num1 = 10;
let num2 = 5;

println!("{} == {} ?", num1, num2); // 출력: 10 == 5 ? false
println!("{} != {} ?", num1, num2); // 출력: 10 != 5 ? true


```

* **> (보다 크다):** 첫 번째 값이 두 번째 값보다 크면 `true`, 그렇지 않으면 `false`를 반환합니다. 예: `10 > 5` 는 `true`.
* **< (보다 작다):** 첫 번째 값이 두 번째 값보다 작으면 `true`, 그렇지 않으면 `false`를 반환합니다. 예: `5 < 10` 는 `true`.

```rust
let num1 = 10;
let num2 = 5;
println!("{} > {} ?", num1, num2); // 출력: 10 > 5 ? true


```


* **>= (보다 크거나 같다):** 첫 번째 값이 두 번째 값보다 크거나 같으면 `true`, 그렇지 않으면 `false`를 반환합니다. 예: `10 >= 10` 는 `true`.
* **<= (보다 작거나 같다):** 첫 번째 값이 두 번째 값보다 작거나 같으면 `true`, 그렇지 않으면 `false`를 반환합니다. 예: `5 <= 10` 는 `true`.


### 🤯 논리 연산자: 코드의 '만약' 문을 만들어 주는 마법!

논리 연산자는 여러 조건을 연결하여 복잡한 로직을 구현하는 데 사용됩니다.  💡 이들을 활용하면 코드가 더욱 풍부하고 정교해질 수 있습니다!


* **&& (AND):** 두 값이 모두 `true`이면 `true`, 그렇지 않으면 `false`를 반환합니다. 예: `10 > 5 && 5 < 10` 는 `true`.

```rust
let num1 = 10;
let num2 = 5;

println!("{} > {} && {} < {}", num1, num2, num2, 10); // 출력: 10 > 5 && 5 < 10 ? true



```


* **|| (OR):** 두 값 중 하나라도 `true`이면 `true`, 그렇지 않으면 `false`를 반환합니다. 예: `10 > 5 || 5 > 10` 는 `true`.

```rust
let num1 = 10;
let num2 = 5;
println!("{} > {} || {} > {}", num1, num2, num2, 10); // 출력: 10 > 5 || 5 > 10 ? true



```


* **! (NOT):** 값이 `true`이면 `false`, `false`이면 `true`를 반환합니다. 예: `!(10 > 5)` 는 `false`.

```rust
let num1 = 10;
let num2 = 5;

println!("!({} > {})", num1, num2); // 출력: !(10 > 5) ? false




```


### ✨ 연산자 우선순위: 코드의 명령어 순서를 결정하는 법!

연산자는 우선 순위에 따라 계산됩니다.  
예를 들어 `a + b * c` 라면, 곱셈이 먼저 이루어지고 그 결과에 더하기가 적용됩니다!


**다음은 일반적인 연산자 우선순위입니다.**

1. **( )**: 부호로 계산 순서를 지정
2. ***, /, %**: 나눗셈, 곱셈, 나머지 연산 (왼쪽에서 오른쪽)
3. **+, -**: 덧셈, 뺄셈 연산 (왼쪽에서 오른쪽)


### 💯 실무 활용: 코드를 더욱 효율적으로 만들어봐요!

연산자는 간단한 계산을 넘어서 복잡한 로직을 구현하는 데 필수적입니다.  이제까지 배운 연산자들을 사용하여 다양한 프로그램을 만들 수 있습니다.


**예시:**

```rust
let price = 100;
let discountRate = 0.1;
let discountedPrice = price * (1 - discountRate);
println!("{}원 상품에서 {}% 할인 후 가격은 {}원 입니다.", price, discountRate*100, discountedPrice);

```
