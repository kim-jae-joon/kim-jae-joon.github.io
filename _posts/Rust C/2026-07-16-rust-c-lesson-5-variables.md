##  🔥5강: 변수 선언과 할당 - Rust의 기본, 정복하기!🚀

안녕하세요, 최고의 Rust C 일타 강사, 저에게 오신 것을 환영합니다! 😎 지금부터 우리는 Rust의 기초를 다질게요. 곧, 당신도 "Rust는 제가 만든 것처럼 가볍고 강력해!"라고 외칠 수 있을 거예요!

### **1. 변수: 프로그램의 도구장에 들어갈 마법의 상자 ✨**
이제부터 뭐든지 만들어내기 위한 최고의 도구, 바로 "변수"를 배우겠습니다. 변수는 프로그램에서 정보를 저장하고 사용하는 공간이죠.  마치 우리가 책상 위에 물건들을 놓아서 관리하는 것과 같다는 생각하면 이해하기 쉬우답니다.

예를 들어, 게임의 캐릭터 이름을 "철수"라고 저장하려면 변수를 만들고 그 안에 "철수"라는 정보를 담으면 됩니다! 이렇게 변수는 프로그램에서 값을 조작하고 사용하는 데 필수적인 역할을 해요.

### **2. 변수 선언: 우리의 마법 상자 이름 정하기 🪄**
 Rust에서 변수를 만드려면 "선언"이라는 과정을 거쳐야 합니다.  변수에는 이름을 부여하고, 어떤 종류의 정보(숫자, 문자열 등)를 저장할지 정하는 단계죠!

```rust
let 철수_나이 = 25; // 변수 선언: "철수_나이"라는 이름으로 숫자 데이터 저장! 💡
```

* `let`: Rust에서 변수를 선언할 때 사용하는 키워드입니다. '마법 상자 만들기'라는 의미 같죠? 🧙‍♂️
* `"철수_나이"`: 우리가 만든 변수의 이름입니다.  
* `= 25;`: 변수에 저장할 값을 지정합니다. 이 경우, 숫자 25를 저장합니다.

**🚨 실무주의보:** Rust에서는 변수 이름은 알파벳, 숫자와 언더바(_)로 시작해야 합니다.  중요한 변수 이름을 잘 정하면 코드 읽기가 한결 편해진답니다! 😉


### **3. 할당: 마법 상자에 정보 채우기 ✨**
변수를 선언하면 마치 비어있는 상자가 만들어졌죠? 이제는  이 상자 안에 정보(값)를 넣어야겠죠? 이것을 "할당"이라고 합니다.

```rust
let 이름 = "철수"; // 변수 "이름"에 문자열 "철수"를 저장!
```

* `=`: 할당 연산자가 있습니다.  '마법 상자에 정보 넣기'라는 역할을 해요!


### **4. 상수: 바뀌지 않는 마법의 상자 😎**
모든 변수가 변경될 수 있는 것은 아닙니다. Rust는 "상수" 라는 특별한 종류의 변수를 지원합니다.  상수는 값이 변경되지 않도록 보장하는 특별한 마법 상자입니다!

```rust
const PI = 3.141592653589793; // 상수 선언: "PI"라는 이름으로 변하지 않는 숫자값 저장!  
```


* `const`: Rust에서 상수를 선언할 때 사용하는 키워드입니다. '변경 불가능'이라는 의미를 가집니다! 🔒

### **💡 초보자 폭풍 질문!**
"상수는 왜 필요해요?" 🤔  예를 들어, Pi처럼 항상 같은 값을 가지고 있어야 하는 상황에서 유용하게 사용됩니다. 또한, 오류를 방지하는 데에도 효과적이죠!

### **5. 가리키는 변수: 마법의 주소지 😎**
Rust에서 "가리키는 변수"라는 개념은 다른 변수의 메모리를 직접 참조할 수 있도록 하는 매우 강력한 도구입니다!  마치 집 주소지를 알고 있어서 그 집에 들어갈 수 있는 것과 같죠! 

```rust
let 철수_나이 = 25; // "철수_나이" 변수 선언 및 할당

let 나이_주소 = &철수_나이; // 가리키는 변수 선언: "나이_주소"가 "철수_나이"의 주소를 가리킵니다!
```

* `&`:  "참조 연산자"입니다. 다른 변수의 주소를 저장하는 역할을 합니다.


### **6. 실무 적용: Rust의 마법으로 현실 만들기 🚀**

Rust의 변수 선언과 할당은 간단하지만, 이 기본 개념이 컴퓨터 프로그램을 만드는 가장 기본적인 블록돌입니다!  다음 장르에 가이드를 추가하면 더욱 효율적이고 명확한 코딩을 가능하게 하죠! 

* **게임 개발:** 게임 캐릭터의 위치, 점수, 체력 등을 변수로 관리합니다.
* **웹 애플리케이션:** 사용자 입력 정보를 저장하고 처리합니다.
* **데이터 분석:** 데이터를 분석하고 결과를 표시하기 위해 변수를 사용합니다.
