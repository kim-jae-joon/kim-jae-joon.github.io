## 🔥 12강: 튜플 배우기 - 여러 자료형을 하나로 묶는 마법! 🚀

안녕하세요, Rust C 탐험대 여러분! 😎 오늘은  **"튜플 (Tuple)"** 이라는 신비로운 개념을 접할 거예요. 이름만 들어봤다면 "뭔가 복잡한 거 같아.." 라고 생각하실 수 있지만, 진짜 간단하고 유용해요! 🚀

우선 튜플은 여러 자료형을 하나로 묶어서 관리하는 방법이에요. 마치 다양한 종류의 음료를 담는 깔끔한 컵처럼 💡 여러 가지 정보를 한 번에 효율적으로 저장할 수 있습니다!

###  🤔 왜 튜플이 필요해?

만약 내가 파스타 레시피를 적을 때, "면", "소스", "치즈" 세 가지 재료가 필요하다고 말하는 상황이라고 가정해 보세요. 이런 정보들을 각각 분리해서 쓰는 건 어떨까요? 🍝

* `면`: 
  '봉게티'
* `소스`: 
  '토마토 소스'
* `치즈`: 
  '모짜렐라 치즈'

이런식으로 하나씩 적으면 보기에 지저분하고, 코드를 읽을 때도 복잡해지죠? 🤔 그래서 이때 "튜플"이 등장하는 거예요! 🦸‍♂️


###   ✨ 튜플의 매력 ✨

* **간결성:** 여러 자료들을 하나로 담아 간결하게 표현할 수 있습니다.
* **효율성:** 코드가 더욱 명확하고 읽기 쉬워 효율성을 높여줍니다.
* **유연성:** 원하는 데이터 유형을 튜플에 들어갈 수 있으므로 다양한 상황에서 활용 가능합니다.

###  👩‍💻 튜플 만들어 보세요!

Rust C 에서는 생략되어도 되지만, 자료형을 명시하면 더 명확해요! 🔥

```rust
let my_pasta: (String, String, String) = ("봉게티", "토마토 소스", "모짜렐라 치즈");
```

* `(String, String, String)` :  문자열 자료형을 세 개 사용하는 튜플입니다. 각 문자열은 파스타의 '면', '소스', '치즈' 정보를 나타냅니다.
* `let my_pasta`: 변수 이름은 'my_pasta' 입니다.
* `= ("봉게티", "토마토 소스", "모짜렐라 치즈")`: 튜플에  문자열을 담아서 변수에 할당합니다.

###    🔑 튜플 접근하기 🔑


```rust
println!("나는 {} 먹고 싶어요!", my_pasta.0); // 결과: 나는 봉게티 먹고 싶어요!
println!("소스는 {} 입니다.", my_pasta.1);   // 결과: 소스는 토마토 소스 입니다.
println!("치즈는 {} 이죠?", my_pasta.2);  // 결과: 치즈는 모짜렐라 이죠?
```

* `my_pasta.0`: 튜플의 첫 번째 요소 (index 0)를 접근합니다.
* `my_pasta.1`: 튜플의 두 번째 요소 (index 1)를 접근합니다.
* `my_pasta.2`: 튜플의 세 번째 요소 (index 2)를 접근합니다.

###    🚨 실무주의보 🚨


 Rust C 에서는  튜플을 사용하여 함수가 여러 값을 반환하는 경우에도 효율적으로 코드를 작성할 수 있습니다! 예를 들어, 두 개의 정수를 계산하고 결과를 함께 반환하는 함수를 만들어 볼 수 있습니다.

```rust
fn calculate_sum(a: i32, b: i32) -> (i32, i32) {
    (a + b, a * b) 
}
let (sum, product): (i32, i32) = calculate_sum(5, 3); // 결과: (8, 15)

println!("합계: {}, 곱셈 결과: {}", sum, product);
```


* `fn calculate_sum(a: i32, b: i32) -> (i32, i32)` : 두 개의 정수 입력을 받아서 합과 곱을 반환하는 함수입니다.
* `-> (i32, i32)` : 함수가 (정수, 정수) 형태로 값을 반환한다는 의미입니다.
* `let (sum, product): (i32, i32) = calculate_sum(5, 3);`:  `calculate_sum` 함수를 호출하여 합과 곱을 받아서 변수에 할당합니다.
