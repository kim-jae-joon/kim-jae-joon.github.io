### 🤔 배열이란 무엇일까요?

배열은 같은 종류의 데이터를 **순서대로 저장하고 관리**하는 멋진 도구죠. 옛날에는 책장에 책들을 순서대로 꽂는 것처럼, Rust에서도 배열을 사용해서 데이터를 정돈합니다. 📚

예를 들어, 학생들의 이름 목록을 저장해야 할 때, 각 이름을 하나의 "칸"에 담아서 서로 연결된 "배열" 형태로 관리하면 편리하고 효율적이죠! ✨

###  💥 동적 배열 vs 고정 크기 배열: Rust에서 두 얼굴의 배열 🤯

Rust는 **동적 배열**과 **고정 크기 배열** 두 가지 유형을 제공합니다. 각각은 장단점을 가지고 있어서 적절한 상황에 맞게 사용해야 합니다. 🚀


> > 동적 배열: 데이터를 추가하거나 제거할 수 있는 "탄력적인" 배열! 😎
> > 고정 크기 배열: 미리 정해진 크기만큼만 저장 가능하지만, 속도가 빠르고 메모리를 효율적으로 사용하는 "강인한" 배열! 💪

### 1. 동적 배열 :  'Vec' - Rust의 변화를 사랑하는 배열 친구 🤩

`Vec<T>`는 Rust에서 가장 많이 사용되는 동적 배열입니다. 'Vector'라는 단어에서 따온 이름으로, 마치 화살처럼 데이터들을 방향성 있게 연합시키는 느낌을 주죠! 🏹  

**✨ Vec의 장점:**
* **사이즈 조절 가능**: 필요에 따라 크기를 변경할 수 있어 유연하게 사용 가능합니다. 👍
* **쉽고 간편한 API 제공**: 데이터 추가, 삭제, 접근 등 다양한 작업을 간단하게 수행할 수 있습니다. 🤩

**🔥 예제:** `Vec`를 활용해서 학생들의 이름 목록을 저장하는 코드를 살펴보세요!

```rust
let mut students = Vec::new(); // 새로운 비어있는 Vec를 생성합니다.

students.push("철수");      // "철수"를 배열에 추가합니다.
students.push("영희");      // "영희"를 배열에 추가합니다.
students.push("민수");      // "민수"를 배열에 추가합니다.

println!("학생 목록: {:?}", students); // 출력 결과: 학생 목록: ["철수", "영희", "민수"]
```


* **`Vec::new()`**: 새로운 비어있는 `Vec`를 생성하는 함수입니다. 
* **`push(name)`**: 지정된 이름 (`name`)을 배열의 마지막에 추가하는 함수입니다.

### 2. 고정 크기 배열: 'Fixed Size Array' - 예측 가능한 성능을 원할 때 최고의 선택! 💪

Rust에서는 특정 크기를 정해진 채로 데이터를 저장하는 **고정 크기 배열**도 제공합니다. 이런 배열은 동적 배열처럼 크기를 바꿀 수 없지만, 메모리 사용량이 예측 가능하고 연산 속도가 빠르다는 장점이 있습니다! ⚡️

 **✨ 고정 크기 배열의 장점:**
* **메모리 효율성**: 미리 정해진 크기에 따라 메모리를 할당하기 때문에, 동적 배열처럼 추가/삭제 작업으로 인한 메모리 분할 및 재할당 없이 작동합니다. 🚀
* **CPU 속도 향상**: 고정 크기가 정의되어 있으므로 컴퓨터가 데이터를 접근하는 데 필요한 시간이 단축됩니다.  ⚡️

**🔥 예제:** 고정 크기 배열을 사용해서 주사위 눈금을 저장하는 코드를 살펴보세요!


```rust
let dice_rolls = [1, 2, 3, 4, 5, 6]; // 미리 정해진 크기(6)의 배열

println!("주사위 결과: {:?}", dice_rolls); // 출력 결과: 주사위 결과: [1, 2, 3, 4, 5, 6]
```



* **`[1, 2, 3, 4, 5, 6]`**: 배열의 크기(6)와 각 요소의 값을 명시적으로 지정합니다.
