##  🔥 7강: 조건문 이해하기: if 문과 else 문 사용하기 🔥

**안녕하세요, Rust C 일타 최고의 선생님(나 자신!)이에요! 😎 ** 지금부터 차근차근 Rust C 프로그래밍의 기본, 조건문까지 배워볼게요. 컴퓨터를 마치 인간처럼 생각하는 방법을 배우는 건 정말 신기한 일이죠? 진짜 🤯 

### 🎯 어제 오늘 할 일: 조건문 💪

지난 강에서는 변수와 연산자, 자료형에 대해 배웠죠! 이제부터는 프로그램이 **특정 상황에 따라 다른 코드를 실행**하는 방법을 알아보려고 해요. 마치 사람이 "오늘 날씨가 좋아"면 "외출하자!" 라고 말하지만, "비 오네?" 라고 하면 "집에서 게임 하자!"라고 결정하는 것과 같은 생각이죠! 💡

### ✨ 조건문의 영웅: if 문 🚀

**if 문은 특정 조건이 참(True)이면 코드 블록을 실행하고, 거짓(False)이면 건너뛰는 역할을 합니다.**  
마치 "만약"이라는 단어로 시작하는 조건을 놓고 판단하는 것과 같아요.

```rust
if 날씨가 좋다 { // 만약 날씨가 좋은 경우라면...
    println!("오늘은 외출하기 좋은 날!"); // 이 코드를 실행한다! 😎
}
```


*  `if` 키워드는 조건문의 시작을 알립니다. 
* `날씨가 좋다` 는 **조건식**이라고 불리며, 참(True) 또는 거짓(False)으로 평가될 수 있습니다.
* `{}` 는 조건식이 **참일 때 실행되는 코드 블록**입니다. 여기서는 `println!()` 함수를 사용하여 "오늘은 외출하기 좋은 날!" 라고 출력하는 코드가 있습니다.


### 🤔 잠깐 질문: 만약 조건식이 거짓이면? 🤔

그럼 **else 문**을 활용하면 좋습니다! 🎉 

```rust
if 날씨가 좋다 { // 만약 날씨가 좋은 경우라면...
    println!("오늘은 외출하기 좋은 날!"); // 이 코드를 실행한다! 😎
} else { // 아니면, 즉 날씨가 좋지 않을 때...
    println!("집에서 영화 보자!"); // 이 코드를 실행한다! 😌
}
```

* `else` 키워드는 조건식이 **거짓일 때** 실행되는 코드 블록을 나타냅니다.


### 🌟 핵심: if 문과 else 문의 시너지 🚀

if 문과 else 문은 함께 사용하면 더욱 강력한 조건문을 만들 수 있습니다! 예를 들어, 학생의 성적이 높으면 "훌륭한 성적!", 그렇지 않으면 "좀더 노력해 보자!" 라고 말할 수 있죠.


```rust
let 성적 = 90; // 학생의 성적

if 성적 >= 80 {  // 만약 성적이 80점 이상인 경우라면...
    println!("훌륭한 성적!"); // 이 코드를 실행한다! 🎉
} else { // 아니면, 즉 성적이 80점 미만일 때...
    println!("좀 더 노력해 보자!"); // 이 코드를 실행한다! 💪
}
```
