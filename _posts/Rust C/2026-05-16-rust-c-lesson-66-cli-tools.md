##  1단계: 명령줄 인터페이스 개요 🗺️

Rust에서 CLI 도구를 개발하기 전에 먼저 **명령줄 인터페이스(CLI)** 자체에 대해 간략하게 이해해야 합니다. CLI는 사용자가 단어, 문자열을 입력하여 프로그램을 제어하고 결과를 얻을 수 있는 방식입니다.  텍스트 기반으로 모든 작업이 이루어지기 때문에 효율적이고 직관적인 인터페이스로 자주 활용됩니다. 💻

예를 들어, 우리가 가장 익숙한 `ls` 명령은 파일 목록을 표시하는 CLI 도구의 대표적인 예입니다. 사용자는 간단한 명령어 `ls`만 입력하면 현재 디렉터리에 있는 모든 파일 및 디렉토리를 확인할 수 있습니다.

*  **명령어:** 핵심 행동을 나타내는 단어 또는 문자열입니다 (예: `ls`, `mkdir`, `cd`).
* **인수:** 명령어를 실행하는데 필요한 추가 정보입니다 (예: `ls /home/user`에서 `/home/user` 가 인수).
* **플래그:** 명령어의 동작을 조정하거나 특정 기능을 활성화하는 기호 또는 단어입니다 (예: `ls -l`).

##  2단계: Rust로 CLI 도구 만들기 💪

Rust에서는  **`clap` 라이브러리**를 통해 CLI 도구 개발을 간편하게 해줍니다. Clap은 명령줄 인자 파싱, 플래그 설정 등의 복잡한 작업을 손쉽게 처리해 주는 강력한 도구입니다. 💥

###  2.1 clap 라이브러리 사용하기

먼저 `clap` 라이브러리를 프로젝트에 추가해야 합니다. Cargo를 사용하는 Rust 개발자라면 다음 명령어로 간단하게 설치할 수 있습니다.

```bash
cargo add clap
```

이제 `clap` 라이브러리가 프로젝트에 적용되어 우리가 CLI 도구 개발에 필요한 기능을 사용할 준비가 되었습니다! 🎉


###  2.2 명령어 구조 정의하기

**`derive(command)`**: 이 어트리뷰트를 사용하여 Clap이 Rust 구조체를 CLI 명령으로 자동으로 변환해 주는 기능을 활성화합니다. 🦸‍♂️
*  명령어 이름을 설정합니다 (예: `my_cli`).

```rust
#[derive(command)]
struct MyCli {}
```

**`MyCli`**: 우리가 만든 CLI 도구의 구조체입니다. Clap은 이 구조체를 명령으로 변환하며, 각 필드는 인수나 플래그로 사용될 수 있습니다.


###  2.3 인자와 플래그 정의하기

명령줄 인자와 플래그를 정의하여 사용자가 입력할 수 있는 정보를 받을 수 있습니다.

```rust
#[derive(command)]
struct MyCli {
    #[arg(short, long)] // short: '-f', long: '--file'
    filename: String, 

    #[arg]  // 추가 인자 설정
    message: String,
}
```

* **`#[arg]`**: 명령줄에서 입력받을 인자를 정의합니다. `short`, `long` 어트리뷰트는 인자가 사용할 수 있는 단축 표시와 길어진 표시를 지정합니다.
* **`filename: String`**:  "file"라는 이름의 인자로 파일명을 받아들이며, 
   String 자료형으로 저장됩니다.

**💡 초보자 폭풍 질문!**: "왜 `#[arg]` 어트리뷰트를 사용해야 하는 걸까요?" 🤔

Clap은 명령줄 입력을 자동으로 파싱하기 위해 구조체를 기반으로 정의된 기능들을 활용합니다. `#[arg]` 어트리뷰트는 인자에 대한 정보를 Clap에게 전달하여, 해당 정보를 명확하게 처리하고 사용할 수 있도록 합니다. 


###  2.4 CLI 도구 실행하기

```rust
fn main() {
    MyCli::parse(); // clap 라이브러리를 통해 명령줄 입력 파싱 및 처리
}
```

`MyCli::parse()` 함수는 Clap을 활용하여 사용자가 입력한 명령줄 인자와 플래그를 파싱하고,  `MyCli` 구조체의 필드에 저장합니다. 💾 이제 `filename`과 `message` 데이터에 접근하여 CLI 도구의 로직을 실행할 수 있습니다!


##  3단계: 실제 코드 예시 분석 👀

```rust
#[derive(command)]
struct MyCli {
    #[arg(short, long)]
    filename: String, 

    #[arg] 
    message: String,
}

fn main() {
    let cli = MyCli::parse(); // 명령줄 입력 파싱

    println!("파일명: {}", cli.filename);  // 파일명 출력
    println!("메시지: {}", cli.message);   // 메시지 출력 
}
```

* **도움 정보 제공**: `--help` 또는 `/?` 명령으로 도움 메시지를 표시하여 사용자에게 CLI를 활용하는 방법을 설명합니다. 📚
