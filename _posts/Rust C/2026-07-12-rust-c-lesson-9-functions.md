### 🔑 함수: 코딩에서 절대 빠질 수 없는 파우어 업 🚀

함수라고 하면, 마치 자동차와 같이 생각해 보세요. 자동차는 연료, 엔진, 제어 시스템 등 다양한 부품들이 조합되어 특정 목표를 달성하는 강력한 도구죠! 함수도 똑같이! 코딩의 작은 자동차라고 할 수 있겠네요.

* **목적**: 특정 작업을 수행하기 위한 명령 집합
* **이름**: 자신을 나타내는 이름 (예: `계산`, `출력`, `사용자 입력 받기`)
* **매개변수**: 함수가 진행하는 작업에 필요한 정보 (옵션)

### 🤯 매개변수: 함수의 "식재료" 📦

매개변수는 함수가 작업을 수행하기 위해 필요한 정보를 담고 있는 '식재료'라고 생각해 보세요. 예를 들어, 🎂 케이크 만들기 함수에 ‘밀가루’와 ‘설탕’ 같은 재료를 매개변수로 전달하면 달콤하고 부드러운 케이크를 만들 수 있겠죠?

```rust
fn calculate_area(width: i32, height: i32) -> i32 {
    width * height 
}
```

* 위 코드는 `calculate_area`라는 함수를 정의하고 있습니다.

> > 🚨 실무주의보: Rust에서는 변수 선언 시 자료형을 명시해야 합니다!  `i32`은 32비트 정수형입니다.


* `width` 와 `height` 는 매개변수로, 함수가 면적을 계산하기 위해 필요한 길이 정보를 담고 있습니다.
* `-> i32` 는 반환값의 자료형을 나타냅니다. 이 함수는 넓이를 정수 형태(i32)로 반환합니다!

###  📤 반환값: 함수의 "완성도" 🌟

반환값은 함수가 작업을 마쳤을 때 생성하는 결과물입니다. 마치 요리사가 케이크를 완전히 만들고 맛있게 나오면 손님에게 케이크 자체(반환값)를 선물하는 것과 같죠! 😎


```rust
fn add_numbers(a: i32, b: i32) -> i32 {
    a + b
}

let result = add_numbers(5, 3); 
println!("합계는 {} 입니다!", result); // 출력 결과: 합계는 8 입니다!
```


* 위 코드에서 `add_numbers` 함수는 두 개의 정수를 더한 값을 반환합니다.
*  `let result = add_numbers(5, 3);` 에서 `add_numbers` 함수에 5와 3를 매개변수로 전달하고, 결과값을 `result` 변수에 저장합니다.
