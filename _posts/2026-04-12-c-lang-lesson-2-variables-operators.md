## 🚀 2강: C 언어의 뼈대: 변수, 자료형, 연산자

### 1. 뼈대 1: 변수 (Variable) - 데이터의 '상자'

프로그램을 만들다 보면 수많은 데이터를 다루게 됩니다. 예를 들어, 사용자의 나이, 게임 점수, 상품 가격 등등이요. 이런 값들을 컴퓨터 메모리에 저장하고, 필요할 때 꺼내 쓰고, 또 바꾸기도 해야겠죠? 이때 사용하는 것이 바로 **변수**입니다.

변수는 데이터를 담는 '상자'라고 생각하면 이해하기 쉬워요. 이 상자에는 이름이 붙어 있고, 그 이름으로 상자를 찾아 안에 있는 데이터를 넣거나 꺼낼 수 있습니다.

#### 📦 변수 선언하기

변수를 사용하려면 먼저 컴퓨터에게 "이런 이름의 상자를 만들게요!" 하고 알려줘야 합니다. 이것을 **변수 선언**이라고 합니다. 선언할 때는 상자에 어떤 종류의 데이터를 담을 건지 (자료형)와 상자의 이름 (변수 이름)을 정해줍니다.

**문법:** `자료형 변수이름;`

```c
#include <stdio.h> // printf 함수를 사용하기 위해 포함합니다.

int main() {
    int age; // 'int'형 데이터를 저장할 'age'라는 변수를 선언합니다. (정수형 상자를 만들게요!)
    double height; // 'double'형 데이터를 저장할 'height'라는 변수를 선언합니다. (소수점 있는 숫자형 상자를 만들게요!)
    char initial; // 'char'형 데이터를 저장할 'initial'이라는 변수를 선언합니다. (문자형 상자를 만들게요!)

    printf("변수 선언 완료!\n"); // 상자만 만들고 아직 뭘 넣지는 않았어요.
    return 0;
}
```

#### 📦 변수에 값 저장하기 (할당 / 초기화)

변수를 선언했으니 이제 상자에 데이터를 넣어볼까요? 변수에 값을 넣는 것을 **할당(Assignment)**이라고 합니다. 이때 사용하는 기호는 `=` (등호)입니다.

**주의!** C 언어에서 `=`는 '같다'가 아니라 '오른쪽의 값을 왼쪽 변수에 넣어라!'는 의미입니다.

```c
#include <stdio.h>

int main() {
    int score; // score라는 정수형 변수 선언
    score = 100; // score 변수에 100이라는 값을 할당 (100점 상자에 넣기!)

    printf("제 점수는 %d점입니다.\n", score); // %d는 정수형 변수의 값을 출력하라는 의미입니다.

    score = 95; // 값을 다시 할당할 수 있습니다. (상자 안의 100점을 95점으로 바꾸기!)
    printf("변경된 제 점수는 %d점입니다.\n", score);

    return 0;
}
```

**💡 팁:** 변수를 선언함과 동시에 값을 할당하는 것을 **초기화(Initialization)**라고 합니다. 대부분의 경우 이렇게 선언과 초기화를 함께 하는 것이 좋습니다!

```c
#include <stdio.h>

int main() {
    int year = 2024; // 선언과 동시에 2024로 초기화
    float pi = 3.14f; // 소수점 변수도 선언과 동시에 초기화! (float은 뒤에 'f'를 붙이는 습관을 들이세요!)
    char grade = 'A'; // 문자 변수도 선언과 동시에 초기화! (문자는 작은따옴표 ' '로 감싸요!)

    printf("현재 연도: %d년\n", year);
    printf("파이 값: %f\n", pi); // %f는 float형 변수의 값을 출력하라는 의미입니다.
    printf("성적: %c학점\n", grade); // %c는 char형 변수의 값을 출력하라는 의미입니다.

    return 0;
}
```

#### 📌 변수 이름 짓는 규칙 (Naming Rules)

변수 이름은 아무렇게나 지을 수 있는 건 아니에요. 몇 가지 규칙이 있습니다.

1.  **영문자, 숫자, 언더스코어(`_`)만 사용 가능합니다.**
2.  **숫자로 시작할 수 없습니다.** (예: `1age`는 안 돼요, `age1`은 가능!)
3.  **대소문자를 구분합니다.** (`age`와 `Age`는 다른 변수입니다!)
4.  **C 언어의 예약어(keyword)는 사용할 수 없습니다.** (예: `int`, `double`, `if`, `for` 등)
5.  **변수 이름은 의미 있게 짓는 것이 좋습니다.** (예: `a`보다는 `appleCount`가 나중에 코드를 이해하기 쉽겠죠?)

### 2. 뼈대 2: 자료형 (Data Type) - 상자의 '종류'

아까 변수를 "데이터를 담는 상자"라고 설명했죠? 그런데 상자도 종류가 다양하잖아요. 작은 보석 상자, 큰 냉장고 상자, 길쭉한 택배 상자 등등.

**자료형**은 변수가 어떤 종류의 데이터를 (정수, 소수, 문자 등) 얼마나 큰 공간에 담을 것인지 정의합니다. 컴퓨터는 자료형을 보고 메모리에 상자를 얼마나 크게 만들지, 그리고 그 상자에 담긴 데이터를 어떻게 해석해야 할지 알게 됩니다.

#### 💡 왜 자료형이 필요할까요?

1.  **메모리의 효율적인 사용:** 정수를 저장하는데 소수점까지 저장할 수 있는 아주 큰 공간을 할당하는 건 낭비겠죠? 필요한 만큼만 메모리를 사용하기 위함입니다.
2.  **정확한 연산:** 숫자와 문자를 다루는 방식은 완전히 다릅니다. 컴퓨터가 "이건 숫자니까 더하기를 해줘야지!" 혹은 "이건 문자니까 그대로 출력해야지!" 하고 정확히 판단할 수 있도록 도와줍니다.

#### 📊 주요 자료형 살펴보기

| 자료형     | 설명                             | 예시             | `printf` 형식 지정자 |
| ---------- | -------------------------------- | ---------------- | -------------------- |
| `int`      | **정수** (Integer)를 저장합니다. | 10, -500         | `%d`                 |
| `char`     | **문자** (Character) 하나를 저장합니다. | 'A', 'z', '5'    | `%c`                 |
| `float`    | **실수** (부동소수점)를 저장합니다. (단정밀도) | 3.14f, -0.5f     | `%f`                 |
| `double`   | **실수** (부동소수점)를 저장합니다. (배정밀도, `float`보다 정밀도가 높고 더 큰 범위의 수를 저장) | 3.141592, 123.456 | `%lf` (입력 시), `%f` (출력 시) |

**예제: 다양한 자료형 사용하기**

```c
#include <stdio.h>

int main() {
    // 1. int (정수)
    int studentCount = 30;
    int examScore = 95;
    printf("학생 수: %d명, 시험 점수: %d점\n", studentCount, examScore);

    // 2. char (문자)
    char myInitial = 'S'; // 문자는 반드시 작은따옴표 ' '로 감싸야 합니다.
    char newLineChar = '\n'; // 특수 문자도 char로 표현 가능 (줄바꿈)
    printf("제 이니셜은 %c입니다.%c", myInitial, newLineChar);

    // 3. float (단정밀도 실수)
    float temperature = 25.7f; // float형 리터럴(값)에는 'f'를 붙이는 것이 좋습니다.
    printf("현재 온도: %.1f도\n", temperature); // %.1f는 소수점 첫째 자리까지 출력하라는 의미입니다.

    // 4. double (배정밀도 실수)
    double gravity = 9.80665;
    printf("지구의 중력 가속도: %.5f m/s^2\n", gravity); // %.5f는 소수점 다섯째 자리까지 출력하라는 의미입니다.

    return 0;
}
```

#### 📏 `sizeof` 연산자로 자료형 크기 알아보기

`sizeof` 연산자를 사용하면 변수나 자료형이 메모리에서 얼마나 많은 공간을 차지하는지 (바이트 단위로) 알 수 있습니다.

```c
#include <stdio.h>

int main() {
    int a;
    char b;
    float c;
    double d;

    printf("int형 변수 a의 크기: %lu 바이트\n", sizeof(a));
    printf("char형 변수 b의 크기: %lu 바이트\n", sizeof(b));
    printf("float형 변수 c의 크기: %lu 바이트\n", sizeof(c));
    printf("double형 변수 d의 크기: %lu 바이트\n", sizeof(d));

    printf("\n자료형 자체의 크기도 알 수 있어요!\n");
    printf("int의 크기: %lu 바이트\n", sizeof(int));
    printf("char의 크기: %lu 바이트\n", sizeof(char));
    printf("long long의 크기: %lu 바이트\n", sizeof(long long)); // 더 큰 정수형
    printf("short의 크기: %lu 바이트\n", sizeof(short));     // 더 작은 정수형

    return 0;
}
```
**💡 참고:** `sizeof` 연산자의 결과는 운영체제나 컴파일러에 따라 약간 다를 수 있습니다. `%lu`는 `unsigned long` 타입을 출력하기 위한 형식 지정자인데, `sizeof`의 반환형이 보통 이렇습니다.

### 3. 뼈대 3: 연산자 (Operator) - 데이터 가지고 '놀기'

변수는 데이터를 담는 상자이고, 자료형은 상자의 종류를 정한다고 했죠? 그럼 연산자는 뭘까요? 바로 이 상자 안에 있는 데이터를 가지고 **다양한 작업을 수행하는 '도구'**입니다! 더하기, 빼기, 곱하기, 나누기, 비교하기 등등 우리가 할 수 있는 거의 모든 데이터 조작이 연산자를 통해 이루어집니다.

연산자는 정말 종류가 많지만, 오늘은 C 언어를 배우는 초반에 가장 많이 쓰이는 필수 연산자들만 집중적으로 알아볼게요!

#### ➕➖✖️➗ 산술 연산자 (Arithmetic Operators)

가장 기본적이고 친숙한 연산자들입니다. 우리가 일상생활에서 쓰는 사칙연산과 같습니다.

| 연산자 | 설명     | 예시         | 결과 |
| ------ | -------- | ------------ | ---- |
| `+`    | 덧셈     | `5 + 2`      | `7`  |
| `-`    | 뺄셈     | `5 - 2`      | `3`  |
| `*`    | 곱셈     | `5 * 2`      | `10` |
| `/`    | 나눗셈   | `5 / 2`      | `2` (정수 나눗셈!) |
| `%`    | 나머지   | `5 % 2`      | `1`  |

**💡 중요! 정수 나눗셈:** 정수형 변수끼리 `/` 연산을 하면 결과도 정수가 나옵니다. 소수점 이하는 버려져요! 정확한 소수점 결과를 원한다면 `float`이나 `double` 자료형을 사용해야 합니다.

```c
#include <stdio.h>

int main() {
    int num1 = 10;
    int num2 = 3;

    printf("덧셈: %d + %d = %d\n", num1, num2, num1 + num2);
    printf("뺄셈: %d - %d = %d\n", num1, num2, num1 - num2);
    printf("곱셈: %d * %d = %d\n", num1, num2, num1 * num2);

    // 정수 나눗셈 예시: 10 / 3 = 3 (소수점 버림)
    printf("나눗셈 (정수): %d / %d = %d\n", num1, num2, num1 / num2);

    // 나머지 연산 예시: 10 % 3 = 1
    printf("나머지: %d %% %d = %d\n", num1, num2, num1 % num2); // 참고: %를 출력하려면 %%를 사용합니다.

    // 실수 나눗셈을 하려면? (둘 중 하나라도 실수형이어야 해요!)
    float f_num1 = 10.0f;
    float f_num2 = 3.0f;
    printf("나눗셈 (실수): %.1f / %.1f = %.2f\n", f_num1, f_num2, f_num1 / f_num2);

    return 0;
}
```

#### `=` 대입 연산자 (Assignment Operator) & 복합 대입 연산자

우리가 변수에 값을 넣을 때 사용했던 `=`가 바로 대입 연산자입니다.

그리고 값을 변경할 때 더 편리하게 사용할 수 있는 **복합 대입 연산자**도 있습니다.

| 연산자 | 설명             | 예시          | 동등한 표현     |
| ------ | ---------------- | ------------- | --------------- |
| `=`    | 값 할당          | `a = 10;`     |                 |
| `+=`   | 더하고 할당      | `a += b;`     | `a = a + b;`    |
| `-=`   | 빼고 할당        | `a -= b;`     | `a = a - b;`    |
| `*=`   | 곱하고 할당      | `a *= b;`     | `a = a * b;`    |
| `/=`   | 나누고 할당      | `a /= b;`     | `a = a / b;`    |
| `%=`   | 나머지 구하고 할당 | `a %= b;`     | `a = a % b;`    |

```c
#include <stdio.h>

int main() {
    int total = 100;
    printf("초기 total: %d\n", total); // 100

    total += 50; // total = total + 50; 과 같습니다.
    printf("50을 더한 total: %d\n", total); // 150

    total -= 30; // total = total - 30; 과 같습니다.
    printf("30을 뺀 total: %d\n", total); // 120

    total *= 2; // total = total * 2; 와 같습니다.
    printf("2를 곱한 total: %d\n", total); // 240

    total /= 4; // total = total / 4; 와 같습니다.
    printf("4로 나눈 total: %d\n", total); // 60

    return 0;
}
```

#### `++` `--` 증감 연산자 (Increment/Decrement Operators)

변수의 값을 1씩 늘리거나 줄일 때 사용하는 아주 간편한 연산자입니다. C 언어에서 정말 많이 쓰이는 연산자 중 하나예요!

| 연산자 | 설명       | 예시      | 동등한 표현 |
| ------ | ---------- | --------- | ----------- |
| `++`   | 1 증가     | `a++;`    | `a = a + 1;` |
| `--`   | 1 감소     | `a--;`    | `a = a - 1;` |

**💡 중요! 전위(Prefix)와 후위(Postfix)의 차이:**
증감 연산자는 변수 앞에 붙느냐 (전위, `++a`) 뒤에 붙느냐 (후위, `a++`)에 따라 동작 방식이 미묘하게 다릅니다.

*   **전위 연산자 (`++a`, `--a`):** **먼저 변수의 값을 변경한 후**, 그 변경된 값을 연산에 사용합니다.
*   **후위 연산자 (`a++`, `a--`):** **먼저 변수의 현재 값을 연산에 사용한 후**, 연산이 끝난 뒤 변수의 값을 변경합니다.

```c
#include <stdio.h>

int main() {
    int count = 5;
    printf("초기 count: %d\n", count); // 5

    // 전위 증가 연산자: 먼저 증가하고, 그 값을 사용
    printf("전위 증가 (++count): %d\n", ++count); // count는 6이 되고, 6을 출력
    printf("++count 후 count: %d\n", count); // 6

    printf("\n");

    int value = 5;
    printf("초기 value: %d\n", value); // 5

    // 후위 증가 연산자: 먼저 값을 사용하고, 그 다음에 증가
    printf("후위 증가 (value++): %d\n", value++); // value는 5를 출력하고, 그 다음에 6이 됨
    printf("value++ 후 value: %d\n", value); // 6

    printf("\n");

    int num = 10;
    // 전위 감소
    printf("전위 감소 (--num): %d\n", --num); // 9
    // 후위 감소
    printf("후위 감소 (num--): %d\n", num--); // 9 (출력 후 8로 변경)
    printf("num-- 후 num: %d\n", num); // 8

    return 0;
}
```

#### ⚖️ 비교 연산자 (Relational Operators) & 🧠 논리 연산자 (Logical Operators)

이 두 연산자들은 주로 두 값의 크기를 비교하거나, 여러 조건을 조합할 때 사용하며, 결과로 **참(True) 또는 거짓(False)**을 반환합니다. (C 언어에서는 0이 거짓, 0이 아닌 모든 값은 참으로 간주됩니다.)

이 연산자들은 나중에 배울 `if`문이나 `while`문처럼 '흐름을 제어하는 문법'과 함께 사용할 때 진가를 발휘하니, 여기서는 "아, 이런 것도 있구나!" 정도로만 알아두고 넘어갈게요. 다음 강의에서 더 자세히 다룰 기회가 있을 겁니다!

| 연산자 | 설명         | 예시          | 결과 (True/False) |
| ------ | ------------ | ------------- | ----------------- |
| `==`   | 같다         | `a == b`      | `a`와 `b`가 같으면 참 |
| `!=`   | 같지 않다    | `a != b`      | `a`와 `b`가 다르면 참 |
| `>`    | 크다         | `a > b`       | `a`가 `b`보다 크면 참 |
| `<`    | 작다         | `a < b`       | `a`가 `b`보다 작으면 참 |
| `>=`   | 크거나 같다  | `a >= b`      | `a`가 `b`보다 크거나 같으면 참 |
| `<=`   | 작거나 같다  | `a <= b`      | `a`가 `b`보다 작거나 같으면 참 |
| `&&`   | 논리 AND     | `a && b`      | `a`와 `b` 모두 참일 때 참 |
| `||`   | 논리 OR      | `a || b`      | `a` 또는 `b` 둘 중 하나라도 참일 때 참 |
| `!`    | 논리 NOT     | `!a`          | `a`가 참이면 거짓, 거짓이면 참 |

### 🌟 총정리: 작은 프로젝트 - '간단한 물건 구매 계산기'

배운 내용을 한 번에 써먹는 미니 프로젝트를 해볼까요? 간단한 물건 구매 계산기를 만들어 봅시다!

```c
#include <stdio.h>

int main() {
    // 1. 변수 선언 및 초기화: 필요한 데이터를 담을 상자들을 준비합니다.
    char productName[20] = "스마트폰"; // 상품명 (문자열은 char 배열로 다루지만, 일단 이렇게 쓴다고 알아두세요!)
    int price = 1200000;              // 단가 (정수)
    int quantity = 1;                 // 수량 (정수)
    float discountRate = 0.05f;       // 할인율 (5%, 실수)
    char customerGrade = 'V';         // 고객 등급 (문자)

    printf("--- [간단한 물건 구매 계산기] ---\n");
    printf("상품명: %s\n", productName);
    printf("단가: %d원\n", price);
    printf("수량: %d개\n", quantity);

    // 2. 연산자를 사용하여 계산: 상자 안의 데이터를 가지고 요리합니다.
    int subTotal = price * quantity; // (곱셈 연산자) 단가 * 수량으로 총액 계산
    printf("-------------------------------\n");
    printf("총액 (할인 전): %d원\n", subTotal);

    // 고객 등급이 'V' (VIP)이면 5% 할인 적용
    // (여기서는 if문을 배우지 않았으니, 조건이 만족했다고 가정하고 진행할게요!)
    // 실제로는 if (customerGrade == 'V') { ... } 처럼 사용합니다.
    float discountAmount = subTotal * discountRate; // (곱셈 연산자) 할인 금액 계산
    float finalPrice = subTotal - discountAmount;   // (뺄셈 연산자) 최종 결제 금액 계산

    printf("할인율: %.0f%%\n", discountRate * 100); // 소수점 없이 % 표시
    printf("할인 금액: %.0f원\n", discountAmount);   // 소수점 없이 출력 (원 단위)
    printf("최종 결제 금액: %.0f원\n", finalPrice);    // 소수점 없이 출력 (원 단위)

    // 수량 변경 시뮬레이션: += 연산자 사용
    quantity += 1; // 1개 더 추가! (quantity = quantity + 1; 과 같음)
    printf("\n--- 수량 변경 시뮬레이션 (1개 추가) ---\n");
    printf("새로운 수량: %d개\n", quantity);
    subTotal = price * quantity; // 총액 다시 계산
    discountAmount = subTotal * discountRate;
    finalPrice = subTotal - discountAmount;
    printf("총액 (할인 전): %d원\n", subTotal);
    printf("최종 결제 금액: %.0f원\n", finalPrice);

    return 0;
}
```

### 🎊 마무리: 오늘의 핵심 정리!

오늘 우리는 C 언어의 튼튼한 뼈대, **변수**, **자료형**, **연산자**에 대해 알아봤습니다.

*   **변수:** 데이터를 담는 '상자'. 이름을 가지고 있고, 값을 저장하거나 변경할 수 있어요.
*   **자료형:** 상자의 '종류'. `int`(정수), `char`(문자), `float`/`double`(실수) 등 어떤 데이터를 얼마나 크게 담을지 정해줍니다.
*   **연산자:** 데이터를 가지고 '작업'하는 도구. 덧셈, 뺄셈 같은 산술 연산자부터, 값을 할당하는 대입 연산자, 1씩 증감시키는 증감 연산자까지 다양하게 배웠습니다.

이 세 가지 개념은 C 언어 프로그래밍의 가장 기본적인 토대입니다. 이들을 잘 이해하고 능숙하게 다룰 수 있어야만 더 복잡한 프로그램을 만들 수 있어요.

**가장 중요한 것은 '직접 해보는 것'입니다!** 오늘 배운 예제 코드를 직접 입력해보고, 변수 값을 바꿔보고, 연산자를 다르게 사용해보면서 익숙해지는 것이 중요해요.
