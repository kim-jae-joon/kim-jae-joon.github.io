## 🔥 19강: 클래스 기본 개념 - 객체 지향 프로그래밍의 진짜 비밀이 여기 있대요! 🚀

안녕하세요, 파이썬 일타 강사 최고(본인이라고 부르는 걸 좋아하는) 멋진 전국 시니어 개발자 X입니다! 😎 오늘은  "클래스 기본 개념" 에 대해 알아볼 거예요. 객체 지향 프로그래밍의 진짜 비밀이 풀리는 중요한 강좌라니까요, 집중력 높여서 들어주세요! 👀

**💡 초보자에게 간단하게 설명해 주시겠어요? 🤔**
*클래스* 는 마치 레시피 같은 거예요.  레시피를 보고 쿠키를 만들 수 있듯이, 클래스의 코드를 보고 프로그램 객체를 만들 수 있다는 거죠! 🍪

그럼 이 레시피에 어떤 재료가 들어갈까요? 🤔 바로 **멤버 변수**와 **메서드**죠!  
*멤버 변수* 는 쿠키의 재료 같아요. (설탕, 버터 등) 클래스 내부에서 저장되는 데이터를 말하는 거예요. 🍪🥛
*메서드* 는 쿠키를 만드는 과정 자체죠. *(재료들을 섞고, 구운다, ...) *  클래스에서 정의된 함수라고 생각하면 됩니다. 👨‍🍳

**✅ 예시 코드로 뽀모도로 타임!**
어떻게 만들까요? 단순히 멤버 변수를 정의하고 메서드를 정의하는 거예요.  🤯 초보자도 알겠지요?

```python
class Dog: # 클래스 정의 시작
    def __init__(self, name, age): # 생성자 (Constructor) : 객체 만들 때 자동으로 실행되는 메서드! 🐶
        self.name = name # 멤버 변수 'name' 에 강아지 이름을 저장!
        self.age = age # 멤버 변수 'age' 에 강아지 나이를 저장!

    def bark(self): # 메서드 정의 : 강아지가 짖는 행동을 나타내는 함수! 🐶
        print(f"{self.name} 은 {self.age}살입니다!") # 출력문: 강아지 이름과 나이 출력

# 객체 생성 ( 인스턴스화 ) -  클래스 레시피로 쿠키 만들기!🍪
my_dog = Dog("멍뭉이", 3) # "멍뭉이"라는 이름, 3살인 강아지를 생성!

my_dog.bark() # '강아지가 짖는 행동'을 시행! 출력 :  “멍뭉이는 3살입니다!”
```

* **`class Dog:`** : 🐶 "Dog" 라는 클래스를 정의합니다. 이제부터 우리는 강아지를 나타내는 객체를 만들 수 있습니다.
* **`def __init__(self, name, age):`**:  생성자! 객체 생성할 때 자동으로 실행되어 `name`, `age` 를 멤버 변수에 저장하는 역할을 합니다. (강아지의 이름과 나이) 🤔

**🚨 실무주의보:**
클래스를 이용하면 코드 재사용도가 높아집니다. 여러 개의 강아지 객체를 만들더라도, 각 강아지에게는 다른 이름과 나이가 있겠지만,  코드 구조 자체는 동일하게 유지됩니다! 🤯


**🤔 "클래스 생성자"에 대해 더 자세히 알고 싶어요? 🤔**
생성자 `__init__(self, name, age)` 는 클래스의 객체를 만들 때 자동으로 실행되는 함수예요. 마치 강아지를 태어낼 때 부모가 이름과 나이를 정하는 것처럼, 생성자가 멤버 변수들을 초기화하는 역할을 합니다. 😎

**✅ 추가적인 메서드 활용:**
클래스는 단순히 데이터만 저장하는 것이 아닌, 객체의 행동을 나타내는 기능까지 담을 수 있어요!  🔥
예를 들어 "멍?" 라고 말하는 메서드나, "꼬리 흔들기" 같은 메서드를 추가할 수 있죠. 🚀
