**2강: 하는 엘소 : 운각러요 (종머울북호야물할)**

안녕하세요, 파이썬을 배우는 친구들! 저는 '파이썬 일타 강사'와 '15년 차 시니어 개발자'로 두 가지 역할을兼顾하고 있습니다. 내일부터는 진짜 실무에 쓸 수 있는 법칙을 알려드리겠습니다. 파이썬은 20~30대 후반의 중장년들을 위해 만들어진 언어입니다. 그만큼 실무 경험과 지식이 풍부한 분들에게 도움이 더 많이 됩니다. 하지만 저희는 초보자도 가르쳐 주고, 중급자도 배울 거리 많습니다.

**1. 파이썬의 장점**

파이썬은 쉽게 배우고, 짧고 간결한 코드로 개발할 수 있습니다. 또한, 많은 라이브러리가 제공되어 다양한 분야에 사용될 수 있습니다.

### 💡 초보자 폭풍 질문!

*   파이썬의 장점을 왜 이제야 알았어요?

## 2. 데이터 타입

파이썬에는 숫자, 문자열, 리스트, 딕셔너리 등 다양한 데이터 타입이 있습니다.

### 🚨 실무주의보

데이터 타입은 프로그램의 성능과 안정성에 직접적인 영향을 미칩니다. 적절한 데이터 타입을 선택하여 프로그래밍하는 것이 중요합니다.

### 2.1 숫자

숫자는 정수와 실수 두 가지로 나눌 수 있습니다.

#### 👉 코드 예제

```python
# 정수
a = 10
print(a)  # 출력: 10

# 실수
b = 3.14
print(b)  # 출력: 3.14
```

#### 👉 개념 설명

*   정수는 숫자가 정해진 값을 나타내는 데 사용됩니다.
*   실수는 소수점이 포함된 숫자를 나타냅니다.

### 2.2 문자열

문자열은 하나 또는 그 이상의 글자를 포함하는 데이터 타입입니다.

#### 👉 코드 예제

```python
# 단일 문자열
c = "A"
print(c)  # 출력: A

# 다중 문자열
d = "Hello, World!"
print(d)  # 출력: Hello, World!
```

#### 👉 개념 설명

*   문자열은 문자 하나 이상으로 구성된 데이터 타입입니다.
*   문자열 내에서 문자를 접근할 때 인덱스(0부터 시작)를 사용합니다.

### 2.3 리스트

리스트는 가변 길이의 목록을 나타내는 데이터 타입입니다.

#### 👉 코드 예제

```python
# 빈 리스트 생성
e = []
print(e)  # 출력: []

# 리스트에 요소 추가
e.append(1)
e.append(2)
e.append(3)
print(e)  # 출력: [1, 2, 3]

# 리스트의 특정 인덱스 값 추출
print(e[0])  # 출력: 1
```

#### 👉 개념 설명

*   리스트는 요소 여러 개를 저장하는 데 사용됩니다.
*   리스트 내에서 요소를 접근할 때 인덱스를 사용합니다.

### 2.4 딕셔너리

딕셔너리는 키-값 쌍으로 구성된 데이터 타입입니다.

#### 👉 코드 예제

```python
# 빈 딕셔너리 생성
f = {}
print(f)  # 출력: {}

# 딕셔너리에 요소 추가
f["name"] = "John"
f["age"] = 30
print(f)  # 출력: {'name': 'John', 'age': 30}

# 딕셔너리의 특정 키에 해당하는 값을 추출
print(f["name"])  # 출력: John
```

#### 👉 개념 설명

*   딕셔너리는 키-값 쌍을 저장하는 데 사용됩니다.
*   딕셔너리 내에서 특정 키에 해당하는 값을 접근할 때 키를 사용합니다.

## 3. 제어 구조

파이썬의 제어 구조는 다음과 같습니다.

### 3.1 조건문

조건문은 프로그램 흐름을 조절하는 데 사용됩니다.

#### 👉 코드 예제

```python
# 조건식의 참, 거짓 판별
x = 5
if x > 10:
    print("x는 10보다 큽니다.")
else:
    print("x는 10보다 작습니다.")

# 출력: x는 10보다 작습니다.
```

#### 👉 개념 설명

*   조건문은 프로그램의 흐름을 조절하는 데 사용됩니다.
*   `if` 절을 사용하여 참, 거짓 판별할 수 있습니다.

### 3.2 반복문

반복문은 특정 코드 블록을 여러 번 실행하는 데 사용됩니다.

#### 👉 코드 예제

```python
# for 루프를 사용한 반복문
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# 출력: apple, banana, cherry
```

#### 👉 개념 설명

*   반복문은 특정 코드 블록을 여러 번 실행하는 데 사용됩니다.
*   `for` 루프를 사용하여 반복할 수 있습니다.

### 3.3 함수

함수는 프로그램 내에서 재사용 가능한 코드 블록을 정의하는 데 사용됩니다.

#### 👉 코드 예제

```python
# 함수 정의
def greet(name):
    print(f"Hello, {name}!")

# 함수 호출
greet("John")
# 출력: Hello, John!
```

#### 👉 개념 설명

*   함수는 프로그램 내에서 재사용 가능한 코드 블록을 정의하는 데 사용됩니다.
*   `def` 키워드를 사용하여 함수를 정의할 수 있습니다.

## 4. 모듈과 패키지

파이썬의 모듈과 패키지는 다음과 같습니다.

### 4.1 모듈

모듈은 프로그램 내에서 재사용 가능한 코드 블록을 정의하는 데 사용됩니다.

#### 👉 코드 예제

```python
# math 모듈 import
import math

# math 모듈 내의 함수 호출
print(math.pi)
```

#### 👉 개념 설명

*   모듈은 프로그램 내에서 재사용 가능한 코드 블록을 정의하는 데 사용됩니다.
*   `import` 문으로 모듈을 import할 수 있습니다.

### 4.2 패키지

패키지는 여러 모듈을 포함한 프로그램 구조입니다.

#### 👉 코드 예제

```python
# math 패키지 내의 모듈 import
from math import sin, cos

# math 패키지 내의 함수 호출
print(sin(30))
print(cos(45))
```

#### 👉 개념 설명

*   패키지는 여러 모듈을 포함한 프로그램 구조입니다.
*   `from` 키워드를 사용하여 패키지 내의 모듈을 import할 수 있습니다.

## 5. 예외 처리

파이썬의 예외 처리는 다음과 같습니다.

### 5.1 try/except 블록

try/except 블록은 프로그램에서 예외를 발생시킬 때 처리하는 데 사용됩니다.

#### 👉 코드 예제

```python
# try/except 블록
try:
    x = 5 / 0
except ZeroDivisionError:
    print("오류가 발생했습니다.")
```

#### 👉 개념 설명

*   try/except 블록은 프로그램에서 예외를 발생시킬 때 처리하는 데 사용됩니다.
*   `try` 블록 내의 코드를 실행한 후, 예외가 발생할 경우 `except` 블록이 실행됩니다.
