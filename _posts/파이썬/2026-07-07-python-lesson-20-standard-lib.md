## 1. 내 컴퓨터의 지배자: os 모듈

먼저 os 모듈입니다. os는 Operating System의 약자예요. 말 그대로 여러분의 운영체제(윈도우, 맥, 리눅스 등)와 파이썬이 대화를 나눌 수 있게 해주는 통역사라고 생각하면 됩니다.

비유를 들자면, os 모듈은 내 컴퓨터라는 거대한 서류함의 관리자입니다. "어느 폴더에 뭐가 있지?", "새 폴더 하나 만들어줘!", "이 파일 이름 좀 바꿔줘!" 같은 명령을 내릴 때 사용합니다.

### os 모듈 활용법 3가지

os 모듈을 제대로 쓰려면 최소한 이 세 가지 기능은 마스터해야 합니다.

#### 첫 번째: 경로 확인 및 변경 (어디 있는지 찾기)
우리가 코딩을 하다 보면 "지금 내 프로그램이 컴퓨터 어디에서 실행되고 있지?"라는 궁금증이 생깁니다. 이때 사용하는 것이 현재 작업 디렉토리 확인 기능입니다.

```python
import os

# 1. 현재 내가 작업하고 있는 폴더 경로를 가져옵니다.
current_path = os.getcwd()
print("현재 위치는 여기입니다: " + current_path)

# 2. 작업 경로를 다른 곳으로 옮겨봅니다. (경로는 본인 컴퓨터에 맞게 수정하세요)
# os.chdir("C:/Users/Documents") 
# print("경로를 변경했습니다!")

# 3. 변경된 경로를 다시 확인합니다.
print("변경된 위치: " + os.getcwd())
```
- `import os`: os라는 도구 상자를 가져오겠다는 선언입니다.
- `os.getcwd()`: Get Current Working Directory의 약자로, 지금 파이썬이 바라보고 있는 현재 폴더 경로를 알려줍니다.
- `os.chdir()`: Change Directory의 약자로, 작업 폴더를 강제로 이동시킵니다. 마치 윈도우 탐색기에서 폴더를 더블클릭해서 들어가는 것과 같습니다.

#### 두 번째: 폴더 내용물 훔쳐보기 (목록 확인)
특정 폴더에 어떤 파일들이 들어있는지 리스트로 쫙 뽑아내고 싶을 때 사용합니다.

```python
import os

# 확인하고 싶은 폴더 경로를 설정합니다.
target_dir = "." # 마침표는 '현재 폴더'라는 뜻입니다.

# 폴더 안에 있는 모든 파일과 폴더 이름을 리스트 형태로 가져옵니다.
file_list = os.listdir(target_dir)

print("현재 폴더의 구성원들입니다:")
for file in file_list:
    print("- " + file)
```
- `os.listdir()`: 지정한 경로에 있는 모든 파일과 폴더 이름을 리스트로 반환합니다.
- `for file in file_list`: 리스트에 담긴 파일 이름들을 하나씩 꺼내서 출력하는 반복문입니다.

#### 세 번째: 폴더 생성 및 관리 (내 집 마련)
코드로 폴더를 직접 만들고 관리하는 방법입니다.

```python
import os

# 생성할 폴더 이름을 정합니다.
new_folder_name = "재준봇_학습자료"

# 폴더가 이미 존재하는지 확인하고, 없을 때만 만듭니다.
if not os.path.exists(new_folder_name):
    os.mkdir(new_folder_name)
    print(new_folder_name + " 폴더를 성공적으로 만들었습니다!")
else:
    print("이미 폴더가 존재해서 만들지 않았습니다.")

# 여러 단계의 폴더를 한 번에 만들고 싶을 때는 makedirs를 씁니다.
os.makedirs("python_study/day20/library", exist_ok=True)
print("하위 폴더까지 한 번에 생성 완료!")
```
- `os.path.exists()`: 해당 경로에 파일이나 폴더가 이미 있는지 확인합니다. 무작정 만들려다가 이미 있으면 에러가 나기 때문에 꼭 확인하는 습관을 들여야 합니다.
- `os.mkdir()`: 폴더를 하나 생성합니다.
- `os.makedirs()`: 상위 폴더가 없어도 하위 폴더까지 한 번에 몽땅 만들어버리는 아주 강력한 기능입니다. `exist_ok=True` 옵션을 주면 폴더가 이미 있어도 에러를 내지 않고 그냥 넘어갑니다.

> **초보자 폭풍 질문!**
> Q: 선생님, `os.mkdir`를 썼는데 `FileExistsError`가 났어요! 왜 그런가요?
> A: 그건 이미 똑같은 이름의 폴더가 그 자리에 있기 때문입니다! 컴퓨터는 똑같은 이름의 폴더를 한 곳에 두 개 만들 수 없거든요. 그래서 제가 코드에 넣은 것처럼 `if not os.path.exists()`로 먼저 확인을 하거나, `os.makedirs(..., exist_ok=True)`를 사용하는 것이 정답입니다!

## 2. 파이썬의 엔진 컨트롤러: sys 모듈

다음은 sys 모듈입니다. sys는 System의 약자입니다. 앞서 배운 os가 윈도우 같은 운영체제를 다루는 것이라면, sys는 파이썬 인터프리터(파이썬 엔진) 자체를 제어하는 도구입니다.

비유를 하자면, os가 집안의 가구를 배치하는 것이라면, sys는 집의 전기 배전반을 만지는 것과 같습니다. 조금 더 깊숙한 설정과 제어를 담당하죠.

### sys 모듈 활용법 3가지

#### 첫 번째: 실행 시 인자값 받기 (sys.argv)
프로그램을 실행할 때 외부에서 데이터를 바로 던져주고 싶을 때 사용합니다.

```python
import sys

# sys.argv는 프로그램 실행 시 입력한 인자들을 리스트로 저장합니다.
# 실행 방법: python test.py 홍길동 20
print("입력받은 인자 리스트: " + str(sys.argv))

if len(sys.argv) > 1:
    name = sys.argv[1]
    age = sys.argv[2]
    print("안녕하세요 " + name + "님! 나이는 " + age + "살이시군요!")
else:
    print("인자값을 입력해주세요!")
```
- `sys.argv`: 프로그램 이름과 함께 입력한 값들이 리스트 형태로 저장됩니다. `sys.argv[0]`은 항상 파일 이름이 들어가고, `sys.argv[1]`부터 우리가 입력한 실제 값이 들어갑니다.

#### 두 번째: 프로그램 강제 종료 (sys.exit)
특정 조건이 되었을 때 프로그램을 즉시 멈추게 하고 싶을 때 사용합니다.

```python
import sys

user_input = input("계속하시겠습니까? (y/n): ")

if user_input == 'n':
    print("프로그램을 즉시 종료합니다. 안녕히 가세요!")
    sys.exit() # 여기서 바로 프로그램이 종료됩니다.

print("계속 진행합니다!")
```
- `sys.exit()`: 이 함수가 호출되는 순간, 파이썬은 뒤에 어떤 코드가 남아있든 상관없이 그 즉시 실행을 중단하고 종료합니다. 비상정지 버튼 같은 존재죠.

#### 세 번째: 파이썬 경로 및 버전 확인 (sys.path, sys.version)
내 파이썬이 어디서 모듈을 찾고 있는지, 버전은 무엇인지 확인할 때 씁니다.

```python
import sys

# 1. 현재 설치된 파이썬의 버전을 확인합니다.
print("파이썬 버전: " + sys.version)

# 2. 파이썬이 모듈을 찾기 위해 검색하는 경로 리스트를 확인합니다.
print("모듈 검색 경로:")
for path in sys.path:
    print("- " + path)
```
- `sys.version`: 현재 실행 중인 파이썬의 상세 버전 정보를 알려줍니다.
- `sys.path`: 파이썬이 `import` 명령어를 만났을 때, 어떤 폴더들을 뒤져서 모듈을 찾을지 결정하는 경로 목록입니다. 가끔 내가 만든 모듈을 못 찾을 때 이 리스트에 경로를 추가해주면 해결됩니다.

> **실무주의보**
> 주의: `sys.exit()`는 매우 강력합니다. 하지만 너무 남발하면 프로그램의 흐름을 파악하기 어렵게 만들 수 있어요. 가급적이면 함수에서 `return`을 사용해 자연스럽게 종료하거나, 예외 처리(`try-except`)를 통해 우아하게 종료하는 습관을 들이는 것이 고수의 길입니다!

## 3. 시간의 마법사: datetime 모듈

마지막으로 datetime 모듈입니다. 날짜와 시간을 다루는 모듈이죠. 코딩을 하다 보면 "현재 시간이 몇 시지?", "오늘부터 디데이가 며칠 남았지?", "로그 파일에 기록할 시간을 남겨야겠는데?" 같은 상황이 정말 많이 옵니다.

비유하자면, datetime은 아주 정밀한 디지털 시계와 달력이 합쳐진 도구입니다.

### datetime 모듈 활용법 3가지

#### 첫 번째: 현재 날짜와 시간 가져오기
가장 기본 중의 기본입니다. 지금 이 순간을 포착하는 방법입니다.

```python
from datetime import datetime

# 현재 날짜와 시간을 모두 가져옵니다.
now = datetime.now()

print("현재 전체 시간: " + str(now))
print("현재 년도: " + str(now.year))
print("현재 월: " + str(now.month))
print("현재 일: " + str(now.day))
```
- `from datetime import datetime`: `datetime` 모듈 안에 또 `datetime` 클래스가 들어있어서 이렇게 가져옵니다.
- `datetime.now()`: 현재 시스템의 지역 시간을 가져오는 함수입니다.

#### 두 번째: 시간 차이 계산하기 (timedelta)
"지금부터 100일 뒤는 언제지?" 혹은 "내 생일까지 며칠 남았지?" 같은 계산을 할 때 사용합니다.

```python
from datetime import datetime, timedelta

# 1. 현재 시간 기준
today = datetime.now()

# 2. 100일 뒤의 날짜를 계산합니다.
after_100_days = today + timedelta(days=100)
print("오늘부터 100일 뒤는: " + str(after_100_days))

# 3. 1주일 전의 날짜를 계산합니다.
before_1_week = today - timedelta(weeks=1)
print("1주일 전은: " + str(before_1_week))
```
- `timedelta`: 말 그대로 '시간의 차이'를 나타내는 객체입니다. 일(days), 주(weeks), 시간(hours), 분(minutes) 단위로 설정해서 더하거나 뺄 수 있습니다.

#### 세 번째: 날짜 포맷 변경하기 (strftime, strptime)
컴퓨터가 이해하는 날짜 형식을 사람이 보기 편한 문자열로 바꾸거나, 반대로 문자열을 날짜 객체로 바꾸는 방법입니다.

```python
from datetime import datetime

now = datetime.now()

# 1. 날짜 객체 -> 문자열 (String Format Time)
# %Y: 년도(4자리), %m: 월, %d: 일, %H: 시간, %M: 분, %S: 초
formatted_time = now.strftime("%Y년 %m월 %d일 %H시 %M분 %S초")
print("보기 좋게 바꾼 시간: " + formatted_time)

# 2. 문자열 -> 날짜 객체 (String Parse Time)
date_string = "2024-12-25"
parsed_date = datetime.strptime(date_string, "%Y-%m-%d")
print("문자열을 날짜 객체로 변경: " + str(parsed_date))
```
- `strftime()`: "String Format"의 약자로, 날짜를 내가 원하는 모양의 문자열로 예쁘게 포장하는 것입니다.
- `strptime()`: "String Parse"의 약자로, 문자열을 분석해서 다시 파이썬이 계산할 수 있는 날짜 객체로 되돌리는 것입니다.

> **초보자 폭풍 질문!**
> Q: 선생님, `strftime`랑 `strptime` 이름이 너무 비슷해서 헷갈려요! 어떻게 구분하죠?
> A: 아주 좋은 질문입니다! 끝 글자를 보세요. `f`는 Format(형식)의 약자입니다. "날짜를 문자열 형식으로 포맷팅하겠다!"라는 뜻이죠. `p`는 Parse(분석)의 약자입니다. "문자열을 분석해서 날짜로 만들겠다!"라는 뜻입니다. **f는 포장지, p는 해체**라고 외우면 절대 안 잊어버리겠죠?
