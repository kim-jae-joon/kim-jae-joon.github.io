## 🔥 파이썬, 나와 함께 시작하는 로봇 시대 🚀

오늘은 '파이썬 기초: 변수와 자료형' 에 대해 배우겠습니다! 처음 프로그래밍을 접한다면 마치 컴퓨터에게 속삭이는 언어를 배우는 것과 같아, 흥미진진한 여정인 거예요! 🎉

### 💻 변수란 무엇일까요? 🤔

변수는 데이터를 저장하는 '상자'라고 생각해보세요. 이 상자에 우리가 원하는 값을 담고, 필요할 때마다 값을 읽어올 수 있습니다. 마치 실제 공간에서 가방이나 상자처럼!  🎒

```python
# 변수 '이름'에 "홍길동" 라는 문자열을 저장해 보세요
이름 = "홍길동" 

# 변수 '나이'에 25를 저장해 보세요
나이 = 25

print(이름, 나이) # 출력 결과: 홍길동 25 
```

- `이름 = "홍길동"` : 이 부분에서 '이름'이라는 변수를 만들고 그 안에 "홍길동"이라는 문자열을 저장했습니다.  💬
- `나이 = 25` : '나이' 라는 변수에 숫자 25를 담았습니다. 🔢

**💡 초보자 폭풍 질문!**: 변수 이름은 무엇이든 되나요? 🤔

네, 아직 개발하는 범위 내에서는 대부분 자유롭지만,  파이썬의 규칙에 맞춰서 작성해야 합니다. 예를 들어, '123변수'는 불가능하지만, '변수_123' 처럼 언더스코어(_) 로 구분하면 가능합니다!

### ✨ 자료형? 변수를 분류하는 방법이죠! ✨


데이터를 저장할 때 종류에 따라 어떤 방식으로 정리되어야 할까요? 🌈 바로 '자료형'이 그것입니다! 변수가 가지고 있는 데이터의 종류를 나타내는 것으로, 파이썬에서는 여러 종류의 자료형을 지원합니다.

- **숫자형 (Numeric Types)** : 
    * `int` : 정수, 양수와 음수 (예: 10, -5)  🔢
    * `float` : 소수점을 가진 실수 (예: 3.14, -2.7) 📏

- **문자열형 (String Types)** : 글자로 이루어진 데이터를 저장합니다. (예: "Hello", "Python")💬
- **논리형 (Boolean Types)** :  `True` 또는 `False` 로만 나타내는 자료형입니다. 조건을 판단하는 데 자주 사용됩니다. 🚦

**🚨 실무주의보**: 숫자를 문자열로, 문자열을 숫자로 변환할 때는 각 데이터 타입에 맞는 함수를 이용해야 합니다! (예: `int("10")`, `str(3.14)`)


### 📌 자료형 변환 🚀

```python
# int 형의 값을 string으로 변환하는 예시
숫자 = 10
문자열_숫자 = str(숫자)
print(문자열_숫자) # 출력 결과: 10

# float 형의 값을 string으로 변환하는 예시
소수점 = 3.14
문자열_소수점 = str(소수점)
print(문자열_소수점) # 출력 결과: 3.14
```
