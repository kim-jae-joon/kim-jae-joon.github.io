## 🤔 TCP/UDP: 네트워크 통신의 두 주인공!

자, 우리가 인터넷으로 정보를 주고받으면서 느끼는 속도와 안정성을 생각해 보세요. 이 모든 것을 가능하게 하는 것은 바로 **TCP(Transmission Control Protocol)** 와 **UDP(User Datagram Protocol)**입니다! 🤝

* **TCP:** 📦  '우리 회사'처럼 신뢰할 수 있고, 메시지를 정확히 전달하고 싶을 때 사용하는 프로토콜이에요. 하나의 데이터를 여러 개의 작은 패킷으로 나눠서 보내고, 각 패킷의 번호를 통해 뭉뚱그려 올리는 방식으로 '완전'하게 데이터를 보장합니다! 🔒

* **UDP:**  💌  간단하고 빠르게 메시지를 전달하고 싶을 때 사용하는 프로토콜이에요. 회신 확인 없이 바로 정보를 전달하기 때문에 TCP보다 속도가 빠르지만, 데이터 손실 가능성도 있습니다. 🚀

## 💻 파이썬으로 소켓 프로그래밍!

파이썬은 네트워크 소켓 프로그래밍을 위한 훌륭한 도구입니다! 🎉  `socket` 모듈을 이용하면 TCP/UDP 프로토콜을 사용하여 서버와 클라이언트를 구현할 수 있습니다.


### 1️⃣  서버 만들기: '파이썬 전설'의 커피숍

**💡 초보자 폭풍 질문!** 🤔
 * "요즘은 온라인에서 모든 것을 해결하는 세상이고, 서버와 클라이언트가 뭘 의미할까요?"
 * "그래서 TCP/UDP를 배우면 어떻게 되나요?"

우선 간단한 서버 프로그램부터 만들어 보겠습니다. '파이썬 전설'의 커피숍을 가정해보세요!  ☕️ 우리는 클라이언트가 주문하는 메뉴를 받고, "주문 접수되었습니다!" 라는 메시지를 응답하는 서버를 만들 수 있습니다.

```python
import socket

HOST = '127.0.0.1'    # 표준 루프백 IP 주소 (localhost)
PORT = 65432         # 포트 번호 설정

with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s: # TCP 소켓 생성
    s.bind((HOST, PORT))   # 서버와 연결할 IP 주소와 포트 지정
    s.listen()            # 클라이언트 접속을 기다림

    while True:
        conn, addr = s.accept()     # 클라이언트 접속 요청 수락
        with conn:
            print(f'클라이언트 연결됨: {addr}') 
            while True:
                data = conn.recv(1024).decode('utf-8')  # 메시지 받기 (최대 1024바이트)
                if not data:
                    break
                print(f'클라이언트가 보낸 메시지: {data}')   
                conn.sendall(b'주문 접수되었습니다!')      # 응답 전달



```

* **`socket.socket(socket.AF_INET, socket.SOCK_STREAM)`**: TCP 소켓을 생성합니다! `AF_INET`은 IPv4 주소를 사용하며, `SOCK_STREAM`은 TCP 프로토콜을 나타냅니다.
* **`s.bind((HOST, PORT))`**: 서버가 연결할 IP 주소와 포트 번호를 설정합니다. 
    *  `127.0.0.1` 은 'localhost' (나만의 컴퓨터)를 의미합니다. 다른 사용자에게 서비스를 제공하려면 외부 IP 주소를 입력해야 합니다!

* **`s.listen()`**: 서버가 클라이언트 연결을 기다리는 모드로 전환됩니다. 
* **`conn, addr = s.accept()`**:  클라이언트의 접속 요청이 있으면 연결을 수립하고, `conn`은 연결 객체, `addr`는 클라이언트의 IP 주소와 포트 번호를 담습니다.

* **`data = conn.recv(1024).decode('utf-8')`**:  클라이언트로부터 데이터를 받아옵니다. 
    * `conn.recv(1024)`는 최대 1024 바이트까지 데이터를 받습니다.  
    * `.decode('utf-8')`: UTF-8 문자 인코딩으로 변환하여 읽기 쉽게 합니다.

* **`conn.sendall(b'주문 접수되었습니다!')`**: 클라이언트에게 응답 메시지를 전송합니다.
    * `b'...'`: 바이너리 형식으로 데이터를 전달해야 합니다.


### 2️⃣  클라이언트 만들기: '파이썬 전설'의 고객

지금까지 서버만 만들었고, 이제 우리는 '파이썬 전설'의 커피숍에 주문을 보내는 클라이언트를 만들어보겠습니다. 🚀


```python
import socket

HOST = '127.0.0.1'  # 서버 IP 주소
PORT = 65432        # 서버 포트 번호

with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:  
    s.connect((HOST, PORT)) # 서버와 연결
    s.sendall(b'커피 주세요!')   # 메시지 전송
    data = s.recv(1024).decode('utf-8')  # 응답 받기
    print(f'서버의 응답: {data}') 

```

* **`s.connect((HOST, PORT))`**: 클라이언트가 서버와 연결합니다.
