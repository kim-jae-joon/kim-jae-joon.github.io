안녕하세요! 여러분의 파이썬 여정을 함께하는 최고의 일타 강사, 🐍파이썬 마스터님입니다. 오늘부터는 파이썬 프로그램의 주요 구성 요소 중 하나인 **메서드 정의와 호출**에 대해 자세히 알아보겠습니다! 🤩

눈치채셨나요? 코딩은 마치 요리사처럼 🧑‍🍳, 레시피(프로그램)를 통해 메뉴(결과물)를 만들어내는 과정이죠. 이때 "요리를 하다", "칼을 날카롭게 가꾸다"와 같은 단위 작업들을 **함수**라고 부르고, 그 함수들이 모여서 전체 레시피를 구성하는 것처럼, 파이썬 프로그램 역시 여러 개의 함수들로 나뉘어져 있습니다.

그런데 이번 강의에서는 요리사가 칼을 손에 쥐고 재료들을 준비하듯, **메서드 정의**와 호출 방법을 익히는 것을 목표로 합니다! ✨ 매력적인 비유를 통해 쉽게 이해할 수 있도록 설명해 드릴 테니, 제 말씀 잘 들어 주세요 😉.

###  ✨ 메서드 정의 - 파이썬 프로그램의 "특별한 요리법" ✨

우선 메서드는 우리가 **특정 작업을 정의하고 실행**할 수 있는 기능이라고 생각하면 좋습니다. 🍲 예를 들어, 감자튀김 레시피에는 "감자가 썰고 준비하는 법", "설탕과 계란을 반죽하는 법" 등 여러 단계 작업이 있죠? 각각의 작업은 특정한 메서드에 해당하며, 파이썬 프로그램에서는 이러한 **메서드를 정의**하여 코드를 구성합니다.

파이썬에서 메서드를 정의하는 방법은 다음과 같습니다:

```python
def 함수명(인자1, 인자2, ...):
    # 작업 내용 작성
    return 결과값  # 반환값 설정 (선택 사항)
```

* `def` 키워드를 사용하여 메서드를 시작합니다. 🐍

* `함수명`은 우리가 메서드에 붙여줄 이름을 정의하는 부분입니다. 파이썬에서는 알파벳, 숫자, 언더바(_)만 사용할 수 있습니다. 예시로 '감자튀김' 같은 이름을 쓸 수도 있지만, 조금 더 명확하고 간결한 이름이 좋습니다.

* `인자1`, `인자2` 등은 메서드가 실행될 때 필요한 입력 값을 나타냅니다. 마치 요리할 때 감자와 양념 같은 재료처럼 🤪!
* `# 작업 내용 작성`: 이 부분에 우리가 원하는 작업 코드를 작성합니다.

* `return 결과값`:  메서드가 끝난 후, 특정 값을 반환할 수 있습니다. 마치 요리의 완성된 음식처럼 😋!


**💡 초보자 폭풍 질문!:**
> '인자'는 무엇이죠? 예를 들어 '감자튀김' 메서드에 필요한 인자는 뭔가요??

**✔️ 답변:**  좋은 질문입니다! '인자'는 우리가 메서드가 실행될 때 주어진 정보들입니다. '감자튀김' 메서드의 경우, 감자와 기름, 설탕 등이 인자로 활용될 수 있습니다. 😉

### 💻 실제 코드 예시 - 메서드 정의: "숫자 더하기" 프로그램 만들기

```python
def add(num1, num2):  # 'add'라는 이름의 함수를 정의합니다. 두 개의 인자 (num1, num2)를 받습니다.
    result = num1 + num2   # num1과 num2를 더해서 'result' 변수에 저장합니다.
    return result        # 'result' 값을 반환합니다.

sum_value = add(5, 3)  # 'add' 메서드를 호출하고 결과값을 'sum_value' 변수에 저장합니다.
print(sum_value)     # 출력 결과: 8
```


**코드 설명:**

1. `def add(num1, num2):`: `add`라는 이름의 함수를 정의하고 두 개의 인자 (`num1`, `num2`)를 받습니다.
2. `result = num1 + num2`: 함수 내부에서 `num1`과 `num2`를 더하여 `result` 변수에 저장합니다.
3. `return result`: 함수가 종료될 때, `result` 값을 반환합니다. 

* 위 예시처럼 메서드는 코드를 재사용하고 효율적으로 관리할 수 있게 해줍니다! 💪

메서드 호출은 괄호 `()` 안에 메서드 이름과 필요한 인자들을 입력하여 실행하는 것을 의미합니다. 마치 요리 레시피에서 "감자가 썰고 준비하다"라고 명령하는 것처럼 😉.
