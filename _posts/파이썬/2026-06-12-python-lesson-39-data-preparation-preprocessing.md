자, 생각해보세요. 요리하기 전엔 재료들을 준비하고 깔끔하게 손질하는 게 중요죠? 데이터 분석이나 머신러닝을 할 때도 마찬가지예요! 우리는 **데이터를 원하는 형태로 변형하고 정제**하는 과정, 바로 **"데이터 준비 및 전처리"**에 집중할 거예요. 

그리고 이 "주방의 요리사" 역할을 하게 될 건 바로 NumPy!  👨‍🍳 NumPy는 데이터를 효율적으로 저장하고 처리하는 데 최적화된 라이브러리죠. 여러분이 알고 있는 Python 리스트 대신, NumPy의 배열은 **속도와 효율성** 면에서 월등히 우위에 있습니다. 😎

## 1. 🍎 바나나를 썰어요! - NumPy 배열 이해하기

NumPy는 데이터를 저장하는 가장 기본적인 요소인 **"배열(Array)"**을 제공합니다.  Python의 리스트와 유사하지만, NumPy 배열은 같은 자료형으로 구성되어 있어 더 빠르고 효율적으로 처리할 수 있습니다! 🍎 바나나는 잘게 썰어서 준비하는 것처럼 데이터도 필요한 형태로 변환해야 할 거예요.

```python
import numpy as np

# NumPy 배열 생성: (3, 2) 크기의 배열을 만들고 각 요소에 숫자를 채워넣습니다.
arr = np.array([[1, 2], [3, 4], [5, 6]])  
print(arr) 

# 출력 결과: [[1 2]
            #          [3 4]
            #          [5 6]]
```

💡 **초보자 폭풍 질문!**: "numpy.array([ ... ])" 에서 그냥 리스트를 `array()` 함수로 돌려서 배열을 만들 수 있나요? 🤔 정답은 네, 가능합니다! 하지만 NumPy 배열은 리스트보다 더욱 강력한 기능들을 가지고 있어 작업 속도와 효율성이 훨씬 좋습니다.

## 2. 🧂 데이터 전처리의 소금 - NumPy 활용하기

데이터는 때로는 우리가 원하는 형태로 오지 않아서, 조금 "간을" 해주거나 "소금" 을 뿌려야 할 때도 있죠! 이때, NumPy가 사용되어 데이터를 정제하고 변환합니다.  🧂

### 2-1. 🪄  데이터 타입 변환 - 파리의 날개처럼 가볍게!
   NumPy는 데이터 형태를 바꿀 수 있는 다양한 함수들을 제공합니다. 숫자, 문자열 등 다양한 유형의 데이터를 효율적으로 변환할 수 있습니다!

```python
# 숫자 데이터 타입 변환: string을 int로 변환
data = np.array(['1', '2', '3'], dtype=int)  # 먼저, '1', '2', '3' 를 NumPy 배열로 만들고 형태를 정의합니다!
print(data) # 출력 결과: [1 2 3]


```

### 2-2. 📏 데이터형 변환 - 바른 길을 찾아요!
   NumPy는 데이터 사이즈 조정, 행렬 추가/뺄셈 등 다양한 연산을 수행하며, 특히 **데이터 정규화** 에 유용하게 사용됩니다.

```python
# 데이터 정규화: 모든 값을 0~1 사이로 변환하여 표준화
data = np.array([10, 20, 30])  
normalized_data = (data - np.min(data)) / (np.max(data) - np.min(data))  
print(normalized_data) # 출력 결과: [0.         0.5       1.        ]
```

🚨 **실무주의보**: 데이터 정규화는 머신러닝 알고리즘 학습 성능 향상에 큰 도움이 되지만, 어떤 알고리즘에는 맞지 않는 경우도 있으니 주의하세요!
