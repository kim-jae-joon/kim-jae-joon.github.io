###  📦 패키지, 그 존재의 의미

먼저, **패키지**라는 개념을 이해하는 것이 중요합니다. 🤔 마치 레스토랑에서 주문하는 음식처럼, "파이썬"은 여러 기능을 가진 레시피 (프로그램)를 만들 수 있는 도구입니다. 하지만 레시피만으로는 아무것도 만들 수 없죠! 🍳  기본 재료들을 더해서 완성된 요리를 만들듯이, 파이썬 프로그램 개발을 위해서는 다양한 기능을 제공하는 "패키지"를 활용해야 합니다.

예를 들어, 웹사이트 구축에는 HTML, CSS, JavaScript와 같은 패키지가 필요하고, 데이터 분석에는 Pandas, NumPy 등의 패키지를 사용합니다.  🤯 이러한 패키지는 파이썬 프로젝트의 효율성을 높이고 개발 시간을 단축하는 데 큰 역할을 합니다!

### 🔌 PIP: 파이썬 패키지 관리자

이제부터 가장 중요한 주인공, **PIP** 에 대해 알아보겠습니다. PIP는 파이썬에서 패키지를 설치하고 관리하는 도구입니다. 마치 온라인 쇼핑몰처럼, 다양한 라이브러리를 원하는 대로 선택하여  🐍 파이썬 프로젝트에 편리하게 추가할 수 있습니다!

###  🚀 PIP 활용하기: 실제로 코드를 실행해 보세요!

#### 1. 설치 확인하기

먼저, "pip"가 이미 설치되어 있는지 확인하는 것은 매우 중요합니다. 다음 명령어를 사용하여 확인하세요.
```bash
python -m pip --version 
```

💡  출력 결과에 버전 번호가 표시되면 PIP가 성공적으로 설치되었음을 의미합니다. 🥳

#### 2. 패키지 설치하기

만약, "pip"가 설치되지 않았다면, 파이썬 공식 사이트에서 다운로드하여 설치하세요. (링크는 제 강의 노트에 안내드릴게요!) 😎  설치 완료 후, 원하는 패키지를 설치하는 것은 매우 간단합니다!

예를 들어 "requests" 라이브러리를 설치하고 싶다면 다음 명령어를 실행하면 됩니다.
```bash
pip install requests
```
💡 이 코드는 파이썬 패키지 관리 시스템(PIP)에 "requests"라는 패키지를 설치하는 명령입니다!

#### 3. 패키지 업데이트하기

기존에 설치된 패키지를 최신 버전으로 업데이트하려면 다음 명령어를 사용하세요.
```bash
pip install --upgrade requests
```
🚨 **주의:** 이 명령은 기존의 "requests" 라이브러리를 삭제하고 최신 버전으로 재설치합니다! 

#### 4. 패키지 제거하기

특정 패키지를 삭제해야 할 경우 다음 명령어를 사용하세요.
```bash
pip uninstall requests
```
