### 1.3.5.10.1. 파이썬의 역사와 개념

파이썬은 1991년 귀도 반 로메르(Guido van Rossum)가 개발했습니다. 그 당시에는 다른 프로그래밍 언어를 만들겠다는 목표였지만, **의미있는 이름을 찾지 못해** 파이썬이라고 정했는데... 🤣

파이썬은 interpreted language입니다. 즉, 실행할 때마다 코드를 컴파일하지 않고, 직접 인터프리터에 의해서 실행되는 언어입니다. 이점으로 인해 **개발 시간**을 줄이고, **디버깅**이 dễ합니다! 💡

### 파이썬의 특징

*   **명료한 문법**: 파이썬은 다른 프로그래밍 언어보다 명료한 문법을 가지고 있습니다. 예를 들어, `x = 5`는 변수 x에 5라는 값을 할당하는 문입니다.
*   **유연한 자료형**: 파이썬은 다양한 데이터 타입을 지원합니다. 예를 들어, 숫자, 문자열, 리스트 등이 있습니다.
*   **오류가 적음**: 파이썬은 런타임 오류가 적습니다. 하지만, 개발자의 실수인 경우에는 발생할 수 있습니다.

### 파이썬의 단점

*   **성능이 좋지 않음**: 파이썬은 interpreted language이기 때문에 컴파일 후 실행하는 다른 언어보다 성능이 떨어집니다.
*   **확장성이 제한적**: 파이썬은 interpreted language이기 때문에 확장을 하기가 어렵습니다.

### 1.3.5.10.2. 파이썬의 구조

파이썬은 모듈 단위로 구조화되어 있습니다. 모듈은 함수, 클래스, 변수 등으로 구성됩니다.

*   **함수**: 함수는 코드를 실행할 때 호출되는 블록입니다. 예를 들어, `def hello_world(): print("Hello World!")`는 hello_world 함수를 정의합니다.
*   **클래스**: 클래스는 객체를 생성하고, 메서드를 정의하는 블록입니다. 예를 들어, `class Car: def \_\_init__(self): self.color = "Red"`는 Car 클래스를 정의합니다.
*   **변수**: 변수는 값을 저장할 때 사용되는 블록입니다. 예를 들어, `x = 5`는 x라는 이름을 가진 변수에 5라는 값을 할당합니다.

### 파이썬 프로그래밍

파이썬은 간결하고 명료한 문법을 가지고 있습니다. 이점으로 인해 개발 시간을 줄이고, 디버깅이 쉽습니다!

*   **변수**: `x = 5`는 x라는 이름을 가진 변수에 5라는 값을 할당합니다.
*   **함수**: `def hello_world(): print("Hello World!")`는 hello_world 함수를 정의합니다.
*   **클래스**: `class Car: def \_\_init__(self): self.color = "Red"`는 Car 클래스를 정의합니다.

### 실무 예시

실무에서는 파이썬을 사용하여 다양한 업무를 처리할 수 있습니다. 예를 들어, **웹 개발**, **데이터 분석**, ** MACHINE LEARNING** 등이 있습니다!

*   **웹 개발**: 파이썬은 Flask나 Django와 같은 웹 프레임워크를 사용하여 웹 애플리케이션을 개발할 수 있습니다.
*   **데이터 분석**: 파이썬은 Pandas와 NumPy와 같은 라이브러리를 사용하여 데이터를 분석하고 처리할 수 있습니다.
*   ** MACHINE LEARNING**: 파이썬은 Scikit-learn과 TensorFlow와 같은 라이브러리를 사용하여 머신 러닝을 진행할 수 있습니다.

### 초보자 폭풍 질문! 🤔

**Q: 파이썬은 어떻게 실행할까요?**

A: 파이썬은 인터프리터에 의해 실행됩니다. 따라서, `python hello_world.py`와 같은 명령어를 사용하여 프로그램을 실행할 수 있습니다.

**Q: 파이썬에서 변수는 어떻게 선언하나요?**

A: 파이썬에서는 `x = 5`라는 코드로 변수 x에 5라는 값을 할당합니다.

**Q: 파이썬은 다양한 자료형을 지원하는데요, 다른 언어와 어떤 차이가 있나요?**

A: 파이썬은 다양한 자료형을 지원하는 점에서 다른 언어와 다릅니다. 예를 들어, 숫자, 문자열, 리스트 등이 있습니다.
