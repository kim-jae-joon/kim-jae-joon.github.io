##  🚀 20강: 인스턴스 생성 및 객체 변수 - 파이썬의 마법을 배우자!

안녕하세요 여러분, 저는 대한민국 최고의 파이썬 일타 강사이자 15년차 시니어 개발자인 **파이썬 마스터** 😎입니다. 오늘은 파이썬 프로그래밍의 심장부에 있는 '인스턴스 생성 및 객체 변수'를 배우겠습니다!

> "인스턴스? 객체 변수? 어떤 걸까요?" 🤔 걱정하지 마세요! 내가 이 모든 것을 당신에게 **시각적으로**, **간결하게**, 그리고 **유머러스하게** 설명해 드리겠습니다. 🔥

###  👶 파이썬, 오브젝트를 이해하기 - 그들의 생태계에 한번 들어보자! 🧬

상상해 보세요. 당신은 맛있는 초코칩 쿠키를 만들고 싶어요. 쿠키 반죽에는 **밀가루, 설탕, 계란, 초콜릿** 등 다양한 재료들이 필요합니다. 이들의 조합이 바로 "쿠키"라는 완성된 결과물을 만든답니다!

파이썬에서도 마찬가지입니다! 🍫 '객체'는 쿠키와 같은 하나의 개체를 나타내고, '클래스'는 쿠키 반죽 레시피처럼 객체를 만들 수 있는 지침을 말합니다. 그리고 **인스턴스**는 실제로 레시피를 따라 쿠키를 만든 것과 같습니다. 각각의 쿠키가 유일한 특징을 가지고 있듯이, 인스턴스도 클래스에 정의된 변수 값을 기반으로 독자적인 특성을 지니게 됩니다.

> 예를 들어 "강아지"라는 클래스는 `name`, `breed`, `age`와 같은 속성을 가질 수 있습니다. 그리고 각 강아지는 이러한 속성들을 가지고 **유일한 개체**로 존재하게 된답니다! 🐶

###  📝 인스턴스 생성하기 - 파이썬, 마법의 명령어를 사용해 보세요!

파이썬에서는 `class` 키워드를 사용하여 클래스를 정의하고, `instance = ClassName()` 문을 통해 인스턴스를 생성합니다. 🪄

```python
# 강아지 클래스 정의
class Dog:
    def __init__(self, name, breed, age):
        self.name = name
        self.breed = breed
        self.age = age

# 강아지 인스턴스 생성
my_dog = Dog("뽀로로", "푸들", 3)  
```

**설명:**

- `class Dog:`: 'Dog'라는 이름의 클래스를 정의합니다.
- `def __init__(self, name, breed, age):`: 이는 생성자 함수입니다! 인스턴스가 생성될 때 자동으로 실행되어 강아지 객체에 속성을 할당합니다. ⚙️
- `self.name = name`: 'my_dog' 이름이라는 인자가 입력되면  `my_dog` 객체의 `name` 변수에 저장됩니다! 💪
- `my_dog = Dog("뽀로로", "푸들", 3)`: "뽀로로" 라는 이름, "푸들" 종류, 3살짜리 강아지 인스턴스를 만들어서  `my_dog` 변수에 저장합니다.

###  🔎 객체 변수 - 우리 강아지를 더 자세히 알아보자! 👀

이제 `my_dog` 객체에 접근하여 강아지의 정보를 확인할 수 있습니다! 🐶

```python
print(my_dog.name)   # 출력: 뽀로로
print(my_dog.breed)  # 출력: 푸들
print(my_dog.age)    # 출력: 3
```

`my_dog.`는 'my_dog' 객체의 속성에 접근하는 방법입니다. 💪
