##  26강: 웹 프로그래밍 기초 (Flask) 🚀 - 인공지능보다 훨씬 빠르게 웹사이트 만드는 비밀!

**여러분 안녕하세요! 👋 대한민국 최고 파이썬 일타 강사, 저를 따라 하면 누구나 웹 개발 천재가 될 수 있습니다. 😎 오늘부터 "Flask"라는 마법의 주문을 배우겠습니다. 🧙‍♂️ 이 주문을 외우면 아무도 의심하지 않게 당신만의 특별한 웹사이트를 만들어낼 수 있답니다!**

### 🤔 Flask, 그게 뭔가요?

FlasK는 **Python으로 웹 애플리케이션을 만드는 데 사용하는 프레임워크**입니다. "프레임워크"란 생각보다 어려운 단어죠? 걱정 마세요!  말로만 설명하기보다, 그림으로 설명해 드릴게요!

**🤯 예를 들어,** 우리가 집을 짓고 싶다고 가정해봅시다. 🧱 다리부터 시작해서 벽을 세우고 창문을 설치하는 과정이 있죠? 
Flask는 **웹사이트의 건축 계획 (프레임워크)**과 **설비(파이썬, HTML 등)들을 제공**하여 우리가 웹 개발 프로세스를 효율적으로 진행하도록 도와줍니다.

`Flask`:  🏗️ 웹 사이트 건물을 짓는 플랫폼!
`Python`: 💪 건축 자재 (강철, 나무, 창문)
`HTML`: 📑 지붕과 벽을 만드는 설계도 

### 🔥 Flask의 매력점은?

1. **간편하고 배우기 쉬운 디자인**:  초보자도 웹 개발에 빠르게 적응할 수 있도록 편안한 환경을 제공합니다! 🤩
2. **다용도성:** 작고 간단한 웹사이트부터 복잡한 애플리케이션까지 다양하게 활용 가능합니다. 🎯
3. **능동적인 커뮤니티**: 문제가 생길 때 도움을 주는 많은 개발자들이 함께 있습니다. 🙌

### 🚀 Flask 기초 만들기! - Hello, World! 🤩

마지막으로, 우리 웹사이트에 "안녕하세요, 세계!"라는 문구를 출력하는 간단한 코드를 살펴볼까요? 🌎💬


```python
from flask import Flask

app = Flask(__name__)

@app.route("/")
def hello_world():
  return "Hello, World!"

if __name__ == "__main__":
  app.run(debug=True)
```

**🧐 코드 설명:**

- `from flask import Flask`: Flask 모듈을 불러옵니다! 📚
- `app = Flask(__name__)`: Flask 애플리케이션 객체를 생성합니다. 이게 우리 웹사이트의 기본 프레임워크죠! 💪
- `@app.route("/")`:  "/"(루트 경로)에 접근했을 때 실행할 함수인 `hello_world`를 지정합니다. 🛣️
- `def hello_world():`: "Hello, World!" 문자열을 반환하는 함수입니다! 🎉
- `if __name__ == "__main__":`: 파일이 직접 실행되었을 경우만 웹 서버를 시작합니다.

**💡 초보자 폭풍 질문!**:  'app.run(debug=True)' 부분은 무슨 기능을 하는 걸까요? 🤔 답변: 진짜 개발 중인 사람들은 디버깅 모드를 사용해서 코드 오류를 빠르게 찾아내는 거예요! 🔥

**🚨 실무주의보:**  `debug=False` 로 설정해야 실제 웹사이트 배포 시 안전한 환경을 확보할 수 있습니다! 🔐
