**7강: 하는 엘소 : 솠곤라요 (종머울북호야물할)**

오늘은 '함수'라는 가장 중요한 핵심 개념을 한 번에 끝장내겠습니다! 🎉 함수는 프로그램의 기본 단위로, 코드를 재사용하고, 유지보수가 쉬워지게 하는데 사용됩니다. 하지만, 왜 이런 이름이 붙었는지, 그리고 어떻게 만들고 활용할 수 있는지 궁금하죠? 🤔

**함수란 무엇인가요?**

함수는 일련의 명령을 하나로 묶은 것을 말합니다. 예를 들어, 어떤 사람이 주스를 만드는 과정을 설명해 보겠습니다.

1.  유기농 딸기를 사옵니다.
2.  딸기를 물론이고, 다져서 블렌더에 넣어줍니다.
3.  설탕과 레몬즙을 추가합니다.
4.  전자레인지에 돌려 주스를 만듭니다.

그런데, 이 사람을 한 번 더 불러 와야 하겠습니다. 새로운 사람에게 똑같은 과정을 설명해 줘야 합니다. 이때, 우리는 "주스 만들기"라는 이름의 함수를 정의하고, 그 내용만 호출해서 사용하면 됩니다! 📚

**함수 이름이 왜 이런가요?**

함수는 어떤 것을 수행하는지에 따라 '명사'나 '동사'로 지어집니다. 예를 들어, `add`, `delete`, `save` 같은 함수가 있습니다. 주소록을 만들 때 사용하는 `create_contact` 함수도 있죠! 📱

이러한 함수 이름은 사용하기 편리하고, 코드의 가독성을 높여줍니다.

**함수는 어떻게 만드나요?**

함수는 다음과 같이 정의할 수 있습니다:

```python
def add(a, b):
    return a + b
```

여기서 `a`와 `b`는 함수가 받을 인수를 의미합니다. `return` 키워드는 함수가 반환할 값을 지정합니다.

```python
print(add(3, 5)) # 출력: 8
```

함수는 위에서처럼 `def` 키워드로 정의한 후, 호출하여 사용하면 됩니다!

**함수가 왜 필요하죠?**

함수를 사용하는 이유는 코드를 재사용하기 쉽게 만들 수 있기 때문입니다. 예를 들어, 아래와 같이 두 개의 함수를 만들었다고 가정해 보겠습니다:

```python
def add(a, b):
    return a + b

def multiply(a, b):
    return a * b
```

이러한 두 개의 함수를 하나로 합쳐보겠습니다.

```python
def arithmetic(a, b, operator):
    if operator == "add":
        return a + b
    elif operator == "multiply":
        return a * b
    else:
        raise ValueError("Invalid Operator")
```

함수는 이러한 방식으로 재사용할 수 있습니다!

**🚨 실무주의보**

만약, 함수를 너무 많이 사용한다면 코드의 가독성이 저하될 수도 있으므로, 유의하시기 바랍니다! 💡

**💡 초보자 폭풍 질문! (계속)**

함수에 관한 마지막으로, 인수를 몇 개까지 받을 수 있는지 궁금하실 것 같아요!

```python
def add(a, b):
    return a + b

print(add(1)) # 에러 발생!
```

이때는 다음과 같이 `*args`를 사용하면 됩니다.

```python
def add(*args):
    result = 0
    for num in args:
        result += num
    return result
```

마지막으로, 함수가 몇 개 있는지 궁금하실 것 같아요!

```python
def function1():
    pass

def function2():
    pass

print(function1.__name__) # 출력: function1
print(function2.__name__) # 출력: function2
```

이러한 `__name__` 속성은 함수 이름을 반환합니다.

**💬 결론**

함수는 프로그램의 기본 단위로, 코드를 재사용하고 유지보수가 쉬워지게 하는데 사용됩니다. 함수 이름은 어떤 것을 수행하는지에 따라 '명사'나 '동사'로 지어집니다. 함수는 `def` 키워드로 정의한 후, 호출하여 사용하면 됩니다! 💪
