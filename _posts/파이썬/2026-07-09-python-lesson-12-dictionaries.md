## 12강: 딕셔너리 -  🔑 값 찾는 마법사!

🔥 안녕하세요 파이썬 애호가 여러분! 🔥 난 최고의 파이썬 강사, 당신의 코딩을 불태울 '파이' 몬스터! 😎 이번 주는 데이터를 정리하고 관리하는 멋진 도구인 **딕셔너리**에 대해 알아보겠습니다. 🤯

### 🤔 왜 딕셔너리는 필요할까요?

평소에 물건을 찾을 때 어떻게 하는지 생각해 보세요. 장바구니에 있는 과자는 이름으로, 책상 위의 노트는 색깔로 구분하는 걸 아닌가요?  인간은 키와 값을 연결하여 정보를 효율적으로 관리합니다. 🧠 파이썬에서도 이처럼 데이터를 **키와 값**으로 매핑하는 방식으로 사용할 수 있는 놀라운 도구가 바로 딕셔너리입니다!

예를 들어, 친구들의 연락처를 저장해보세요. 이름(키)을 통해 연락처(값)를 찾을 수 있겠죠? 이럴 때 딕셔너리가 정말 유용하다는 것을 느낄 수 있습니다. 👍

###  💻 파이썬 딕셔너리 - 작동 방식은?

파이썬에서 딕셔너리는 `{}` 로 표현하며, 키와 값을 함께 입력하여 정의합니다.
```python
person = {
    "name": "김민수",  
    "age": 25,  
    "hobby": "coding"   
}
```

* 위 코드처럼 키(예: "name")는 문자열로 시작하며 대괄호([])를 이용하여 값(예: "김민수")을 지정합니다. 🤩
* 각 키-값 쌍은 쉼표(,) 로 구분되어 표현되며, 마지막 쌍 뒤에는 중괄호(}) 를 사용해 정의를 마무리합니다.


###  👀 딕셔너리 활용법 - 다양한 예시

1. **학생 정보 관리:**
```python
student = {
    "name": "이영희",
    "major": "컴퓨터공학",
    "grade": 4,
}
print(student["name"])  # 출력: 이영희
print(student["major"]) # 출력: 컴퓨터공학
```

   * 학생의 이름, 전공, 등급을 키로 정의하고 해당 값을 저장하여 관리할 수 있습니다.


2. **간단한 카페 주문 시스템:**
```python
menu = {
    "커피": 4000,
    "차": 3500,
    "음료": 2500,
}

order = {}
order["커피"] = 2

total_price = 0

for item, quantity in order.items():
    total_price += menu[item] * quantity

print(f"총 주문 금액: {total_price}")
```

   * 각 음료의 가격을 키로 정의하고 그에 따른 값으로 저장하여 카페 주문 시스템을 구현할 수 있습니다.
