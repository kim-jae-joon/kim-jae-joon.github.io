## 🔥 리스트: 생성, 접근 및 조작 (파이썬의 기본 골격을 이해하기!)

안녕하세요! 파이썬 마스터가 되고 싶은 당신들을 위해 다시 돌아왔습니다 😎✨  오늘부터 파이썬의 핵심 요소 중 하나인 **리스트**를 깊이 있게 알아보겠어요! 리스트는 자료를 정렬하고, 여러 데이터를 효율적으로 저장하는 데 있어서 필수적이죠. 이 강의를 통해 파이썬 리스트에 대해 온전히 이해할 수 있도록 최선을 다하겠습니다! 🚀

### 1.  리스트: 파이썬의 '상자'

어떤 물건들을 모아서 보관하고 싶은 상황이라고 해봅시다. 이때, 우리는 여러 개의 공간을 가진 상자를 사용하는 것이죠? 🤔  파이썬에서도 리스트는 바로 그 **상자**와 같은 역할을 합니다!

```python
나의_좋아하는_음식 = ["김치찌개", "불고기", "떡볶이"] 
print(나의_좋아하는_음식)
```

코드가 실행되면, 위처럼 우리는 "김치찌개", "불고기", "떡볶이"를 리스트로 저장하고 출력할 수 있습니다. 각 음식은 리스트 안에 **요소**라 불리며, 숫자나 문자 등 다양한 데이터 타입을 담을 수 있습니다!

### 2.  리스트 생성: '상자'를 만드는 법

새로운 리스트를 만들 때는 중괄호 `[]` 를 사용하는데요. 각 요소는 쉼표 `,` 로 구분합니다. 예를 들어, 좋아하는 과목을 리스트로 만들어 보도록 하죠! 😎

```python
좋아하는_과목 = ["파이썬 프로그래밍", "웹 개발", "디자인 기초"]
print(좋아하는_과목)
```

💡 초보자 폭풍 질문!:  왜 중괄호를 사용해야 하는지 궁금하신가요? 🤔  중괄호는 파이썬에서 리스트를 나타내는 특별한 문자입니다. 바로 마법 지팡이처럼! ✨

### 3.  리스트 접근: '상자' 안의 물건 찾기

리스트 속 요소에 접근하기 위해서는 **인덱스**라는 번호를 사용합니다. 인덱스는 리스트 시작 위치부터 각 요소까지 계산되는데요, 첫 번째 요소는 0번 인덱스, 두 번째는 1번 인덱스로 나타납니다.

```python
색깔 = ["빨강", "초록", "파랑"]
print(색깔[0])  # 출력: 빨강
print(색깔[2])  # 출력: 파랑
```

🚨 실무주의보!: 주의하세요! 인덱스는 0부터 시작하며, 리스트 길이보다 크면 오류가 발생합니다. 마치 상자 안에 없는 물건을 찾으려고 하는 것과 같죠! 😬


### 4.  리스트 조작: '상자' 속 재배치

리스트는 **조작**하여 요소를 추가하거나 제거할 수 있습니다! 💪

#### 4-1. 요소 추가: "새 물건" 담기

* `.append()`: 리스트의 마지막에 새로운 요소를 추가하는 방법입니다.
  ```python
  동물 = ["강아지", "고양이"]
  동물.append("원숭이") 
  print(동물) # 출력: ['강아지', '고양이', '원숭이']
  ```

* `.insert()`: 리스트의 특정 인덱스에 새로운 요소를 추가하는 방법입니다.

```python
곡 = ["사랑", "행복", "슬픔"]
곡.insert(1, "기쁨") 
print(곡) # 출력: ['사랑', '기쁨', '행복', '슬픔']
```


#### 4-2. 요소 제거: "물건" 치우기

* `.remove()`: 리스트에서 특정 값을 찾아 제거하는 방법입니다.

  ```python
  나만의_곡 = ["사랑", "행복", "슬픔", "사랑"]
  나만의_곡.remove("사랑") 
  print(나만의_곡) # 출력: ['행복', '슬픔', '사랑']
  ```

* `.pop()`: 리스트에서 특정 인덱스에 있는 요소를 제거하고 반환하는 방법입니다.


### 5. 리스트 슬라이싱: 부분 가져오기

리스트의 일부만 선택적으로 추출할 수 있습니다! 마치 상자에서 원하는 물건만 골라내는 것과 같죠! 🤔

```python
숫자 = [1, 2, 3, 4, 5, 6]
print(숫자[1:4]) # 출력: [2, 3, 4]  # 2번째부터 4번째 요소까지
```
