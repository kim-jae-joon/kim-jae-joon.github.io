### 📚 Why Plotly? - 다른 시각화 도구들과 달라지는 특별함

물론, 파이썬에는 Matplotlib 같은 기존의 데이터 시각화 도구도 있어요. 그런데 Plotly는 그걸 한 단계 업그레이드한 존재라고 할 수 있습니다! 🤩 몇 가지 장점을 살펴보죠:

* **인터랙티브:** Plotly가 만든 그래프는 단순히 보는 것이 아니라, 확대/축소, 포커스 이동 등 다양한 작업을 통해 데이터를 자세하게 분석할 수 있어요. 마치 시험 문제처럼 여러 앵글에서 접근하는 것과 같은 감각이죠! 💪
* **다채로운 표현 방식:** 선 그래프, 바 차트, 산점도, 히스토그램 등 다양한 그래프 유형을 지원하며, 각 유형마다 설정 가능한 옵션도 매우 풍부해서 원하는 정보를 최대한 효과적으로 전달할 수 있죠! 🔥
* **웹 기반:** Plotly는 웹 브라우저에서 실행되기 때문에, 데이터 시각화 결과물을 다른 사람들과 간편하게 공유할 수 있어요.  compartilhar  🚀

###  💻 코드부터 이해하기 - Plotly의 기본 구조

Plotly를 사용하려면 먼저 `plotly` 라이브러리를 설치해야 합니다! 😉 간단히 `pip install plotly` 명령어로 설치 가능해요.

```bash
pip install plotly
```

이제, Plotly를 사용하는 가장 기본적인 코드를 살펴보겠습니다. 🤔 굉장히 간단한 예시지만, 데이터 시각화의 기본 원리를 이해하는데 효과적입니다!

```python
import plotly.express as px  # Plotly 라이브러리에서 express 모듈을 가져옵니다. 간편하게 사용하기 위해 px를 사용합니다. 💡

df = px.data.iris() # iris 데이터셋 불러오기 (Plotly가 제공하는 데이트)

fig = px.scatter(df, x="sepal_width", y="sepal_length", color="species")  # 산점도 그래프 생성

fig.show() # 그래프 표시 
```

**설명:**

1. `import plotly.express as px`: Plotly 라이브러리의 express 모듈을 가져옵니다. 간편하게 사용하기 위해 'px'라는 alias를 사용합니다. 💡

2. `df = px.data.iris()`: Plotly가 제공하는 iris 데이터셋을 불러옵니다. 이 데이터셋에는 꽃잎 길이, 꽃잎 너비 등의 정보가 포함되어 있습니다. 🌸

3. `fig = px.scatter(df, x="sepal_width", y="sepal_length", color="species")`: 산점도 그래프를 생성합니다. 
    - `df`: 사용할 데이터 프레임입니다.
    - `x="sepal_width"`: x축에 꽃잎 너비 값을 표시합니다.
    - `y="sepal_length"`: y축에 꽃잎 길이 값을 표시합니다.
    - `color="species"`: 꽃 종류에 따라 색상을 다르게 표현합니다.

4. `fig.show()`: 그래프를 화면에 표시합니다.


### 📊 데이터의 이야기는 무엇인가? - Plotly 활용 사례

Plotly의 매력은 바로 **다양한 형태의 시각화** 와 **인터랙티브 기능** 에 있습니다! 🎉 이를 통해 우리는 단순히 숫자만 보고 있는 것이 아니라 데이터 속에 숨겨진 이야기를 찾아낼 수 있죠.


* **경제 분석:** 주식 가격 변동 추세, 경기 침체 예측, 국가 GDP 성장 등 다양한 시각으로 경제 현상을 파악할 수 있습니다.📈

* **온라인 판매 데이터:** 상품 인기 순위, 고객 구매 패턴, 광고 효과 분석 등 판매 데이터를 시각화하여 비즈니스 전략에 활용하는 데 효과적입니다.🛍️

* **사회 통계:** 인구 변화 추이, 질병 확산 현황, 교육 수준 등 사회적인 문제들을 더욱 명확하게 이해할 수 있도록 도와줍니다.🌎
