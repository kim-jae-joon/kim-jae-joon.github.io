### 🎉 오늘의 주제: 파일 읽기 및 쓰기

파일 I/O는 **파이썬 프로그램을 현실 세계와 연결하는 핵심 기술**인데요. 우리가 만들어낸 모든 데이터나 설정 정보를 파일로 저장하고 불러올 수 있답니다!  🎉

예를 들어, 게임 개발자라면 플레이어의 성능 데이터를 파일에 저장해서 다음에 다시 시작할 때 그 정보를 가져와 사용할 수 있어요. 🕹️ 이런 방법으로 게임을 더욱 재미있게 만들고 사용자 경험을 향상시키기도 할 수 있죠!

**💡 초보자 폭풍 질문!: " 파일이랑 프로그래머, 어떤 관계야?" 🤔**


### 🧵 파이썬에서 파일 열기: 'open()' 함수를 활용해봐요!

파이썬은 `open()` 함수를 사용해서 파일을 열고 작업할 수 있답니다. 이 함수는 두 가지 중요한 매개변수를 받습니다. 
1️⃣ **파일 이름**: 우리가 열고 싶은 파이의 이름입니다.  
2️⃣ **모드**: 어떤 작업을 할지 나타내는 코드입니다. 파일을 읽으려면 'r', 쓰려면 'w' (새로운 파일) 또는 'a' (기존 파일 맨 뒤에 추가), 수정하려면 'x'를 사용하세요.

**🚨 실무주의보!:**  파이썬은 자동으로 파일의 경로를 확인합니다. 하지만, 파일이 특정 위치에 있거나 다른 프로젝트 내부에 있는 경우에는 **절대 경로**를 명시해야 합니다!


```python
# '파일이름.txt'라는 파일에 읽기 모드로 접근
file = open('파일이름.txt', 'r') 

# '새로운_파이프 파일.txt' 라는 파일에 쓰기 모드로 접근
new_file = open('새로운_파이프 파일.txt', 'w')  
```


### 📖 파일에서 데이터 읽기: `read()` 함수 사용하기!

열린 파일에서 데이터를 읽으려면 `read()` 함수를 사용합니다. `read()` 함수는 파일에 저장된 모든 내용을 한 번에 문자열로 돌려줍니다. 😎

```python
# '파일이름.txt' 에서 모든 내용을 읽어 변수 'content' 에 저장
content = file.read()
print(content) 

```



### 📝 파일로 데이터 쓰기: `write()` 함수 사용하기!


파일에 새로운 데이터를 쓰고 싶다면 `write()` 함수를 사용하세요. `write()` 함수는 문자열을 파일에 기록합니다.  ✍️

```python
# '새로운_파이프 파일.txt'에 "Hello, World!" 를 저장
new_file.write("Hello, World!") 
```



### 🚀 파일 닫기: `close()` 함수를 사용해서 연결 종료하기!


파일 작업을 마치면 반드시 `close()` 함수를 호출하여 파일을 닫아야 합니다.  🚫 이렇게 하면 데이터가 저장되고 프로그램과 파일 사이의 연결이 제대로 끊어져 다른 문제가 발생하지 않도록 합니다!

```python
# 열린 파일에 적은 내용을 저장하고 연결 해제
new_file.close()

# 모든 작업 완료 후 '파일이름.txt' 를 닫아요!
file.close()
```
