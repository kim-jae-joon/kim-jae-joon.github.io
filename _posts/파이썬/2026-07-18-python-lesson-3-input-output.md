## 🐍 파이썬, 알아두면 좋은 '단순 입력과 출력' 시리즈:  **1강 시작!** 🔥🚀

안녕하세요! 대한민국 최고의 파이썬 일타 강사 😎 나를 만나서 정말 기쁘시죠? 저는 당신의 코딩 여정을 함께 이끌어줄 **파이썬 멘토**가 될 거예요. 🤩 오늘은 파이썬의 기초 중 가장 중요한, '단순 입력과 출력'을 배우고, 파이썬 세계로 발걸음을 내딛도록 도와드리겠습니다!

### **💻 무엇이 '입력과 출력'일까요?** 🤔

상상해보세요. 우리가 컴퓨터에 말하는 것처럼 사람끼리 대화하는 것을 생각해 보세요. 저는 당신에게 질문을 하고, 당신은 제 질문에 대한 답변을 건네게죠? 😎 그래서 '입력'과 '출력'이라는 단어가 생겨났습니다!

* **입력 (Input):** 우리가 컴퓨터에 주는 정보
* **출력 (Output):** 컴퓨터가 우리에게 돌려주는 결과물

쉽게 말하면, 사람이 돈을 지불하고 상점에서 과자를 받는 것과 같습니다. 💰🍫  돈은 입력이고, 과자는 출력입니다! 🎉

### **✨ 파이썬 코드로 직접 실험해보세요!**

지금부터 우리가 `print()` 함수를 이용하여 파이썬에 '말'을 걸어볼게요. 마치 스피커처럼 당신의 메시지를 소리 냈는다고 생각하세요!  🎤

```python
print("안녕하세요! 저는 파이썬입니다!") # 이 줄은 우리가 입력한 내용이죠!

# print() 함수는 ( ) 안에 들어간 값을 출력해주는데요. 💡 초보자에게 조금 복잡하게 느껴질 수 있지만, 생각만 해 보세요!
# 마치 친구에게 "안녕하세요!"라고 말하는 것처럼 간단한걸까요?

```

이 코드를 실행하면,  **"안녕하세요! 저는 파이썬입니다!"** 가 화면에 나오게 될 거예요. 🤯 당신은 컴퓨터와 대화할 수 있다는 사실을 직접 경험했잖아요!

### **💎 다양한 입력과 출력 활용해 보세요!**

'단순 입력과 출력'이란 개념을 응용하여, 더욱 창의적으로 코드를 작성할 수 있습니다. 😉 예를 들어...

```python
# 당신의 이름을 물어보고 인사하는 프로그램 만들기
이름 = input("이름을 알려주세요: ") # ( ) 안에 입력받고 싶은 정보가 적어진 부분!
print(f"안녕하세요, {이름}님!") # f-string 활용해서 이름 출력하기 😎

# 숫자를 입력 받고 더 큰 수를 계산하는 프로그램 만들기
수1 = int(input("첫 번째 수를 입력하세요: ")) # int() 함수로 숫자로 변환! 🧮
수2 = int(input("두 번째 수를 입력하세요: "))
print(f"{수1} + {수2} = {수1 + 수2}")

```

**💡 초보자 폭풍 질문!**:  'int() 함수'가 무엇인지, 'f-string'이 뭐라고 하세요? 🤔 어떤 기능을 하는지 고민이시죠? 다음 강의에서는 이에 대한 자세한 설명과 실습을 통해 더욱 이해하기 쉬운 코드 작성법을 알려드릴게요!
