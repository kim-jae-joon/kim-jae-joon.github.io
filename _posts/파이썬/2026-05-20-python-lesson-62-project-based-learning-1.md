## 🚀 파이썬 마스터 되기 프로젝트 기반 학습: 간단 프로젝트 제작! (62강)

**안녕하세요, 코드하는 당신을 위한 파이썬 강사 '코딩 센타우로스'입니다!🔥  ** 오늘은 파이썬을 배우는 가장 효과적인 방법 중 하나인 **프로젝트 기반 학습**에 대해 알아볼 거예요. 단순히 문법이나 함수만 외우는 것보다 실제 프로젝트를 만들면서 파이썬의 매력을 훨씬 더 느낄 수 있답니다! 😎

💡 **왜 프로젝트 기반 학습이 중요할까요?** 🧐 
> '지식'과 '능력'은 완전히 다른 개념이죠. 지식은 책에서 배운 내용, 능력은 그것을 실제로 적용해서 문제를 해결하는 능력입니다. 프로젝트 기반 학습은 이 능력을 키워주는 가장 효과적인 방법이라고 볼 수 있습니다! 🚀

###  🎉 간단하지만 만족스러운 프로젝트, "숫자 맞추기 게임" 만들어 보세요! 🎉

우선, 초보자가 손쉽게 따라 할 수 있는 '숫자 맞추기 게임'을 만들어 보는 건 어떨까요? 🧠 이 게임은 바로 파이썬의 기본적인 문법과 조건문, 반복문을 활용하여 구현할 수 있습니다!

**1단계: 게임 로직 정하기**

*   컴퓨터가 1에서 100 사이의 임의의 숫자를 선택합니다.
*   사용자가 컴퓨터가 선택한 숫자를 맞히기 위해 추측을 합니다.
*   컴퓨터는 사용자의 추측이 맞으면 "정답입니다!"를 출력하고, 아니면 "더 높거나 낮습니다"라는 메시지를 출력합니다.

**2단계: 파이썬 코드 작성하기**

```python
import random

# 1부터 100 사이의 임의의 숫자를 생성합니다.
컴퓨터가_선택한_숫자 = random.randint(1, 100)

print("1에서 100 사이의 숫자를 맞춰 보세요!")

while True:
    사용자_추측 = int(input("숫자를 입력하세요: "))

    if 사용자_추측 == 컴퓨터가_선택한_숫자:
        print("정답입니다!")
        break  # 게임 종료

    elif 사용자_추측 < 컴퓨터가_선택한_숫자:
        print("더 높습니다.")

    else:
        print("더 낮습니다.")
```

**3단계: 코드 분석하기**

*   `import random`: `random` 모듈을 불러와 임의 숫자 생성 기능을 사용합니다.

*   `컴퓨터가_선택한_숫자 = random.randint(1, 100)`: 1부터 100 사이의 임의의 정수를 선택하여 `컴퓨터가_선택한_숫자` 변수에 저장합니다.
    *   `random.randint(a, b)` 함수는  `a` (포함)부터 `b` (포함) 사이의 랜덤 정수를 반환하는 함수입니다.

*   `while True:`: 무한 루프를 시작합니다. 사용자가 답을 맞출 때까지 게임이 계속됩니다.
*   `사용자_추측 = int(input("숫자를 입력하세요: "))`: 사용자에게 숫자를 입력하도록 요청하고, `int()` 함수로 정수 값으로 변환합니다.
*   `if 사용자_추측 == 컴퓨터가_선택한_숫자:`: 사용자가 맞춘 경우 "정답입니다!" 메시지를 출력하고 `break` 문으로 루프를 종료합니다.

*   `elif 사용자_추측 < 컴퓨터가_선택한_숫자:`: 사용자가 낮은 수를 추측했으면 "더 높습니다." 메시지를 출력합니다.
*   `else:`: 사용자가 높은 수를 추측했으면 "더 낮습니다." 메시지를 출력합니다.

**4단계: 게임 실행하기**

이제 코드를 저장하고 실행하면 숫자 맞추기 게임을 플레이할 수 있습니다! 🎉🎉
