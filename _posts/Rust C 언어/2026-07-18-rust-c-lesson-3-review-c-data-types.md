## 🚀 3강: C 언어 기본 문법 리뷰 - 데이터 타입 마스터하기: 코딩의 기초를 다지다

### 데이터 타입이란?

**데이터 타입**은 컴퓨터에게 "이 정보는 무엇이야?" 라는 질문에 답해주는 역할을 합니다. 마치 옷을 입는 것처럼, 각 데이터는 자신만의 "옷장"을 가지고 있어요. 숫자 데이터는 정수복장, 실수는 정밀복장을 입고, 문자는 텍스트복장을 입는 거죠! 

#### 예시로 이해하기

상상해 보세요, 여러분이 친구들과 파티를 열었는데 각자가 가져온 음식을 정확히 이해해야 합니다.

- **정수형 데이터 (int)**: 친구가 가져온 초콜릿 개수를 셀 때 쓰는 숫자죠. 예를 들어 `int numberOfChocolates = 10;` 은 친구가 가져온 초콜릿 개수가 정확히 10개임을 알려줍니다.
  
- **실수형 데이터 (float, double)**: 케이크의 무게를 측정할 때 필요한 세밀한 수치죠. 예를 들어 `float cakeWeight = 2.5;`는 케이크가 정확히 2.5킬로그램임을 알려줍니다.
  
- **문자형 데이터 (char)**: 파티 이름이나 인사말을 적어두는 작은 카드와 같습니다. `char partyGreeting = '환영합니다';`는 파티에 온 손님들에게 환영 메시지를 전달합니다.
  
- **다양한 타입의 조합**: 때로는 여러 타입을 함께 사용해야 합니다. 예를 들어, `struct Event { int id; char name[20]; float duration; }` 같은 구조체는 파티 이벤트 정보를 한 번에 담아낼 수 있어요. 여기서 `id`는 정수, `name`은 문자 배열, `duration`은 실수로 구성되어 있습니다.

### 데이터 타입의 세계 탐험: 상세 예제들

#### 1. 정수형 데이터 (int)

```c
#include <stdio.h>

int main() {
    // 정수형 변수 선언과 초기화
    int age = 25;        // 나이를 나타내는 정수 변수
    int numberOfStudents = 30;  // 수업에 참가하는 학생 수

    printf("나이: %d\n", age);      // 출력: 나이: 25
    printf("참가 학생 수: %d\n", numberOfStudents);  // 출력: 참가 학생 수: 30

    // 덧셈 예제
    int sum = age + numberOfStudents;  // 나이와 학생 수를 더한 값
    printf("나이와 학생 수의 합: %d\n", sum);  // 출력: 나이와 학생 수의 합: 55

    return 0;
}
```
- **설명**:
  - `int age = 25;`: `age`라는 이름의 정수형 변수를 선언하고 25로 초기화합니다.
  - `printf()` 함수를 사용해 변수의 값을 화면에 출력합니다. 정수형 변수는 %d 포맷 지정자로 출력됩니다.
  - `sum` 변수는 `age`와 `numberOfStudents`를 더한 결과를 저장합니다. 정수형 덧셈은 자연스럽게 동작합니다.

#### 2. 실수형 데이터 (float)

```c
#include <stdio.h>

int main() {
    float price = 19.99;  // 상품 가격을 나타내는 실수 변수
    float discount = 0.10;  // 할인율

    float finalPrice = price * (1 - discount);  // 할인된 최종 가격 계산

    printf("원래 가격: %.2f\n", price);  // 출력: 원래 가격: 19.99
    printf("할인율: %.2f\n", discount * 100);  // 출력: 할인율: 10.00
    printf("최종 가격: %.2f\n", finalPrice);  // 출력: 최종 가격: 17.99

    return 0;
}
```
- **설명**:
  - `float price = 19.99;`: `price` 변수를 선언하고 상품 가격을 실수로 초기화합니다.
  - 할인율을 적용해 최종 가격을 계산합니다. 실수형 연산은 소수점까지 정확하게 처리됩니다.
  - `%.2f` 포맷 지정자를 사용해 소수점 아래 두 자리까지 출력합니다.

#### 3. 문자형 데이터 (char)

```c
#include <stdio.h>

int main() {
    char greeting = 'H';  // 인사말 문자 변수 초기화
    char userName = "Alice";  // 이름 문자 배열 초기화 (주의: 문자 배열은 따옴표 필요)

    printf("인사말: %c\n", greeting);  // 출력: 인사말: H
    printf("이름: %s\n", userName);  // 출력: 이름: Alice

    return 0;
}
```
- **설명**:
  - `char greeting = 'H';`: 문자 `'H'`를 이용해 인사말을 표현합니다.
  - `char userName[] = "Alice";`: 이름을 문자 배열로 저장합니다. 문자 배열은 따옴표로 묶어야 합니다.
  - `%c` 포맷 지정자는 단일 문자를, `%s`는 문자열을 출력할 때 사용합니다.

### 💡 초보자 폭풍 질문! 🤔

**질문**: 문자 배열은 왜 따옴표로 묶어야 하나요?

**답변**: 문자 배열은 여러 문자를 포함할 수 있는 공간을 의미합니다. 따라서 배열 내에 들어갈 문자들은 모두 문자열의 일부로 인식되어야 하므로 따옴표로 묶어줘야 합니다. 예를 들어, `"Alice"`는 배열 내에 정확히 네 개의 문자('A', 'l', 'i', 'c', 'e')를 포함하고 있다는 것을 알려줍니다.
