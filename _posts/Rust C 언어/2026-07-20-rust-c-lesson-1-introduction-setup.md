# 1강: Rust C 언어 기초: 소개 및 설치 – 초보자를 위한 짜릿한 여정 시작!

안녕하세요, 미래의 코드 마스터 여러분! 🤖 오늘부터는 Rust와 C 언어의 세계로 여행을 떠나볼 거예요. 이 여정은 단순한 여행이 아니라, 여러분의 프로그래밍 스킬을 한 단계 업그레이드시켜줄 마법의 열쇠를 찾는 모험입니다. 그럼, 첫 번째 강의로 들어가 볼까요?

## Rust & C 언어: 프로그래밍의 양대 산맥

**진짜 신기하죠?**  
Rust와 C 언어는 각각 독특한 매력을 지니고 있어요. C 언어는 컴퓨터의 언어를 이해하는 데 있어 가장 기본적인 문법을 제공하며, Rust는 안전성과 동시성을 극대화하는 현대적인 접근법으로 무장하고 있습니다. 

### Rust 소개: 안전한 시스템 프로그래밍의 왕자

Rust는 2010년에 등장한 이래로 시스템 프로그래밍의 새로운 기준을 세우고 있어요. **"소유권(Ownership)"**과 **"빌림(Borrowing)"**이라는 개념을 통해 메모리 안전성을 보장합니다. 마치 마법처럼, 코드를 작성하는 동안 메모리 누수나 레이스 컨디션(race condition)을 걱정할 필요가 없어요!

**💡 초보자 폭풍 질문!**
- **Q:** 소유권과 빌림이란 무엇인가요?
- **A:** 소유권은 데이터에 대한 완전한 통제권을 의미해요. 데이터가 더 이상 사용되지 않으면 자동으로 메모리에서 해제되죠. 빌림은 데이터를 잠시 빌려 쓰는 개념으로, 여러 부분에서 동시에 접근할 수 있지만 규칙을 지켜야 합니다. 이렇게 하면 코드의 안전성을 극대화할 수 있어요!

### C 언어 소개: 프로그래밍의 조상

C 언어는 1970년대에 등장하여 오늘날까지 여전히 널리 사용되고 있어요. 간결하고 효율적인 문법으로 시스템 레벨 프로그래밍의 여왕으로 불리죠. 운영 체제나 하드웨어와 직접 상호작용할 때 필수적인 언어예요.

**🚨 실무주의보**
- **C 언어는 하드웨어에 매우 가까워서 메모리 관리에 신경 써야 합니다.** 실수로 인한 메모리 누수나 버퍼 오버플로우는 큰 문제를 일으킬 수 있어요. Rust의 소유권 개념이 이런 문제를 예방하는 데 도움이 될 거예요!

## Rust 설치하기: 첫걸음 내딛기

### Rust 설치 단계

1. **Rust 공식 웹사이트 방문**
   - [Rust 공식 웹사이트](https://www.rust-lang.org/tools/install)에서 최신 버전의 설치 가이드를 확인하세요.

2. **명령 프롬프트 또는 터미널 열기**
   - 윈도우 사용자는 **명령 프롬프트**, macOS나 Linux 사용자는 **터미널**을 열어주세요.

3. **Rust 설치 명령어 실행**
   ```bash
   # 윈도우
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```
   ```bash
   # macOS/Linux
   curl https://sh.rustup.org -sSf | sh
   ```
   - 이 명령어는 Rust와 Cargo (Rust의 패키지 관리자)를 설치합니다. 설치가 완료되면 `rustup` 명령어를 통해 Rust 버전을 관리할 수 있어요.

4. **Rust 버전 확인**
   ```bash
   rustc --version
   ```
   - 정상적으로 설치되었다면 현재 설치된 Rust의 버전이 출력될 거예요! 🎉

### C 언어 설치: 기본 환경 설정

C 언어는 대부분의 개발자 환경에서 기본적으로 설치되어 있지만, 최신 컴파일러를 원한다면 다음과 같이 설치할 수 있어요.

1. **GCC 설치 (Linux/macOS)**
   ```bash
   sudo apt-get update  # Ubuntu/Debian
   sudo apt-get install gcc g++  # GCC 설치
   ```
   ```bash
   brew install gcc  # macOS에서 Homebrew 사용 시
   ```

2. **Visual Studio C++ Build Tools (Windows)**
   - Visual Studio 공식 웹사이트에서 **C++ 빌드 도구**를 다운로드하고 설치하세요.

**💡 초보자 폭풍 질문!**
- **Q:** 왜 C 언어를 설치해야 하나요?
- **A:** C 언어는 시스템 레벨 프로그래밍의 기초를 다지는 데 필수적이에요. 특히 하드웨어와 밀접하게 연관된 프로젝트나 성능 최적화에 매우 유용하죠. Rust를 배우면서 C 언어의 기본 개념을 탄탄히 다질 수 있어요!

## 실용적인 코드 예제: Rust와 C 언어 입문

### Rust 예제 1: 기본 프로그램 출력
```rust
fn main() {
    println!("안녕하세요, Rust 세계에 오신 것을 환영합니다!"); // 첫 메시지 출력

    // 반복문 예시: 5번 반복하며 숫자 출력
    for i in 1..=5 {  // 1부터 5까지 반복
        println!("현재 숫자: {}", i);  // 각 숫자 출력
    }
}
```
- **설명:** `fn main()`은 프로그램의 시작점입니다. `println!` 매크로는 텍스트를 출력하는 함수예요. `for` 반복문은 간단하게 숫자를 출력하는 예시입니다. `1..=5`는 1부터 5까지의 범위를 의미합니다.

### Rust 예제 2: 조건문 활용
```rust
fn main() {
    let age = 20;  // 변수 선언 및 초기화

    if age >= 18 {  // 조건문: 나이가 18세 이상인지 확인
        println!("성인이시네요!");  // 조건 만족 시 메시지 출력
    } else {
        println!("아직 미성년자입니다.");  // 조건 불만족 시 메시지 출력
    }
}
```
- **설명:** `if` 문은 조건을 평가하고 해당 조건에 따라 다른 코드 블록을 실행합니다. `else`는 조건이 만족되지 않을 때 실행되는 부분입니다.

### C 언어 예제 1: 기본 프로그램 출력
```c
#include <stdio.h>  // 표준 입출력 라이브러리 포함

int main() {
    printf("안녕하세요, C 언어의 세계로 오신 것을 환영합니다!\n");  // 문자열 출력

    // 반복문 예시: 5번 반복하며 숫자 출력
    for (int i = 1; i <= 5; i++) {  // for 루프: 변수 i를 1부터 5까지 증가시키며 반복
        printf("현재 숫자: %d\n", i);  // 각 숫자 출력
    }

    return 0;  // 프로그램 종료 표시
}
```
- **설명:** `#include <stdio.h>`는 입출력 함수를 사용할 수 있게 해줍니다. `printf`는 문자열을 출력하는 함수입니다. `for` 루프는 C에서도 비슷하게 작동하지만, 구조가 약간 다를 수 있어요.

### C 언어 예제 2: 조건문 활용
```c
#include <stdio.h>

int main() {
    int age = 20;  // 변수 선언 및 초기화

    if (age >= 18) {  // 조건문: 나이가 18세 이상인지 확인
        printf("성인이시네요!\n");  // 조건 만족 시 메시지 출력
    } else {
        printf("아직 미성년자입니다.\n");  // 조건 불만족 시 메시지 출력
    }

    return 0;  // 프로그램 종료 표시
}
```
- **설명:** C 언어에서도 `if` 문과 `else`는 조건에 따라 다른 동작을 수행하는 데 사용됩니다. `printf` 함수는 Rust의 `println!`과 유사하게 문자열을 출력합니다.
