## 🚀 4강: C 언어 기본 문법 리뷰 - 제어 구조: 당신의 코드를 조종하는 마법의 힘!

안녕하세요, 코딩의 모험가 여러분! 오늘은 우리 함께 **제어 구조**의 세계로 날아가볼게요. 🤸‍♀️ 생각보다 복잡하지 않답니다! 오히려 이 개념을 이해하면, 당신의 코드가 마치 살아 숨 쉬는 생물처럼 움직이게 될 거예요. 준비되셨나요? **진짜 신기하죠?**

### 🤔 제어 구조란 무엇인가요?

제어 구조는 프로그램이 특정 순서대로 실행되도록 지시하는 핵심적인 부분입니다. 쉽게 말해, 컴퓨터에게 "이거 했다가, 저거 했다가, 아니면 이런 상황에서는 저거 해!"라고 명령하는 거죠. 🧙‍♂️ 마치 영화 감독이 영화의 각 장면을 지시하는 것과 비슷해요.

#### 핵심 요소들

1. **조건문 (Conditional Statements)**: 상황에 따라 다른 동작을 수행하게 만듭니다.
2. **반복문 (Loop Statements)**: 특정 코드 블록을 여러 번 반복 실행합니다.

### 🧩 조건문: 만약 문을 열다

#### 1. **if 문**

```c
#include <stdio.h>

int main() {
    int score = 85;  // 학생의 점수

    if (score >= 90) {
        printf("합격! 당신은 최고의 학생이에요.\n");  // 점수가 90점 이상이면
    } else if (score >= 70) {  // 점수가 70점 이상이면
        printf("합격! 조금 더 노력하면 최고야.\n");
    } else {
        printf("개선 필요! 열심히 공부해봐요.\n");  // 그 외의 경우
    }

    return 0;
}
```

**코드 설명:**
- `if (score >= 90)`: 점수가 90점 이상이면 첫 번째 메시지 출력.
- `else if (score >= 70)`: 만약 첫 조건이 아니라면, 점수가 70점 이상이면 두 번째 메시지 출력.
- `else`: 그 외의 경우, 즉 점수가 70점 미만이면 마지막 메시지 출력.

**💡 초보자 폭풍 질문!**
- **Q:** `else if`는 왜 필요한 건가요?
- **A:** `else if`는 첫 번째 조건이 거짓일 때 추가적인 조건을 검사할 수 있게 해줍니다. 여러 조건을 순차적으로 확인할 수 있어요!

#### 2. **if-else 문**

```c
#include <stdio.h>

int main() {
    int age = 22;  // 사용자의 나이

    if (age >= 18) {
        printf("성인이네요!\n");  // 나이가 18세 이상이면
    } else {
        printf("성인이 되려면 조금 더 기다리세요!\n");  // 그 외의 경우
    }

    return 0;
}
```

**코드 설명:**
- `if (age >= 18)`: 나이가 18세 이상이면 "성인이네요!" 출력.
- `else`: 나이가 18세 미만이면 "성인이 되려면 조금 더 기다리세요!" 출력.

#### 3. **switch 문**

```c
#include <stdio.h>

int main() {
    char grade = 'B';  // 학생의 성적 등급

    switch (grade) {
        case 'A':
            printf("장원상!\n");  // A 등급일 때
            break;
        case 'B':
            printf("우수상!\n");  // B 등급일 때
            break;
        case 'C':
            printf("참여상!\n");  // C 등급일 때
            break;
        default:
            printf("성적이 등록되지 않았어요.\n");  // 다른 등급일 때
            break;
    }

    return 0;
}
```

**코드 설명:**
- `switch (grade)`: `grade` 변수의 값에 따라 다양한 케이스를 검사합니다.
- `case 'A'`: `grade`가 'A'일 때 해당 메시지 출력.
- `break`: 각 케이스가 끝나면 `break`로 실행을 중지시켜 다음 케이스로 넘어가지 않게 합니다.
- `default`: 모든 케이스를 통과해도 해당 메시지가 출력됩니다.

### 🏃‍♂️ 반복문: 무한 루프와의 싸움

#### 1. **for 문**

```c
#include <stdio.h>

int main() {
    for (int i = 0; i < 5; i++) {  // 0부터 4까지 반복
        printf("반복 횟수: %d\n", i);
    }

    return 0;
}
```

**코드 설명:**
- `for (int i = 0; i < 5; i++)`: 초기값 `i = 0`, 조건 `i < 5`, 증가식 `i++`를 사용하여 반복합니다.
- `printf("반복 횟수: %d\n", i);`: 각 반복마다 반복 횟수를 출력합니다.

#### 2. **while 문**

```c
#include <stdio.h>

int main() {
    int count = 0;  // 초기 카운트 값

    while (count < 3) {  // 카운트가 3 미만일 때까지 반복
        printf("카운트: %d\n", count);
        count++;  // 카운트를 증가시켜 반복 조건을 바꿈
    }

    return 0;
}
```

**코드 설명:**
- `while (count < 3)`: 카운트가 3 미만일 때까지 반복 실행.
- `printf("카운트: %d\n", count);`: 각 반복마다 현재 카운트를 출력합니다.
- `count++`: 반복마다 카운트를 증가시켜 반복 조건을 업데이트합니다.

#### 3. **do-while 문**

```c
#include <stdio.h>

int main() {
    int guess = 3;  // 초기 추측값
    int number = 7;  // 정답값

    do {
        printf("숫자를 맞춰보세요: %d\n", guess);
        // 사용자 입력 부분은 생략
        guess++;  // 추측값을 증가시켜 반복
    } while (guess <= number);  // 추측값이 정답보다 작거나 같은 동안 반복

    return 0;
}
```

**코드 설명:**
- `do { ... } while (guess <= number);`: 조건이 참인 동안 반복을 보장합니다.
- `printf("숫자를 맞춰보세요: %d\n", guess);`: 추측값을 기반으로 사용자에게 질문합니다.
- `guess++`: 추측값을 증가시켜 다음 반복을 준비합니다.
