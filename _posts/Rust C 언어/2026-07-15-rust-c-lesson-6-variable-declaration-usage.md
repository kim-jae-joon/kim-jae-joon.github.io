## **1. 변수란 무엇인가요?**

**변수**는 간단히 말해 데이터를 저장하는 컨테이너 같은 존재입니다. 프로그래밍에서는 이 컨테이너에 숫자, 문자열, 부울 값 등 다양한 정보를 담아둘 수 있어요. Rust에서 변수를 선언하는 순간, 그 이름 아래에 데이터를 담아둘 공간이 생깁니다.

**왜 변수가 필요할까요?**
- **데이터 관리**: 코드에서 사용할 데이터를 추적하고 관리할 수 있어요.
- **유연성**: 프로그램이 실행되는 동안 값을 쉽게 변경할 수 있어요.

### **2. 변수 선언하기**

Rust에서 변수를 선언하는 기본 문법은 다음과 같습니다:

```rust
// 1. 기본 타입 변수 선언 (i32는 32비트 정수를 의미)
let my_number: i32 = 42;

// 2. 문자열 변수 선언
let my_name: String = String::from("Rust Master");

// 3. 불리언 변수 선언
let is_morning: bool = true;
```

#### **코드 분석**

- **`let` 키워드**: `let`은 Rust에서 변수를 선언할 때 사용하는 키워드입니다. 이 키워드가 등장하면 "새로운 변수가 탄생!"이라고 생각하면 돼요.
  
  - **`let my_number: i32 = 42;`**
    - `my_number`: 변수의 이름입니다.
    - `: i32`: 변수가 어떤 타입의 데이터를 저장할지 지정합니다. 여기서는 32비트 정수 타입입니다.
    - `= 42`: 초기값을 할당합니다. 이 코드는 `my_number` 변수에 42를 저장합니다.

  - **`let my_name: String = String::from("Rust Master");`**
    - `String` 타입은 Rust에서 문자열을 다루는 데 주로 사용됩니다. `String::from()`은 문자열 리터럴을 `String` 타입으로 변환합니다.
    
  - **`let is_morning: bool = true;`**
    - `bool` 타입은 참(`true`) 또는 거짓(`false`) 값을 저장합니다. 여기서는 아침이라는 것을 나타내기 위해 `true`로 설정했습니다.

### **3. 변수 타입과 초기화**

#### **타입 추론 (Type Inference)**
Rust는 매우 똑똑한 언어예요. 타입을 명시하지 않아도 컴파일러가 자동으로 타입을 추론할 수 있습니다. 예를 들어:

```rust
let num = 10; // 컴파일러가 `num`이 i32 타입임을 자동으로 결정합니다.
let text = "Hello Rust!"; // 컴파일러가 `text`가 String 타입임을 알아냅니다.
```

#### **다양한 타입 예제**

- **정수 타입 (`i32`, `u32`, `f32`)**
  ```rust
  let age: i32 = 27; // 32비트 부호 있는 정수
  let score: u32 = 95; // 32비트 부호 없는 정수
  let pi: f32 = 3.14159; // 32비트 부동소수점 숫자
  ```

- **문자열 타입 (`String`, `&str`)**
  ```rust
  let greeting: String = String::from("안녕하세요!"); // 소유권을 가진 문자열
  let greeting_slice: &str = "안녕하세요!"; // 참조 문자열 (가변 길이)
  ```

- **불리언 타입**
  ```rust
  let is_active: bool = true; // 활성화 상태
  ```

### **4. 변수 사용하기**

변수를 선언한 후에는 다양한 연산과 기능을 통해 데이터를 조작할 수 있습니다. 몇 가지 예를 들어볼게요!

#### **4.1 숫자 연산**

```rust
let num1: i32 = 10;
let num2: i32 = 5;

let sum: i32 = num1 + num2; // 덧셈
let diff: i32 = num1 - num2; // 뺄셈
let product: i32 = num1 * num2; // 곱셈
let quotient: f64 = num1 as f64 / num2 as f64; // 나눗셈 (실수 타입으로 변환)

println!("합: {}", sum);
println!("차: {}", diff);
println!("곱: {}", product);
println!("몫: {}", quotient);
```

**코드 분석**
- `num1 + num2`: 두 숫자를 더합니다.
- `num1 * num2`: 두 숫자를 곱합니다.
- `num1 as f64 / num2 as f64`: 나눗셈을 수행할 때 실수 타입으로 변환하여 정확도를 높입니다.

#### **4.2 문자열 조작**

```rust
let greeting: String = String::from("Hello, ");
let name: String = String::from("Alice");

let full_greeting = greeting.clone() + name; // 문자열 연결

println!("{}", full_greeting); // 출력: Hello, Alice
```

**코드 분석**
- `greeting.clone()`: `greeting` 문자열의 복사본을 만듭니다.
- `+ name`: 두 문자열을 연결합니다.

### **💡 초보자 폭풍 질문!**

**Q: 변수 타입을 명시하지 않아도 되나요?**
**A:** 네, Rust는 타입 추론을 지원하므로 명시하지 않아도 컴파일러가 자동으로 타입을 결정합니다. 하지만 명확한 타입 지정은 코드의 가독성을 높이고 오류를 줄여줍니다. 예를 들어, `let value = 10;`와 `let value: i32 = 10;`는 동일한 결과를 내지만, 후자가 더 명확합니다.

### **🚨 실무주의보**

**주의사항**: 변수 이름은 의미 있는 것으로 지정하는 것이 중요합니다. 예를 들어, `x`나 `a`와 같은 애매한 이름 대신 `userAge`나 `customerName`처럼 코드의 맥락을 반영하는 이름을 사용하세요. 이렇게 하면 코드를 유지보수하거나 다른 개발자와 협업할 때 훨씬 이해하기 쉬워집니다.
