## 12강: Rust 벡터와 스트링, 코딩 세계의 탐험대가 되어보자!

안녕하세요, 코딩의 모험가 여러분! 오늘은 **Rust 벡터와 스트링**이라는 신비로운 영역으로 떠나볼 거예요. 이 둘은 마치 컴퓨터의 '마법의 배낭'과 '비밀 일기' 같아요. 벡터는 여러 아이템을 넣고 빼는 데, 스트링은 우리가 쓰는 모든 이야기를 담는 그릇이죠. 초보자분들껜 다소 생소할 수 있지만, 걱정 마세요! 지금부터 여러분을 Rust 세계의 주인공으로 만들어드릴게요.

### 벡터: 마법의 배낭 탐험

#### 개념 이해하기
**벡터(Vector)**는 Rust에서 동적 크기 배열을 의미해요. 쉽게 말해, 필요에 따라 아이템을 추가하거나 제거할 수 있는 마법의 배낭 같은 거죠. 이 배낭은 항상 무엇이 들어 있는지 알려주는 능력도 있어요.

**예제 1: 벡터 생성 및 아이템 추가**
```rust
fn main() {
    // 벡터 생성 (빈 배낭)
    let mut numbers = Vec::new(); // 빈 벡터 생성, "벡터의 시작은 빈 배낭"

    // 아이템 추가 (배낭에 물건 넣기)
    numbers.push(10); // 배낭에 10 추가
    numbers.push(20); // 배낭에 20 추가
    numbers.push(30); // 배낭에 30 추가

    // 현재 배낭 상태 확인
    println!("배낭에 담긴 아이템들: {:?}", numbers); // 출력: 배낭에 담긴 아이템들: [10, 20, 30]

    // 인덱스를 이용한 접근 (특정 위치의 아이템 찾기)
    let first_item = &numbers[0]; // 첫 번째 아이템 가져오기 (0번 인덱스)
    println!("첫 번째 아이템: {}", first_item); // 출력: 첫 번째 아이템: 10
}
```

**코드 설명:**
- `Vec::new()`로 빈 벡터를 생성합니다. 이는 마법의 배낭을 처음 여는 순간이죠!
- `.push()` 메서드는 배낭에 아이템을 하나씩 넣는 역할을 합니다. 예를 들어, 숫자를 추가할 때마다 배낭에 새로운 아이템이 들어가는 모습을 상상해보세요.
- `numbers[0]`은 배낭의 첫 번째 아이템에 접근하는 방법입니다. 인덱스는 0부터 시작하므로 첫 번째 아이템은 항상 0번 위치에 있습니다.

#### 다양한 반복 방법
벡터를 다룰 때 다양한 반복 방법을 알고 있으면 훨씬 강력해집니다.

**예제 2: 반복문 활용**
```rust
fn main() {
    let mut numbers = vec![1, 2, 3, 4, 5]; // 초기화된 벡터

    // for 루프를 이용한 반복
    for number in &numbers { // 참조를 통해 안전하게 반복
        println!("현재 아이템: {}", number);
    }

    // while 루프를 이용한 반복
    let mut index = 0;
    while index < numbers.len() { // 벡터 길이까지 반복
        println!("반복 인덱스 {}: {}", index, numbers[index]);
        index += 1;
    }

    // Iterator 사용
    for number in numbers.iter() { // Iterator를 이용한 반복
        println!("Iterator로 본 아이템: {}", number);
    }
}
```

**코드 설명:**
- **for 루프**: `for number in &numbers`는 벡터의 각 요소를 안전하게 순회할 수 있게 해줍니다. 참조를 사용하면 원본 데이터를 변경할 때 안전합니다.
- **while 루프**: `while index < numbers.len()`은 인덱스를 직접 조작하며 벡터를 순회합니다.
- **Iterator**: `numbers.iter()`는 유연한 반복 방법으로, 다양한 상황에서 유용합니다.

### 스트링: 우리 이야기의 일기장

#### 개념 이해하기
**스트링(String)**은 텍스트 데이터를 저장하고 다루는 데 사용되는 Rust의 핵심 요소입니다. 스트링은 마치 우리의 개인 일기와 같아요. 어떤 이야기든 기록하고 수정하며, 필요할 때 꺼내 읽을 수 있어요.

**예제 3: 스트링 생성 및 조작**
```rust
fn main() {
    // 기본 스트링 생성
    let greeting = String::from("안녕하세요!"); // 문자열 생성, "일기장에 첫 페이지 쓰기"
    println!("초기 스트링: {}", greeting); // 출력: 초기 스트링: 안녕하세요!

    // 스트링 결합 (일기 내용 추가)
    let mut longer_greeting = greeting.clone(); // 원본 스트링 복사, "일기 내용 복사"
    longer_greeting.push_str(", 만나서 반갑습니다!"); // 추가 텍스트 붙이기
    println!("수정된 스트링: {}", longer_greeting); // 출력: 수정된 스트링: 안녕하세요!, 만나서 반갑습니다!

    // 길이 확인
    println!("스트링 길이: {}", longer_greeting.len()); // 출력: 스트링 길이: 24

    // 슬라이스 사용 (특정 부분 읽기)
    let first_word = &longer_greeting[0..5]; // 첫 번째 단어 읽기
    println!("첫 번째 단어: {}", first_word); // 출력: 첫 번째 단어: 안
}
```

**코드 설명:**
- `String::from("텍스트")`로 새로운 스트링을 생성합니다. 일기에 처음 페이지를 쓰는 것과 같죠!
- `.push_str()`은 기존 스트링 끝에 텍스트를 추가하는 방법입니다. 마치 일기에 새로운 내용을 덧붙여 쓰는 것과 같아요.
- `longer_greeting[0..5]`는 스트링의 특정 부분을 슬라이스하여 읽는 방법입니다. 여기서는 "안"이라는 첫 번째 단어를 추출합니다.
