#### # 구조체: 여러 데이터 타입을 하나로 묶어보자!

구조체는 마치 여러 개의 보석을 하나의 아름다운 목걸이에 결합시키는 것과 같아요. 각각의 보석이 서로 다른 특성을 가지고 있지만, 하나의 목걸이로써 완성도 높은 장식품이 되는 것처럼, 구조체는 다양한 데이터 타입을 하나의 단위로 묶어줍니다.

**기본 구조체 예제:**

```c
#include <stdio.h>

// 구조체 정의: 학생 정보를 묶는 구조체
struct Student {
    char name[50];  // 이름 (문자 배열)
    int age;        // 나이 (정수형)
    float gpa;      // 학점 평균 (실수형)
};

int main() {
    // 구조체 변수 선언
    struct Student alice;

    // 멤버 변수 초기화
    strcpy(alice.name, "Alice");    // 이름 설정
    alice.age = 22;                 // 나이 설정
    alice.gpa = 3.8;               // 학점 평균 설정

    // 구조체 데이터 출력
    printf("학생 이름: %s\n", alice.name);
    printf("나이: %d\n", alice.age);
    printf("학점 평균: %.2f\n", alice.gpa);

    return 0;
}
```

**코드 해부:**
1. **구조체 정의**: `struct Student`에서 각 멤버 변수를 정의합니다. 여기서 `name`은 문자열 배열, `age`는 정수, `gpa`는 실수형 데이터 타입입니다.
2. **구조체 변수 선언**: `struct Student alice;`로 구조체 변수를 선언합니다.
3. **멤버 접근**: `.` 연산자를 사용하여 각 멤버 변수에 값을 할당합니다. 예를 들어, `strcpy(alice.name, "Alice");`는 `alice` 구조체의 `name` 멤버에 "Alice"를 복사합니다.
4. **출력**: `.name`, `.age`, `.gpa`를 통해 구조체의 각 멤버에 접근하여 값을 출력합니다.

**왜 이런 방식이 필요할까?**
- **데이터 그룹화**: 복잡한 데이터를 관리하기 쉽게 묶어줍니다.
- **코드 가독성**: 관련 데이터를 한 곳에 모아놓음으로써 코드의 이해가 쉬워집니다.

#### ## 공용체: 공간 절약의 마법사!

공용체는 마치 마법사의 마법 지팡이 같아요. 여러 데이터 타입을 하나의 메모리 공간에 묶어 놓되, 그 공간은 가장 큰 데이터 타입의 크기만큼만 차지합니다. 덕분에 메모리 사용을 효율적으로 관리할 수 있죠!

**기본 공용체 예제:**

```c
#include <stdio.h>
#include <stdlib.h>

// 공용체 정의: 다양한 타입의 데이터를 저장하는 공용체
union Data {
    int intValue;   // 정수형 데이터
    float floatValue; // 실수형 데이터
    char charValue; // 문자 데이터
};

int main() {
    union Data data;

    // 각 타입의 데이터 할당
    data.intValue = 100;
    printf("정수 값: %d\n", data.intValue);

    data.floatValue = 3.14f;
    printf("실수 값: %.2f\n", data.floatValue);

    data.charValue = 'A';
    printf("문자 값: %c\n", data.charValue);

    // 주의사항: 공용체는 하나의 메모리 공간을 공유하기 때문에 주의 필요!
    // 한 번에 하나의 타입만 사용해야 합니다.

    return 0;
}
```

**코드 해부:**
1. **공용체 정의**: `union Data`에서 여러 타입의 데이터 멤버를 정의합니다. 여기서 `intValue`, `floatValue`, `charValue`는 각각 다른 타입의 데이터를 저장할 수 있습니다.
2. **공용체 변수 선언**: `union Data data;`로 공용체 변수를 선언합니다.
3. **타입별 데이터 할당**: 각 타입별로 공용체 멤버에 값을 할당합니다. 예를 들어, `data.intValue = 100;`은 정수형 데이터를 할당합니다.
4. **출력**: 할당된 값을 출력합니다.

**주의사항:**
- **메모리 충돌 위험**: 공용체는 하나의 메모리 공간을 공유하므로, 한 번에 한 가지 타입의 데이터만 사용해야 합니다. 다른 타입으로 변경하면 이전 데이터가 덮어씌워질 수 있습니다.

**💡 초보자 폭풍 질문!**
- **Q**: 공용체를 사용하면 메모리 누수가 발생할까요?
  - **A**: 공용체 자체로 메모리 누수는 발생하지 않지만, 잘못된 관리로 인해 메모리 누수가 생길 수 있습니다. 특히, 동적 메모리 할당과 함께 사용할 때 주의가 필요합니다. 할당된 메모리는 반드시 해제해야 합니다.
  
- **Q**: 구조체와 공용체의 주요 차이점은 무엇인가요?
  - **A**: 구조체는 여러 데이터 타입을 서로 독립적으로 묶어 메모리 공간을 여러 부분으로 나누어 사용합니다. 반면에 공용체는 하나의 메모리 공간을 여러 타입이 공유하므로, 그 공간은 가장 큰 타입의 크기만큼만 차지합니다. 이로 인해 공용체는 메모리 효율성이 뛰어나지만, 동시에 데이터 타입 간의 충돌 위험이 있습니다.
