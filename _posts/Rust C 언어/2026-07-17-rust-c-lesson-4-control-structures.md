## 4강: 🚀 Rust C 언어 기초: 제어 구조 (if, for, while) - 코딩의 마법 세계로 떠나요!

안녕하세요, 여러분의 코딩 탐험가 🧙‍♂️ **개발왕**입니다! 오늘은 우리가 컴퓨터와 좀 더 똑똑하게 대화하는 방법을 배워볼게요. **제어 구조**라는 마법의 열쇠를 쥐게 되면, 프로그램이 마치 살아 숨 쉬는 존재처럼 움직일 수 있게 됩니다. 

### 🧙‍♂️ **if 문: 선택의 마법사**

**만약에...** 라는 말이 떠오르지 않나요? 바로 그 말이 `if` 문을 만드는 핵심입니다! 이 마법의 문은 특정 조건이 맞을 때만 특정 코드 블록을 실행하게 해줍니다. 마치 미로에서 특정 조건을 만족해야만 다음 방으로 이동할 수 있는 문 같은 거죠.

#### 예제 1: 온도 체크 🌡️
```c
#include <stdio.h>

int main() {
    float temperature = 25.0;  // 현재 온도를 설정해봐요

    // 만약 온도가 30도 이상이면 경고 메시지를 출력합니다.
    if (temperature >= 30) {
        printf("경고! 너무 덥습니다!\n");
    } else {
        printf("현재 온도는 %f 도입니다. 시원합니다!\n", temperature);
    }

    return 0;
}
```
- **설명**:
  - `if (temperature >= 30)`: 온도가 30도 이상인지 확인합니다.
  - 만약 참이면 `printf`로 경고 메시지를 출력합니다.
  - 그렇지 않으면 온도를 알려주는 메시지를 출력합니다.
  - `else`는 `if` 조건이 거짓일 때 실행되는 대안 경로를 제공합니다.

#### 예제 2: 퀴즈 도전! 📚
```c
#include <stdio.h>

int main() {
    int score = 85;  // 퀴즈 점수를 설정해봐요

    // 만약 점수가 80점 이상이면 합격 메시지를 출력합니다.
    if (score >= 80) {
        printf("축하해요! 합격입니다.\n");
    } else {
        printf("다음 기회에 꼭 합격하세요!\n");
    }

    return 0;
}
```
- **설명**:
  - `if (score >= 80)`: 점수가 80점 이상인지 검사합니다.
  - 합격 조건이 맞으면 축하 메시지를 출력합니다.
  - 그렇지 않으면 다음 기회를 위해 노력하라는 메시지를 보여줍니다.

### 💡 초보자 폭풍 질문! 💡
**Q: `if` 문 뒤에 `else if`를 추가할 수 있나요?**  
**A:** 네, 물론이죠! 여러 조건을 순차적으로 검사할 수 있어요. 예를 들어,
```c
if (score >= 90) {
    printf("장학금 수혜자!\n");
} else if (score >= 80) {
    printf("우수 성적!\n");
} else {
    printf("계속 노력하세요!\n");
}
```
이렇게 여러 가지 결과를 처리할 수 있습니다.

### ### **for 문: 반복의 마법사**

`for` 문은 일정 횟수만큼 코드를 반복하게 해주는 반복의 마법사입니다. 마치 레시피를 여러 번 반복하는 것처럼요!

#### 예제 1: 숫자 출력 💫
```c
#include <stdio.h>

int main() {
    int i;  // 반복 변수 i를 선언합니다.

    // 1부터 5까지 숫자를 출력합니다.
    for (i = 1; i <= 5; i++) {  // 초기화 (i=1), 조건 (i<=5), 증가 (i++)
        printf("%d\n", i);
    }

    return 0;
}
```
- **설명**:
  - `for (i = 1; i <= 5; i++)`: 반복 시작 조건 (`i=1`), 종료 조건 (`i<=5`), 그리고 각 반복마다 증가값 (`i++`)을 정의합니다.
  - `printf("%d\n", i);`: `i` 값을 출력합니다.

#### 예제 2: 배열 순회 🎲
```c
#include <stdio.h>

int main() {
    int numbers[] = {10, 20, 30, 40, 50};  // 배열 선언
    int count = sizeof(numbers) / sizeof(numbers[0]);  // 배열 원소 개수 계산

    // 배열의 각 요소를 출력합니다.
    for (int j = 0; j < count; j++) {  // 인덱스 0부터 시작하여 배열 끝까지
        printf("%d\n", numbers[j]);
    }

    return 0;
}
```
- **설명**:
  - `for (int j = 0; j < count; j++)`: 배열의 각 요소를 순회하기 위한 반복 구조입니다.
  - `printf("%d\n", numbers[j])`: 배열의 각 요소를 하나씩 출력합니다.

### ### **while 문: 무한 루프의 마법사**

`while` 문은 특정 조건이 참인 동안 계속 반복하게 해줍니다. 마치 타이머처럼 작동하는 마법의 문이죠!

#### 예제 1: 카운터 게임 🎲
```c
#include <stdio.h>

int main() {
    int count = 0;  // 카운트 변수 초기화

    // 사용자가 5를 입력할 때까지 카운트를 증가시킵니다.
    while (count < 5) {  // 조건: count가 5 미만일 때까지 반복
        printf("몇 번째입니까? ");
        scanf("%d", &count);  // 사용자 입력 받기
        printf("현재 카운트: %d\n", count);
    }

    printf("미션 완료!\n");
    return 0;
}
```
- **설명**:
  - `while (count < 5)`: 조건이 참인 동안 반복합니다.
  - `scanf("%d", &count)`: 사용자로부터 정수를 입력받습니다.
  - 반복이 종료되면 "미션 완료!" 메시지를 출력합니다.

#### 예제 2: 사용자 입력 확인 🧐
```c
#include <stdio.h>

int main() {
    char input;  // 사용자 입력을 저장할 변수

    // 사용자가 'y' 또는 'n'을 입력할 때까지 반복합니다.
    while (1) {  // 무한 루프 시작
        printf("계속하시겠습니까? (y/n): ");
        scanf(" %c", &input);  // 공백을 포함하여 입력 받기

        if (input == 'y' || input == 'Y') {
            printf("계속합니다!\n");
            break;  // 특정 조건 만족 시 탈출
        } else if (input == 'n' || input == 'N') {
            printf("그만둡니다.\n");
            break;  // 중지 조건 만족 시 탈출
        } else {
            printf("잘못된 입력입니다. 다시 입력해주세요.\n");
        }
    }

    return 0;
}
```
- **설명**:
  - `while (1)`: 무한 루프를 시작합니다.
  - 사용자 입력을 받아 조건에 따라 다른 동작을 수행합니다.
  - `break;`: 특정 조건을 만족하면 루프를 종료합니다.

### 🚨 실무주의보 🚨
**실무 팁**:  
- **무한 루프 주의**: `while` 문에서 종료 조건이 명확하지 않으면 무한 루프에 빠질 수 있으니 주의하세요! 항상 탈출 조건을 명확히 설정하는 습관을 들이세요.
- **조건 최적화**: 복잡한 조건이 있을 때는 논리 연산자를 잘 활용해 코드를 간결하게 유지하세요. 예를 들어, `if (condition1 && condition2 || condition3)` 대신 필요한 조건만 포함시켜 가독성을 높이세요.
