## 23강: Rust C 언어 심화 응용: 파일 시스템 작업 - 당신의 코드 슈퍼히어로가 되다!

안녕하세요, 멋진 코딩 여정에 오신 것을 환영합니다! 🎉 오늘은 **Rust C 언어로 파일 시스템을 탐험하는 멋진 여정**을 함께 할 거예요. 마치 고대 유적지 탐험가처럼, 우리는 컴퓨터 속 숨겨진 보물 같은 데이터를 찾아내고 조작할 거랍니다! 😎

**진짜 신기하죠?**  파일 시스템은 컴퓨터 세계의 핵심 인프라죠. 파일을 만들고, 읽고, 수정하고 삭제하는 능력은 프로그램이 현실 세계와 소통하는 데 필수적이에요! 지금부터 당신의 코드 슈퍼히어로 실력을 업그레이드할 준비되셨나요? 🦸‍♀️🦸‍♂️

### 1. 파일 시스템 기초: 파일 다루기의 핵심 도구

파일 작업을 시작하기 전에 몇 가지 기본 개념을 알아보죠.

#### 1.1 파일 경로: 목적지를 정확히 알려줘!

컴퓨터는 주소처럼 파일 경로를 통해 파일을 찾아요. 마치 지도를 보고 목적지를 찾아가듯이요!

* **절대 경로:**  하드 드라이브의 뿌리부터 시작하는 완전한 경로 (예: `/home/user/documents/report.txt`)
* **상대 경로:** 현재 위치에서 시작하는 경로 (예: `../projects/data.csv` - 현재 폴더의 부모 폴더로 한 단계 올라가 'projects' 폴더 안의 'data.csv' 파일)

**💡 초보자 폭풍 질문!**  
*  **Q:** 절대 경로와 상대 경로의 차이점은 무엇인가요?
*  **A:**  절대 경로는 파일 시스템의 시작점부터 정확하게 파일 위치까지 지정하는 반면, 상대 경로는 현재 위치를 기준으로 파일 위치를 나타냅니다. 예를 들어, 프로젝트 폴더 내부에서 실행될 코드에서는 상대 경로가 더 편리합니다.

#### 1.2 파일 모드: 읽기, 쓰기, 수정 권한 조절!

파일을 다룰 때 접근 권한을 명확히 해야 합니다. 마치 열쇠를 이용하는 것과 같아요!

* **읽기 전용 (`std::fs::OpenOptions::read`)**: 파일 내용만 읽을 수 있습니다.
* **쓰기 전용 (`std::fs::OpenOptions::write`)**: 파일 내용을 덮어쓸 수 있습니다.
* **추가 및 쓰기 (`std::fs::OpenOptions::append`)**: 기존 내용 뒤에 추가할 수 있습니다.
* **쓰기 및 읽기 모두 (`std::fs::OpenOptions::read` & `std::fs::OpenOptions::write`)**: 파일을 읽고 수정할 수 있습니다.

### 2. Rust로 파일 읽기: 데이터 탐험 시작!

이제 실제로 Rust 코드로 파일 내용을 읽어보겠습니다! 🏹

**예제 1: 기본 파일 읽기**

```rust
use std::fs::File;
use std::io::{self, Read}; // Read trait 필요!

fn main() -> io::Result<()> {
    // 1. 파일 열기 (예: "example.txt")
    let mut file = File::open("example.txt")?; // 오류 처리 중요!

    // 2. 버퍼 준비 (읽을 공간)
    let mut buffer = String::new(); // 문자열 버퍼 생성

    // 3. 읽기 루프: 바이트를 버퍼에 쌓기
    io::stdin().read_to_string(&mut buffer)?; //  💡 주의: 여기서는 예시로 stdin 사용, 실제 파일 읽기에서는 파일 객체 사용!

    // 4. 내용 출력
    println!("파일 내용:\n{}", buffer);

    Ok(())
}
```

**코드 해설:**

1. **`use std::fs::File; use std::io::{self, Read};`**: 필요한 모듈 가져오기! 파일 작업과 데이터 읽기를 위한 필수 요소죠.
2. **`let mut file = File::open("example.txt")?;`**: "example.txt" 파일을 읽기 전용 모드로 열어요. `?` 연산자는 오류 발생 시 코드 중단을 돕습니다. 안전하게 파일 다루는 습관 들이기!
3. **`let mut buffer = String::new();`**: 파일 내용을 저장할 문자열 버퍼 생성. 마치 컵처럼 데이터를 담을 준비를 하는 거예요!
4. **`io::stdin().read_to_string(&mut buffer)?;`**: **여기서 주의!** 실제 파일 읽기에서는 `file` 객체를 사용해야 합니다. 예시에서는 `stdin`을 사용하여 간단하게 보여드리고 있어요.

**개선된 예제 (파일 직접 읽기):**

```rust
use std::fs::File;
use std::io::{self, Read};

fn main() -> io::Result<()> {
    let mut file = File::open("example.txt")?; // 파일 열기
    let mut buffer = String::new(); // 버퍼 준비

    file.read_to_string(&mut buffer)?; // 파일 내용 읽기 (효율적!)
    println!("파일 내용:\n{}", buffer);

    Ok(())
}
```

**💡 실무주의보:**  `read_to_string` 함수는 파일 내용을 한 번에 읽어들입니다. 대용량 파일 처리 시에는 버퍼링 기법을 활용하는 것이 좋습니다. (다음 강좌에서 다룰게요!)

### 3. 파일 쓰기: 당신의 이야기를 세상에!

이제 당신의 창조적인 아이디어를 파일에 담아보세요! 🎨

**예제 2: 텍스트 파일 쓰기**

```rust
use std::fs::File;
use std::io::{self, Write}; // 쓰기 위해서는 Write trait 필요!

fn main() -> io::Result<()> {
    // 1. 파일 열기 (쓰기 모드로!): "my_story.txt" 생성 또는 덮어쓰기
    let mut file = File::create("my_story.txt")?; 

    // 2. 내용 작성
    writeln!(file, "안녕하세요, 세상! 이것은 내가 쓴 첫 번째 파일입니다.")?; 
    writeln!(file, "당신도 할 수 있어요!")?;

    println!("글이 성공적으로 저장되었습니다!");

    Ok(())
}
```

**코드 해부:**

1. **`File::create("my_story.txt")?;`**: 파일이 이미 존재하면 덮어씁니다. 새로운 파일 생성도 가능해요!
2. **`writeln!(file, "...")?;`**:  `writeln!` 함수는 편리하게 줄 바꿈을 처리해줍니다. 파일에 직접 문자열을 쓸 때 유용하죠!

### 4. 디렉토리 탐색: 파일의 가족사진 앨범을 살펴보자!

컴퓨터 내부 파일 구조를 탐색하는 방법도 알아봅시다. 마치 가족 앨범을 펼치는 것처럼요!

**예제 3: 디렉토리 내 파일 목록 가져오기**

```rust
use std::fs::{self, ReadDir};

fn main() {
    // 1. 디렉토리 열기 (예: "." 현재 디렉토리)
    let entries = fs::read_dir(".").expect("디렉토리 읽기 오류"); 

    // 2. 각 엔트리 순회
    for entry in entries {
        // 에러 처리 필요!
        let entry = entry?; 
        let path = entry.path();

        // 3. 파일인지 확인 후 출력
        if path.is_file() {
            println!("파일: {}", path.display());
        } else if path.is_dir() {
            println!("디렉토리: {}", path.display());
        }
    }
}
```

**코드 깊이 파악:**

1. **`fs::read_dir(".").expect("...")`**: 현재 디렉토리의 모든 엔트리 (파일 및 폴더)를 가져옵니다. `expect`는 오류 처리를 간편하게 해주는 부분이에요.
2. **`for entry in entries`**: 각 엔트리를 순회하며 정보를 얻습니다.
3. **`entry.path()`**: 엔트리의 경로 정보를 가져옵니다.
4. **`path.is_file()`, `path.is_dir()`**: 파일인지 디렉토리인지 판별하여 구분해서 출력합니다.

**다양한 반복문 활용 예시:**

파일 작업에서 반복문은 매우 유용합니다.

* **`for` 루프:**  위 예제처럼 디렉토리 내 모든 엔트리를 순차적으로 처리할 때 적합합니다.
* **`while` 루프:**  특정 조건이 만족될 때까지 계속 파일을 처리할 때 유용합니다. (예: 파일 크기 조건에 따른 처리)
* **`do-while` 루프:**  최소 한 번 실행 후 조건을 확인하는 경우에 사용합니다. Rust에는 내장되어 있지 않지만, `while` 루프를 활용해 구현 가능합니다.
