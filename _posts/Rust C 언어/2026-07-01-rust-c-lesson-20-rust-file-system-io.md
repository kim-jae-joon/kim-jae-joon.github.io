### 🌟 파일 시스템 탐험: 기본 개념 알아보기

파일 시스템은 컴퓨터의 디지털 옷장이라고 생각하면 됩니다. 각각의 파일은 옷가지처럼 특정 정보를 담고 있고, 디렉토리(폴더)는 그 옷들을 깔끔하게 분류해주는 서랍들과 같아요. Rust에서는 이 옷장을 멋지게 관리할 수 있는 강력한 도구들을 제공합니다.

#### 📜 핵심 개념

- **파일 (File):** 데이터를 저장하는 단위 (텍스트, 이미지, 바이너리 데이터 등)
- **디렉토리 (Directory)/폴더 (Folder):** 파일들을 구조적으로 정리하는 범주
- **경로 (Path):** 파일이나 디렉토리의 위치를 지정하는 주소 (예: `/home/user/documents/myfile.txt`)

### 💻 Rust로 파일 읽기: 마법의 책 읽기 🪄

파일 읽기는 마치 오래된 도서관에서 원하는 책을 찾아내고 그 내용을 읽는 것과 같습니다. Rust에서는 `std::fs` 모듈을 통해 이 마법을 펼칠 수 있어요!

#### 예제 1: 텍스트 파일 읽기

```rust
use std::fs::File;
use std::io::{self, Read};

fn main() -> io::Result<()> {
    // 1. 마법의 책 (파일)을 열어보자!
    let mut file = File::open("example.txt")?; // "example.txt"는 읽으려는 파일 경로

    // 2. 빈 그릇 (벡터)을 준비해 내용을 담을 준비 완료!
    let mut contents = String::new();

    // 3. 파일에서 내용을 빨아들이는 마법 주문!
    file.read_to_string(&mut contents)?; // 내용을 String으로 읽어들임

    // 4. 마법의 결과를 화면에 출력!
    println!("파일 내용: {}", contents);

    Ok(())
}
```

**코드 해설:**

- **`File::open("example.txt")?`**: `example.txt` 파일을 읽기 모드로 열어요. 에러 발생 시 '?' 연산자가 에러를 처리해요.
- **`read_to_string(&mut contents)?`**: 파일의 내용을 `String` 타입으로 읽어들입니다. `&mut`은 변경 가능한 참조를 의미해요.
- **`println!`**: 읽어들인 내용을 화면에 출력합니다.

#### 실전 연습: 다양한 읽기 방법 🎯

1. **`read()` 메서드 사용:** 바이너리 파일 읽기
   ```rust
   let mut buffer = vec![0; 100]; // 100바이트 버퍼 생성
   file.read(&mut buffer)?; // 버퍼에 데이터 읽기
   ```

2. **반복문 활용:** 큰 파일의 부분적 읽기
   ```rust
   let mut buffer = Vec::new();
   let chunk_size = 1024; // 1KB 단위로 읽기
   while let Ok(n) = file.read(&mut buffer) {
       if n == 0 { break; } // 파일 끝 도달 시 종료
       // 버퍼 내용 처리
       buffer.clear(); // 버퍼 초기화
   }
   ```

### 📄 파일 쓰기: 마법의 붓으로 그림 그리기 🎨

파일 쓰기는 마치 창의적인 붓놀림으로 캔버스에 그림을 그리는 것과 같습니다. Rust에서는 `write()` 함수를 통해 이 마법을 실현할 수 있어요!

#### 예제 2: 텍스트 파일 쓰기

```rust
use std::fs::File;
use std::io::{self, Write};

fn main() -> io::Result<()> {
    // 1. 마법의 캔버스 (파일)을 새로 열거나 기존 파일에 쓰기 모드로 열기
    let mut file = File::open("output.txt")?; // 이미 있는 파일을 업데이트하거나 새 파일 생성
    // 아니면 새로 쓰려면 File::create() 사용

    // 2. 붓 (내용) 준비!
    let text = "안녕하세요, Rust 파일 시스템 마스터입니다!";

    // 3. 붓으로 캔버스에 그림 그리기 (내용 쓰기)
    file.write_all(text.as_bytes())?; // 바이트로 쓰기

    // 4. 완성된 작품 감상!
    println!("내용 작성 완료!");

    Ok(())
}
```

**코드 해설:**

- **`File::open("output.txt")?`**: 파일을 쓰기 모드로 열어요. 이미 존재하는 경우 내용을 덮어씁니다.
- **`write_all(text.as_bytes())?`**: `text` 문자열을 바이트로 변환하여 파일에 씁니다.

#### 실전 연습: 다양한 쓰기 방법 🎯

1. **`write()` 메서드 사용:** 바이트 단위 쓰기
   ```rust
   file.write(b"특수 문자: \x01")?; // 바이트 직접 쓰기
   ```

2. **반복문 활용:** 큰 데이터 쓰기
   ```rust
   let data_chunks: Vec<&[u8]> = vec![b"데이터 조각 1", b"데이터 조각 2"];
   for chunk in data_chunks {
       file.write(chunk)?;
   }
   ```

### 💡 초보자 폭풍 질문! 🚀

**Q:** 파일을 열 때 항상 `?` 연산자를 사용해야 하나요?

**A:** `?` 연산자는 에러 처리를 간편하게 해줍니다. 파일 열기 시 에러가 발생하면 즉시 처리할 수 있어요. 하지만 에러 처리 로직을 직접 구현하는 것도 가능합니다. 상황에 맞게 선택하면 돼요!

### 🚨 실무 주의보 🛡️

- **파일 경로 관리:** 플랫폼에 따라 경로 형식이 다를 수 있으니 주의하세요 (`/` vs `\`). `std::env::current_dir()`를 사용해 현재 디렉토리를 쉽게 얻을 수 있어요.
- **동시성:** 여러 스레드에서 동일한 파일을 동시에 접근할 때는 락(Lock) 메커니즘을 사용해야 데이터 일관성을 유지할 수 있습니다.
