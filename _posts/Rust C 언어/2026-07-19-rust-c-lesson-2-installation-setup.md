## 2강: Rust 설치 마스터하기! 🦀 설치부터 기본 세팅까지, 초보자도 따라하면 고수가 될 거예요!

안녕하세요! **Rust 여정의 빛나는 첫걸음**을 함께 걸을 준비가 되셨나요? 🤝  이제 막 Rust 세계에 발을 들여놓으신 여러분을 위해, 이번 강의에서는 마치 **'Rust 설치 마법사'**가 되어 드릴게요. 쉽고 재미있게 따라 하다 보면, 이 낯선 문법들이 어느새 익숙한 친구처럼 느껴질 거예요!

**🔥 설치, 왜 이렇게 중요할까요?**

Rust를 배우는 데 있어 가장 기본적이면서도 중요한 단계는 바로 **Rust 설치**입니다. 마치 멋진 레고 세트를 조립하기 전에 모든 부품을 제대로 준비하는 것과 같죠! 제대로 설치되지 않으면, 아름다운 프로그램을 만드는 과정에서 예상치 못한 블록이 나타날 수 있답니다. 😅

**# 1단계: 공식 설치 가이드로 Rust 세계에 입장하기**

Rust 공식 웹사이트는 마치 **'설치 지도'** 같아요! 🤗  **[Rust 공식 설치 가이드](https://www.rust-lang.org/tools/install)** 에서 제공하는 단계를 차근차근 따라 하면 됩니다. 걱정 마세요, 복잡하지 않아요!

**- **Windows 사용자:**

  1. **바이너리를 다운로드**: 웹사이트에서 현재 버전의 Rust 설치 파일을 선택하세요. 운영 체제에 맞게 다운로드해야 합니다 (일반적으로 `.exe` 파일).
  2. **런처 실행**: 다운로드한 `.exe` 파일을 실행하고 화면 안내에 따라 설치를 완료하세요. 대부분의 경우 추가 설정 없이 진행해도 괜찮습니다.
  3. **확인!**: 설치가 완료되면 명령 프롬프트나 터미널에서 `rustc --version`을 입력하고 엔터를 누르세요. 정상적으로 설치되었다면 현재 Rust 버전이 출력됩니다! 🎉

**- **macOS 사용자:**

  1. **Homebrew 활용**: 이미 Homebrew가 설치되어 있다면, 터미널에서 다음 명령어를 입력하세요:
     ```bash
     brew install rustup
     ```
     Homebrew가 없다면 먼저 설치해야 합니다: `brew install brew`
  2. **Rustup 설치**: 명령을 실행한 후, `rustup-init` 명령어를 입력하여 Rust 도구 모음(`rustup`)을 설정합니다. 기본 설정을 선택하면 대부분의 경우 문제 없이 진행됩니다.
  3. **설치 완료 확인**: 터미널에서 `rustc --version`을 실행하고 결과를 확인하세요! Rust가 제대로 설치되었음을 확인할 수 있습니다.

**- **Linux 사용자:**

  1. **터미널 열기**: 터미널을 실행하고 아래 명령어를 차례대로 입력하세요:
     ```bash
     curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
     ```
     이 스크립트는 `rustup`을 설치하고 환경 변수를 설정합니다.
  2. **설치 완료**: 스크립트 실행 후, `source $HOME/.cargo/env` 명령어를 실행하여 환경 변수를 활성화하세요.
  3. **확인**: `rustc --version` 명령어를 통해 설치 성공 여부를 확인하세요!

**# 2단계: Cargo 설정하기 - Rust의 '똑똑한 비서'**

Cargo는 Rust 프로젝트를 관리하는 데 필수적인 도구입니다. 마치 당신의 개인 비서처럼 프로젝트 빌드, 테스트, 배포까지 도와줄 거예요!

**Cargo 기본 설정 예시:**

```bash
# Cargo 설정 파일 생성
cargo config --list

# 특정 설정 변경 (예: 컴파일러 최적화 수준)
cargo config set profile nightly

# 프로젝트 생성 (새로운 Rust 프로젝트 시작!)
cargo new my_first_rust_project --bin
cd my_first_rust_project
```

**- **`cargo config --list`**: 현재 설정된 Cargo 설정을 확인합니다. 마치 비서에게 현재 일정을 물어보는 것과 같죠!

**- **`cargo config set profile nightly`**: 여기서는 **`nightly`** 프로파일을 활성화합니다. 이 프로파일은 최신 Rust 언어 기능을 사용할 수 있게 해줍니다. **개발 중인 기능을 빠르게 실험하고 싶다면** 이 옵션을 선택하는 게 좋습니다!

**- **`cargo new my_first_rust_project --bin`**: 이 명령어는 새로운 실행 파일 프로젝트를 생성합니다. `--bin` 옵션을 사용하면 기본 실행 파일을 만들어줍니다. 마치 새로운 오피스 파일을 만드는 것과 같아요!

**💡 초보자 폭풍 질문!**

* **Q: Cargo 설정을 왜 해야 하나요?**
  * **A:** Cargo 설정은 프로젝트의 빌드 환경을 일관되게 관리해줍니다. 컴파일러 옵션, 의존성 관리 등을 자동화하여 개발 과정을 훨씬 편리하게 만들어줍니다. 마치 프로젝트 관리 앱 설정처럼요!

**# 3단계: 기본 Rust 코드로 첫 걸음 다지기**

이제 실제로 Rust 코드를 작성해보며 설치와 설정이 잘 되었는지 확인해봅시다!

**예제 1: Hello, World! 출력하기**

```rust
// `main` 함수는 프로그램의 시작점입니다. 마치 영화 시작 전에 나오는 크레딧처럼요!
fn main() {
    // `println!` 매크로는 화면에 텍스트를 출력합니다. 마치 큰 스피커로 메시지를 외치는 것 같죠!
    println!("Hello, Rust 세계에 오신 것을 환영합니다!");
}
```

**코드 설명:**
1. `fn main() { ... }`: 프로그램의 진입점을 정의합니다.
2. `println!("문자열");`: 텍스트를 콘솔에 출력하는 함수입니다. `!` 마크는 매크로 호출을 의미합니다.

**예제 2: 숫자 더하기**

```rust
fn main() {
    // 변수 `a`와 `b`에 숫자를 할당합니다. 마치 주머니에 동전을 넣는 것 같죠!
    let a = 5;
    let b = 10;

    // `+` 연산자로 두 숫자를 더합니다. 마치 동전을 합쳐 보는 것 같아요!
    let sum = a + b;

    // 결과를 출력합니다. 마치 결과를 친구에게 알려주는 것 같죠!
    println!("{} + {} = {}", a, b, sum);
}
```

**코드 설명:**
1. `let a = 5;`: 변수 `a`에 값 `5`를 할당합니다.
2. `let b = 10;`: 변수 `b`에 값 `10`을 할당합니다.
3. `let sum = a + b;`: 두 변수를 더해 새로운 변수 `sum`에 저장합니다.
4. `println!("{} + {} = {}", a, b, sum);`: 결과를 포맷팅하여 출력합니다.

**예제 3: 반복문으로 숫자 출력하기**

Rust에서는 `for` 루프, `while` 루프, 그리고 `do-while` 유사 구조를 사용할 수 있습니다. 여기서는 `for` 루프를 예로 들어보겠습니다.

```rust
fn main() {
    // `0`부터 `4`까지 반복합니다. 마치 친구들과 함께 숫자 게임을 하는 것 같죠!
    for i in 0..5 {
        println!("현재 숫자: {}", i);
    }

    // `while` 루프 예시: 특정 조건이 만족될 때까지 반복합니다.
    let count = 0;
    while count < 3 {
        println!("while 루프 반복: {}", count);
        count += 1; // 변수 값 증가
    }

    // `do-while` 유사 구조 (Rust에는 직접적인 `do-while`이 없지만, `while` 루프를 활용)
    let mut x = 0;
    loop {
        println!("do-while 유사 루프: {}", x);
        x += 1;
        if x >= 3 { break; } // 조건에 따라 루프 종료
    }
}
```

**코드 설명:**
1. **`for` 루프**: `0..5` 범위 내에서 반복하며 각 숫자를 출력합니다.
2. **`while` 루프**: `count`가 `3` 미만일 때까지 반복하며 값을 출력하고 증가시킵니다.
3. **`do-while` 유사 구조**: `loop`과 `break`를 사용하여 `do-while`과 유사한 동작을 구현합니다.

**🚨 실무 주의보**

* **Rust 버전 관리**: 항상 최신 Rust 버전을 사용하는 것이 좋습니다. `rustup update` 명령어를 통해 쉽게 업데이트할 수 있습니다. 최신 버전은 성능 개선과 보안 업데이트를 포함하고 있으니 놓치지 마세요!
* **의존성 관리**: 프로젝트에 필요한 라이브러리는 `Cargo.toml` 파일을 통해 관리합니다. 필요한 패키지를 추가하고 버전을 지정하면 빌드 과정이 훨씬 원활해집니다.
