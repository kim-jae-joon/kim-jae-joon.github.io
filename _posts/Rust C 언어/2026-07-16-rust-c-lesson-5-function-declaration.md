## 💫 5강: Rust C 언어 기초 - 함수 선언 및 호출: 당신의 코드 마법사로 변신하는 길 🧙‍♂️🔍

### 🚀 함수의 마법 시작하기

**함수**는 코드에서 반복적으로 사용할 수 있는 작은 기능 블록이라고 생각하면 됩니다. 예를 들어, 요리 레시피에서 "계란을 2개 깨서 섞어라"는 부분을 여러 번 반복해야 할 때마다 똑같은 지시를 매번 적는 대신, 한 번 정의해 놓고 필요할 때마다 호출하면 훨씬 효율적이죠. 바로 이런 역할을 함수가 해줍니다!

#### 기본 함수 선언 구조

Rust와 C 언어에서 함수를 선언하는 기본 구조는 다음과 같습니다:

```c
// C 언어 예제
int addNumbers(int a, int b) {
    // 여기서 a와 b를 더하고 결과를 반환합니다.
    int result = a + b;
    return result;
}

// Rust 언어 예제
fn add_numbers(a: i32, b: i32) -> i32 {
    // Rust에서도 비슷한 방식으로 작동합니다.
    let result = a + b;
    result
}
```

**코드 해석:**
1. **함수 선언 부분:**
   - `int addNumbers(int a, int b)`: 함수 이름 `addNumbers`는 두 개의 정수 매개변수 `a`와 `b`를 받습니다.
   - `fn add_numbers(a: i32, b: i32) -> i32`: Rust에서는 `fn` 키워드로 함수를 정의하며, `a: i32`와 `b: i32`는 매개변수 타입을 명시하고, `-> i32`는 반환 타입을 나타냅니다.
   
2. **함수 본문:**
   - 함수 내부에서 `a`와 `b`를 더한 결과를 `result` 변수에 저장하고, 이 값을 `return` 문으로 반환합니다.

### 🤔 함수 호출: 마법의 순간!

함수를 정의했다면 이제 실제로 사용하는 순간이죠! 함수 호출은 마치 마법사가 주문을 외우는 것과 같습니다.

#### 다양한 함수 호출 방법

**반복문과 조건문을 활용해 함수를 다양한 상황에서 호출해보겠습니다.**

1. **간단한 호출 예제:**

   ```c
   #include <stdio.h>

   int main() {
       int result = addNumbers(5, 3);  // 함수 호출
       printf("결과: %d\n", result);  // 출력
       return 0;
   }
   ```

   **코드 해석:**
   - `addNumbers(5, 3)`: 함수를 호출하며 5와 3을 인자로 전달합니다.
   - `printf`로 결과를 화면에 출력합니다.

2. **매개변수와 반환값 활용:**

   ```rust
   fn main() {
       let sum = add_numbers(10, 20);  // 함수 호출 및 반환값 저장
       println!("합계: {}", sum);      // 출력
   }
   ```

   **코드 해석:**
   - `add_numbers(10, 20)`: 함수를 호출하고 결과를 `sum` 변수에 저장합니다.
   - `println!` 매크로로 결과를 출력합니다.

3. **여러 함수 호출 조합:**

   ```c
   #include <stdio.h>

   int multiply(int x, int y) {
       return x * y;
   }

   int main() {
       int num1 = 4;
       int num2 = 6;
       
       int sum_result = addNumbers(num1, num2);  // 첫 번째 함수 호출
       int product_result = multiply(num1, num2);  // 두 번째 함수 호출
       
       printf("합계: %d, 곱셈 결과: %d\n", sum_result, product_result);
       return 0;
   }
   ```

   **코드 해석:**
   - `addNumbers`와 `multiply` 두 함수를 연속적으로 호출하여 다양한 연산을 수행합니다.
   - 각 함수의 결과를 출력합니다.

### 💡 초보자 폭풍 질문!

**Q1: 함수 매개변수와 반환 타입을 어떻게 선택해야 하나요?**
- **A1:** 매개변수 타입은 함수가 받을 입력 데이터의 종류에 따라 결정됩니다. 예를 들어, 숫자를 더하는 함수라면 `int` 타입을 사용합니다. 반환 타입은 함수가 실행 후 어떤 종류의 값을 반환할지 결정합니다. 숫자 결과를 반환한다면 `int` 타입을 사용하면 됩니다. 필요에 따라 `float`, `char`, 포인터 타입 등 다양하게 선택할 수 있어요!

**Q2: 함수를 재귀적으로 호출하는 방법은 뭔가요?**
- **A2:** 재귀 함수는 자기 자신을 호출하는 방식입니다. 예를 들어, 팩토리얼 계산을 재귀로 구현하면 다음과 같습니다:

   ```c
   int factorial(int n) {
       if (n == 1) {
           return 1;  // 기본 조건
       } else {
           return n * factorial(n - 1);  // 재귀 호출
       }
   }
   ```

   **코드 해석:**
   - `n`이 1이면 1을 반환하고, 그렇지 않으면 `n * factorial(n - 1)`로 자기 자신을 호출하여 계산합니다.

### 🚨 실무주의보
