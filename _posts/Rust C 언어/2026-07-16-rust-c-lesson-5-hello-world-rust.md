# 5강: Rust의 첫 발걸음 - "안녕하세요, 세상!" (Hello World) 🚀

여러분, 안녕하세요! 💬 이번 강의에서는 Rust라는 강력하고 안전한 언어를 처음 접하는 분들을 위해 Rust의 세계로 들어가는 문을 활짝 열어드립니다. 바로 **"Hello World"** 프로그램을 만들어보는 시간이죠! 이건 마치 학교 첫 날 선생님이 "안녕하세요, 여러분!"이라고 말씀하시는 것과 똑같아요. 🤝

## Rust 소개: 미래의 안전한 프로그래밍 언어

Rust는 성능과 안전성을 동시에 추구하는 언어입니다. 마치 슈퍼히어로 영화에서 보는 것처럼, **동시에 스피드와 책임감**을 겸비한 캐릭터 같은 존재라고 생각하면 됩니다. 이 언어는 메모리 안전성을 보장하면서도 매우 빠른 실행 속도를 자랑합니다. 덕분에 시스템 프로그래밍이나 고성능 애플리케이션 개발에 딱 맞는 도구가 되었죠.

### 왜 "Hello World"가 중요한지 아세요?

> 진짜 신기하죠? "Hello World"는 프로그래밍의 첫걸음이자 가장 기본적인 목표입니다. 이 프로그램을 성공적으로 작성하면 프로그래밍의 문턱을 넘어선 셈이에요. 그리고 이 경험은 앞으로 복잡한 프로젝트를 다룰 때 큰 자신감을 줄 거예요!

## 첫 Rust 프로그램: 단계별 가이드

이제부터 차근차근 따라해보세요. Rust의 기본적인 구조와 문법을 이해하는 데 도움이 될 것입니다.

### 1. 프로젝트 설정하기

먼저, Rust 환경을 설정해야 합니다. `rustup`을 사용하면 간단해요:

```bash
# 터미널에서 아래 명령어를 실행해주세요.
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

**설명:**
- `curl`을 통해 `rustup` 설치 스크립트를 받아옵니다.
- 이 스크립트를 실행하면 Rust와 Cargo (Rust의 패키지 관리자)가 설치됩니다.

### 2. 첫 Rust 파일 생성

다음으로, 첫 Rust 파일을 만듭니다. 터미널에서 다음 명령어를 실행하세요:

```bash
cargo new hello_rust --bin
```

**설명:**
- `cargo new`는 새로운 Rust 프로젝트를 생성하는 명령어입니다.
- `hello_rust`는 프로젝트 이름이고 `--bin` 옵션은 실행 파일을 생성하도록 지정합니다.

프로젝트 폴더로 이동합니다:

```bash
cd hello_rust
```

### 3. "Hello World" 코드 작성

이제 `src/main.rs` 파일을 열어서 "Hello World" 메시지를 출력하는 코드를 작성해보죠!

#### 예제 1: 기본적인 "Hello World"

```rust
/// 메인 함수는 프로그램의 시작점입니다.
fn main() {
    /// 이 부분에서 '안녕하세요, 세상!'이 출력됩니다.
    println!("안녕하세요, 세상!");
}
```

**코드 분석:**
- `fn main() { ... }`: 프로그램의 진입점입니다. 여기서 시작해서 실행됩니다.
- `println!("안녕하세요, 세상!");`: Rust의 매크로 `println!`을 사용하여 콘솔에 문자열을 출력합니다. 매크로는 간단히 말해 함수처럼 작동하지만 컴파일 시점에 직접 처리됩니다.

#### 예제 2: 변수와 함께 "Hello World"

```rust
fn main() {
    /// 사용자 이름을 변수에 저장합니다.
    let userName = "친구야!";
    /// 변수 값을 출력합니다.
    println!("안녕하세요, {}!", userName);
}
```

**코드 분석:**
- `let userName = "친구야!";`: 문자열 `"친구야!"`를 `userName`이라는 변수에 할당합니다.
- `println!("{}", userName);`: 문자열 포맷팅을 사용해 변수 값을 포함한 메시지를 출력합니다. 이는 유연한 문자열 처리를 가능하게 합니다.

#### 예제 3: 반복문으로 "Hello World" 여러 번 출력

```rust
fn main() {
    /// 5번 반복하며 "안녕하세요"를 출력합니다.
    for i in 1..=5 {
        println!("안녕하세요, 번째 세상!", i);
    }
}
```

**코드 분석:**
- `for i in 1..=5 { ... }`: 반복문을 사용해 1부터 5까지 반복합니다.
- `println!("{}", i);`: 각 반복마다 현재 숫자 `i`를 포함한 메시지를 출력합니다.

### 4. 프로그램 실행하기

이제 작성한 코드를 실행해봅시다:

```bash
cargo run
```

**실행 결과 예시:**
```
안녕하세요, 세상!
안녕하세요, 친구야!
안녕하세요, 1번째 세상!
안녕하세요, 2번째 세상!
안녕하세요, 3번째 세상!
안녕하세요, 4번째 세상!
안녕하세요, 5번째 세상!
```

### 💡 초보자 폭풍 질문! 🚀

**질문:** `println!` 매크로는 어떻게 작동하나요?

**답변:** `println!`은 Rust에서 콘솔 출력을 위한 매크로입니다. 매크로는 코드를 컴파일 시점에 전처리하여 더 간결하고 효율적인 코드를 생성합니다. 여기서 `println!`은 문자열과 인자를 받아서 이를 포맷팅하여 콘솔에 출력합니다. 예를 들어, `println!("{}", userName);`은 문자열 `"{}"`에 `userName` 변수의 값을 채워 넣어 출력합니다.

### 🚨 실무주의보 🛡️

**주의사항:** Rust의 강력한 타입 시스템과 소유권 규칙은 처음에는 다소 복잡하게 느껴질 수 있습니다. 하지만 이 규칙들 덕분에 메모리 오류를 미리 방지할 수 있으니, 천천히 이해하고 실습하면서 익혀나가세요!
