### 🚀 왜 Rust C 언어를 배우는 걸까?

**Rust**는 안전성과 성능을 동시에 잡은 언어예요. 마치 슈퍼히어로 같죠! **C 언어**는 컴퓨터의 심장이라고 불릴 만큼 근본적인 언어로, 시스템 프로그래밍이나 게임 엔진 등 성능이 최우선인 분야에서 널리 사용됩니다. **Rust와 C의 결합**은 개발자들에게 뛰어난 제어력과 안전성을 제공합니다. 

> **초보자 폭풍 질문!** 🤔  Rust와 C는 어떻게 다른가요?
> **답변**: Rust는 메모리 안전성과 동시성 처리를 강화한 최신 언어입니다. 반면, C는 오래된 전통을 지닌 강력한 성능 언어죠. 둘을 함께 배우면 시스템 수준에서 더 안전하고 효율적인 코드를 작성할 수 있어요!

### ### 기본 문법 탐험: 핵심 키워드로 시작하기

#### 1. **변수 선언과 초기화**

코딩에서 가장 기본적인 것은 **변수**를 사용해 데이터를 저장하고 다루는 거예요. Rust와 C에서 변수를 선언하고 초기화하는 방법을 알아볼게요.

**예제 1: 정수 변수 선언**

```c
#include <stdio.h>  // 표준 입출력 라이브러리 포함

int main() {
    int age = 25;  // 'age'라는 이름의 정수 변수를 선언하고 25로 초기화
    printf("나이는 %d살입니다.\n", age);  // 출력: 나이는 25살입니다.
    return 0;  // 프로그램 종료
}
```

**코드 해부**:
- `#include <stdio.h>`: 표준 입출력 라이브러리를 불러옵니다.
- `int main()`: 프로그램의 시작점입니다.
- `int age = 25;`: `int` 타입의 변수 `age`를 선언하고 값 `25`로 초기화합니다.
- `printf()`: 문자열과 변수 값을 출력합니다. `%d`는 정수 포맷 지정자입니다.
- `return 0;`: 프로그램이 성공적으로 종료됨을 나타냅니다.

**예제 2: 문자열 변수 선언**

```c
#include <stdio.h>

int main() {
    char name[10] = "Alice";  // 문자열을 저장할 char 배열 선언 및 초기화
    printf("이름은 %s입니다.\n", name);  // 출력: 이름은 Alice입니다.
    return 0;
}
```

**코드 해부**:
- `char name[10]`: 문자 배열을 선언하고 크기를 명시합니다 (최대 9글자 + 종료 문자 `\0` 포함).
- `"Alice"`: 배열에 문자열을 초기화합니다.
- `printf()`에서 문자열 포맷팅을 수행합니다.

#### 2. **기본 제어 구조**

**반복문**과 **조건문**은 코딩에서 가장 중요한 요소들입니다. 이들을 잘 이해하면 복잡한 문제도 쉽게 해결할 수 있어요!

##### 반복문: 반복의 마법을 펼쳐보세요!

**예제 1: `for` 루프**

```c
#include <stdio.h>

int main() {
    for (int i = 0; i < 5; i++) {  // 0부터 4까지 반복
        printf("반복 횟수: %d\n", i);
    }
    return 0;
}
```

**코드 해부**:
- `for (int i = 0; i < 5; i++)`: 초기화 (`i = 0`), 조건 (`i < 5`), 증감 (`i++`)을 한 줄에 담아 반복을 설정합니다.
- `printf()`로 각 반복 단계에서 값을 출력합니다.

**예제 2: `while` 루프**

```c
#include <stdio.h>

int main() {
    int counter = 0;
    while (counter < 5) {  // 카운터가 5 미만일 때까지 반복
        printf("반복 횟수: %d\n", counter);
        counter++;  // 카운터 증가
    }
    return 0;
}
```

**코드 해부**:
- `while (counter < 5)`: 조건이 참인 동안 반복합니다.
- `counter++`: 반복마다 카운터를 증가시킵니다.

**예제 3: `do-while` 루프 (C에서는 직접 지원하지 않으나 개념 이해를 위한 예시)**

```c
#include <stdio.h>

int main() {
    int i = 0;
    do {  // 적어도 한 번 실행 후 조건 확인
        printf("반복 횟수: %d\n", i);
        i++;
    } while (i < 5);  // 예시로 `do-while` 구조 구현
    return 0;
}
```

**코드 해부**:
- `do-while` 구조는 일반적으로 `while`과 유사하지만, 최소한 한 번은 실행 후 조건 확인을 합니다. 여기서는 `while`으로 구현했지만, 이해를 돕기 위함입니다.

##### 조건문: 선택의 순간!

**예제 1: `if-else` 문**

```c
#include <stdio.h>

int main() {
    int score = 75;  // 점수를 저장하는 변수
    if (score >= 80) {  // 점수가 80 이상이면
        printf("합격!\n");  // 합격 메시지 출력
    } else {
        printf("다시 공부해야 합니다.\n");  // 그렇지 않으면 공부 메시지 출력
    }
    return 0;
}
```

**코드 해부**:
- `if (score >= 80)`: 조건이 참이면 특정 코드 블록을 실행합니다.
- `else`: 조건이 거짓일 때 실행되는 대안 블록입니다.

**예제 2: `switch 문` (Rust에서는 `match` 사용, 여기서는 C 스타일로 예시)

```c
#include <stdio.h>

int main() {
    char grade = 'B';  // 학점을 저장하는 변수
    switch (grade) {  // switch 문으로 학점에 따른 처리
        case 'A':
            printf("최우수!\n");  // A 학점 시 메시지 출력
            break;  // 각 케이스 종료
        case 'B':
            printf("우수!\n");  // B 학점 시 메시지 출력
            break;
        case 'C':
            printf("평균!\n");  // C 학점 시 메시지 출력
            break;
        default:
            printf("기타 학점\n");  // 그 외의 학점 시 메시지 출력
            break;
    }
    return 0;
}
```

**코드 해부**:
- `switch (grade)`: `grade` 변수에 따라 케이스를 선택합니다.
- `case`와 `break`: 각 케이스가 끝날 때 `break`로 제어를 다음 줄로 넘어가지 않게 합니다.
- `default`: 모든 케이스에 해당하지 않을 때 실행되는 기본 케이스입니다.
