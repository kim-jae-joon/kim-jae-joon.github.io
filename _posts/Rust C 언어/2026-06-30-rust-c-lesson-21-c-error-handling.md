#### 🌟 오류 처리의 기초: 왜 중요할까요?

**"이거 모르면 큰일 납니다!"**  
오류 처리는 마치 우리가 길을 걷다가 갑작스러운 돌멩이가 등장할 때 주의 깊게 피하듯이, 코드가 예상치 못한 상황에 직면했을 때 부드럽게 대처하는 방법입니다. 왜 중요할까요? 

- **사용자 경험 향상**: 프로그램이 갑자기 멈추면 사용자는 당황스럽죠. 오류 메시지를 통해 문제의 원인을 알려주면 사용자도 문제 해결에 동참할 수 있습니다.
- **안정성 강화**: 예상치 못한 오류를 처리함으로써 시스템 전체의 안정성이 향상됩니다. 마치 자동차의 안전 벨트와 에어백처럼요! 🚗

#### 💡 개념 이해: 주요 오류 처리 구문

##### 1. **return 문으로의 탈출**
기본적으로 함수가 문제를 감지하면 `return` 문을 통해 제어를 원래 호출 지점으로 돌릴 수 있습니다.

```c
#include <stdio.h>

// 함수 정의: 문제 발생 시 오류 코드 반환
int riskyOperation(int x) {
    if (x <= 0) {
        printf("오류: 음수 또는 0 입력! \n");
        return -1; // 오류 코드 반환
    }
    // 정상 처리 로직
    return x * x; // 정상 반환
}

int main() {
    int result = riskyOperation(-5);
    if (result < 0) {
        printf("처리 중 오류 발생!\n");
    } else {
        printf("결과: %d\n", result);
    }
    return 0;
}
```
**코드 설명:**
- `riskyOperation` 함수는 입력 값이 음수거나 0일 때 `-1`이라는 오류 코드를 반환합니다.
- `main` 함수에서 오류 코드를 체크하고 적절한 메시지를 출력합니다.

##### 2. **예외 처리: `setjmp`와 `longjmp`**
`setjmp`와 `longjmp`는 더 복잡한 예외 처리를 가능하게 합니다. 이 두 함수는 스택을 통해 이전 상태로 돌아가는 기능을 제공합니다.

```c
#include <stdio.h>
#include <setjmp.h>

jmp_buf errorHandler; // 예외 처리를 위한 버퍼

void riskyFunction() {
    if (rand() % 2 == 0) { // 임의의 오류 발생 시뮬레이션
        printf("오류 발생!\n");
        longjmp(errorHandler, 1); // 예외 발생 및 스택으로 복귀
    }
    printf("정상 동작 중...\n");
}

int main() {
    if (setjmp(errorHandler) == 0) { // 초기 호출 시 0 반환
        riskyFunction(); // 함수 호출
    } else {
        printf("예외 처리 완료!\n");
    }
    return 0;
}
```
**코드 설명:**
- `setjmp`로 예외 처리 버퍼를 설정하고, `longjmp`로 예외 상황에서 스택의 이전 상태로 돌아갑니다.
- `setjmp` 호출이 두 번째로 이루어지면 (`== 0`이 아닌 경우) 예외 처리가 완료된 것으로 간주합니다.

##### 3. **조건문과 오류 핸들링**
가장 간단하면서도 효과적인 방법 중 하나는 조건문을 활용한 오류 핸들링입니다.

```c
#include <stdio.h>
#include <stdlib.h>

int divide(int numerator, int denominator) {
    if (denominator == 0) {
        fprintf(stderr, "오류: 분모가 0입니다!\n");
        return EXIT_FAILURE; // 실패 코드 반환
    }
    return numerator / denominator; // 정상 처리
}

int main() {
    int result = divide(10, 0); // 예제 호출
    if (result == EXIT_FAILURE) {
        printf("오류 발생!\n");
    } else {
        printf("결과: %d\n", result);
    }
    return 0;
}
```
**코드 설명:**
- `divide` 함수는 분모가 0일 때 오류 메시지를 출력하고 실패 코드를 반환합니다.
- `main` 함수에서 이 실패 코드를 체크하여 적절한 메시지를 출력합니다.
