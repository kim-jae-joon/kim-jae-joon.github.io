#### **💡 기초 다지기: 다차원 배열이란 무엇인가요?**

**다차원 배열**은 우리가 일상생활에서 자주 접하는 구조를 프로그래밍 세계로 옮겨놓은 거예요. 가령, **식당 메뉴**를 생각해봅시다. 각 메뉴에는 **종류** (예: 면, 밥)와 **세부 항목** (예: 불고기 면, 비빔밥)이 있죠. 이와 비슷하게, 다차원 배열은 하나의 배열이 다른 배열들을 포함하는 구조예요.

##### 예제 1: 2차원 배열 - 식당 메뉴처럼 생각하기
```c
#include <stdio.h>

int main() {
    // 2차원 배열: 각 행은 메뉴 종류, 각 열은 세부 메뉴 항목
    int menu[3][4] = {  // 3개의 메뉴 종류, 각 종류에 4개의 세부 메뉴
        {"면", "볶음밥", "비빔밥", "파스타"},
        {"불고기 면", "계란 볶음밥", "김치 비빔밥", "치즈 파스타"},
        {"크림 파스타", "스테이크 면", "매운 비빔밥", "새우 볶음밥"}
    };

    // 출력하기
    for (int i = 0; i < 3; i++) {  // 메뉴 종류 행 순회
        for (int j = 0; j < 4; j++) {  // 세부 메뉴 열 순회
            printf("메뉴: %s\n", menu[i][j]);
        }
        printf("\n");  // 각 메뉴 종류 사이에 빈 줄 추가
    }

    return 0;
}
```
- **설명**:
  - `menu[3][4]`: 3개의 행(메뉴 종류)과 각 행에 4개의 열(세부 메뉴 항목)을 가진 배열입니다.
  - 중첩된 `for` 루프를 사용하여 각 요소를 순회하고 출력합니다. 첫 번째 루프는 행을, 두 번째 루프는 열을 순회합니다.
  - 각 메뉴 종류별로 빈 줄을 넣어 구분합니다.

#### **📐 포인터로 다차원 배열을 조작하기**

다차원 배열을 포인터로 다루는 것은 마치 **마법의 지팡이**를 휘두르는 것과 같아요. 이 지팡이로 배열의 깊은 곳까지 자유자재로 움직일 수 있죠!

##### 예제 2: 포인터로 다차원 배열 접근하기
```c
#include <stdio.h>

int main() {
    int scores[2][3] = {{85, 90, 92}, {78, 88, 95}};  // 2x3 배열
    int *ptr;  // 포인터 선언

    // 첫 번째 행의 첫 번째 요소를 가리키는 포인터 설정
    ptr = &scores[0][0];
    printf("scores[0][0] = %d\n", *ptr);  // *ptr으로 값을 참조

    // 포인터를 이용해 다른 요소 접근
    ptr++;  // 포인터를 다음 위치로 이동
    printf("다음 요소: %d\n", *ptr);  // 다음 요소 참조

    // 두 번째 행 접근 (이중 포인터 사용)
    ptr = scores[1];  // 첫 번째 행을 넘어서 두 번째 행의 시작 주소로 이동
    for (int j = 0; j < 3; j++) {
        printf("scores[1][%d] = %d\n", j, *(ptr + j));  // 이중 포인터 활용
    }

    return 0;
}
```
- **설명**:
  - `int *ptr;`: 포인터 변수 `ptr`을 선언합니다.
  - `ptr = &scores[0][0];`: 첫 번째 요소의 주소를 포인터에 저장합니다.
  - `*ptr`: 포인터가 가리키는 값을 참조합니다.
  - `ptr++;`: 포인터를 다음 위치로 이동시켜 배열을 순회합니다.
  - 이중 포인터 기법 (`*(ptr + j)`)을 사용해 두 번째 행의 요소들을 순회합니다.

#### **🛠️ 다양한 반복 구조로 다차원 배열 다루기**

다차원 배열을 효과적으로 다루기 위해 다양한 반복 구조를 활용해볼까요?

##### 예제 3: `for` 루프를 이용한 다차원 배열 순회 (3가지 방식)
```c
#include <stdio.h>

int main() {
    int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};

    // 1. 중첩된 for 루프 (전통적 방법)
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            printf("행 %d, 열 %d: %d\n", i, j, matrix[i][j]);
        }
    }

    // 2. 단일 포인터를 이용한 방법
    int *ptr = matrix[0];  // 첫 번째 행의 첫 번째 요소 주소
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            printf("행 %d, 열 %d: %d\n", i, j, *(ptr + j));
        }
        ptr += 3;  // 다음 행으로 이동
    }

    // 3. `while` 루프를 이용한 방법
    i = 0;
    while (i < 2) {
        j = 0;
        while (j < 3) {
            printf("행 %d, 열 %d: %d\n", i, j, matrix[i][j]);
            j++;
        }
        i++;
    }

    return 0;
}
```
- **설명**:
  - **중첩된 `for` 루프**: 가장 직관적인 방법으로 행과 열을 순차적으로 순회합니다.
  - **단일 포인터 사용**: 포인터를 이용해 행을 건너뛰면서 열을 순회합니다. 이 방법은 메모리 주소를 직접 다루는 느낌이 들어요.
  - **`while` 루프**: 반복 구조를 다르게 적용해 보는 방법입니다. `i`와 `j`를 직접 관리하며 순회합니다.

- **답변**: 포인터를 사용하면 메모리 주소를 직접 다룰 수 있어, 동적 메모리 할당이나 복잡한 데이터 구조에서 효율적으로 데이터를 조작할 수 있어요. 특히 큰 데이터셋을 다룰 때 성능 최적화에 큰 도움이 됩니다.
