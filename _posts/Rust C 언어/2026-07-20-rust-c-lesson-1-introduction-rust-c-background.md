## 1강: Rust 소개 & C 언어에서 온 당신을 위한 친절한 로드맵 🚀

**안녕하세요, 코딩 모험가 여러분!**  👋  C 언어의 익숙한 길을 걸으며 멋진 코드 조각들을 만들어 온 당신을 위해, 오늘은 **Rust라는 신비로운 언어 세계로의 첫걸음**을 안내할게요. 마치 낡은 지도를 쥐고 탐험을 떠나는 기분이지 않나요? 😎  C 언어는 기초를 다지는 탄탄한 토대였지만, Rust는 그 다음 단계의 탐험을 위한 첨단 도구 상자랍니다!

**C 언어, 너의 trusty companion**

먼저, 우리의 믿음직한 동반자 **C 언어**를 살짝 되돌아봅시다. C는 컴퓨터 하드웨어와 가장 가까이 닿는 언어라고 할 수 있죠. 마치 자동차 엔진룸을 직접 만지는 것처럼 섬세하고 강력하지만, 그만큼 조심스럽게 다뤄야 하는 존재죠. 메모리 관리는 당신의 책임이고, 실수는 때때로 치명적인 결과를 초래할 수 있어요. 하지만 그만큼 배우는 과정에서 얻는 성취감은 엄청나죠! 

> **C 언어의 핵심 무기:**
> - **낮은 수준 제어:** 하드웨어 직접 제어 가능! 🚀  
> - **효율성:** 최고의 성능을 요구하는 분야에서 사랑받는 이유! 💪
> - **널리 사용:** 시스템 프로그래밍, 게임 엔진 등 다양한 분야에서 기반 기술로 활용!

**하지만... 한계도 있죠!**

C 언어의 강력함 뒤에는 '메모리 누수'와 같은 잠재적인 함정이 숨어 있어요. 마치 탐험 중 길을 잃는 것처럼, 잘못된 메모리 관리는 프로그램을 불안정하게 만들 수 있죠. 💥

**Rust, 구원자 등장! 🎉**

여기서 **Rust**가 등장합니다! Rust는 C 언어의 후예이자, 동시에 그 한계를 극복하기 위해 탄생한 혁신적인 언어예요. **"소유권(Ownership)"**과 **"빌림(Borrowing)**이라는 개념을 통해 메모리 안전성을 보장하면서도 고성능을 유지하는 놀라운 균형을 찾아냈죠. 마치 마법처럼 안전하면서도 빠르게 코드를 작성할 수 있게 해주는 언어랍니다!

### Rust의 핵심 마법: 소유권과 빌림

**1. 소유권 (Ownership):**

Rust는 변수 하나에 **단 하나의 소유자만 존재**한다고 가정합니다. 이 소유자가 변수를 더 이상 사용하지 않으면 메모리에서 자동으로 해제됩니다. 마치 친구에게 장난감을 빌려주고 반납 날짜를 정하는 것과 같아요!

```rust
fn main() {
    // 'hello' 문자열 소유권이 'message' 변수에 넘어감
    let message = String::from("Hello, Rust!"); 
    // 'message'가 더 이상 필요하지 않아지면 컴파일러가 자동으로 메모리 해제!
}
```

* **코드 해설:**  
    - `String::from("Hello, Rust!")`는 문자열 "Hello, Rust!"를 소유하는 새로운 `String` 변수 `message`를 생성합니다.
    - `message` 변수가 범위 밖으로 나가거나 더 이상 사용되지 않으면 Rust 컴파일러가 자동으로 메모리를 해제합니다. 이 과정에서 **누수 문제**가 발생하지 않도록 보호해줘요!

**2. 빌림 (Borrowing):**

Rust는 임시로 코드 외부에서 데이터에 접근할 수 있게 해줍니다. 마치 도서관에서 책을 잠시 빌려 읽는 것과 같죠! 빌림은 소유권을 바꾸지 않고 안전하게 데이터를 공유할 수 있게 해줍니다.

```rust
fn greet(name: &str) {
    println!("Hello, {}!", name); // '&str' 타입은 문자열을 참조하는 빌림입니다.
}

fn main() {
    let userName = String::from("Alice");
    greet(&userName); // userName의 소유권을 유지하면서 greet 함수에 빌림 제공
}
```

* **코드 해설:**
    - `&str` 타입은 문자열을 참조하는 빌림을 나타냅니다. `greet` 함수는 `userName`의 소유권을 직접 받지 않고 문자열 내용에 접근할 수 있어요.
    - 빌림 규칙 덕분에 `userName`이 여전히 유효한 상태에서 `greet` 함수 내에서 안전하게 사용 가능합니다.

### Rust로 만드는 간단한 예제: 주사위 게임 🎲

이제 Rust의 기본을 직접 체험해 볼까요? 간단한 주사위 던지기 게임을 만들어 보겠습니다.

```rust
use rand::Rng; // 난수 생성 라이브러리

fn roll_dice() -> u8 { // 주사위 굴리기 함수
    let mut rng = rand::thread_rng(); // 난수 생성기 초기화
    rng.gen_range(1..=6) // 1부터 6까지의 난수 생성
}

fn main() {
    let dice_roll = roll_dice(); // 주사위 던지기
    println!("당신은 {}를 굴렸습니다!", dice_roll); // 결과 출력

    // 만약 더 복잡한 게임 로직이 필요하다면?
    // 반복문이나 조건문을 활용해볼 수 있습니다!
    // 예시: 3번 던져서 합 계산
    let mut total = 0;
    for _ in 0..3 { // 반복문: 0부터 2까지 반복
        total += roll_dice(); // 각 던지기 결과를 더함
    }
    println!("총합: {}", total);

    // 조건문 예시: 주사위 합이 18 이상이면 승리!
    if total >= 18 {
        println!("축하합니다! 승리했습니다!");
    } else {
        println!("다시 시도해 보세요!");
    }
}
```

* **코드 해설:**
    - **`use rand::Rng;`:** 외부 라이브러리 `rand`를 사용하기 위한 수입니다. 난수 생성에 필요해요.
    - **`roll_dice()` 함수:** 난수 생성기를 사용하여 1부터 6까지의 주사위 숫자를 반환합니다.
    - **`for` 반복문:** 코드에서 3번 주사위를 던지고 각 결과를 `total`에 누적합니다.
    - **`if-else` 조건문:** 주사위 합이 특정 조건 (18 이상)을 만족하면 승리 메시지를 출력합니다.

- **작은 프로젝트부터 시작:** 큰 프로젝트에 바로 뛰어들기보다는 간단한 프로그램부터 시작하며 Rust의 특징을 차근차근 익혀나가세요.
