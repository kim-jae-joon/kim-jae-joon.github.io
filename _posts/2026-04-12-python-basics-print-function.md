**파이썬 기본 구조**
-------------------

파이썬은 프로그래밍 언어 중 하나로, 다른 언어들과는 달리 매우 간결하고 readability가 좋습니다. 파이썬의 기본 구조를 이해하면 다양한 기능을 구현하는 데 도움이 됩니다.

### **Indention** (들여쓰기)

파이썬에서는 들여쓰기를 통해 코드의 블록을 구분합니다. 들여쓰기는 4칸(8공간)으로 제한되어 있습니다. 예를 들어, 다음 코드를 살펴보겠습니다.
```markdown
# 파이썬은 들여쓰기敏感 언어입니다.

def greet(name):
    print("Hello, " + name + "!") # 들여쓰기가 4칸인 블록입니다.

greet("John")
```
위의 코드에서 `greet` 함수를 호출할 때 `print` 문을 포함하는 들여쓰기된 블록이 실행됩니다. 만약 들여쓰기를 빼먹었다면, 파이썬은 다음과 같은 오류 메시지를 출력합니다.
```markdown
SyntaxError: expected an indented block (greet.py, line 3)
```
### **Comment**

파이썬에서 주석을 사용하면 코드의 이해를 돕고, 개발자의 의도와 목적을 설명할 수 있습니다. 주석은 `#` 기호로 시작하며, 줄 또는 블록으로 작성할 수 있습니다.
```markdown
# 이 줄에 달린 주석입니다.

"""
이블록에 달린 주석입니다.
만약 여러 줄 주석이 필요하다면 triple quotes를 사용하세요.
"""
```
**출력 함수**
------------

파이썬에서 출력을 위한 다양한 함수가 존재합니다. 여기서는 `print()` 함수와 `input()` 함수를 살펴보겠습니다.

### **`print()` 함수**

`print()` 함수는 입력된 값들을 콘솔에 출력합니다.
```markdown
# print() 함수 예제

print("Hello, World!") # 문자열을 출력합니다.
print(123) # 숫자를 출력합니다.
print(True) # 불리언 값을 출력합니다.

# 줄바꿈을 원한다면, 다음의 코드를 사용하세요.
print("Hello, \nWorld!")
```
위의 예제에서 `print()` 함수는 문자열, 숫자, 불리언 값을 콘솔에 출력합니다. 또한 줄바꿈을 원할 때 `\n` 을 사용하여 줄을 바꿀 수 있습니다.

### **`input()` 함수**

`input()` 함수는 사용자로부터 입력을 받습니다.
```markdown
# input() 함수 예제

name = input("이름을 입력하세요: ")
print("인사합니다, " + name + "!")

age = int(input("나이를 입력하세요 (숫자만): "))
print("당신의 나이는", age, "살입니다.")
```
위의 예제에서 `input()` 함수는 사용자로부터 이름과 연령을 입력받습니다. `int()` 함수를 사용하여 숫자만 입력받도록 할 수 있습니다.

### **`format()` 메서드**

`format()` 메서드는 문자열에 변수를 삽입할 때 사용합니다.
```markdown
# format() 메서드 예제

name = "John"
age = 30

print("이름: {} 나이: {}".format(name, age))
```
위의 예제에서 `format()` 메서드는 이름과 연령을 문자열에 삽입하여 출력합니다.

### **f-string**

파이썬 3.6 버전 이상부터는 f-string이라는 새로운 문법을 지원합니다.
```markdown
# f-string 예제

name = "John"
age = 30

print(f"이름: {name} 나이: {age}")
```
위의 예제에서 f-string은 이름과 연령을 문자열에 삽입하여 출력합니다. f-string은 매우 편리하며, 가독성이 좋습니다.
