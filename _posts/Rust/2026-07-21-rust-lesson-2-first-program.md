# 2강: Rust 기초 - 당신의 첫 번째 프로그램 작성하기: 시작하는 여정

안녕하세요, 여러분! Rust 5년차 주니어 개발자인 저와 함께하는 이 시간이 얼마나 신나는지 느껴지시나요? 오늘은 Rust 프로그래밍의 첫 걸음을 내딛는 순간을 함께 나누려 합니다. 마치 컴퓨터 세계로의 여행을 시작하는 것 같지 않나요? 🤩

## 준비운동: 왜 Rust인가요?

**Rust**는 성능과 안전성을 동시에 추구하는 언어입니다. 시스템 프로그래밍에 최적화되어 있어 메모리 관리가 자동화되면서도 컴파일 타임에 오류를 검출해주는 강점이 있어요. 이건 마치, 코딩하면서 자동으로 안전모를 쓰고 작업하는 것과 같아요! **"이거 모르면 큰일 납니다!"** 안전한 코드를 작성하는 능력은 개발자의 필수 아이템이니까요.

## 첫 번째 프로그램: "안녕, 세상!"

우리가 작성할 첫 프로그램은 정말 간단하지만, 그만큼 중요한 의미가 있어요. "안녕, 세상!"이라는 메시지를 출력하는 프로그램을 만들어보겠습니다.

### 코드 예시 1: 기본 출력

```rust
fn main() {
    println!("안녕, 세상!");  // 이 줄은 콘솔에 "안녕, 세상!"을 출력합니다.
}
```

#### 코드 분해

- `fn main() { ... }`: Rust 프로그램의 진입점입니다. `main` 함수는 프로그램이 실행되면 가장 먼저 호출됩니다.
- `println!("안녕, 세상!");`: `println!` 매크로는 문자열을 출력하는 데 사용됩니다. 따옴표 안의 내용이 그대로 콘솔에 나타납니다.

### 💡 초보자 폭풍 질문!

**Q: `fn`은 무엇을 의미하나요?**
- **A**: `fn`은 함수를 정의하는 키워드입니다. 여기서는 `main` 함수를 정의하는 데 사용되었어요. 함수는 코드의 블록을 묶어서 재사용할 수 있게 해줍니다.

### 🚨 실무주의보

**Rust에서는 코드의 안전성과 효율성을 위해 컴파일 타임에 많은 검사가 이루어집니다.** 따라서 간단한 오타나 문법 오류도 미리 잡아낼 수 있어요. 예를 들어, `println!` 대신 `println`만 쓰면 컴파일 오류가 발생합니다. 항상 `!`를 잊지 마세요!

## 확장해보기: 변수와 데이터 타입

이제 조금 더 나아가서 변수를 사용해보죠. 변수는 데이터를 저장하는 공간이라고 생각하면 됩니다. 마치 주머니에 물건을 넣어두듯이요!

### 코드 예시 2: 변수 사용

```rust
fn main() {
    let name = "Rustacean";  // 문자열 변수 선언
    let age = 25;            // 정수 변수 선언
    println!("제 이름은 {}이고, {}살이에요.", name, age);  // 변수를 출력
}
```

#### 코드 분해

- `let name = "Rustacean";`: `let` 키워드는 변수를 선언합니다. 여기서 `name`이라는 변수에 문자열 `"Rustacean"`을 할당합니다.
- `let age = 25;`: 정수형 변수 `age`를 선언하고 값 `25`를 할당합니다.
- `println!("제 이름은 {}이고, {}살이에요.", name, age);`: 문자열 내의 `{}`는 플레이스홀더로, 실제 변수 값이 대체됩니다.

### 💡 초보자 폭풍 질문!

**Q: 변수 이름은 어떻게 정해야 하나요?**
- **A**: 변수 이름은 명확하고 의미 있는 단어를 사용하는 것이 좋습니다. 예를 들어, `user_age` 대신 `age`와 같이 간결하면서도 이해하기 쉬운 이름을 선택하세요. 단, Rust는 대소문자를 구분하므로 주의하세요!

### 코드 예시 3: 다양한 데이터 타입

Rust는 다양한 데이터 타입을 지원합니다. 몇 가지 예를 들어보겠습니다.

#### 정수 타입

```rust
fn main() {
    let is_active: bool = true;  // 불리언 타입
    let height: u32 = 180;       // 부호 없는 32비트 정수
    println!("활성화 상태: {}, 키: {}cm", is_active, height);
}
```

#### 코드 분해

- `let is_active: bool = true;`: `bool` 타입은 참/거짓 값을 저장합니다.
- `let height: u32 = 180;`: `u32`는 부호 없는 32비트 정수를 나타냅니다.

#### 문자열과 튜플

```rust
fn main() {
    let greeting: String = String::from("환영합니다!");  // String 타입
    let point: (i32, i32) = (10, 20);                   // 튜플
    println!("{}", greeting);
    println!("포인트 좌표: ({}, {})", point.0, point.1);
}
```

#### 코드 분해

- `String::from("환영합니다!")`: `String` 타입은 가변적인 문자열을 저장합니다.
- `let point: (i32, i32) = (10, 20);`: 튜플은 여러 타입의 데이터를 한 번에 저장할 수 있습니다.

### 🚨 실무주의보

**타입 안전성**은 Rust의 핵심 가치 중 하나입니다. 컴파일 타임에 타입이 맞지 않는 경우 오류가 발생하므로, 항상 변수 타입을 명시하는 습관을 들이세요. 이는 장기적으로 코드의 안정성을 크게 향상시킵니다.
