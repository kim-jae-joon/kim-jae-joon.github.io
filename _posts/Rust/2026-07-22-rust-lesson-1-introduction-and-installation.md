# 1강: Rust 기초: 소개 및 설치 - 당신의 코딩 여정을 위한 첫걸음!

안녕하세요, 여러분! 오늘은 코딩 세계로의 흥미진진한 여행을 시작하는 날입니다. 특히 Rust라는 언어를 통해 안전하고 고성능의 소프트웨어를 개발하는 법을 배우게 될 거예요. Rust는 마치 마법사의 지팡이처럼, 성능과 안전성 사이에서 균형을 잡는 마법 같은 언어랍니다. 🪄

## Rust, 왜 선택해야 할까?

Rust는 주로 시스템 프로그래밍에 최적화되어 있습니다. **안전성**과 **성능**이라는 두 마리 토끼를 동시에 잡을 수 있는 언어라는 점에서 독보적입니다. 이 언어는 메모리 안전성을 보장하면서도 C나 C++처럼 빠르게 동작할 수 있어요.

- **메모리 안전성**: 메모리 누수나 데이터 레이스 같은 오류를 컴파일 타임에 잡아냅니다.
- **고성능**: 시스템 레벨에서의 빠른 실행 속도를 제공합니다.
- **소유권 시스템**: 복잡한 데이터 관리를 간결하게 처리할 수 있는 독특한 개념입니다.

### 💡 초보자 폭풍 질문!
**Q: "Rust가 처음인데, 기존 언어(예: Python)와 비교해 어떤 점이 어려울까요?"**

**A:** 처음에는 소유권 시스템과 생명주기 관리가 조금 복잡하게 느껴질 수 있어요. 하지만 이는 장기적으로 코드의 안정성과 성능을 크게 향상시키는 핵심 요소입니다. Python처럼 직관적인 메모리 관리를 기대하신다면 초반에 적응 기간이 필요할 수 있지만, 곧 그 가치를 체감하실 거예요!

## Rust 설치하기: 첫 번째 관문 통과!

Rust를 시작하기 위해선 먼저 환경을 세팅해야 합니다. 걱정하지 마세요, 설치 과정은 정말 간단합니다!

### Windows, macOS, Linux 모두를 위한 설치 가이드

1. **공식 웹사이트 방문**: [Rust 다운로드 페이지](https://www.rust-lang.org/tools/install)로 이동합니다.
2. **스크립트 실행**:
   - **Windows/Linux**: 터미널을 열고 아래 명령어를 실행합니다.
     ```bash
     curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
     ```
   - **macOS**: 터미널에서 아래 명령어를 실행합니다.
     ```bash
     brew install rust
     ```
     혹은 공식 스크립트를 사용해도 좋습니다:
     ```bash
     curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
     ```

3. **설치 완료 후 설정**: 설치가 끝나면 터미널에서 다음 명령어를 실행해 Rust가 제대로 설치되었는지 확인합니다.
   ```bash
   rustc --version
   ```
   정상적으로 버전 정보가 출력되면 설치 완료!

### 🚨 실무주의보
**실무에서는 일관된 환경 유지가 중요합니다.** `rustup`을 사용하면 다양한 Rust 버전을 쉽게 관리할 수 있으니, 프로젝트마다 필요한 Rust 버전을 쉽게 설정할 수 있어요.

## Hello, Rust! 첫 코드 작성하기

이제 기본 세팅이 완료되었으니, Rust로 첫 코드를 작성해볼까요? 가장 기본적인 "Hello, World!" 프로그램부터 시작합니다.

### 코드 예제 1: `hello_world.rs`
```rust
fn main() {
    println!("Hello, Rust 세계!");  // 여기서 `println!`은 출력 매크로입니다.
}
```

- **`fn main()`**: Rust 프로그램의 진입점입니다. 모든 실행은 이 함수에서 시작됩니다.
- **`println!`**: 문자열을 출력하는 매크로입니다. C 언어의 `printf`와 유사하지만, 더 안전하고 타입 안전을 제공합니다.

### 실행 방법
터미널에서 해당 파일이 있는 디렉토리로 이동 후:
```bash
rustc hello_world.rs
./hello_world
```
출력:
```
Hello, Rust 세계!
```

### 💡 초보자 폭풍 질문!
**Q: `println!` 대신 다른 방법으로 출력할 수 있나요?**

**A:** 네, 물론이죠! Rust에는 다양한 출력 방식이 있습니다. 예를 들어, 문자열 포맷팅을 사용하는 방법도 있어요:

#### 코드 예제 2: 포맷팅된 출력
```rust
fn main() {
    let name = "Rust";
    println!("환영합니다, {}!", name);  // 변수 삽입
}
```

- **`{}`**: 플레이스홀더로, 여기에 변수 값이 삽입됩니다.

## 소유권과 기본 데이터 타입 이해하기

Rust의 핵심 개념 중 하나는 **소유권(Ownership)**입니다. 이 개념을 이해하면 메모리 관리의 복잡성을 크게 줄일 수 있어요.

### 기본 데이터 타입
- **정수**: `i32`, `u32` 등
- **문자**: `char`
- **불리언**: `bool`

#### 코드 예제 3: 기본 데이터 타입 사용
```rust
fn main() {
    let age = 25u32;  // `u32`: 부호 없는 32비트 정수
    let name = "Rustacean";  // 문자열 슬라이스 (string slice)
    let is_beginner = true;  // 불리언 타입

    println!("나이: {}, 이름: {}, 초보자인가요? {}", age, name, is_beginner);
}
```

- **`let`**: 변수를 선언하는 키워드입니다.
- **타입 명시**: Rust는 정적 타이핑 언어이므로, 타입을 명시하는 것이 일반적입니다 (하지만 컴파일러가 종종 타입을 추론합니다).

### 🚨 실무주의보
**소유권과 생명주기 이해의 중요성**:
- 소유권 규칙을 잘 이해하면 메모리 누수나 데이터 레이스 같은 문제를 미리 방지할 수 있습니다. 초기에는 조금 복잡하게 느껴질 수 있지만, 장기적으로는 코드의 안정성과 성능을 크게 향상시킵니다.
