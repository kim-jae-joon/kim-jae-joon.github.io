## 연산자: 데이터 마법사의 주문

연산자는 숫자, 문자열, 불리언 값 등을 조작하고 비교하는 데 사용되는 특별한 기호들입니다. Rust에서는 다양한 종류의 연산자가 있어, 코드를 더욱 풍부하고 유연하게 만들어 줍니다. 그럼, 몇 가지 주요 연산자들을 하나씩 살펴볼게요!

### 산술 연산자: 숫자의 춤

산술 연산자는 숫자를 다루는 기본적인 마법 주문입니다.

```rust
fn main() {
    let a = 10; // 첫 번째 숫자, 10을 저장
    let b = 3;  // 두 번째 숫자, 3을 저장

    let sum = a + b; // 덧셈: 두 숫자를 더합니다
    let diff = a - b; // 뺄셈: 첫 번째 숫자에서 두 번째 숫자를 뺍니다
    let prod = a * b; // 곱셈: 두 숫자를 곱합니다
    let quot = a / b; // 나눗셈: 첫 번째 숫자를 두 번째 숫자로 나눕니다
    let rem = a % b;  // 나머지: 나눗셈의 나머지를 구합니다

    println!("합: {}", sum); // 출력: 합: 13
    println!("차: {}", diff); // 출력: 차: 7
    println!("곱: {}", prod); // 출력: 곱: 30
    println!("몫: {}", quot); // 출력: 몫: 3
    println!("나머지: {}", rem); // 출력: 나머지: 1
}
```

**코드 설명:**
- `a + b`: 두 숫자를 더하는 마법 주문입니다.
- `a - b`: 첫 번째 숫자에서 두 번째 숫자를 빼는 주문입니다.
- `a * b`: 두 숫자를 곱하는 주문입니다.
- `a / b`: 나눗셈을 통해 몫을 얻습니다.
- `a % b`: 나눗셈의 나머지를 구하는 주문입니다.

**💡 초보자 폭풍 질문!**
- **Q: 나눗셈에서 실수 결과를 얻고 싶다면 어떻게 해야 하나요?**
  - **A:** Rust에서는 `f64` 타입을 사용하면 됩니다. 예를 들어:
    ```rust
    let a: f64 = 10.0;
    let b: f64 = 3.0;
    let float_quot = a / b; // 실수 나눗셈
    println!("실수 나눗셈 결과: {}", float_quot); // 출력: 실수 나눗셈 결과: 3.3333333333333335
    ```

### 비교 연산자: 진실의 검

비교 연산자는 두 값이 같은지, 더 큰지, 작은지 등을 판단하는 데 사용됩니다.

```rust
fn main() {
    let x = 5;
    let y = 10;

    let is_equal = x == y; // 같음 비교
    let is_not_equal = x != y; // 같지 않음 비교
    let is_greater = x > y; // x가 y보다 큰지
    let is_less = x < y; // x가 y보다 작은지
    let is_greater_or_equal = x >= y; // x가 y보다 크거나 같은지
    let is_less_or_equal = x <= y; // x가 y보다 작거나 같은지

    println!("x == y: {}", is_equal); // 출력: x == y: false
    println!("x != y: {}", is_not_equal); // 출력: x != y: true
    println!("x > y: {}", is_greater); // 출력: x > y: false
    println!("x < y: {}", is_less); // 출력: x < y: true
    println!("x >= y: {}", is_greater_or_equal); // 출력: x >= y: false
    println!("x <= y: {}", is_less_or_equal); // 출력: x <= y: true
}
```

**코드 설명:**
- `==`: 두 값이 같은지 확인합니다.
- `!=`: 두 값이 다른지 확인합니다.
- `>`: 첫 번째 값이 두 번째 값보다 큰지 확인합니다.
- `<`: 첫 번째 값이 두 번째 값보다 작은지 확인합니다.
- `>=`: 첫 번째 값이 두 번째 값보다 크거나 같은지 확인합니다.
- `<=`: 첫 번째 값이 두 번째 값보다 작거나 같은지 확인합니다.

**🚨 실무주의보**
- 비교 연산자는 조건문이나 반복문에서 자주 사용됩니다. 예를 들어, 사용자 입력을 검증할 때 매우 유용합니다.

### 논리 연산자: 복잡한 판단의 힘

논리 연산자는 불리언 값(참/거짓)을 조합하는 데 사용됩니다.

```rust
fn main() {
    let flag1 = true;
    let flag2 = false;

    let and_result = flag1 && flag2; // AND 연산자: 둘 다 참일 때만 참
    let or_result = flag1 || flag2;  // OR 연산자: 하나라도 참이면 참
    let not_result = !flag1;         // NOT 연산자: 참이면 거짓, 거짓이면 참

    println!("AND 결과: {}", and_result); // 출력: AND 결과: false
    println!("OR 결과: {}", or_result);  // 출력: OR 결과: true
    println!("NOT 결과: {}", not_result); // 출력: NOT 결과: false
}
```

**코드 설명:**
- `&&`: 두 조건이 모두 참일 때만 참을 반환합니다.
- `||`: 두 조건 중 하나라도 참이면 참을 반환합니다.
- `!`: 불리언 값을 반전시킵니다.

**💡 초보자 폭풍 질문!**
- **Q: 논리 연산자를 사용해 복잡한 조건을 어떻게 만드나요?**
  - **A:** 예를 들어, 사용자가 나이가 18세 이상이고, 회원가입을 완료했는지 확인하는 조건을 만들어 볼 수 있습니다:
    ```rust
    let age = 20;
    let is_member = true;

    if age >= 18 && is_member {
        println!("환영합니다, 회원님!");
    } else {
        println!("조건을 충족하지 못했습니다.");
    }
    ```

### 할당 연산자: 데이터의 이동 마법사

할당 연산자는 변수에 값을 할당하는 데 사용되며, 때로는 값을 계산하고 할당하는 데도 활용됩니다.

```rust
fn main() {
    let mut score = 10; // 초기 점수 설정

    // 단순 할당
    score = 20;
    println!("단순 할당 후 점수: {}", score); // 출력: 단순 할당 후 점수: 20

    // 덧셈 후 할당
    score += 5; // score = score + 5와 동일
    println!("덧셈 후 할당 후 점수: {}", score); // 출력: 덧셈 후 할당 후 점수: 25

    // 뺄셈 후 할당
    score -= 3; // score = score - 3와 동일
    println!("뺄셈 후 할당 후 점수: {}", score); // 출력: 뺄셈 후 할당 후 점수: 22

    // 곱셈 후 할당
    score *= 2; // score = score * 2와 동일
    println!("곱셈 후 할당 후 점수: {}", score); // 출력: 곱셈 후 할당 후 점수: 44

    // 나눗셈 후 할당
    score /= 4; // score = score / 4와 동일
    println!("나눗셈 후 할당 후 점수: {}", score); // 출력: 나눗셈 후 할당 후 점수: 11
}
```

**코드 설명:**
- `=`: 기본 할당 연산자입니다.
- `+=`, `-=`, `*=`, `/=`: 각각 덧셈 후 할당, 뺄셈 후 할당, 곱셈 후 할당, 나눗셈 후 할당을 수행합니다.

**🚨 실무주의보**
- 할당 연산자는 코드를 간결하게 만들어주지만, 명확성을 잃지 않도록 주의하세요. 너무 복잡한 표현은 유지보수에 어려움을 줄 수 있습니다.
