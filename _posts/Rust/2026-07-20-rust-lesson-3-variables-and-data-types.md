# 3강: Rust 기초: 변수와 데이터 타입 - 당신의 코딩 여정의 첫걸음!

안녕하세요, Rust의 신비로운 세계로 안내할 5년차 주니어 개발자입니다! 오늘은 Rust 프로그래밍의 핵심 중 핵심, 바로 **변수와 데이터 타입**에 대해 함께 탐험해볼 시간입니다. 코딩의 첫걸음을 내딛는 당신을 위해, 정말 쉽고 재미있게 설명해볼게요. 준비됐나요? 그럼 시작해볼까요!

## 왜 변수가 중요한가요?

생각해보세요. 당신이 요리를 하는데 재료를 일일이 "오늘 쓸 재료 목록"이라고 부르는 대신, "양파", "대파", "소금" 같은 이름을 붙여놓으면 훨씬 더 직관적이고 관리하기 쉽겠죠? 변수는 바로 그런 역할을 합니다. 프로그램 내에서 정보를 저장하고 관리하는 '이름표'라고 생각하면 됩니다.

### 기본 변수 선언하기

Rust에서 변수를 선언하는 방법은 간단합니다. 다음 예제를 보세요:

```rust
// 변수 선언 예시
fn main() {
    // 정수형 변수 `age` 선언 및 초기화
    let age = 25;  // 나이를 25로 설정
    println!("제 나이는 {}살입니다.", age);  // 출력: 제 나이는 25살입니다.

    // 문자열 변수 `name` 선언 및 초기화
    let name = "Rusty";  // 이름을 "Rusty"로 설정
    println!("제 이름은 {}입니다.", name);  // 출력: 제 이름은 Rusty입니다.
}
```

- `let age = 25;`: `let` 키워드는 변수를 선언하는 데 사용되며, `age`라는 이름의 변수에 정수 `25`를 저장합니다.
- `println!` 매크로는 출력을 위한 함수로, 중괄호 `{}` 안에 변수 값을 삽입할 수 있습니다.

### 데이터 타입의 중요성

Rust는 강력한 타입 시스템을 가지고 있어, 변수에 어떤 종류의 데이터를 저장할지 명확히 지정해야 합니다. 이건 왜 중요할까요?

- **안전성**: 타입을 명시함으로써 컴파일 타임에 오류를 잡아낼 수 있습니다. 예를 들어, 숫자를 문자열로 잘못 처리하는 실수를 미리 방지할 수 있습니다.
- **성능**: 컴파일러가 데이터 타입을 알고 있으면 최적화를 더 잘 할 수 있습니다.

#### 주요 데이터 타입들

1. **정수형 (Integer Types)**
   ```rust
   let year: i32 = 2023;  // `i32`는 32비트 정수형
   println!("현재 연도: {}", year);
   ```
   - `i32`, `u32`, `i64`, `u64` 등 다양한 크기의 정수 타입이 있습니다. `i`는 부호 있는 정수, `u`는 부호 없는 정수를 의미합니다.

2. **부동소수점 (Floating-Point Types)**
   ```rust
   let pi: f64 = 3.14159;  // `f64`는 64비트 부동소수점
   println!("파이의 값: {:.2}", pi);  // 소수점 둘째 자리까지 출력
   ```
   - `f32`와 `f64`가 주로 사용되며, 부동소수점 숫자를 나타냅니다.

3. **문자열 (String Types)**
   ```rust
   let greeting: String = String::from("Hello, Rust!");
   println!("{}", greeting);
   ```
   - `String` 타입은 가변적인 문자열을 다룹니다. `String::from` 메서드를 사용해 문자열 리터럴을 생성할 수 있습니다.

### 💡 초보자 폭풍 질문!

**Q: 변수명을 어떻게 정해야 할까요?**

A: 변수명은 의미 있고 명확해야 합니다. 예를 들어, `userAge` 대신 `age`만 쓰는 것보다는 `userAge`처럼 그 변수가 무엇을 나타내는지 명확히 해주는 것이 좋습니다. 하지만 너무 길게 만들지 않도록 주의하세요!

### 🚨 실무주의보

**타입 어노테이션의 중요성**

Rust에서는 명시적인 타입 어노테이션이 매우 중요합니다. 컴파일 타임 오류를 줄이고 코드의 가독성을 높이는 데 큰 도움이 됩니다. 예를 들어, 다음과 같이 타입을 명시하지 않으면 컴파일 에러가 발생할 수 있습니다:

```rust
// 컴파일 에러 예시 (타입 명시 누락)
fn main() {
    let x = 5;  // 타입이 자동으로 추론되지만, 명시적으로 지정하는 것이 좋습니다.
    let y = "10";
    // 아래 줄은 컴파일 에러를 발생시킬 수 있습니다 (타입 불일치)
    // println!("합: {}", x + y);  // 에러: 덧셈 연산은 타입이 일치해야 합니다.
}
```

대신, 타입을 명시하면 더 안전합니다:

```rust
fn main() {
    let x: i32 = 5;
    let y: i32 = 10;
    println!("합: {}", x + y);  // 정상적으로 작동
}
```

## 실전 활용 예제: 가계부 앱

이제 실제로 변수와 데이터 타입을 활용해 간단한 가계부 앱을 만들어 볼까요?

### 코드 예제: 가계부 앱

```rust
fn main() {
    // 사용자 정보
    let user_name: String = String::from("Rusty");
    let user_age: u32 = 28;

    // 가계부 항목
    let mut balance: f64 = 1000.50;  // 초기 잔액

    // 지출 항목
    let expense_item: String = String::from("커피");
    let expense_amount: f64 = 3.50;

    // 지출 처리
    balance -= expense_amount;
    println!("{}님의 현재 잔액: ${:.2}", user_name, balance);

    // 영수증 출력
    println!("지출 내역: {} - ${:.2}", expense_item, expense_amount);
}
```

- **변수 설명**:
  - `user_name`: 사용자의 이름을 저장하는 문자열 변수.
  - `user_age`: 사용자의 나이를 저장하는 부호 없는 정수 변수.
  - `balance`: 현재 잔액을 저장하는 부동소수점 변수 (`mut` 키워드로 가변성 부여).
  - `expense_item`: 지출 항목을 저장하는 문자열 변수.
  - `expense_amount`: 지출 금액을 저장하는 부동소수점 변수.

이 코드를 통해 간단한 가계부 기능을 구현해보았습니다. 변수와 데이터 타입이 어떻게 실제 애플리케이션에 적용되는지 이해하셨나요?
