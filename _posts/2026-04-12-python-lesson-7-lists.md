# 파이썬 기초: 리스트(list)

안녕하세요! 오늘은 파이썬에서 매우 중요한 자료구조인 '리스트(list)'에 대해 알아보도록 하겠습니다. 목록이나 배열 같은 개념과 비슷하며, 여러 아이템을 하나의 변수 안에 저장할 수 있게 해줍니다.

## 1. 리스트란 무엇인가요?

리스트는 파이썬에서 가장 기본적인 자료구조 중 하나로, 순서가 있는 변경 가능한 요소들의 컬렉션입니다. 각 요소는 인덱스를 통해 접근하며, 리스트에는 같은 유형의 데이터뿐만 아니라 다른 유형의 데이터도 함께 저장할 수 있습니다.

**예제 코드:**
```python
# 리스트 생성
fruits = ['사과', '바나나', '오렌지']

# 리스트 출력
print(fruits)  # ['사과', '바나나', '오렌지']
```

## 2. 리스트에 접근하기

리스트의 요소는 인덱스를 통해 접근할 수 있습니다. 파이썬에서 인덱싱은 0부터 시작합니다.

**예제 코드:**
```python
# 첫 번째 요소에 접근
print(fruits[0])  # 사과

# 마지막 요소에 접근
print(fruits[-1])  # 오렌지
```

## 3. 리스트 슬라이싱하기

슬라이싱은 리스트의 일부분을 추출하는 방법입니다. 시작 인덱스부터 끝 인덱스까지를 지정하여 원하는 부분만 가져올 수 있습니다.

**예제 코드:**
```python
# 첫 번째 요소부터 두 번째 요소까지만 슬라이싱
print(fruits[0:2])  # ['사과', '바나나']

# 마지막 두 요소를 가져오기
print(fruits[-2:])  # ['바나나', '오렌지']
```

## 4. 리스트에 아이템 추가하기

리스트는 동적으로 요소를 추가하거나 삭제할 수 있습니다. `append()` 메서드로 맨 뒤에 새로운 요소를 추가할 수 있습니다.

**예제 코드:**
```python
# 리스트에 새로운 과일 '키위' 추가
fruits.append('키위')
print(fruits)  # ['사과', '바나나', '오렌지', '키위']
```

## 5. 리스트에서 아이템 삭제하기

`remove()` 메서드를 사용하여 특정 요소를 삭제할 수 있습니다.

**예제 코드:**
```python
# '오렌지'라는 과일을 삭제
fruits.remove('오렌지')
print(fruits)  # ['사과', '바나나', '키위']
```

## 6. 리스트 정렬하기

리스트를 오름차순이나 내림차순으로 정렬할 수 있습니다. `sort()` 메서드를 사용하면 원본 리스트가 변경되고, `sorted()` 함수는 새로운 정렬된 리스트를 반환합니다.

**예제 코드:**
```python
# 오름차순 정렬
fruits.sort()
print(fruits)  # ['키위', '사과', '바나나']

# 내림차순 정렬
fruits.sort(reverse=True)
print(fruits)  # ['바나나', '사과', '키위']
```

## 7. 리스트 합치기

두 개 이상의 리스트를 합쳐 새로운 리스트를 만들 수 있습니다. `+` 연산자를 사용하거나, `extend()` 메서드를 사용할 수 있습니다.

**예제 코드:**
```python
# 두 리스트를 + 연산자로 합치기
list1 = [1, 2, 3]
list2 = [4, 5, 6]
combined_list = list1 + list2
print(combined_list)  # [1, 2, 3, 4, 5, 6]

# extend() 메서드로 합치기
list1.extend(list2)
print(list1)  # [1, 2, 3, 4, 5, 6]
```

## 8. 리스트 반복하기

리스트를 반복하여 각 요소에 접근하거나 작업을 수행할 수 있습니다.

**예제 코드:**
```python
# for문으로 리스트의 각 요소 출력하기
for fruit in fruits:
    print(fruit)

# 결과:
# 바나나
# 사과
# 키위
```

## 9. 리스트 길이 구하기

리스트에 포함된 요소 개수를 구하려면 `len()` 함수를 사용합니다.

**예제 코드:**
```python
print(len(fruits))  # 3
```
