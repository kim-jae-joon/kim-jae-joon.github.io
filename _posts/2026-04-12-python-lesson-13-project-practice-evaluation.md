# 파이썬 프로젝트 실습 및 평가

안녕하세요! 오늘은 초보자를 위한 파이썬 프로젝트 실습과 평가에 대해 다루겠습니다. 이번 과정에서는 기본적인 파이썬 문법을 복습하고, 간단한 프로젝트를 만들어 보면서 실제 코딩 능력을 향상시켜보도록 하겠습니다.

## 1. 파이썬 프로젝트 선택

파이썬은 다양한 분야에서 활용할 수 있는 강력한 언어입니다. 프로젝트 선택에 있어서는 다음과 같은 조건을 고려해 보세요:

- **개인의 관심사**: 자신이 가장 좋아하는 주제를 선택하면 더욱 즐겁게 할 수 있습니다.
- **복잡성**: 초보자라면 간단한 프로젝트부터 시작해보는 것이 좋습니다. 점차 복잡도를 높여나갈 수 있습니다.
- **실용성**: 배운 것을 실제로 적용할 수 있는 프로젝트가 더 의미가 큽니다.

## 2. 기본적인 파이썬 문법 복습

프로젝트 시작 전에, 파이썬의 기본적인 문법을 간단히 리뷰해보겠습니다. 다음은 몇 가지 핵심 개념과 예제 코드입니다.

### 2.1 변수와 자료형
```python
# 변수 선언
name = "Alice"
age = 30

# 데이터 타입 확인
print(type(name))  # <class 'str'>
print(type(age))   # <class 'int'>
```

### 2.2 조건문
```python
# if-else 문
if age > 18:
    print("성인입니다.")
else:
    print("미성년자입니다.")
```

### 2.3 반복문
```python
# for 문
for i in range(5):
    print(i)

# while 문
count = 0
while count < 5:
    print(count)
    count += 1
```

### 2.4 함수
```python
def greet(name):
    return f"안녕, {name}!"

print(greet("Alice"))  # 안녕, Alice!
```

## 3. 간단한 프로젝트 실습

이제 실제로 파이썬을 사용해 보면서 배운 내용을 적용해보겠습니다. 예를 들어, **온도 변환 계산기**라는 프로젝트를 만들어 보도록 하겠습니다.

### 3.1 프로젝트 요구사항
- 섭씨 온도를 입력받아 화씨 온도로 변환하는 기능
- 결과를 출력

### 3.2 코드 작성
```python
# 온도 변환 함수 정의
def celsius_to_fahrenheit(celsius):
    return (celsius * 9/5) + 32

# 사용자로부터 섭씨 온도 입력 받기
celsius = float(input("섭씨 온도를 입력하세요: "))

# 화씨 온도 계산
fahrenheit = celsius_to_fahrenheit(celsius)

# 결과 출력
print(f"{celsius}°C는 {fahrenheit}°F입니다.")
```

### 3.3 실행 결과
```python
섭씨 온도를 입력하세요: 25
25°C는 77.0°F입니다.
```

## 4. 프로젝트 평가

프로젝트를 완성한 후에는 다음과 같은 점을 고려해보세요:

- **코드 가독성**: 변수 이름이 명확하고, 코드가 간결하게 작성되었는지 확인합니다.
- **함수 사용**: 필요한 부분에 함수를 정의하여 중복 코드를 줄였는지 검토합니다.
- **오류 처리**: 입력값이 올바르게 들어왔는지 예외 상황을 고려한 처리가 되었는지 점검합니다.

## 5. 추가 도전과제

프로젝트를 완성했다면, 다음의 도전 과제들을 시도해보세요:

- **단위 변환 계산기**: 다양한 단위(길이, 무게 등)를 변환할 수 있는 프로그램을 작성해 보세요.
- **가계부 앱**: 월간 지출을 기록하고 총 합계를 계산하는 간단한 가계부 앱을 만들어보세요.
