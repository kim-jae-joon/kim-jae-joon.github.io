**7강: 함수와 람다 함수**

함수는 코딩에서 가장 기본이면서도 강력한 개념입니다. 오늘은 함수와 람다 함수의 기초부터 심화까지 전반적인 내용을 담아 보겠습니다.

**1. 함수란 무엇인가? 🤔**

함수는 일련의 코드를 묶어 하나의 단위로 만든 것을 말합니다. 이 코드 단위가 특정 동작을 수행하거나, 데이터를 처리하는 등의 작업을 함으로써 다양한 목적에 사용됩니다.

**1.1 함수의 용도 🔥**

- 프로그램의 일부 코드를 다시 호출할 수 있게 만드는 기능
- 반복적인 코드를 줄여서 유지보수와 확장성을 향상시키는 기능

**1.2 함수의 구성 📝**

함수는 다음과 같은 구조로 구성됩니다.

```python
def 함수명(매개변수):
    # 함수 본문(코드)
```

- `def` 키워드로 함수를 정의합니다.
- 함수명은 함수를 식별하기 위한 이름입니다.
- 매개변수는 함수에게 전달되는 값으로, 함수 내에서 사용됩니다.

**2. 람다 함수 🔥**

람다 함수는 이름이 없는 함수를 뜻합니다. 작은 코드 블록을 즉시 실행할 때 사용하는 함수입니다. 람다 함수는 `lambda` 키워드를 사용하여 정의합니다.

```python
lamda 매개변수: 표현식
```

- `lambda` 키워드로 람다 함수를 정의합니다.
- 매개변수는 함수에게 전달되는 값으로, 함수 내에서 사용됩니다.
- 표현식은 리턴할 값을 지정합니다.

**3. 예제 1 - 람다 함수 🔍**

```python
numbers = [1, 2, 3, 4, 5]
double_numbers = list(map(lambda x: x*2, numbers))
print(double_numbers) # [2, 4, 6, 8, 10]
```

- `map()` 함수는 지정된 함수를 입력 자료형에 적용합니다.
- 람다 함수 `lambda x: x*2`는 각 원소를 두 배로 만드는 기능을 합니다.

**4. 예제 2 - 내장함수 🔍**

```python
def sum_numbers(numbers):
    return sum(numbers)

numbers = [1, 2, 3, 4, 5]
print(sum_numbers(numbers)) # 15
```

- `sum()` 함수는 입력 자료형의 원소들을 더해주는 기능을 합니다.
- 내장함수는 파이썬에서 이미 구현되어 있는 함수를 말합니다.

**5. 실무 예제 - 정렬과 필터링 🔥**

```python
students = [
    {"name": "김철수", "age": 15},
    {"name": "이영희", "age": 16},
    {"name": "박자바", "age": 17}
]

# 학생 이름을 오름차순으로 정렬한 결과
sorted_students = sorted(students, key=lambda x: x['name'])
print(sorted_students)
# [
#     {'name': '김철수', 'age': 15},
#     {'name': '이영희', 'age': 16},
#     {'name': '박자바', 'age': 17}
# ]

# 학생 이름이 '김'으로 시작하는 학생들
filtered_students = list(filter(lambda x: x['name'].startswith('김'), students))
print(filtered_students)
# [{'name': '김철수', 'age': 15}]
```

- `sorted()` 함수는 입력 자료형의 원소들을 정렬한 결과를 반환합니다.
- `filter()` 함수는 지정된 조건을 만족하는 원소들만 추출하여 반환합니다.

**🚨 실무주의보!**
