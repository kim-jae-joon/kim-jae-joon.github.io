# 1강: 파이썬 소개 및 설치

안녕하세요! 본 강의에서는 초보자를 위한 파이썬에 대한 기본적인 소개와 설치 방법을 다루겠습니다. 파이썬은 쉽고 직관적인 문법으로 많은 분야에서 활용되는 인기 있는 프로그래밍 언어입니다. 이번 강의를 통해 파이썬의 특징과 장점, 그리고 설치 과정을 알아보도록 하겠습니다.

## 1. 파이썬이란?

파이썬은 1991년 제안된 고급 프로그래밍 언어로, 인간이 읽고 쓸 수 있는 코드를 작성하기 위한 인터프리터 기반의 동적 타입 언어입니다. 파이썬은 간결한 문법으로 복잡한 알고리즘을 쉽게 구현할 수 있어 다양한 분야에서 활용되고 있습니다.

| 특징      | 설명                                |
|---------|-------------------------------------|
| 간단한 문법   | 영어 단어를 기반으로 한 명령어 사용        |
| 다중 프로그래밍 언어 지원 | 객체지향, 함수형 등 다양한 스타일 지원     |
| 크로스 플랫폼  | 윈도우, 맥OS, 리눅스 등 다양한 운영 체제에서 실행 가능 |

## 2. 파이썬의 장점

파이썬은 다음과 같은 이유로 많은 사람들에게 사랑받고 있습니다.

- **높은 생산성**: 간결한 문법으로 코드를 작성할 수 있어 개발 속도가 빠릅니다.
- **다양한 라이브러리**: 데이터 분석, 웹 개발, 머신 러닝 등 다양한 분야에서 사용 가능한 다양한 라이브러리를 제공합니다.
- **커뮤니티 지원**: 활성화된 커뮤니티로 언제든지 도움을 받을 수 있습니다.

## 3. 파이썬 설치

파이썬을 설치하기 전에 운영 체제를 확인해주세요. 아래에서는 윈도우와 맥OS에서 파이썬을 설치하는 방법을 설명하겠습니다.

### 3.1 윈도우에서 파이썬 설치

1. 파이썬 공식 웹사이트(https://www.python.org/)로 이동합니다.
2. **Downloads** 메뉴를 클릭하고 **Windows installer (64-bit)** 또는 **Windows installer (32-bit)**를 다운로드합니다.
3. 다운로드한 설치 파일을 실행합니다.
4. **Add Python 3.x to PATH** 옵션을 체크하고 **Install Now** 버튼을 클릭합니다.

### 3.2 맥OS에서 파이썬 설치

1. 터미널을 엽니다.
2. 다음 명령어를 실행하여 Homebrew 패키지 관리자를 설치합니다:

    ```bash
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```

3. Homebrew를 사용하여 파이썬을 설치합니다:

    ```bash
    brew install python
    ```

## 4. 파이썬 실행

파이썬이 성공적으로 설치되었다면, 터미널이나 명령 프롬프트에서 다음과 같이 입력해보세요.

```python
print("Hello, Python!")
```

**출력:**

```
Hello, Python!
```

이번 강의에서는 파이썬에 대한 기본적인 소개와 설치 방법을 살펴봤습니다. 간단한 문법과 다양한 라이브러리 덕분에 파이썬은 많은 분야에서 활용되며, 앞으로도 계속해서 성장할 것으로 예상됩니다.
