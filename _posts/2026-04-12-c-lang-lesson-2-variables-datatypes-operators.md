## 🚀 2강: C 언어의 기본기: 변수, 자료형 그리고 연산자

### 1. 변수 (Variable): 데이터가 머무는 집

프로그램을 만들다 보면 수많은 데이터를 다루게 됩니다. 예를 들어, 사용자의 나이, 게임 점수, 학생의 학점 등이죠. 이 데이터들을 컴퓨터 메모리 어딘가에 저장하고, 필요할 때마다 꺼내 쓰고, 또 값을 변경하기도 해야 합니다.

이때 필요한 것이 바로 **변수**입니다! 변수는 말 그대로 '값이 변할 수 있는 수'라는 뜻이지만, 사실 더 정확하게는 **데이터를 저장하는 메모리 공간에 붙여진 '이름'**이라고 생각하시면 돼요.

상상해 보세요. 메모리라는 넓은 창고에 수많은 빈 상자들이 있습니다. 이 상자들 중 하나에 '나이'를 저장하고 싶다면, 그 상자에 "age"라는 이름표를 붙이는 거죠. 그리고 그 안에 숫자 30을 넣으면 됩니다.

#### 📝 변수 선언과 초기화

C 언어에서 변수를 사용하려면 먼저 어떤 데이터를 저장할지 컴퓨터에게 알려줘야 합니다. 이를 **변수 선언(Declaration)**이라고 합니다. 그리고 변수를 선언하면서 처음으로 값을 넣어주는 것을 **변수 초기화(Initialization)**라고 해요.

**[변수 선언 문법]**
```c
자료형 변수이름;
```

**[변수 초기화 문법]**
```c
자료형 변수이름 = 초기값;
```

**[변수 이름 짓는 규칙 (식별자 규칙)]**
변수 이름을 지을 때는 몇 가지 규칙을 지켜야 해요.
*   **알파벳, 숫자, 언더스코어(`_`)**만 사용할 수 있습니다.
*   **숫자로 시작할 수 없습니다.** (예: `1age`는 안 됨, `age1`은 됨)
*   **공백을 포함할 수 없습니다.** (예: `my age`는 안 됨, `my_age`는 됨)
*   C 언어에서 미리 정의된 **예약어(Keywords)**는 사용할 수 없습니다. (예: `int`, `if`, `for` 등)
*   **대소문자를 구분**합니다. (`age`와 `Age`는 다른 변수입니다!)

#### 💡 예제 코드: 변수 선언 및 사용

```c
#include <stdio.h> // printf 함수를 사용하기 위해 포함하는 헤더 파일입니다. (나중에 자세히 설명!)

int main() {
    // 1. 변수 선언: 'score'라는 이름을 가진 정수형 상자를 준비합니다.
    int score; 

    // 2. 변수 초기화: 준비된 상자에 100이라는 값을 넣습니다.
    score = 100;

    // 변수 'score'의 값을 출력합니다.
    // %d는 '정수형 데이터를 출력할 거야!'라고 printf 함수에게 알려주는 서식 지정자입니다.
    printf("현재 점수는 %d점입니다.\n", score); 

    // 3. 변수 값 변경: 상자 안의 값을 95로 바꿉니다.
    score = 95;
    printf("변경된 점수는 %d점입니다.\n", score);

    // 선언과 동시에 초기화할 수도 있습니다.
    int total_students = 30; // 'total_students'라는 정수형 변수를 선언하고 30으로 초기화
    printf("전체 학생 수는 %d명입니다.\n", total_students);

    return 0; // 프로그램이 성공적으로 끝났음을 운영체제에 알립니다.
}
```

**[실행 결과]**
```
현재 점수는 100점입니다.
변경된 점수는 95점입니다.
전체 학생 수는 30명입니다.
```

### 2. 자료형 (Data Type): 상자의 종류와 크기

메모리 창고에 있는 상자에 어떤 종류의 데이터를 넣을지에 따라 상자의 크기와 모양이 달라져야겠죠? 예를 들어, 작은 숫자 하나를 넣는데 엄청 큰 상자를 준비하면 공간 낭비일 테고, 아주 큰 숫자를 넣는데 작은 상자를 준비하면 다 들어가지 않을 거예요.

**자료형**은 변수에 저장할 **데이터의 종류** (정수, 실수, 문자 등) 와 **크기** (메모리를 얼마나 차지할지)를 정의합니다. C 언어의 주요 기본 자료형은 다음과 같습니다.

| 자료형 (Type) | 설명 (Description)                                 | 저장하는 데이터 예시      | 크기 (바이트) | 서식 지정자 |
| :------------ | :------------------------------------------------- | :------------------------ | :------------ | :---------- |
| `int`         | **정수** (Integer): 양수, 음수, 0                | `10`, `-5`, `0`           | 4 바이트      | `%d`        |
| `char`        | **문자** (Character): 한 글자                      | `'A'`, `'z'`, `'5'`       | 1 바이트      | `%c`        |
| `float`       | **실수** (Floating-point): 소수점 있는 수 (단정밀도) | `3.14f`, `-0.5f`          | 4 바이트      | `%f`        |
| `double`      | **실수** (Double floating-point): 소수점 있는 수 (배정밀도, 더 정밀함) | `123.456`, `9.806`        | 8 바이트      | `%lf`       |

> **Tip:** C 언어의 `char`는 사실 문자의 ASCII 코드 값(정수)을 저장하는 자료형입니다. 그래서 정수처럼 다룰 수도 있어요! 그리고 `float`형 상수에 `f`를 붙여주는 이유는, 그냥 `3.14`라고 쓰면 기본적으로 `double`형으로 인식하기 때문입니다. 명확하게 `float`형이라고 알려주는 거죠.

#### 💡 예제 코드: 다양한 자료형 사용 및 크기 확인

`sizeof` 연산자를 사용하면 각 자료형이나 변수가 메모리에서 몇 바이트를 차지하는지 알아볼 수 있습니다.

```c
#include <stdio.h>

int main() {
    // 다양한 자료형의 변수 선언 및 초기화
    int     age = 25;
    char    grade = 'A'; // 문자는 반드시 작은따옴표(')로 감쌉니다.
    float   pi_float = 3.141592f; // float형 상수임을 명시하기 위해 'f'를 붙입니다.
    double  pi_double = 3.1415926535; // double은 float보다 더 정밀한 소수점 값을 저장합니다.

    printf("나이: %d살\n", age);
    printf("학점: %c\n", grade);
    printf("파이 (float): %f\n", pi_float);
    printf("파이 (double): %lf\n", pi_double); // double형은 출력 시 %lf를 사용합니다.

    printf("\n--- 각 자료형의 크기 확인 (바이트) ---\n");
    printf("int 의 크기: %lu 바이트\n", sizeof(int));      // %lu는 unsigned long int를 출력할 때 사용합니다.
    printf("char 의 크기: %lu 바이트\n", sizeof(char));
    printf("float 의 크기: %lu 바이트\n", sizeof(float));
    printf("double 의 크기: %lu 바이트\n", sizeof(double));

    // 변수의 크기를 직접 확인할 수도 있습니다.
    printf("age 변수의 크기: %lu 바이트\n", sizeof(age));

    return 0;
}
```

**[실행 결과]** (컴퓨터 환경에 따라 `int`의 크기가 다를 수 있습니다. 보통은 4바이트입니다.)
```
나이: 25살
학점: A
파이 (float): 3.141592
파이 (double): 3.141593

--- 각 자료형의 크기 확인 (바이트) ---
int 의 크기: 4 바이트
char 의 크기: 1 바이트
float 의 크기: 4 바이트
double 의 크기: 8 바이트
age 변수의 크기: 4 바이트
```

### 3. 연산자 (Operator): 데이터를 다루는 도구

변수와 자료형으로 데이터를 저장했다면, 이제 이 데이터를 가지고 덧셈, 뺄셈, 비교 등 다양한 작업을 수행해야겠죠? 이때 필요한 것이 바로 **연산자**입니다. 연산자는 데이터를 가지고 특정한 작업을 수행하게 해주는 '도구'라고 생각하시면 돼요.

C 언어에는 정말 다양한 연산자들이 있지만, 오늘은 초보자들이 가장 많이 사용하는 핵심 연산자들을 위주로 살펴볼게요!

#### 3.1. 산술 연산자 (Arithmetic Operators)

가장 익숙한 사칙연산과 나머지를 구하는 연산자입니다.

| 연산자 | 설명     | 예시         | 결과  |
| :----- | :------- | :----------- | :---- |
| `+`    | 덧셈     | `10 + 3`     | `13`  |
| `-`    | 뺄셈     | `10 - 3`     | `7`   |
| `*`    | 곱셈     | `10 * 3`     | `30`  |
| `/`    | 나눗셈   | `10 / 3`     | `3`   |
| `%`    | 나머지   | `10 % 3`     | `1`   |

> **주의!** 정수와 정수의 나눗셈(`int / int`)은 결과도 **정수**입니다. 소수점 이하를 버리고 정수 부분만 취해요. 만약 실수 결과를 원한다면 적어도 둘 중 하나를 `float`나 `double`로 바꿔줘야 합니다. (예: `10.0 / 3` 또는 `(double)10 / 3`)

#### 💡 예제 코드: 산술 연산자

```c
#include <stdio.h>

int main() {
    int a = 10;
    int b = 3;

    printf("a + b = %d\n", a + b); // 13
    printf("a - b = %d\n", a - b); // 7
    printf("a * b = %d\n", a * b); // 30
    printf("a / b = %d (정수 나눗셈)\n", a / b); // 3 (소수점 버림)
    printf("a %% b = %d (나머지 연산)\n", a % b); // 1 (나머지)
    // 참고: printf에서 '%' 문자를 출력하려면 '%%' 두 개를 써야 합니다.

    // 실수 나눗셈 예시
    double c = 10.0;
    double d = 3.0;
    printf("c / d = %lf (실수 나눗셈)\n", c / d); // 3.333333

    return 0;
}
```

**[실행 결과]**
```
a + b = 13
a - b = 7
a * b = 30
a / b = 3 (정수 나눗셈)
a % b = 1 (나머지 연산)
c / d = 3.333333 (실수 나눗셈)
```

#### 3.2. 대입 연산자 (Assignment Operators)

변수에 값을 '대입'할 때 사용하는 연산자입니다. 가장 기본은 `=`이고, 산술 연산과 대입을 한 번에 하는 복합 대입 연산자도 있습니다.

| 연산자 | 설명           | 예시 (동일한 의미)  |
| :----- | :------------- | :------------------ |
| `=`    | 대입           | `x = 5;`            |
| `+=`   | 덧셈 후 대입   | `x += 5;` (`x = x + 5;`) |
| `-=`   | 뺄셈 후 대입   | `x -= 5;` (`x = x - 5;`) |
| `*=`   | 곱셈 후 대입   | `x *= 5;` (`x = x * 5;`) |
| `/=`   | 나눗셈 후 대입 | `x /= 5;` (`x = x / 5;`) |
| `%=`   | 나머지 후 대입 | `x %= 5;` (`x = x % 5;`) |

#### 💡 예제 코드: 대입 연산자

```c
#include <stdio.h>

int main() {
    int num = 10;
    printf("초기 num 값: %d\n", num); // 10

    num += 5; // num = num + 5; -> num은 15
    printf("num += 5 후: %d\n", num);

    num -= 3; // num = num - 3; -> num은 12
    printf("num -= 3 후: %d\n", num);

    num *= 2; // num = num * 2; -> num은 24
    printf("num *= 2 후: %d\n", num);

    num /= 4; // num = num / 4; -> num은 6
    printf("num /= 4 후: %d\n", num);

    num %= 5; // num = num % 5; -> num은 1
    printf("num %= 5 후: %d\n", num);

    return 0;
}
```

**[실행 결과]**
```
초기 num 값: 10
num += 5 후: 15
num -= 3 후: 12
num *= 2 후: 24
num /= 4 후: 6
num %= 5 후: 1
```

#### 3.3. 증감 연산자 (Increment/Decrement Operators)

변수의 값을 1 증가시키거나 1 감소시킬 때 사용하는 아주 유용한 연산자입니다.

| 연산자 | 설명                                       |
| :----- | :----------------------------------------- |
| `++`   | 값을 1 증가시킵니다.                       |
| `--`   | 값을 1 감소시킵니다.                       |

이 연산자들은 변수 앞에 붙을 때(`++a`, `--a`)와 뒤에 붙을 때(`a++`, `a--`) 동작 방식에 미묘한 차이가 있습니다. 이것을 **전위(Prefix)**와 **후위(Postfix)**라고 부릅니다.

*   **전위 (Prefix): `++변수` 또는 `--변수`**
    *   **먼저** 변수의 값을 1 증가/감소시킨 후, 그 **변경된 값**을 사용합니다.
*   **후위 (Postfix): `변수++` 또는 `변수--`**
    *   **먼저** 변수의 **현재 값**을 사용한 후, 그 다음 줄이나 연산이 끝난 후에 변수의 값을 1 증가/감소시킵니다.

이 부분이 초보자들이 헷갈리기 쉬우니 예제 코드를 통해 꼭 이해하고 넘어가세요!

#### 💡 예제 코드: 증감 연산자

```c
#include <stdio.h>

int main() {
    int a = 10;
    int b = 10;
    int result1, result2;

    // 1. 전위 증가 (Pre-increment)
    // 먼저 a를 1 증가(11)시키고, 그 증가된 값을 result1에 대입합니다.
    result1 = ++a; 
    printf("전위 증가 (++a) 결과: a = %d, result1 = %d\n", a, result1); // a: 11, result1: 11

    // 2. 후위 증가 (Post-increment)
    // 먼저 b의 현재 값(10)을 result2에 대입하고, 그 다음에 b를 1 증가(11)시킵니다.
    result2 = b++; 
    printf("후위 증가 (b++) 결과: b = %d, result2 = %d\n", b, result2); // b: 11, result2: 10

    int x = 20;
    int y = 20;

    // 3. 전위 감소 (Pre-decrement)
    result1 = --x; 
    printf("전위 감소 (--x) 결과: x = %d, result1 = %d\n", x, result1); // x: 19, result1: 19

    // 4. 후위 감소 (Post-decrement)
    result2 = y--; 
    printf("후위 감소 (y--) 결과: y = %d, result2 = %d\n", y, result2); // y: 19, result2: 20

    return 0;
}
```

**[실행 결과]**
```
전위 증가 (++a) 결과: a = 11, result1 = 11
후위 증가 (b++) 결과: b = 11, result2 = 10
전위 감소 (--x) 결과: x = 19, result1 = 19
후위 감소 (y--) 결과: y = 19, result2 = 20
```
이 차이를 정확히 이해하는 것이 중요합니다! 헷갈린다면 여러 번 연습해 보세요.

#### 3.4. 비교 연산자 (Relational Operators) - 맛보기!

두 값의 크기를 비교하거나 같은지 다른지 비교할 때 사용합니다. 결과는 참(1) 또는 거짓(0)으로 나옵니다. 이 연산자들은 나중에 배울 `if`문과 같은 '조건문'에서 아주 중요하게 사용됩니다!

| 연산자 | 설명           | 예시         |
| :----- | :------------- | :----------- |
| `==`   | 같다 (두 개!)  | `a == b`     |
| `!=`   | 같지 않다      | `a != b`     |
| `>`    | 크다           | `a > b`      |
| `<`    | 작다           | `a < b`      |
| `>=`   | 크거나 같다    | `a >= b`     |
| `<=`   | 작거나 같다    | `a <= b`     |

> **주의!** 대입 연산자 `=`와 비교 연산자 `==`를 헷갈리지 마세요! `a = 10`은 `a`에 `10`을 대입하는 것이고, `a == 10`은 `a`가 `10`과 같은지 비교하는 것입니다.

#### 3.5. 논리 연산자 (Logical Operators) - 맛보기!

여러 개의 조건을 조합할 때 사용합니다. 이 역시 `if`문에서 아주 유용하게 쓰입니다.

| 연산자 | 설명          | 예시             |
| :----- | :------------ | :--------------- |
| `&&`   | 논리 AND (둘 다 참일 때만 참) | `(a > 0) && (b < 10)` |
| `||`   | 논리 OR (둘 중 하나라도 참이면 참) | `(a == 0) || (b == 0)` |
| `!`    | 논리 NOT (참이면 거짓, 거짓이면 참) | `!(a > 0)`       |

#### 3.6. 연산자 우선순위 (Operator Precedence)

하나의 수식에 여러 연산자가 있을 때, 어떤 연산자를 먼저 계산할지 정해진 규칙이 있습니다. 이를 **연산자 우선순위**라고 합니다. 예를 들어, `10 + 2 * 3`은 `(2 * 3)`을 먼저 계산해서 `10 + 6 = 16`이 됩니다. (곱셈이 덧셈보다 우선순위가 높아요!)

만약 우선순위를 무시하고 원하는 순서대로 계산하고 싶다면 **괄호 `()`**를 사용하면 됩니다. 괄호 안의 연산이 가장 먼저 수행됩니다.

#### 💡 예제 코드: 연산자 우선순위

```c
#include <stdio.h>

int main() {
    int result1 = 10 + 20 * 2;   // 20 * 2 먼저 계산 -> 10 + 40 = 50
    int result2 = (10 + 20) * 2; // (10 + 20) 먼저 계산 -> 30 * 2 = 60

    printf("우선순위에 따른 계산 결과: %d\n", result1);
    printf("괄호를 사용한 계산 결과: %d\n", result2);

    return 0;
}
```

**[실행 결과]**
```
우선순위에 따른 계산 결과: 50
괄호를 사용한 계산 결과: 60
```
모든 연산자의 우선순위를 외울 필요는 없지만, `*`, `/`, `%`가 `+`, `-`보다 높다는 정도는 알아두시고, 헷갈릴 때는 **과감하게 괄호를 사용하는 습관**을 들이는 것이 좋습니다!
